## Changes

### src/pty-manager.js
- `_getLastActivity(w, workerName)` 전면 재작성
  - raw screen 텍스트(ScreenBuffer) 패턴 매칭 전부 제거
  - `logs/events-<worker>.jsonl`에서 최근 20줄 읽어 tool_name 기반 요약 생성
  - "Edit: foo.js, Write: bar.js" 형태, 중복 제거, 최대 5개, 120자 제한
  - 폴백: `_taskText` 첫줄 (80자), JSONL 없거나 비어있을 때
  - 시그니처 변경: `_getLastActivity(w)` -> `_getLastActivity(w, workerName)`
- `healthCheck()` stall 감지 추가
  - `_interventionState` 있는 워커: 즉시 `notifyStall` 호출
  - `_taskText` 있고 5분+ 무출력: `notifyStall` 호출
  - alive 상태 결과에 `idleMs` 필드 추가

### src/notifications.js
- `notifyStall(workerName, reason)` 메서드 추가

  - Slack 미설정 시 `{ sent: false }` 반환
