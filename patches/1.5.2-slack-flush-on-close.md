# Patch 1.5.2 - Worker close 시 Slack 버퍼 강제 flush (5.4)

## Date: 2026-04-05

## Summary
Worker exit 시 notifyTaskComplete()이 호출되지만, alertOnly 모드에서 완료 메시지가 버퍼에만 남고 실제 전송되지 않는 문제 수정. notifyTaskComplete 직후 _flushAll()을 호출하여 버퍼를 즉시 전송.

## Changes

### src/pty-manager.js
- Worker close 핸들러에서 `notifyTaskComplete()` 호출 직후 `_flushAll()` 추가
- `.catch(() => {})` 로 flush 실패 시에도 worker close 흐름 차단하지 않음

### tests/slack-flush-on-close.test.js
- 신규 테스트 파일: 5개 테스트 케이스
- alertOnly 모드에서 _flushAll이 버퍼를 비우는지 검증
- 멀티 채널 flush 검증
- 빈 버퍼 flush 안전성 검증
