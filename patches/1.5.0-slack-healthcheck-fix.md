# Patch 1.5.0 - notifyHealthCheck 상태 누락 수정 (4.20)

## Date: 2026-04-04

## Summary
Slack notifyHealthCheck에서 `restarted` 워커가 alive 목록에서, `restart_failed` 워커가 dead 목록에서 누락되어 Slack 알림에 표시되지 않던 문제 수정.

## Root Cause
- alive 필터가 `state === 'busy' || state === 'idle'`만 검사하여 `restarted` 워커 제외
- dead 필터가 `state === 'exited'`만 검사하여 `restart_failed` 워커 제외

## Changes

### src/notifications.js
- `notifyHealthCheck()`: alive 필터에 `restarted` 상태 추가
- `notifyHealthCheck()`: dead 필터에 `restart_failed` 상태 추가
- LANG에 `restarted` / `restartFailed` 라벨 추가 (ko: '재시작됨' / '재시작 실패', en: 'restarted' / 'restart failed')

### tests/slack-activity.test.js
- 4개 유닛 테스트 추가 (총 12개)
  - restarted 워커 alive 목록 포함 확인
  - restart_failed 워커 dead 목록 포함 확인
  - mixed 상태 워커 분류 확인
  - LANG 라벨 표시 확인
