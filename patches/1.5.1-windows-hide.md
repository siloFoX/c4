# Patch 1.5.1 - Windows 콘솔 창 숨김 (4.25)

## Date: 2026-04-05

## Summary
Windows에서 execSync/spawn 호출 시 콘솔 창이 잠깐 나타나는 문제 해결. `windowsHide: true` 옵션을 일괄 적용하고, conpty 관련 이슈를 방지하기 위해 `useConpty: false` 추가.

## Changes

### src/pty-manager.js
- `execSyncSafe` 래퍼 함수 추가: `windowsHide: true`를 기본 옵션으로 적용
- 기존 모든 `execSync` 호출을 `execSyncSafe`로 교체 (18개소)
- `pty.spawn` 호출에 `useConpty: false` 옵션 추가

### src/daemon-manager.js
- `spawn()` (데몬 프로세스 생성)에 `windowsHide: true` 추가
- `execSync()` (taskkill)에 `windowsHide: true` 추가
