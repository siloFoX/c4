# Patch 1.5.0 - Slack task 요약 절단 버그 수정 (4.19)

## Date: 2026-04-04

## Summary
Slack 알림에서 task 요약 텍스트가 파일명의 `.`(dot)에서 잘리던 버그 수정. `split(/[.\n]/)` 패턴이 줄바꿈뿐 아니라 마침표에서도 분할하여 "Fix bug in daemon.js"가 "Fix bug in daemon"으로 잘리는 문제.

## Root Cause
`_fmtWorker()`, `notifyTaskComplete()`, `notifyError()`에서 task 요약 첫 줄 추출 시 `split(/[.\n]/)`을 사용하여 마침표(.)도 분할자로 처리됨.

## Changes

### src/notifications.js
- `_fmtWorker()`: `split(/[.\n]/)` -> `split('\n')`
- `notifyTaskComplete()`: 동일 수정
- `notifyError()`: 동일 수정

### src/pty-manager.js
- `_getLastActivity()`: `split(/[.\n]/)` -> `split('\n')` (fix-slack 브랜치)

### tests/notifications.test.js
- 5개 유닛 테스트 추가
  - 파일명 포함 task 요약 보존 확인 (daemon.js, config.json 등)
  - multi-line task에서 첫 줄만 추출 확인
  - dot-only 텍스트 보존 확인
