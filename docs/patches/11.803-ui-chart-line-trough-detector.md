# 11.803 UI: chart-line-trough-detector primitive (v1.11.821)

## Summary
Pure-SVG dual-panel React/TS primitive that marks **local minima of `close`
below a rolling SMA-stddev threshold** across the lookback. Top 60% panel
plots the price line; bottom 40% panel plots a binary 0/1 trough indicator
and renders a marker on each strict local minimum. Mirror of
`chart-line-peak-detector` (11.802) with strict `<` comparisons and
`SMA - factor * stdDev` threshold.

## Formula
```
mean[i]      = SMA(close, length)[i]
stdDev[i]    = sqrt( (1/length) * sum_{k=i-length+1..i} (close[k] - mean[i])^2 )
threshold[i] = mean[i] - threshFactor * stdDev[i]
trough[i]    = (close[i] < close[i-1])
             AND (close[i] < close[i+1])
             AND (close[i] < threshold[i])
             ? 1 : 0
```
`trough[i]` is `null` at boundaries (`i == 0` or `i == n - 1`), during SMA
warmup (`i < length - 1`), and whenever any input is non-finite.

## Defaults
- `length = 14`
- `threshFactor = 1`
- `width = 720`
- `height = 460`
- `panelGap = 12`

## Bit-exact testability anchors
- **CONST close** (`close[i] == K` for every `i`): `stdDev = 0`,
  `threshold = K`, and the strict `< threshold` predicate fails -> every
  valid bar has `trough = 0`. Verified across `K in {0, 1, 5, 100, -3}`
  and every `(length, threshFactor)` combination.
- **ALTERNATING [0, 1, 0, 1, ...]** with `threshFactor = 0`: even-index
  inner bars (`close = 0`) have neighbours `1`, `mean = 0.5`, `threshold
  = 0.5`, and `0 < 0.5` -> `trough = 1`. Odd-index inner bars stay at 0.

## Design notes
- Strict `<` everywhere (so a flat plateau never reports a trough).
- Trough axis is fixed to `[0, 1]` regardless of data, so the indicator
  panel reads as a binary gate.
- Markers only render where `trough == 1` (zero-valued bars don't
  clutter the panel with empty dots).
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards before access.
- Default colors: trough marker / indicator line `#16a34a` (green),
  no-trough `#0ea5e9` (blue), tooltip threshold/trough `#86efac` -- the
  mirror palette of peak-detector (which used red).
