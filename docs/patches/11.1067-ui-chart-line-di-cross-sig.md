# 11.1067 -- UI: chart-line-di-cross-sig primitive (v1.11.1085)

## Scope

Add a pure-SVG dual-panel `ChartLineDiCrossSig` component
that renders the close in the top panel and the Average
Directional Index +DI and -DI directional indicators in
the bottom panel, marking bullish (+DI crosses up through
-DI -- trend direction change up; bulls overtake bears) /
bearish (+DI crosses down through -DI -- trend direction
change down; bears overtake bulls) DI-vs-DI crossover
trigger events with bias coloring derived from the
(+DI - -DI) slope at the trigger bar.

The DI cross is the **oldest and simplest signal** in J.
Welles Wilder Jr's 1978 ADX system: when the positive
directional indicator crosses the negative directional
indicator, the dominant directional pressure flips. This
is the canonical trading-rule application predating ADX
itself.

## DI cross pipeline

```
upMove[i]    = high[i] - high[i-1]
downMove[i]  = low[i-1] - low[i]
+DM[i]       = (upMove > downMove && upMove > 0)
               ? upMove : 0
-DM[i]       = (downMove > upMove && downMove > 0)
               ? downMove : 0
tr[i]        = max(high - low,
                   |high - close[i-1]|,
                   |low  - close[i-1]|)
smPlus[i]    = SMA(+DM, period)
smMinus[i]   = SMA(-DM, period)
smTr[i]      = SMA(tr,  period)
plusDI[i]    = (smTr > 0) ? 100 * smPlus / smTr : 0
minusDI[i]   = (smTr > 0) ? 100 * smMinus / smTr : 0

bullish (DI-cross up) :
  prev plusDI <= prev minusDI &&
  cur plusDI > cur minusDI
bearish (DI-cross down) :
  prev plusDI >= prev minusDI &&
  cur plusDI < cur minusDI

regime       : 'bullish' when plusDI >= minusDI
               'bearish' when plusDI <  minusDI
               'none'    when either is null
bias         : (plusDI - minusDI)[i] vs prev -> up /
               down / flat / none
```

Default `period = 14` -- canonical Wilder ADX tuning.

## Sibling family

The ADX signal family now has five primitives:

- **chart-line-adx-trend-cross v1.11.1047** -- ADX
  itself vs the 20/25 threshold (trend strength
  emerging)
- **chart-line-adx-pos-neg-divergence v1.11.1075** --
  compares +DI direction vs -DI direction (5-state
  regime, trend confirmation)
- **chart-line-adx-pos-cross v1.11.1083** -- +DI alone
  vs zero (uptrend emerging from indecision)
- **chart-line-adx-neg-cross v1.11.1084** -- -DI alone
  vs zero (downtrend emerging from indecision)
- **chart-line-di-cross-sig v1.11.1085** (this
  primitive) -- +DI vs -DI direct cross (the original
  Wilder DI cross trading signal)

The DI cross is **the oldest member of the family** --
predating ADX itself. Wilder's original 1978 system
used DI crosses as primary entry/exit signals, with the
ADX added later as a trend-strength filter.

Distinct from the adx-pos-neg-divergence sibling
v1.11.1075:
- **divergence**: compares +DI direction vs -DI
  direction (both rising, both falling, or one rising
  while the other falls) -- 5-state regime
- **cross-sig** (this primitive): compares +DI value
  vs -DI value (direct crossover) -- 3-state regime

Uses SMA-based smoothing (matching the adx-* family
convention) rather than Wilder's exponential smoothing,
for bit-exact integer DI values on linear input.

Warmup is `period + 1 = 15` for the default tuning: DM
and TR seed at i = 1, SMA window-fills at i = period =
14, and cross detection needs the previous bar's +DI
and -DI, so the first potential cross lands at i =
period + 1 = 15.

## Bit-exact anchors

All anchors use HLC input.

- **CONST band** `high = K + 1`, `low = K - 1`, `close
  = K`: upMove = 0, downMove = 0 -> +DM = -DM = 0.
  TR = 2. smPlus = smMinus = 0, smTr = 2. plusDI =
  minusDI = 0. plusDI === minusDI every bar -> no
  cross. regime `bullish` (via >=). 0 crosses.
  Verified across K in {0, 1, 50, 200, 1234}.
- **LINEAR UP** `high = i + 1`, `low = i - 1`, `close
  = i`: upMove = +1, downMove = -1. +DM = 1, -DM = 0.
  TR = 2. plusDI = 50, minusDI = 0 (constants). plusDI
  - minusDI = +50 (bulls dominate). No cross (no
  transition from below). 0 crosses. regime
  `bullish`.
- **LINEAR DOWN** `high = -i + 1`, `low = -i - 1`,
  `close = -i`: upMove = -1, downMove = +1. +DM = 0,
  -DM = 1. plusDI = 0, minusDI = 50 (constants).
  plusDI - minusDI = -50 (bears dominate). No cross.
  0 crosses. regime `bearish`.

All three steady-state anchors produce zero crosses
because +DI and -DI remain on the same side of each
other throughout. Real-world crosses fire when the
balance of directional pressure shifts -- typically the
first bar of a trend reversal.

## Risk

- Pure-SVG, deterministic `.toFixed(2)`, no canvas, no
  chart lib dependency.
- 3-state regime classifier follows the cross-sig
  family convention (bullish on `===` via `>=`).
- SMA-based smoothing (not Wilder) for bit-exact
  testability. Documented as a family-level tuning
  choice.
- The DI cross signal is famously prone to whipsaws in
  range-bound markets -- many traders combine it with
  the ADX threshold (chart-line-adx-trend-cross
  v1.11.1047) to filter out low-conviction crosses.
  This primitive ships the raw cross signal; consumers
  can filter externally.
- Zero-guard on `smTr > 0` prevents NaN on CONST input
  -- returns 0 for both DI values. Matches pandas-ta
  and ta-lib behaviour.
- On steady LINEAR input +DI and -DI saturate at
  constants (50/0 for LINEAR UP, 0/50 for LINEAR DOWN)
  with flat slopes -- no crosses fire. Real-world
  crosses fire when the directional balance shifts,
  typically at trend reversals.
- Completes the ADX signal family at five primitives.
  This is the canonical Wilder DI cross -- the
  original 1978 signal -- and the simplest cross-sig
  family member that uses the ADX system.
