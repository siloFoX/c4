## Summary

`ChartLineSpikeDetector` is a pure-SVG dual-panel React primitive
that plots the close on the top panel and a Price Spike Detector
on the bottom panel. The detector is the Z-score of the bar-to-bar
change relative to its rolling mean and standard deviation:

```
delta[i]    = close[i] - close[i - 1]    (i >= 1)
mean[i]     = SMA(delta, length)[i]
stdDev[i]   = sqrt(sum((delta - mean)^2) / length)  (population)
spike[i]    = (delta[i] - mean[i]) / stdDev[i]
```

Defaults: `length = 14`. Bars before `i = length` are warmup
nulls (`delta` is undefined at bar 0, so the rolling window
must lie in `[1, n]` -- this requires `i >= length`). When
`stdDev == 0` (every delta in the window is identical) the
spike is `null`.

A reading above +1.5 marks an up-spike; below -1.5 marks a
down-spike; magnitudes >= 3 are extreme outliers.

## Bit-exact anchors

- **CONST close** (`close = K`): every `delta = 0`, `stdDev = 0`,
  and the divide-by-zero guard surfaces `null` at every bar --
  the singular "no variation" case.
- **ALTERNATING close** (`[0, 1, 0, 1, ...]`) with even `length`:
  deltas alternate `[+1, -1, +1, -1, ...]`. The rolling sum over
  an even-sized window of paired `+1` / `-1` deltas is exactly
  zero, so `mean = 0`. Every squared residual is `(±1 - 0)^2 = 1`,
  the sum of squares is exactly `length`, and
  `stdDev = sqrt(length / length) = 1` bit-exact. Therefore:

      spike[i] = (delta[i] - 0) / 1 = delta[i] = +/- 1

  bit-exact. The integration sweep verifies this across even
  lengths in `{4, 6, 10, 14, 20}`.

## Zone classifier

Z-score thresholds:

- `extreme-up` -- spike >= 3
- `spike-up` -- 1.5 <= spike < 3
- `normal` -- 0 < |spike| < 1.5 (no spike)
- `at` -- spike == 0
- `spike-down` -- -3 < spike <= -1.5
- `extreme-down` -- spike <= -3
- `none` -- spike is null (warmup or singular)
