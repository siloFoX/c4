## Summary

New `ChartLineCdf` UI primitive at
`web/src/components/ui/chart-line-cdf.tsx` -- a pure-SVG empirical
cumulative distribution function (ECDF) curve rising from zero to one
across the sorted series. The x-axis is the sample value, the y-axis
the cumulative probability 0 to 1, with a dashed median reference
line.

## Statistics

`computeLineCdf(values)` is the empirical cumulative distribution
function. The finite sample values are sorted ascending and grouped
by distinct value; each distinct value carries:

```
count       = its multiplicity in the sample
cumulative  = number of sample values at or below it
probability = cumulative / n
```

The result is a non-decreasing step function whose last step always
reaches 1. `computeLineCdfMedian(steps)` reports the smallest value at
which the ECDF reaches 0.5 or more (null only for an empty step list,
since a non-empty ECDF always reaches 1).

### Worked fixture

`values = [30, 10, 20, 10, 40]` -> sorted `[10, 10, 20, 30, 40]`,
`n = 5`:

```
value 10: count 2, cumulative 2 -> probability 2/5 = 0.4
value 20: count 1, cumulative 3 -> probability 3/5 = 0.6
value 30: count 1, cumulative 4 -> probability 4/5 = 0.8
value 40: count 1, cumulative 5 -> probability 5/5 = 1.0
```

4 distinct values; `median = 20` (the first probability >= 0.5).

## API

- `getLineCdfFiniteValues(values)` -- keep the finite sample values
  (order preserved).
- `computeLineCdf(values)` -- the ECDF step array. Filters non-finite
  values and sorts internally.
- `computeLineCdfMedian(steps)` -- the empirical median value.
- `runLineCdf(values)` -- drop non-finite values, sort ascending,
  compute the ECDF, return `{ values, n, steps, distinctCount,
  minValue, maxValue, median, ok }`. `ok` is false for fewer than two
  finite values.
- `computeLineCdfLayout(options)` -- build the rising staircase path,
  the step markers, and the projected median reference line.
- `describeLineCdfChart(data)` -- ARIA description string.

## Component

`ChartLineCdf` is a `forwardRef` region (`role="region"`) with an
ARIA-linked description, a `role="img"` SVG, a config badge (sample
size + median), a two-series legend (ECDF / Median) with toggle, the
rising step path, step markers at each distinct value, a dashed
median reference line, a hover tooltip exposing the value, cumulative
probability, count, and cumulative count for the hovered step, and a
`motion-safe:animate-fade-in` entrance. Every node carries a
`data-section` attribute.

## Distinctness

This is a statistics chart distinct from `chart-line-survival` and
`chart-line-distribution`. The survival curve *descends* from one to
zero and handles censored observations; the ECDF *ascends* from zero
to one and every observation contributes a plain `1/n` jump with no
censoring. A distribution chart shows the *density*; the ECDF shows
the *cumulative fraction* -- the proportion of the sample at or below
any value, read straight off the curve.
