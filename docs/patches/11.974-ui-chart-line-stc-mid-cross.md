## Summary

Adds the `ChartLineStcMidCross` primitive
(`web/src/components/ui/chart-line-stc-mid-cross.tsx`): a pure SVG
dual-panel chart that visualizes Doug Schaff's Schaff Trend Cycle
(STC) crossings through the 50 midline for double-smoothed MACD
cycle centerline regime transitions with bias coloring. Companion
to `chart-line-schaff-zero-cross` (TODO 11.951): shares the same
STC pipeline (MACD -> double-stochastic + 0.5-EMA smoothing pair)
and defaults (cycle = 10, fast = 23, slow = 50, threshold = 50);
reframes the ARIA / describe text from "baseline regime
transition" to "centerline regime transitions with bias coloring"
so it slots into the broader mid-cross family (RSI / Stoch / CCI /
MFI / Stoch-RSI / Williams %R / UO / CMO / RMI / DPO / QStick).

## Pipeline

STC takes the MACD line and applies a stochastic + smoothing pair
twice, producing a fast, bounded `[0, 100]` cycle oscillator:

1. `emaFast = EMA(close, fast)` (SMA-seeded)
2. `emaSlow = EMA(close, slow)` (SMA-seeded)
3. `macd = emaFast - emaSlow`
4. First pass (stochastic + half-EMA over a `cycle` window of MACD):
   - `k1 = (macd_high == macd_low) ? prev_k1 : 100 * (macd - lo) / (hi - lo)`
   - `d1 = prev_d1 + 0.5 * (k1 - prev_d1)` (seed 50)
5. Second pass (stochastic + half-EMA over a `cycle` window of d1):
   - `k2 = (d1_high == d1_low) ? prev_k2 : 100 * (d1 - lo) / (hi - lo)`
   - `d2 = prev_d2 + 0.5 * (k2 - prev_d2)` (seed 50)
   - `stc = d2`
6. Regime classifier (strict `>=`):
   - `stc >= 50` -> bullish
   - `stc <  50` -> bearish
7. Cross detector emits a `bullish` cross when
   `stc[i-1] <= 50 && stc[i] > 50` and a `bearish` cross when
   `stc[i-1] >= 50 && stc[i] < 50`. Strict inequality on the
   current side means the boundary value 50 does not double-fire.

## Defaults

- `cycle = 10`, `fast = 23`, `slow = 50` (Schaff canonical)
- `threshold = 50` (neutral centerline)
- Warmup boundary under SMALL = `{cycle: 3, fast: 5, slow: 10}`:
  bar `slow + 2 * (cycle - 1) - 1 = 13`, so the first 13 samples
  (40-bar run) are null and the remaining 27 are populated.

## Bit-exact anchors

- CONST `close = K` for all `K in {0, 1, 42, 100, 1234}` ->
  `stc[i] === 50` literal for every `i >= 13`. Regime classifies
  as bullish because 50 sits on the threshold (`stc >= threshold`).
- LINEAR UP `close = i` -> after the SMA-seeded EMAs reach their
  exact steady-state lag, MACD is constant; the stochastic of a
  constant holds the seed; stc settles at 50. All bullish at
  boundary, 0 crosses.
- LINEAR DOWN `close = -i` -> same constant-MACD argument yields
  stc = 50 by symmetry.
- MACD is verifiably constant for LINEAR UP after the EMA seed
  (`first ~= macd[10] ~= macd[20] ~= ...`), confirming the
  steady-state lag math.
- CONST K full-run counters under SMALL, 40 bars:
  `noneCount = 13`, `bullishCount = 27`, `bearishCount = 0`,
  `crosses.length = 0`.
- `pricePath` and `stcPath` are byte-identical across repeated
  layout calls, with `.toFixed(2)` precision and `M ... L ...`
  command shapes. The stc path uses a single `M` for post-warmup
  data.
- `thresholdY` at 50 sits exactly at the panel midpoint
  ((oscTop + oscBottom) / 2) -- the canonical centerline.
- Cross detector boundary: strict inequality (`> T`, `< T`).
