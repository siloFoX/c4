## Math anchor (bit-exact)

```
k[i]     = close-only stochastic %K with degenerate=50 fallback
priceUp  = close[i] > close[i-window]
stochUp  = k[i]     > k[i-window]
states: aligned-bullish, aligned-bearish, divergent-bullish, divergent-bearish, none
crosses suppressed when prev or cur state == 'none'
```

- **CONST close = K**: k=50 (degenerate) -> priceUp=false,
  stochUp=false -> aligned-bearish. 0 crosses.
- **LINEAR UP close = i**: k=100 saturated -> priceUp=true,
  stochUp=false -> divergent-bearish (canonical bearish
  divergence). 0 crosses (prev=none).
- **LINEAR DOWN close = -i**: k=0 -> aligned-bearish. 0 crosses.

For 50 bars: noneCount=18, post-warmup regime count=32.
