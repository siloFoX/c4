# Patch 11.922 -- UI: chart-line-rsi-oversold-cross primitive

## Summary

Added `<ChartLineRsiOversoldCross />` -- pure-SVG dual-panel
React/TS primitive that renders the close (top panel) with
bullish (exit upward) / bearish (entry downward) arrow overlays
at every RSI oversold threshold cross trigger, and the close-
only RSI line (bottom panel) on a fixed 0-100 oscillator with
30 / 50 reference bands. Single-threshold cross variant of the
RSI family that flags the discrete level-30 entry and exit
events distinct from RSI regime classification. Mirror of
11.921 `chart-line-rsi-overbought-cross` using the oversold
level instead of the overbought level.

## Algorithm

```
delta[i]   = close[i] - close[i-1]
gain[i]    = max(delta, 0)
loss[i]    = max(-delta, 0)
avgGain[i] = Wilder smoothing of gain over length
avgLoss[i] = Wilder smoothing of loss over length
rsi[i]     = avgGain + avgLoss == 0
              ? 50  (neutral fallback)
              : avgLoss == 0
                ? 100
                : 100 - 100 / (1 + avgGain / avgLoss)
bullish   : prev <= threshold && cur > threshold  (exit upward)
bearish   : prev >= threshold && cur < threshold  (entry downward)
```

Defaults `length = 14` (canonical RSI window), `threshold = 30`
(oversold level). Regime classifier `bearish` (RSI < threshold,
oversold), `neutral` (RSI >= threshold, normal), `none` (RSI
null).

## Bit-exact anchor

CONST close = K: every delta = 0 -> gain = loss = 0 ->
avgGain = avgLoss = 0 -> RSI uses the neutral fallback 50.
50 > 30 so regime is `neutral` and the threshold is never
crossed. cross count = 0. Verified across K = 0, 1, 5, 17,
100, 1234.

LINEAR DOWN yields RSI = 0 every settled bar (avgGain = 0).
0 < 30 -> regime `bearish` (oversold). LINEAR UP yields RSI
= 100 -> regime `neutral`.

## Visual

- Two stacked panels with the osc panel pinned to 0-100.
- Top: close line + bullish (exit) / bearish (entry) polygon
  arrows at threshold cross bars.
- Bottom: RSI line + cross dot markers + threshold / 50
  reference bands.
- Tooltip lists x / close / RSI / regime / exits / entries /
  crosses.
- Config badge + togglable legend (close / RSI).
- Cross marker is `role="graphics-symbol"` with `tabIndex={0}`.
