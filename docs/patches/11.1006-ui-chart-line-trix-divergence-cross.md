## TRIX pipeline (Hutson 1983)

```
ema1   = SMA-seeded EMA(close, length=15)
ema2   = SMA-seeded EMA(ema1,  length=15)
ema3   = SMA-seeded EMA(ema2,  length=15)
trix[i] = 100 * (ema3[i] - ema3[i-1]) / ema3[i-1]
       = 0 when ema3[i-1] === 0 (degenerate fallback)
```

The SMA-seeded EMA helper is exported as
`applyLineTrixDivergenceCrossSmaSeededEma`: the first `length`
contiguous valid samples seed the recurrence with their
arithmetic mean, after which `ema[i] = alpha * v + (1 - alpha) *
ema[i-1]` with `alpha = 2 / (length + 1)`. Any null gap aborts
further output -- consistent with the SMA-seeded EMA in
`chart-line-tsi-divergence-cross`.

Warmup is `3 * (length - 1) + 1 = 43`: ema1 becomes valid at
index `length - 1 = 14`, ema2 needs another `length - 1` samples
past ema1's seed (index 28), ema3 likewise (index 42), and TRIX
itself needs one additional bar for the 1-period rate of change
(index 43).

## Divergence regime model

5-state classifier exactly like CMO / RSI / Stoch / TSI
divergence:

```
priceUp = close[i] > close[i - divergenceWindow]
trixUp  = trix[i]  > trix[i - divergenceWindow]
state   = aligned-bullish    if priceUp && trixUp
        | aligned-bearish    if !priceUp && !trixUp
        | divergent-bullish  if !priceUp && trixUp   (price down, TRIX up)
        | divergent-bearish  if priceUp && !trixUp   (price up, TRIX down)
        | none               if either input is null
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
```

Crosses suppress when either `prev` or `cur` is `none`, so
warmup transitions never fire spuriously. Strict inequality
guarantees boundary equality falls into the aligned-bearish
state instead of double-firing.

Defaults: `length = 15` (canonical Hutson TRIX window),
`divergenceWindow = 5`.

## Bit-exact anchors

| input                 | trix (post-warmup)    | regime (post i=48)  | crosses |
| --------------------- | --------------------- | ------------------- | ------- |
| CONST close = K       | 0                     | aligned-bearish     | 0       |
| LINEAR UP, close=i    | 100 / (i - 22) decay  | divergent-bearish   | 0       |
| LINEAR DOWN, close=-i | 100 / (i - 22) decay  | aligned-bearish     | 0       |

CONST K verified across K = 0..1234 via parameterised test --
for K > 0 the EMA chain collapses to K so the numerator is zero
(`100 * (K - K) / K = 0`); for K = 0 the EMA chain collapses to
0 and the degenerate fallback returns +0. priceUp = false
because `close[i] === close[i-5]` (strict `>`).

LINEAR UP exercises the steady-state algebra: each EMA layer
introduces a `(length - 1) / 2` lag, so ema3 settles at
`i - 3 * (L - 1) / 2 = i - 21` for L = 15. trix[i] = 100 /
(i - 22), which is positive (the EMA is non-zero) and
monotonically decaying after warmup. priceUp = true, trixUp =
false (decay -> trix[i] < trix[i-5]) -> regime
`divergent-bearish` for the full 32 valid samples on the
80-bar fixture. The cross count stays at 0 because the regime
is entered from `none` (prev = none is never a valid cross
precursor).

LINEAR DOWN exhibits the algebraic symmetry: ema3 settles at
`-i + 21`, so trix[i] = `100 * (-1) / (-(i-1) + 21) = 100 /
(i - 22)` -- same positive magnitude as LINEAR UP, also
monotonically decaying. priceUp = false, trixUp = false ->
regime `aligned-bearish`.
