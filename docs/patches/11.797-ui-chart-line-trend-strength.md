## Summary

`ChartLineTrendStrength` is a pure-SVG dual-panel React primitive
that plots the close on the top panel and a Trend Strength
oscillator on the bottom panel. The oscillator measures how
dominant the linear trend is relative to the residual noise:

```
For window of `length` bars ending at i:
  x = 0, 1, ..., length - 1
  y = close[i - length + 1 .. i]
  meanX = (length - 1) / 2
  meanY = sum(y) / length
  sumXY = sum((x[k] - meanX) * (y[k] - meanY))
  sumXX = sum((x[k] - meanX)^2)
  slope     = sumXY / sumXX
  intercept = meanY - slope * meanX
  residual[k] = y[k] - (intercept + slope * x[k])
  stdErr    = sqrt(sum(residual^2) / length)
  trendStrength = |slope| / (|slope| + stdErr)
```

Defaults: `length = 14`. Bars before `i = length - 1` are warmup
nulls. When `|slope| + stdErr == 0` (singular: a flat constant
series) strength is `null`.

The result is bounded in `[0, 1]`: 0 means slope is zero relative
to noise (no detectable trend); 1 means perfect linear fit (no
residual noise).

## Bit-exact anchor

For **PERFECT LINE** (`close[i] = a * i` for integer `a != 0`):

- The window-relative `y` is a perfect linear function of `x`:
  `y[k] = (a * (i - length + 1)) + a * k`.
- Least squares fits this exactly: `slope = a`,
  `intercept = a * (i - length + 1)`, every residual = 0.
- `stdErr = 0`, `|slope| + stdErr = |a|`,
  `trendStrength = |a| / |a| = 1` bit-exact.

The arithmetic stays exact because all values (slope, intercept,
residuals) are integers when `a` is integer and the lookback is
an integer window of integers; even when `meanX` and `meanY` are
half-integers, they remain dyadic and the resulting slope is the
integer `a`.

The integration sweep verifies this across `a` in
`{1, 2, 3, -1, -5}` and `length` in `{3, 5, 7, 14}`.

For **CONST close** (`close = K`): slope is 0, residuals are 0,
denominator is 0, and the divide-by-zero guard surfaces `null` --
the singular case.

## Zone classifier

- `strong` -- strength >= 0.75 (clean linear regime)
- `firm` -- 0.5 <= strength < 0.75
- `soft` -- 0.25 <= strength < 0.5
- `choppy` -- strength < 0.25 (noise dominates the slope)
- `none` -- strength is null (warmup or constant series)

## Out of scope

- No new dependencies.
- No canvas, no third-party chart libs.
- This primitive uses ratio-form normalization (`|slope| /
  (|slope| + stdErr)`) so the reading is bounded in [0, 1].
  A scale-aware t-statistic variant (`slope / stdErr` without
  normalization) is a separate measure and not in scope here.
