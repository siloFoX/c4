# Patch 11.958 - UI: chart-line-stoch-rsi-oversold-cross primitive

Version: 1.11.976

## Summary

Add `web/src/components/ui/chart-line-stoch-rsi-oversold-cross.tsx` --
a pure-SVG dual panel line chart primitive that visualizes price
alongside the Chande-Kroll Stochastic RSI %K oscillator, with explicit
cross markers and overlay arrows at every %K crossing of the canonical
20 oversold threshold. Bullish (= exit from oversold) when %K crosses
up through 20; bearish (= entry into oversold) when %K crosses down
through 20. Mirror of the overbought-cross primitive with
threshold=20.

## Pipeline

Identical to the overbought variant except the threshold default is
20:

1. Standard RSI on the close-only series with Wilder smoothing.
2. Stochastic over the rolling RSI window of `stochLength` with
   `range = 0` -> hold previous (seed = 50).
3. Classify: null -> none, stochK >= threshold -> bullish, stochK <
   threshold -> bearish.
4. Detect: strict `prev <= T && cur > T` -> bullish (exit); strict
   `prev >= T && cur < T` -> bearish (entry).

## Bit-exact anchors

| Pattern                  | RSI / stochK (after warmup) | Regime  |
|--------------------------|------------------------------|---------|
| CONST close = K          | 50 / 50 (locked at seed)     | bullish |
| LINEAR UP close = i      | 100 / 50 (locked at seed)    | bullish |
| LINEAR DOWN close = -i   | 0 / 50 (locked at seed)      | bullish |

CONST anchor tested bit-exactly across K in `{0, 1, 42, 100, 1234}`.
Unlike the overbought variant (50 < 80 -> bearish), here 50 >= 20 so
all CONST/LINEAR cases are `bullish`. Cross events live in transients:

* `rise-then-decline`: stochK drops from 50 (locked) to 0 at the
  transition -> bearish entry through 20.
* `rise-decline-rise` (60 bars): both bearish entry through 20 (at
  start of decline) AND bullish exit through 20 (at start of second
  rise).

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), %K panel below.
* %K panel uses a fixed `[0, 100]` range with the threshold band drawn
  at its absolute level (20 = lower portion of panel, closer to
  oscBottom).
* Cross markers drawn as circles on the %K panel at `(cx, cyOsc)` and
  as arrows on the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="Stoch RSI Oversold Cross
  chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only
  text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
