## Summary

Add `web/src/components/ui/chart-line-awesome-oversold-cross.tsx` --
a pure-SVG dual-panel React/TypeScript chart primitive that renders
the close in the top panel and a close-only Awesome Oscillator (AO)
line in the bottom panel, marking AO discrete oversold-threshold
crossover events at a configurable negative lower band (default
`-0.5`, the mirror of the canonical "alligator awake" overbought
level). This is the oversold-cross variant of the Awesome Oscillator
family, mirroring `chart-line-awesome-overbought-cross` but with the
threshold flipped from `+0.5` to `-0.5` so analysts can flag
momentum slump entry / exit events at the lower band.

The threshold is configurable so the band can be tuned to the
volatility scale of the underlying close series.

## Math anchor (bit-exact)

```
fastSma = SMA(close, fastLength)   // default fastLength = 5
slowSma = SMA(close, slowLength)   // default slowLength = 34
ao[i]   = fastSma[i] - slowSma[i]
bullish : prev ao <= -0.5 && cur ao >  -0.5  // slump out
bearish : prev ao >= -0.5 && cur ao <  -0.5  // slump in
```

- **CONST close = K**: fastSma = slowSma = K every bar via the SMA
  `min === max` short-circuit -> ao = 0. ao = 0 sits well above the
  threshold -0.5 -- regime `bullish` (0 >= -0.5, mirror of the
  overbought variant where 0 < 0.5 gave bearish). 0 crosses.
- **LINEAR UP close = i**: at steady state SMA(close, n) = i - (n-1)/2
  so ao = (i - 2) - (i - 16.5) = 14.5 constant. 14.5 >> -0.5 ->
  regime `bullish`. 0 crosses.
- **LINEAR DOWN close = -i**: ao = -14.5 constant. -14.5 << -0.5 ->
  regime `bearish` (in oversold). 0 crosses (ao jumps from null
  directly to -14.5, no `prev >= T && cur < T` transient).

Strict inequality on `cur` (`> T` for bullish, `< T` for bearish)
means boundary equality (ao == -0.5) never double-fires a cross.

  Oversold Cross chart" aria-describedby="...">` with the sr-only

  lower band" framing.

  tabIndex={0}>` for keyboard focus.

  chevron overlay) carries `role="graphics-symbol"`, `tabIndex={0}`,

  readers can step through events.

  Enter, or Space.
