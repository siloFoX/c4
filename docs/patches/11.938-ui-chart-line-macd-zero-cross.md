# 11.938 -- UI: chart-line-macd-zero-cross primitive (v1.11.956)

## Summary

Adds `ChartLineMacdZeroCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and the close-only
MACD line in the bottom panel. Marks bullish (cross up
through zero) / bearish (cross down through zero) fast EMA
over slow EMA convergence-divergence baseline events. Zero-
line cross variant of the MACD family that flags the discrete
MACD crossing of the zero baseline distinct from the
canonical MACD / signal crossover.

## Pipeline

For close-only data with defaults `fastLength = 12`,
`slowLength = 26`:

```
fastEma  = EMA(close, fastLength)
slowEma  = EMA(close, slowLength)
macd[i]  = fastEma[i] - slowEma[i]
bullish  : prev macd <= 0 && cur macd > 0
bearish  : prev macd >= 0 && cur macd < 0
```

EMA seeds use SMA of the first `length` values; CONST
short-circuit + posZero normalization on each smoothing step.

## Bit-exact anchor

- CONST `close = K` for any finite K -> fastEma = slowEma =
  K every bar -> macd = 0. macd = 0 sits on the threshold
  but the strict-inequality detector never fires. regime
  `bullish` (macd >= 0). cross count = 0. Verified across K
  in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` -> at steady state EMA(close, n)
  tracks `close - (n-1)/2` so fastEma - slowEma converges
  to `(slowLength - fastLength) / 2 = 7` with default
  windows. 7 > 0, regime `bullish`. 0 crosses (macd jumps
  from null to a positive value once warm).
- LINEAR DOWN `close = -i` -> macd converges to
  `-(slowLength - fastLength) / 2 = -7`. -7 < 0, regime
  `bearish`. 0 crosses.
- Decline followed by sustained rise -> MACD transitions
  from negative to positive -> 1 bullish zero-line cross.
- Rise followed by sustained decline -> 1 bearish cross.

## Regime classifier

| macd range            | regime      |
|-----------------------|-------------|
| macd >= 0             | bullish     |
| macd < 0              | bearish     |
| null                  | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- One horizontal reference band inside the oscillator panel
  (threshold at zero, the baseline).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind.
- Oscillator range auto-fits to observed MACD values with
  10% padding above and below; MACD is unbounded so the
  panel cannot use a fixed range.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.

## Defaults

- fastLength = 12
- slowLength = 26
- threshold = 0
- oscillator range = auto-fit (MACD is unbounded)
- width 720, height 460, padding 44, panelGap 12
