# 11.160 -- UI: HeroCard featured info-card primitive (v1.11.178)

## Component

- `web/src/components/ui/hero-card.tsx` -- large featured info card
  for top-of-page welcome / onboarding surfaces.
  - Exports: `HeroCard` (forwardRef to `HTMLElement` -- outer
    `<section>`), `HeroCardProps`, `HeroCardTone`, `HeroCardSize`.
  - Props: `{ icon?, title, description?, cta?, secondaryCta?,
    tone? = 'primary', size? = 'md', className?, ...HTMLAttributes }`.
  - `tone`: `primary` | `success` | `info` | `muted` -- drives the
    subtle `bg-gradient-to-br` overlay and the icon badge color.
  - `size`: `sm` | `md` | `lg` -- drives outer padding
    (`p-4` / `p-6` / `p-8`), icon badge size, and h2 title size.
  - Layout: a `<section>` with `relative overflow-hidden rounded-xl
    border border-border bg-card shadow-sm`. A `pointer-events-none
    absolute inset-0 -z-0 bg-gradient-to-br from-*/20 via-*/5
    to-transparent` overlay sits behind a `relative z-10` content
    column (icon badge above title; description under title; CTA row
    at the bottom).
  - Data attributes: `data-hero-card` (root), `data-tone`,
    `data-size`, `data-hero-card-gradient`, `data-hero-card-icon`,
    `data-hero-card-title`, `data-hero-card-description`,
    `data-hero-card-cta-row` -- used by tests and downstream
    assertion patterns.
  - `cn()` from `web/src/lib/cn.ts` merges class names.
- `web/src/components/ui/index.ts` -- adds
  `export * from './hero-card'`.

## Tests

- `web/src/components/ui/hero-card.test.tsx` (12 cases): title text
  renders; description renders when provided; description element is
  omitted when absent; icon prop renders inside the icon container;
  cta renders; secondaryCta renders alongside cta; `tone='success'`
  exposes `data-tone="success"` and applies a `from-success` gradient
  class; default `tone` is `primary` (exposed via `data-tone`); size
  drives `p-4` / `p-6` / `p-8` padding; caller `className` merges
  onto the root; `forwardRef` returns a live `<section>` element;
  `tone='muted'` applies the muted gradient class.
