# Patch 11.905 -- UI: chart-line-bb-cross-sig primitive

## Summary

Added `<ChartLineBbCrossSig />` -- pure-SVG dual-panel React/TS
primitive that renders the close + upper / middle / lower
Bollinger Bands (top panel) with bullish / bearish arrow
overlays at every close vs middle band cross trigger, and a
`close - middle` deviation oscillator (bottom panel) with cross
dot markers and a dashed zero baseline. Signal-cross variant of
the Bollinger Bands family that flags mean reversion or trend
continuation trigger events distinct from band touches.

## Algorithm

```
mid[i]    = SMA(close, length)
sdev[i]   = sample stddev over the window
upper[i]  = mid + mult * sdev
lower[i]  = mid - mult * sdev
diff[i]   = close[i] - mid[i]
bullish  : (close - mid) crosses up   (prev <= 0, cur > 0)
bearish  : (close - mid) crosses down (prev >= 0, cur < 0)
```

Defaults `length = 20`, `mult = 2` (canonical Bollinger Bands).
Sample stddev uses `n - 1`. The SMA short-circuits when the
window is constant via the `min === max` precision fix. Regime
classifier `bullish` (close > mid), `bearish` (close < mid),
`neutral` (close === mid), `none` (mid null).

## Bit-exact anchor

CONST close = K: mid = K via SMA short-circuit; sdev = 0
every bar; upper = lower = K, diff = close - mid = 0 every bar.
Regime is `neutral`, cross count = 0. Verified across
K = 0, 1, 5, 17, 100, 1234.

LINEAR UP also has a clean anchor: monotonically increasing
close always sits above its rolling mid (the SMA lags the
trend), so diff > 0 and regime is `bullish`.

- Two stacked panels, ARPS design system colors via tokens.
- Top: close line + upper / middle / lower BB envelope + bullish
  /bearish polygon arrows at cross bars.
- Bottom: (close - middle) deviation line + cross dot markers +
  dashed zero baseline.
- Tooltip lists x / close / upper / mid / lower / diff / regime
  / counts / crosses.
- Config badge + togglable legend (close / mid / upper / lower).
- Cross marker is `role="graphics-symbol"` with `tabIndex={0}`.
