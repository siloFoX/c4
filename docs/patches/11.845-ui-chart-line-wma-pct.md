# Patch 11.845 -- ChartLineWmaPct primitive (v1.11.863)

## Summary

Add `<ChartLineWmaPct />`, a pure-SVG dual-panel line chart with a
WMA Percent-Change oscillator beneath a price line and the
underlying WMA overlay on the top panel. Parallel to 11.836
chart-line-ema-pct and 11.837 chart-line-sma-pct, swapping the EMA
or rolling SMA for the linearly-weighted Weighted Moving Average:

```
wma[i]    = sum_{k=0..length-1} (length - k) * close[i - k]
              / (length * (length + 1) / 2)
wmaPct[i] = wma[i] === 0 ? null : (close[i] - wma[i]) / wma[i] * 100
```

Weights run from `length` at the current bar down to `1` at the
oldest bar in the window. `wmaPct[i]` is `null` during WMA warmup
(`i < length - 1`) and whenever `wma[i]` is zero.

## Defaults

| Knob              | Value     |
|-------------------|-----------|
| length            | 14        |
| bullishThreshold  | 0         |
| bearishThreshold  | 0         |
| wmaColor          | `#84cc16` (lime) |
| pctColor          | `#c026d3` (magenta-purple) |

Lime + magenta-purple disambiguates the top-panel WMA overlay and
the bottom-panel oscillator from the parallel SMA / EMA pct
primitives (emerald + purple for SMA, orange + violet for EMA).

## WMA helper precision

The WMA helper carries a `winMin === winMax` short-circuit so
constant-window slices return their exact value without 1-ULP drift.

## Bit-exact anchors

- `CONST close = K, K != 0`: weighted sum = `K * sum(weights)`,
  divided by `sum(weights)` returns `K` exactly via the helper's
  min/max short-circuit. `wmaPct = (K - K) / K * 100 = 0` bit-exact
  post-warmup. Verified across `(K, length)` sweeps.
- `CONST close = 0`: `wma = 0`, divide-by-zero guard -> `null`.
- `LINEAR UP close = i + 1, L = 7 at i = 6`: weighted sum =
  `7*7 + 6*6 + ... + 1*1 = sum of squares 1..7 = 140`, sum of
  weights = `7 * 8 / 2 = 28`, `wma = 140 / 28 = 5` exact.
  `delta = 7 - 5 = 2` exact. `wmaPct = 2 / 5 * 100 ~= 40`. The
  intermediate `2 / 5` is not dyadic so the result may carry 1
  ULP -- asserted with `toBeCloseTo(40, 10)`.

## Surface

Standard primitive surface mirroring 11.836 / 11.837:

- `ChartLineWmaPctPoint = { x, close }`
- `runLineWmaPct(data, options?) -> Run` with
  `ok = series.length >= length`.
- `computeLineWmaPctLayout({ data, width?, height?, padding?, panelGap?, length?, bullishThreshold?, bearishThreshold? }) -> Layout`.
  Top panel renders close + WMA overlay; bottom panel renders the
  wmaPct oscillator. Bottom y-axis auto-fits but always includes 0.
- `describeLineWmaPctChart(data, opts?) -> string` for the ARIA
  description.
- `<ChartLineWmaPct />` `forwardRef<HTMLDivElement>` with three
  legend toggles (`price`, `wma`, `pct`), config badge, optional
  threshold band + zero-line overlay, markers on cross events, and
  a tooltip surfacing `close / wma / delta / wmaPct / zone / cross`.
