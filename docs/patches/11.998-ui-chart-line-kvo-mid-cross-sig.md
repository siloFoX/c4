## Summary

Add `web/src/components/ui/chart-line-kvo-mid-cross-sig.tsx` --
a pure-SVG dual-panel React/TypeScript chart primitive marking
discrete bullish (KVO signal crosses up through midline 0, volume
momentum centerline trigger) / bearish (KVO signal crosses down
through midline 0) KVO midline-over-signal crossover events.
First volume-aware primitive in the chart-line family: requires
HLCV (high, low, close, volume) input.

## Math anchor (bit-exact)

```
typical[i]   = (high + low + close) / 3
direction[i] = sign(typical[i] - typical[i-1])
vf[i]        = direction[i] * volume[i]
fastSma      = SMA(vf, fastLength)
slowSma      = SMA(vf, slowLength)
kvo          = fastSma - slowSma
signal       = SMA(kvo, kSmoothing)
bullish      : prev signal <= 0 && cur signal >  0
bearish      : prev signal >= 0 && cur signal <  0
```

- **CONST H=L=C=K, V=const**: typical=K -> direction=0 -> vf=0
  -> fast=slow=kvo=0 -> signal=0 (Object.is +0) -> bullish at
  boundary (0 crosses). Verified across K = 0..1234 and V = 1,
  100, 1000, 1234.
- **LINEAR UP H=L=C=i, V=const**: direction=+1, vf=+V -> kvo=0
  -> signal=0 -> bullish (0 crosses).
- **LINEAR DOWN H=L=C=-i, V=const**: direction=-1, vf=-V ->
  kvo=0 -> signal=0 -> bullish (0 crosses).

For 100 bars at defaults (fastLength=34, slowLength=55,
kSmoothing=13): noneCount=67, bullishCount=33.

## API

```ts
interface ChartLineKvoMidCrossSigProps {
  data: { x: number; high: number; low: number; close: number; volume: number }[];
  fastLength?: number;     // default 34
  slowLength?: number;     // default 55
  kSmoothing?: number;     // default 13
  threshold?: number;      // default 0 (midline)
}
```
