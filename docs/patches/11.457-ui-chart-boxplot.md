## Summary

Add a new `<ChartBoxplot>` UI primitive in
`web/src/components/ui/chart-boxplot.tsx` -- pure-SVG
box-and-whisker plot (Tukey). Each series renders as one
box from Q1 to Q3 (IQR) with a median line; whiskers
extend to the lowest and highest non-outlier values using
the configurable IQR fence multiplier (default 1.5);
outlier values beyond the fences plot as dots. Optional
mean marker (dashed) inside the box. Hovering a box opens
a tooltip with the five-number summary (min, Q1, median,
Q3, max) plus mean and outlier count. Per-box click
handler.

## API

```ts
interface ChartBoxplotSeries {
  id: string;
  label: string;
  data: readonly number[];
  color?: string;
}

interface ChartBoxplotProps {
  series: readonly ChartBoxplotSeries[];
  width?: number;                  // default 560
  height?: number;                 // default 320
  padding?: number;                // default 40
  boxGap?: number;                 // default 16
  outlierMultiplier?: number;      // default 1.5
  showLabels?: boolean;            // default true
  showOutliers?: boolean;          // default true
  showMean?: boolean;              // default true
  showTooltip?: boolean;           // default true
  showAxisTicks?: boolean;         // default true
  animate?: boolean;               // default true
  className?: string;
  ariaLabel?: string;              // default 'Box plot'
  ariaDescription?: string;
  formatValue?: (v: number) => string;
  defaultBoxColor?: string;        // default '#2563eb'
  tickCount?: number;              // default 5
  onBoxClick?: (args: {
    series: ChartBoxplotSeries;
    index: number;
    stats: BoxplotStats;
  }) => void;
}

interface BoxplotStats {
  min: number;
  q1: number;
  median: number;
  q3: number;
  max: number;
  mean: number;
  lowerWhisker: number;
  upperWhisker: number;
  outliers: number[];
  count: number;
}
```

## Behaviour

- Renders a `<div role="region">` wrapping `<svg
  role="img">` carrying the same `aria-label`.
- Stats: `getBoxplotStats` filters non-finite values,
  sorts the remaining ascending, and computes the
  five-number summary via linear-interpolation
  quantiles (R-7 / Excel default). Outliers = values
  beyond `Q1 - k*IQR` or `Q3 + k*IQR` where `k =
  outlierMultiplier` (default 1.5). Whiskers extend to
  the most extreme value within those fences.
- Box geometry: each series occupies a vertical slot
  of width
  `(innerWidth - boxGap * (n - 1)) / n`. The box
  itself is 60% of the slot width, centred. The IQR
  rectangle spans `[Q1, Q3]`. A horizontal line at
  the median crosses the box. Optional dashed mean
  marker overlays at the same x-extent.
- Whiskers: vertical lines from Q1 down to
  `lowerWhisker` and from Q3 up to `upperWhisker`,
  capped with horizontal segments.
- Outliers: small filled dots at the box's centre x
  for every value outside the fences (white stroke
  for legibility).
- Hover / focus: thickens the box border + lightens
  the fill, and opens a tooltip with the full
  five-number summary + mean (when `showMean=true`) +
  outlier count (when > 0). `onBoxClick` receives the
  full `BoxplotStats` payload.
- Mount animation:
  `motion-safe:animate-fade-in` per series group on
  mount. `animate=false` removes the class.
- Default ARIA description: `"Box plot with N
  series. <label> median <m> IQR <q1>-<q3>, ..."` --
  `ariaDescription` overrides; empty input produces
  `"No data"`.

## Pure helpers (exported)

```ts
DEFAULT_CHART_BOXPLOT_WIDTH               = 560
DEFAULT_CHART_BOXPLOT_HEIGHT              = 320
DEFAULT_CHART_BOXPLOT_PADDING             = 40
DEFAULT_CHART_BOXPLOT_BOX_GAP             = 16
DEFAULT_CHART_BOXPLOT_OUTLIER_MULTIPLIER  = 1.5
DEFAULT_CHART_BOXPLOT_TICK_COUNT          = 5
DEFAULT_CHART_BOXPLOT_BOX_COLOR           = '#2563eb'

getBoxplotQuantile(sorted, q): number
getBoxplotStats(data, outlierMultiplier?): BoxplotStats
getBoxplotBounds(series): { min, max }
getBoxplotTicks(min, max, count?): number[]
describeBoxplotChart(series, formatValue?): string
```

- `getBoxplotQuantile` uses linear interpolation
  (R-7); single-element arrays return the value;
  empty returns 0; `q` clamps to `[0, 1]`.
- `getBoxplotStats` filters non-finite, sorts, and
  computes the full summary in one pass. Empty / all-
  non-finite series returns a zero-stats record. The
  outlier multiplier is configurable; a tighter
  multiplier yields more outliers, a looser one
  yields fewer.
- `getBoxplotBounds` ignores non-finite values;
  empty fallback `(0, 1)`; collapsed range expands
  by `+/- 0.5`.
- `getBoxplotTicks` is the standard chart-tick helper
  (evenly-spaced; `[min]` for collapsed; min count 2).
- `describeBoxplotChart` produces the default ARIA
  summary; honours `formatValue`.

## Tests (58 cases)

  linear interpolation, clamp q to [0,1].

  five-number summary, mean, outlier classification
  via 1.5 IQR, no outliers for uniform data, whiskers
  within data range, custom multiplier toggles
  outlier count.

  collapsed expansion.
