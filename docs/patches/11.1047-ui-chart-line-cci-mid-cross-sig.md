# 11.1047 -- UI: chart-line-cci-mid-cross-sig primitive (v1.11.1065)

## Scope

Add a pure-SVG dual-panel `ChartLineCciMidCrossSig`
component that renders the close in the top panel and the
Commodity Channel Index (CCI) plus its smoothed SMA signal
line in the bottom panel, marking bullish / bearish CCI-
over-signal crossover trigger events with bias coloring
derived from the CCI slope at the trigger bar.

Donald Lambert's 1980 CCI is the canonical mean-reversion
oscillator: it measures the deviation of the current
typical price from its rolling SMA, scaled by the rolling
mean absolute deviation. The 0.015 scaling constant
ensures roughly 75% of values fall within `[-100, +100]`
for typical-volatility instruments, making `+/- 100` the
conventional overbought / oversold boundaries and `0` the
natural centerline. This primitive watches the CCI vs its
own SMA-smoothed signal line to detect centerline trend
trigger events.

## CCI pipeline

```
typical[i]   = (high[i] + low[i] + close[i]) / 3
smaTp[i]     = SMA(typical, period)
meanDev[i]   = sum(|typical[j] - smaTp[i]|,
                    j = i - period + 1 .. i) / period
cci[i]       = meanDev[i] > 0
               ? (typical[i] - smaTp[i]) /
                 (0.015 * meanDev[i])
               : 0
signal[i]    = SMA(cci, signalLength)

bullish      : prev cci <= prev signal &&
               cur cci > cur signal
bearish      : prev cci >= prev signal &&
               cur cci < cur signal
regime       : 'bullish' when cci >= signal
               'bearish' when cci <  signal
               'none'    when either is null
bias         : cci[i] vs cci[i-1] -> up / down / flat /
               none
```

Default `period = 14`, `signalLength = 3` -- canonical
Lambert tuning paired with the family-conventional short
SMA signal.

Warmup is `period + signalLength - 2 = 15` for the
default tuning: smaTp seeds at i = period - 1 = 13, CCI is
valid from i = 13, then the signal SMA needs signalLength
- 1 = 2 more bars.

The zero-guard `meanDev > 0 ? formula : 0` prevents 0/0
NaN on CONST input (all deviations are 0, so mean
deviation is 0; the centerline is the natural fallback).

## Bit-exact anchors

All anchors use HLC input.

- **CONST band** `high = K + 1`, `low = K - 1`, `close
  = K`: typical = K, smaTp = K -> typical - smaTp = 0.
  meanDev = 0 (all deviations are 0). The zero-guard
  returns CCI = 0 (instead of 0/0 NaN). signal = 0.
  CCI === signal -> regime `bullish` (>=) for every
  valid bar. 0 crosses. Verified across K in {0, 1, 50,
  200, 1234}.
- **LINEAR UP** `high = i + 1`, `low = i - 1`, `close
  = i`: typical = i, smaTp = i - 6.5. typical - smaTp
  = `+6.5` (constant). meanDev = sum of `|j - (i -
  6.5)|` for j in `[i - 13, i]` = `2 * (0.5 + 1.5 +
  2.5 + 3.5 + 4.5 + 5.5 + 6.5) / 14 = 49 / 14 = 3.5`.
  CCI = `6.5 / (0.015 * 3.5) = 6.5 / 0.0525 = 2600 /
  21 ~= 123.81`. signal = CCI (constant) -> regime
  `bullish` (>=) for the `===` reason. 0 crosses. The
  positive CCI level reflects the canonical "typical
  price above its rolling SMA" mean-reversion signal
  on a steady linear uptrend.
- **LINEAR DOWN** `high = -i + 1`, `low = -i - 1`,
  `close = -i`: mirror -> CCI = `-2600 / 21 ~= -123.81`.
  signal = CCI -> regime `bullish` (>=) for the same
  `===` reason. 0 crosses.

## Risk

- Pure-SVG, deterministic `.toFixed(2)`, no canvas, no
  chart lib dependency.
- 3-state regime classifier follows the cross-sig family
  convention (bullish on `===`, bearish on `<`); the
  divergence-cross sibling family uses a 5-state model.
- Zero-guard on meanDev = 0 prevents NaN on CONST input
  -- returns the natural centerline value of 0. This is
  the canonical handling in TA-Lib and pyalgotrade.
- CCI is *not* hard-bounded -- in trending markets it can
  exceed `+/- 200`. The layout auto-ranges over actual
  CCI values, clamping only to guarantee that the
  centerline (0) is always within view.
- On steady LINEAR input CCI pins at `+/- 2600/21` and
  produces zero crosses, which is correct canonical
  behaviour -- real-world price data with noise and
  trend-direction changes produces typical CCI cross
  patterns.
- Complementary to the sibling SMA / EMA / Bollinger /
  Keltner / Donchian mid-cross-sig primitives -- CCI adds
  a mean-reversion-flavoured oscillator to the cross-sig
  family, distinct from the trend-flavoured price-against-
  itself siblings.
