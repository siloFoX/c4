Average (SMA) plus its smoothed SMA signal line in the bottom

## SMA-over-signal pipeline

```
sma[i]    = SMA(close, period)
signal[i] = SMA(sma, signalLength)
bullish   : prev sma <= prev signal && cur sma > cur signal
bearish   : prev sma >= prev signal && cur sma < cur signal
regime    : bullish (sma >= signal), bearish (sma < signal)
bias      : up/down/flat/none from sma[i] vs sma[i-1]
```

Plain SMA with a CONST short-circuit (min === max in the
window returns the constant bit-exactly). The signal line is
a second SMA of the indicator -- the lighter-weight (no EMA
recurrence) sibling of `chart-line-ema-cross-sig`.

Defaults: `period = 14`, `signalLength = 3`. Bias-coloured
cross markers match the convention used by
`chart-line-ema-cross-sig` / `chart-line-tema-cross-sig`.

Warmup is `period + signalLength - 2 = 15` for the default
tuning: SMA seeds at `period - 1 = 13`, then the signal SMA
adds another `signalLength - 1 = 2` bars.

## Bit-exact anchors

| input              | sma (post-warmup) | signal (post-warmup) | sma - signal | regime  | crosses |
| ------------------ | ----------------- | -------------------- | ------------ | ------- | ------- |
| CONST close = K    | K                 | K                    | 0            | bullish | 0       |
| LINEAR UP, close=i | i - 6.5           | i - 7.5              | +1           | bullish | 0       |
| LINEAR DOWN, close=-i | -i + 6.5       | -i + 7.5             | -1           | bearish | 0       |

CONST K verified across K = 0..1234 -- SMA of K = K
exactly via the `min === max` short-circuit; signal = SMA(K,
3) = K. sma === signal -> regime `bullish` (>=).

LINEAR UP exercises the SMA centroid-lag algebra: for period
= 14, the SMA centroid lag is `(period - 1) / 2 = 6.5`, so
sma = `i - 6.5`. The signal SMA of length 3 adds another
`(signalLength - 1)/2 = 1` bar of lag, giving signal = `i -
7.5`. sma - signal = `+1` constant -> regime `bullish`.

LINEAR DOWN is the algebraic mirror: sma = `-i + 6.5`,
signal = `-i + 7.5`, sma - signal = `-1` -> regime
`bearish`.
