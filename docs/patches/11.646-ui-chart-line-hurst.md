## Summary

New `ChartLineHurst` UI primitive at
`web/src/components/ui/chart-line-hurst.tsx` -- a pure-SVG
two-panel chart with a rolling Hurst exponent panel classifying
the series as trending or mean reverting.

## Indicator math

The Hurst exponent measures the long-term memory of a series.

- `computeLineHurstRescaledRange` -- rescaled range (R/S)
  analysis of a single window. The window is mean-adjusted; the
  cumulative deviation is tracked bar by bar; the range `R` is
  the spread of that cumulative deviation; `S` is the population
  standard deviation of the window. The rescaled range is
  `R / S`. A flat window has `S = 0` and an undefined (NaN)
  rescaled range.

- `computeLineHurst` -- the rolling estimate. For each bar the
  trailing window of `period` values is analyzed and the Hurst
  exponent is the single-window R/S estimate:

```
H = log(R/S) / log(period)
```

  clamped into the unit interval `[0, 1]`. `H > 0.5` marks a
  persistent (trending) series, `H < 0.5` an anti-persistent
  (mean reverting) series, `H` near 0.5 a random walk. Bars
  before the window is full -- and flat windows -- are null.

### Worked fixture

`H = log(R/S)/log(period)` involves `log` and `sqrt` and is
generally irrational, so it is asserted with `toBeCloseTo` plus
exact anchors. The rescaled range is exact-testable on its own:

- Step window `[8,8,12,12]`: mean 10, cumulative deviation
  `[-2,-4,-2,0]`, `R = 4`, `S = 2`, `R/S = 2`.
- Alternating window `[12,8,12,8]`: cumulative deviation
  `[2,0,2,0]`, `R = 2`, `S = 2`, `R/S = 1` -- so
  `H = log(1)/log(4) = 0`, exactly. An alternating series is
  maximally mean reverting.
- Trending step `[8,8,8,8,12,12,12,12]`: `R = 8`, `S = 2`,
  `R/S = 4` -- so `H = log(4)/log(8) ~ 0.667`, trending.
- A flat window gives `R/S` NaN, so its Hurst exponent is null.

## API

- `getLineHurstFinitePoints(points)` -- drop points with a
  non-finite `x` or `value`.
- `normalizeLineHurstPeriod(period, fallback)` -- coerce to an
  integer of at least 2 (so `log(period)` stays above zero).
- `computeLineHurstRescaledRange(window)` -- the
  `{ range, stddev, rs }` of a single window.
- `computeLineHurst(values, period)` -- the
  `{ hurst, rs }` rolling series.
- `runLineHurst(points, options?)` -- sort finite points by `x`,
  run the pipeline, return `{ series, period, hurst, rs,
  samples, hurstFinal, hurstMin, hurstMax, trendingCount,
  revertingCount, ok }`. Each sample is classified `trending` /
  `reverting` / `random`. `ok` is false for fewer than two
  points.
- `computeLineHurstLayout(options)` -- stack the price panel
  above a Hurst panel with a fixed `[0, 1]` y-domain; project
  the price and Hurst paths plus markers and the 0.5 reference
  line.
- `describeLineHurstChart(data, options?)` -- ARIA description.

## Component

`ChartLineHurst` is a `forwardRef` region (`role="region"`) with
an ARIA-linked description, a `role="img"` SVG, two stacked
panels (price on top, Hurst below) with their own axis ticks and
grid, panel labels, a config badge showing the period, a
two-series legend (Price / Hurst) with toggle, a hover/focus
tooltip exposing the value, rescaled range, Hurst and class, and
a `motion-safe:animate-fade-in` entrance. It consumes
`{ x, value }` points. The Hurst panel has a fixed `[0, 1]`
y-domain with a dashed 0.5 reference line (the random-walk level)
and class-coloured markers (green trending, red reverting, grey
random). Every node carries a `data-section` attribute.

## Distinctness

- **vs the oscillator panels (roofing / trendflex / reflex /
  pmo)** -- those draw a zero-centred oscillator with a
  data-driven y-domain. The Hurst panel is a fixed unit-interval
  gauge with a 0.5 midline -- the Hurst exponent is bounded to
  `[0, 1]` by definition.
- **vs chart-line-trendflex** -- the Trendflex measures trend
  strength from a Super Smoother's displacement. The Hurst
  exponent measures the long-term memory of the raw series via
  rescaled range analysis -- a statistical, not a filter-based,
  classification.
- **vs the band primitives** -- those draw a band around the
  price. The Hurst panel is a classification gauge.
