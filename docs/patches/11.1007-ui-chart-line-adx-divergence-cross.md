# 11.1007 -- UI: chart-line-adx-divergence-cross primitive (v1.11.1025)

## Scope

Add a pure-SVG dual-panel `ChartLineAdxDivergenceCross`
component that renders the close in the top panel and the
Wilder Average Directional Index (ADX) in the bottom panel,
marking discrete price-versus-ADX direction disagreement events
as bullish / bearish divergence cross transitions. Divergence
variant of the directional movement family; pairs the canonical
Wilder ADX (smoothed +/- DI -> DX -> ADX) with the same 5-state
regime model already used by `chart-line-cmo-divergence-cross`,
`chart-line-rsi-divergence-cross`,
`chart-line-tsi-divergence-cross`, and
`chart-line-trix-divergence-cross`. First primitive in the
divergence family that consumes HLC inputs instead of close-only.

## ADX pipeline (Wilder 1978)

```
highMove[i]  = high[i] - high[i-1]
lowMove[i]   = low[i-1] - low[i]
plusDM[i]    = highMove > lowMove && highMove > 0 ? highMove : 0
minusDM[i]   = lowMove  > highMove && lowMove > 0 ? lowMove  : 0
tr[i]        = max(high - low,
                   |high - prevClose|,
                   |low  - prevClose|)
smTr / smPlusDm / smMinusDm = SMA-seeded Wilder RMA over `length`
plusDI[i]    = 100 * smPlusDm  / smTr   (0 when smTr === 0)
minusDI[i]   = 100 * smMinusDm / smTr   (0 when smTr === 0)
dx[i]        = 100 * |plusDI - minusDI| / (plusDI + minusDI);
              0 when (plusDI + minusDI) === 0 (degenerate)
adx[i]       = SMA-seeded Wilder RMA of dx over `length`
```

The SMA-seeded Wilder RMA helper is exported as
`applyLineAdxDivergenceCrossSmaSeededRma`: the first `length`
contiguous valid samples seed the recurrence with their
arithmetic mean, after which `rma[i] = rma[i-1] * (1 - 1/length)
+ value[i] / length`. Any null gap aborts further output. The
alpha is `1/length` rather than the `2/(length+1)` of a standard
EMA -- this matches Wilder's smoothing convention.

Warmup is `2 * length - 1 = 27`: smTr, smPlusDm, smMinusDm
become valid at index `length = 14`, then DX (computed from
those smoothed values) feeds another `length`-bar Wilder RMA to
produce ADX, which is valid starting at index `length + length -
1 = 27`.

## Divergence regime model

5-state classifier exactly like the other divergence primitives:

```
priceUp = close[i] > close[i - divergenceWindow]
adxUp   = adx[i]   > adx[i - divergenceWindow]
state   = aligned-bullish    if priceUp && adxUp
        | aligned-bearish    if !priceUp && !adxUp
        | divergent-bullish  if !priceUp && adxUp   (price down, ADX up)
        | divergent-bearish  if priceUp && !adxUp   (price up, ADX down)
        | none               if either input is null
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
```

Defaults: `length = 14` (Wilder's canonical ADX window),
`divergenceWindow = 5`. Crosses suppress when either `prev` or
`cur` is `none`, so warmup transitions never fire spuriously.
Strict inequality guarantees boundary equality (price flat or
ADX saturated) falls into the aligned-bearish state instead of
double-firing.

## Bit-exact anchors

| input                          | adx (post-warmup) | regime (post i=32)  | crosses |
| ------------------------------ | ----------------- | ------------------- | ------- |
| CONST high=low=close=K         | 0                 | aligned-bearish     | 0       |
| LINEAR UP (h=i+1, l=i-1, c=i)  | 100 (saturated)   | divergent-bearish   | 0       |
| LINEAR DOWN (h=-i+1, l=-i-1, c=-i) | 100 (saturated) | aligned-bearish    | 0       |

CONST HLC = K verified across K = 0..1234 -- TR collapses to 0
(`max(K-K, |K-K|, |K-K|)`), so does +DM / -DM. The smoothed
+/- DI denominator goes to 0 -> degenerate +DI = -DI = 0 -> DX
denominator also 0 -> degenerate DX = 0 -> ADX = 0.

LINEAR UP exercises the saturated-trend case: every bar gives
TR = 2, +DM = 1, -DM = 0 (because `highMove = 1 > lowMove = -1`
and `lowMove < 0`). Wilder smoothing settles immediately
(SMA-of-constants), so smTr = 2, smPlusDm = 1, smMinusDm = 0,
+DI = 50, -DI = 0, DX = 100, and the final Wilder smoothing of
DX produces ADX = 100. priceUp = true, adxUp = false (100 ===
100) -> regime `divergent-bearish` for 48 valid samples on the
80-bar fixture. The cross count stays at 0 because the regime
is entered from `none` (prev = none is never a valid cross
precursor).

LINEAR DOWN is the algebraic mirror: TR = 2, +DM = 0, -DM = 1
-> +DI = 0, -DI = 50, DX = 100, ADX = 100. priceUp = false,
adxUp = false -> regime `aligned-bearish`.
