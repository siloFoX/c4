# 11.1078 -- UI: chart-line-ichimoku-kijun-cross primitive (v1.11.1096)

## Scope

Add a pure-SVG dual-panel `ChartLineIchimokuKijunCross`
component that renders the close in the top panel and the
Ichimoku tenkan-sen, kijun-sen, and kumo (senkou A /
senkou B cloud) in the bottom panel, marking bullish
(price closes up through the kijun-sen baseline --
baseline trigger up) / bearish (price closes down through
the kijun-sen baseline -- baseline trigger down)
kijun-cross events with **cloud-color bias** at the
trigger bar.

The kijun (baseline) cross is a classic Ichimoku trend
signal: the kijun-sen is the 26-period midpoint and acts
as a dynamic support/resistance level. When the close
crosses above the baseline, it signals a shift to
bullish equilibrium; below, bearish. Distinct from the
sibling tenkan-cross (v1.11.1061) which detects the
tenkan/kijun TK-cross; this primitive detects the
price/baseline cross.

## Kijun baseline cross pipeline

```
tenkan[i]    = (HH(high, tenkanPeriod) +
                LL(low,  tenkanPeriod)) / 2
kijun[i]     = (HH(high, kijunPeriod) +
                LL(low,  kijunPeriod)) / 2
senkouA[i]   = (tenkan[i] + kijun[i]) / 2
senkouB[i]   = (HH(high, senkouBPeriod) +
                LL(low,  senkouBPeriod)) / 2

bullish (kijun cross up) :
  prev close <= prev kijun && cur close > cur kijun
bearish (kijun cross down) :
  prev close >= prev kijun && cur close < cur kijun

regime : 'bullish' when close >= kijun
         'bearish' when close <  kijun
         'none'    when either is null

bias (cloud color at trigger bar) :
  'bullish' when senkouA >  senkouB (green cloud)
  'bearish' when senkouA <  senkouB (red cloud)
  'flat'    when senkouA === senkouB (flat cloud)
  'none'    when either is null
```

Default `tenkanPeriod = 9`, `kijunPeriod = 26`,
`senkouBPeriod = 52` -- canonical Goichi Hosoda (1969)
Ichimoku Kinko Hyo tuning. The tenkan line is still
computed and rendered (it feeds senkouA) but is not part
of the trigger -- the trigger is close vs kijun.

## Sibling family

The Ichimoku family:

- **chart-line-ichimoku-tenkan-cross v1.11.1061** --
  tenkan/kijun TK-cross with cloud bias
- **chart-line-ichimoku-mid-cross-sig v1.11.1067** --
  kumo midline vs SMA signal cross
- **chart-line-ichimoku-divergence-cross** -- kumo
  midline direction vs price direction
- **chart-line-ichimoku-kijun-cross v1.11.1096** (this
  primitive) -- price/kijun baseline cross with cloud
  bias

Distinct from tenkan-cross: the tenkan-cross compares
two indicator lines (tenkan vs kijun), while this
primitive compares price (close) against the kijun
baseline. Both use cloud-color bias at the trigger bar.

Warmup is `kijunPeriod = 26` for kijun cross detection
(kijun valid at i and i-1; close is always present).
Cloud-color bias additionally requires senkouB valid,
which seeds at i = senkouBPeriod - 1 = 51. Crosses that
fire before i = 51 carry `bias = 'none'`.

## Bit-exact anchors

All anchors use HLC input.

- **CONST band** `high = K + 1`, `low = K - 1`,
  `close = K`: kijun = K (constant). close = K. close
  === kijun every bar -> no cross. senkouA = senkouB =
  K -> cloud flat. regime `bullish` (via >=). 0
  crosses. Verified across K in {0, 1, 50, 200, 1234}.
- **LINEAR UP** `high = i + 1`, `low = i - 1`,
  `close = i`: kijun = (i + 1 + i - 26) / 2 = i - 12.5.
  close = i. close - kijun = +12.5 (constant spread,
  close always above kijun) -> no cross. senkouA = i -
  8.25, senkouB = i - 25.5 -> bullish cloud. regime
  `bullish`. 0 crosses.
- **LINEAR DOWN** `high = -i + 1`, `low = -i - 1`,
  `close = -i`: kijun = -i + 12.5. close = -i. close -
  kijun = -12.5 (close always below kijun) -> no cross.
  senkouA = -i + 8.25, senkouB = -i + 25.5 -> bearish
  cloud. regime `bearish`. 0 crosses.
- **PEAK** (linear up then linear down): the close
  rises above the lagging kijun (bullish regime), then
  after the peak falls below the lagging kijun
  baseline, producing a bearish kijun cross. At least
  one bearish cross fires.

All three steady-state anchors produce 0 crosses because
the close stays on the same side of the kijun throughout
(a constant +/-12.5 spread). The PEAK anchor exercises
the cross detector with a real price reversal through
the baseline.

## Risk

- Pure-SVG, deterministic `.toFixed`, no canvas, no
  chart lib dependency.
- 3-state regime classifier follows the cross-sig
  family convention (bullish on `===` via `>=`).
- The cloud-color bias (rather than indicator slope) is
  the idiomatic Ichimoku confirmation, matching the
  tenkan-cross sibling.
- Current-bar reads (no 26-bar forward displacement)
  match the sibling convention. A production Ichimoku
  chart would plot senkouA/B shifted forward 26 bars,
  but the divergence/cross detection here uses
  same-bar values for deterministic testability.
- The signal is close-vs-kijun, distinct from the
  tenkan-vs-kijun TK-cross sibling. Reuses the exact
  same Ichimoku channel computation
  (computeLineIchimokuKijunCross) so the tenkan/kijun/
  senkou values match across the two primitives.
- Steady LINEAR input keeps the close a constant +/-
  12.5 from the kijun (never crossing), so 0 crosses
  fire -- real crosses fire only on actual price
  reversals through the baseline, exercised by the
  PEAK anchor.
- Completes the Ichimoku cross family with the
  price/baseline trigger alongside the tenkan/kijun
  TK-cross, the kumo-midline signal cross, and the
  kumo-divergence detector.
