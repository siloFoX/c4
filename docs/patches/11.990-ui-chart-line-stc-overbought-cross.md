## Summary

Add `web/src/components/ui/chart-line-stc-overbought-cross.tsx`
-- a pure-SVG dual-panel React/TypeScript chart primitive that
renders the close in the top panel and the close-only Schaff
Trend Cycle (STC) line in the bottom panel, marking discrete
bullish (STC crosses up through threshold 75, cycle overbought
trigger) / bearish (STC crosses down through threshold 75) STC
overbought-crossover events with bias coloring derived from the
STC slope at the trigger bar. Overbought-threshold variant of
the STC family: mirror of `chart-line-stc-mid-cross` (11.974)
with the default threshold flipped from 50 to the canonical
overbought level 75, and cross markers are tinted by the STC
slope bias (`up` / `down` / `flat`) so a bullish entry on
rising STC reads brighter than one on a flat STC.

## Math anchor (bit-exact)

```
macd      = SMA(close, fast) - SMA(close, slow)
k1        = stochastic(macd, cycleLength)  (degenerate = 50)
d1        = EMA(k1, factor)                (seed = first valid)
k2        = stochastic(d1, cycleLength)    (degenerate = 50)
stc       = EMA(k2, factor)                (seed = first valid)
bullish   : prev stc <= 75 && cur stc >  75
bearish   : prev stc >= 75 && cur stc <  75
bias      : sign(stc[i] - stc[i-1]) -- 'up' / 'down' / 'flat' / 'none'
```

- **CONST close = K**: macd = 0 -> stochastic degenerate ->
  k1 = 50, d1 = 50 (EMA seed), k2 = 50 (degenerate), stc = 50.
  50 < 75 -> regime `bearish`. 0 overbought crosses. Verified
  across K = 0..1234.
- **LINEAR UP close = i**: macd = 13.5 constant (fast 23 lag
  11, slow 50 lag 24.5). stochastic over constant macd ->
  degenerate -> k1 = 50, ..., stc = 50. 50 < 75 -> regime
  `bearish`. 0 crosses. The full pipeline collapses on
  monotonic ramps -- canonical STC behavior on trending
  inputs after the cycle window saturates.
- **LINEAR DOWN close = -i**: macd = -13.5 constant. Same
  cascade -> stc = 50. regime `bearish`. 0 crosses.

For 80 bars at defaults (fastLength=23, slowLength=50,
cycleLength=10, factor=0.5):
- macd null for i = 0..48 (49 entries)
- k1 / d1 null for i = 0..57 (58 entries)
- k2 / stc null for i = 0..66 (67 entries)
- post-warmup regime count = 13 (i = 67..79)

EMA factor = 0.5 keeps EMA bit-exact on constant inputs: ema
= 0.5 * v + 0.5 * v = v.

Strict inequality on `cur` (`cur > 75` for bullish, `cur < 75`
for bearish) means boundary equality (stc == 75) never double-
fires.

Layout uses fixed oscMin=0 / oscMax=100 (STC canonical range).

## Accessibility

- Wrapping `<div role="region" aria-label="STC Overbought Cross
  chart" aria-describedby="...">` with the sr-only description
  copy carrying the full STC parameter set (fastLength,
  slowLength, cycleLength, factor, threshold) plus a "cycle
  overbought trigger / bias coloring" framing.
- `<svg role="img" aria-labelledby="..." aria-describedby="..."
  tabIndex={0}>` for keyboard focus.
- Each cross marker (both the STC-panel dot and the price-panel
  chevron overlay) carries `role="graphics-symbol"`,
  `tabIndex={0}`, an `aria-label="<bullish|bearish> cross at <x>
  bias <up|down|flat|none>"` and a `data-bias` attribute so
  screen readers and CSS-based tooling can both expose the
  bias.
- Legend buttons (2 series: price, stc) expose `aria-pressed`
  and toggle via mouse, focus, Enter, or Space.
