# 11.764 chart-line-delta-volume (v1.11.782)

## Summary

Pure-SVG dual-panel React/TypeScript chart primitive with a
**Delta Volume** oscillator panel beneath the close. The
oscillator signs each bar's volume by the direction of the
close relative to the prior close. Added as the
`<ChartLineDeltaVolume />` UI primitive in
`web/src/components/ui/`.

## Algorithm

For each bar (with prior close `pc`):

```
sign[i]  = +1   if close[i] > pc
         = -1   if close[i] < pc
         =  0   otherwise   (unchanged or seed)
delta[i] = sign[i] * volume[i]
```

The seed bar (`i = 0`) has no prior close and is treated as
`sign = 0`, `delta = 0`. A bar whose close or volume is
non-finite is nulled; the prior reference is preserved so the
next finite bar resumes signing.

## Anchors

- **Rising (close > prior)**: `sign = +1`, `delta = +volume[i]`
  bit-exact. Verified on a `rising(5)` fixture: deltas
  `0, +100, +100, +100, +100`.
- **Falling (close < prior)**: `sign = -1`,
  `delta = -volume[i]` bit-exact. Verified on `falling(5)`:
  deltas `0, -100, -100, -100, -100`.
- **CONST_CLOSE**: `sign = 0`, `delta = 0` at every bar
  bit-exact.
- **Seed bar**: `delta[0] = 0` (no prior reference).
- **Antisymmetry**: reversing the direction flips the sign of
  every delta.
- **Variable volume**: deltas scale linearly with volume
  bit-exact.
- **Zigzag**: alternating up/down bars produce alternating
  deltas with the same magnitude.
- **Worked sum**: 3 up bars at vol 100 plus 2 down bars at
  vol 50 nets `300 - 100 = 200` bit-exact.
