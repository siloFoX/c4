# 11.573 -- web/src/components/ui/chart-line-keltner.tsx

Pure-SVG line chart with **Keltner channels** -- a
volatility envelope built from an EMA centre line and an
ATR-scaled band. The middle line is an exponential moving
average of the price; the upper and lower bands sit a
multiple of the Average True Range above and below it.

## EMA centre + ATR band

- `computeLineKeltnerEMA(values, period)` -- the
  exponential moving average. The smoothing factor is
  `alpha = 2 / (period + 1)`; the series is seeded with its
  first value, so the EMA is defined at every index.
- `computeLineKeltnerATR(values, period)` -- the Average
  True Range, Wilder-smoothed. For a single-value series the
  true range at index `i` is the absolute period-over-period
  change `|v[i] - v[i-1]|`. The first ATR (at index
  `period`) is the simple mean of the first `period` true
  ranges; subsequent values use Wilder smoothing
  `(prev * (period - 1) + tr) / period`. Indices before the
  window is full are `null`.
- `runLineKeltner` combines them: the **middle** line is the
  EMA, the **upper** band is `middle + multiplier * ATR` and
  the **lower** band is `middle - multiplier * ATR` (defined
  only where the ATR is).

The component fixture (EMA 3, ATR 2, multiplier 2 on a
6-point series) makes every value exact: the EMA is
`[10, 15, 22.5, 21.25, 30.625, 30.3125]`, the ATR is
`[null, null, 10, 10, 15, 12.5]`, so the band at index 2 is
`2.5 .. 42.5` (`22.5 +/- 2 * 10`).

## Distinct from `chart-line-bollinger`

Keltner and Bollinger are both volatility envelopes, but
they differ in **both** components:

- **Centre.** Bollinger centres on a *simple* moving average
  (SMA); Keltner centres on an *exponential* moving average
  (EMA), which reacts faster to recent moves.
- **Band width.** Bollinger scales the band by the
  *standard deviation* of price -- a dispersion measure.
  Keltner scales it by the *Average True Range* -- a
  range-based volatility measure, Wilder-smoothed. ATR
  reacts to the size of each step, not the spread of values
  around a mean.

It also differs from `chart-line-ewmsd` (EWMA centre +
exponentially-weighted *standard deviation* band): again the
band is deviation-based there and range-based (ATR) here.

All exported helpers carry a `LineKeltner` infix so the
`export *` barrel line cannot collide.

  channel parameters (defaults 20 / 10 / 2).

  false), `showMiddle`, `showChannel`, `showTooltip`,

- Standard sizing + format + a11y props. `formatValue` /
  `formatX`.
- `onPointClick({point})` / `onSeriesToggle({seriesId,
  hidden})`.

## Tooltip / badge / legend

Hovering a price dot shows x, price, the EMA value, the
band range and the ATR at that index. Config badge:
`KC ema=<emaPeriod> atr=<atrPeriod> x<multiplier>`. The
legend has three toggle items (Price / EMA / Channel) plus a
banded-point count.
