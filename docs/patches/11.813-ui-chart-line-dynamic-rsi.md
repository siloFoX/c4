# 11.813 UI: chart-line-dynamic-rsi primitive (v1.11.831)

## Summary
Pure-SVG dual-panel React/TS primitive that plots a **Dynamic RSI**
oscillator beneath the close. The RSI lookback adapts each bar to the
ratio of short-window volatility over long-window volatility, then
recomputes a Wilder-style RSI (SMA smoothing of up/down moves) over
the adaptive window.

## Formula
```
shortStd[i]   = popStdDev(close, shortVolLength)[i]
longStd[i]    = SMA(shortStd, longVolLength)[i]
ratio[i]      = longStd[i] > 0 ? shortStd[i] / longStd[i] : 1
dynLength[i]  = clamp(round(baseLength / ratio[i]),
                      minLength, maxLength)
up[k]         = max(0, close[k] - close[k - 1])
down[k]       = max(0, close[k - 1] - close[k])
avgUp[i]      = mean(up[i - dynLength + 1 .. i])
avgDown[i]    = mean(down[i - dynLength + 1 .. i])
rsi[i]        = (avgUp + avgDown > 0)
                  ? 100 * avgUp / (avgUp + avgDown)
                  : 50    (neutral fallback when both are zero)
```
`rsi[i]` is `null` during warmup (when dynLength is not yet defined
or when `i - dynLength < 0`).

## Defaults
- `baseLength = 14`
- `shortVolLength = 5`, `longVolLength = 10`
- `minLength = 5`, `maxLength = 30`
- `overbought = 70`, `oversold = 30`

## Zones and crosses
- `overbought`: `rsi >= overbought`
- `oversold`: `rsi <= oversold`
- `neutral`: in between
- `none`: warmup / non-finite
- `crossed`: `'up'` when entering overbought (prev `< overbought`,
  current `>= overbought`), `'down'` when entering oversold. Markers
  fire on these threshold transitions only.

## Bit-exact testability anchors
- **CONST close** (`close = K`): every up and down is `0` ->
  `avgUp = avgDown = 0` -> neutral fallback -> `rsi = 50` (bit-exact)
  across `K in {1, 5, 100, -3}`.
- **MONOTONIC UP** (`close[k] = k + 1`): every up = `1`, every down =
  `0` -> `avgUp = 1`, `avgDown = 0` -> `rsi = 100` (bit-exact).
- **MONOTONIC DOWN** (`close[k] = N - k`): mirror -> `rsi = 0`
  (bit-exact).

## Design notes
- The simple SMA approach (rather than Wilder recursive smoothing
  with a fixed seed) makes the indicator recomputable per bar with
  the adaptive window without tracking long state -- and keeps the
  bit-exact MONO and CONST anchors clean.
- Neutral fallback when `avgUp + avgDown == 0` keeps the CONST anchor
  finite at 50 rather than returning `NaN` from `0/0`.
- RSI axis is **fixed at [0, 100]** since the metric is bounded.
- Overbought and oversold threshold lines dashed; midline dashed
  (different stroke pattern) for the 50 reference.
- Default colors: RSI line `#0d9488` (teal), thresholds `#9333ea`
  (purple), midline `#475569`, bullish `#16a34a`, bearish `#dc2626`.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
