# 11.823 UI: chart-line-trend-detector primitive (v1.11.841)

## Summary
Pure-SVG dual-panel React/TS primitive that compares the SMA slope
against a threshold to flag each bar as `uptrend` (slope >= threshold),
`downtrend` (slope <= -threshold), or `ranging`. The slope panel
plots the per-bar slope of the SMA over `slopeLookback` bars and
marks regime transitions.

## Formula
```
mean[i]   = SMA(close, length)[i]
slope[i]  = (mean[i] - mean[i - slopeLookback]) / slopeLookback
regime[i] = slope[i] >=  threshold ?  1
          : slope[i] <= -threshold ? -1
          : 0
```
`slope[i]` and `regime[i]` are `null` during warmup (`i < length - 1 +
slopeLookback`) or when any input is non-finite.

## Defaults
- `length = 14`, `slopeLookback = 5`, `threshold = 0.1`
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `uptrend`: `regime == 1`
- `downtrend`: `regime == -1`
- `ranging`: `regime == 0`
- `none`: warmup / non-finite
- `crossed`: `'up'` when entering uptrend from any other regime,
  `'down'` when entering downtrend. Markers fire on regime transitions.

## Bit-exact testability anchors
- **CONST close=K**: `mean = K`, `slope = 0`, `regime = 0` everywhere
  across `K in {0, 1, 5, 100, -3}` and `length in {3, 4, 7, 10}`.
- **LINEAR UP** (`close[i] = i + 1`): SMA of consecutive integers
  over `length` bars equals `i + 1 - (length - 1) / 2`. The slope
  between two such means separated by `slopeLookback` bars is exactly
  `slopeLookback / slopeLookback = 1` (integer-exact). Verified
  across `(length, slopeLookback) in {(3, 2), (3, 3), (3, 5), (4, 2),
  ..., (7, 5)}`.
- **LINEAR DOWN** (`close[i] = N - i`): mirror -> `slope = -1`,
  `regime = -1`.
- **Threshold gates regime**: `LINEAR UP` with `threshold = 2` ->
  `slope = 1 < 2` -> `regime = 0` (ranging, not uptrend). Verified.

## Design notes
- The bit-exact LINEAR slope identity follows from the integer
  arithmetic of the SMA: SMA of consecutive integers is `(sum) /
  length = ((i + 1) + (i + 1 - length + 1)) / 2 = i + 1 - (length -
  1) / 2`, so the difference between SMAs separated by `slopeLookback`
  bars is exactly `slopeLookback`, and the slope is `slopeLookback /
  slopeLookback = 1`. Both numerator and denominator are exact
  integers (or dyadics) in IEEE 754 within `MAX_SAFE_INTEGER`.
- Slope axis auto-scales with a `max(threshold * 1.5, 0.5)` cushion
  on each side so the threshold reference lines stay visible even
  when slopes are small.
- Threshold lines dashed `#9333ea` (purple), zero line dashed
  `#475569` (different stroke pattern).
- Default colors: slope `#0ea5e9` (sky blue), bullish `#16a34a`,
  bearish `#dc2626`, thresholds `#9333ea`.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
