## Indicator

Welles Wilder's Accumulative Swing Index (ASI) is the running
total of his Swing Index, which scores each bar-to-bar price
swing. It needs the full OHLC bar, so `ChartLineAsiPoint` carries
`{x, open, high, low, close}`.

The Swing Index for a bar, given the current `(O, H, L, C)` and
the prior `(Op, Cp)`:

- `K = max(|H - Cp|, |L - Cp|)`.
- `R` is chosen from three candidates -- `|H - Cp|`, `|L - Cp|`
  and `|H - L|` -- by which is largest, each branch adding
  `0.25 * |Cp - Op|`.
- `N = (C - Cp) + 0.5 * (C - O) + 0.25 * (Cp - Op)`.
- `SI = 50 * (N / R) * (K / limitMove)`, defined as zero when `R`
  is zero.

Helpers:

- `computeLineAsiSwingIndex(bars, limitMove)` -- the per-bar
  Swing Index; the opening bar is null.
- `computeLineAsi(bars, limitMove)` -- the Accumulative Swing
  Index, the running cumulative sum, starting at zero.
- `runLineAsi` -- sorts the finite bars by x, runs the pipeline,
  and returns per-bar samples classified `up` / `down` / `flat`,
  the zone counts and the final reading; `ok` is false for fewer
  than two bars.
- `computeLineAsiLayout` -- stacks the price panel above an
  Accumulative Swing Index panel on a dynamic, zero-centred
  scale, with the cumulative line and one marker per scored bar.
- `describeLineAsiChart` -- an off-screen sentence summary for
  assistive technology.

## Testability

The Swing Index divides by `R` and by the limit move, so the
fixture pins both to powers of two: every bar's high-to-low range
is 8 and the limit move is 4. The numerator and `R` are integer
sums with 0.5 and 0.25 weights -- all exact binary fractions for
integer prices -- so every Swing Index lands on a quarter and the
cumulative Accumulative Swing Index stays exact, asserted with
`toEqual`. The main fixture uses doji bars so its R picks exercise
each of the three branches (c, c, a, b, c); a second fixture with
open differing from close exercises the open-weighted numerator
term and the `0.25 * |Cp - Op|` term in R; a flat fixture drives
`R` to zero to check the divide-by-zero guard.
