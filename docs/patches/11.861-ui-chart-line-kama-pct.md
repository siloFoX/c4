## Summary

Added `<ChartLineKamaPct />` -- a pure-SVG, dual-panel React/TS
primitive that overlays the close with the Kaufman Adaptive
Moving Average in the top panel and renders the KAMA percent
deviation in the bottom panel. The efficiency-weighted KAMA
adapts its smoothing constant between fast and slow extremes,
producing a non-uniform divergence from price that the percent
panel surfaces directly:

```
ER[i]      = |close[i] - close[i - erLength]| /
             sum(|close[k] - close[k-1]|) over i-erLength+1..i
fastSC     = 2 / (fastLength + 1)
slowSC     = 2 / (slowLength + 1)
SC[i]      = (ER[i] * (fastSC - slowSC) + slowSC) ^ 2
KAMA[seed] = close[erLength]
KAMA[i]    = KAMA[i-1] + SC[i] * (close[i] - KAMA[i-1])
kamaPct[i] = KAMA[i] === 0 ? null
                           : (close[i] - KAMA[i]) / KAMA[i] * 100
```

Regime classifier: `above` (kamaPct > 0), `below` (< 0),
`at` (= 0), `none` (null). Defaults: `erLength = 10`,
`fastLength = 2`, `slowLength = 30`, kamaColor `#7c3aed`
(violet), pctColor `#0ea5e9` (sky), zero line `#475569`.

## Bit-exact anchor

- **CONST close = K (K > 0)**: every price change is 0 -> ER = 0
  -> SC = `slowSC^2`. KAMA[seed] = K, then
  `K + SC * (K - K) = K`. KAMA stays K bit-exactly,
  `close - KAMA = 0`, `kamaPct = 0 / K * 100 = 0`. Verified
  across multiple K and erLength.
- **CONST close = K = 0**: same as above but the divide-by-zero
  guard leaves `kamaPct = null` (regime `none`).

## Soft anchors

- **LINEAR UP close = i + 1**: KAMA is seeded at the warmup bar
  and from the next bar onward the KAMA increment is less than
  the close increment of 1, so `close - KAMA > 0` and
  `kamaPct > 0` for all subsequent bars.
- **LINEAR DOWN close = N - i**: mirror image, `kamaPct < 0`.
