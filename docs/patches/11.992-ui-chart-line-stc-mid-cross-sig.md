## Summary

Add `web/src/components/ui/chart-line-stc-mid-cross-sig.tsx` --
a pure-SVG dual-panel React/TypeScript chart primitive that
renders the close in the top panel and the close-only Schaff
Trend Cycle (STC) plus its smoothed signal line in the bottom
panel, marking discrete bullish (STC signal crosses up through
midline 50, cycle centerline trigger) / bearish (STC signal
crosses down through midline 50) STC midline-over-signal
crossover events with bias coloring derived from the signal
slope at the trigger bar. Signal-line variant of the STC family:
smooths the STC line with an additional SMA so the trigger
fires only when the smoothed cycle centerline crosses the
midline 50, suppressing the raw-STC whipsaws that plague the
unfiltered variant.

## Math anchor (bit-exact)

```
macd      = SMA(close, fast) - SMA(close, slow)
k1        = stochastic(macd, cycleLength)  (degenerate = 50)
d1        = EMA(k1, factor)
k2        = stochastic(d1, cycleLength)    (degenerate = 50)
stc       = EMA(k2, factor)
signal    = SMA(stc, kSmoothing)
bullish   : prev signal <= 50 && cur signal >  50
bearish   : prev signal >= 50 && cur signal <  50
bias      : sign(signal[i] - signal[i-1]) -- up/down/flat/none
```

- **CONST close = K**: macd = 0 -> cycle collapses through
  stochastic and EMA cascade to stc = 50. signal = SMA(50, 3)
  = 50 via the SMA `min === max` short-circuit. signal == 50
  -> regime `bullish` (boundary inclusive). 0 crosses. Verified
  across K = 0..1234.
- **LINEAR UP close = i**: macd = 13.5 constant. cycle
  collapses to stc = 50. signal = 50. regime `bullish`. 0
  crosses.
- **LINEAR DOWN close = -i**: macd = -13.5. stc = 50. signal
  = 50. regime `bullish`. 0 crosses.

For 80 bars at defaults (fastLength=23, slowLength=50,
cycleLength=10, factor=0.5, kSmoothing=3):
- macd null for i = 0..48
- stc null for i = 0..66
- signal null for i = 0..68 (additional 2-bar SMA warmup)
- post-warmup regime count = 11 (i = 69..79)

Strict inequality on `cur` (`cur > 50` for bullish, `cur < 50`
for bearish) means boundary equality (signal == 50) never
double-fires.

Layout uses fixed oscMin=0 / oscMax=100 (STC canonical range).
