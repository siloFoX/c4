## Summary

Add `web/src/components/ui/chart-line-dpo-cross-sig.tsx` -- a
pure-SVG dual-panel React/TypeScript chart primitive that
renders the close in the top panel and the close-only Detrended
Price Oscillator (DPO) plus its smoothed signal line (SMA of
DPO) in the bottom panel, marking discrete DPO-over-signal-line
crossover events. Signal-line variant of the DPO family that
flags the discrete DPO crossing of its own SMA-smoothed signal
line -- the canonical trigger analysts use to filter raw-DPO
whipsaws while preserving the detrended-momentum responsiveness.

## Math anchor (bit-exact)

```
shift     = floor(length / 2) + 1
sma[i]    = SMA(close, length)[i]
dpo[i]    = close[i - shift] - sma[i]
signal[i] = SMA(dpo, kSmoothing)
bullish   : prev dpo <= prev signal && cur dpo >  cur signal
bearish   : prev dpo >= prev signal && cur dpo <  cur signal
```

- **CONST close = K**: close[i-shift] = K, sma = K -> dpo = 0
  constant for i >= length - 1. signal = SMA(0, 9) = 0 via the
  SMA `min === max` short-circuit. dpo == signal == 0 -> regime
  `bullish` (boundary inclusive). 0 crosses. Verified across K =
  0..1234.
- **LINEAR UP close = i**: close[i-shift] = i - 11, sma[i] = i
  - 9.5 (length=20 constant lag) -> dpo = -1.5 constant for i
  >= 19. signal = SMA(-1.5, 9) = -1.5. dpo == signal == -1.5 ->
  regime `bullish`. 0 crosses. The negative DPO reflects "the
  look-back close sits 1.5 units below the centered SMA" --
  detrended momentum is symmetric around zero for a clean linear
  ramp.
- **LINEAR DOWN close = -i**: close[i-shift] = -(i-11), sma[i]
  = -(i - 9.5) -> dpo = 1.5 constant. signal = 1.5. dpo ==
  signal == 1.5 -> regime `bullish`. 0 crosses.

For 60 bars at defaults (length=20, kSmoothing=9, shift=11):
- dpo null for i = 0..18 (19 entries)
- signal null for i = 0..26 (27 entries)
- post-warmup regime count = 33 (i = 27..59)

Strict inequality on `cur` (`cur > signal` for bullish, `cur <
signal` for bearish) means boundary equality (dpo == signal)
never double-fires.

Layout uses symmetric range around 0 (oscMin = -1.1 * span,
oscMax = +1.1 * span where span = max(|dpoMin|, |dpoMax|))
combining both DPO and signal extrema, with fallback to [-1, 1]
when all values collapse to exactly 0.

## Accessibility

- Wrapping `<div role="region" aria-label="DPO Signal Cross
  chart" aria-describedby="...">` with the sr-only description
  copy carrying the length / kSmoothing / shift parameters plus
  a "smoothed detrended momentum trigger" framing.
- `<svg role="img" aria-labelledby="..." aria-describedby="..."
  tabIndex={0}>` for keyboard focus.
- Each cross marker (both the DPO-panel dot and the price-panel
  chevron overlay) carries `role="graphics-symbol"`,
  `tabIndex={0}`, and an `aria-label="<bullish|bearish> cross at
  <x>"` so screen readers can step through events.
- Legend buttons (3 series: price, dpo, signal) expose
  `aria-pressed` and toggle via mouse, focus, Enter, or Space.
