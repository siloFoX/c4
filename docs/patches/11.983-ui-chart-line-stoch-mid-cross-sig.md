## Summary

Add `web/src/components/ui/chart-line-stoch-mid-cross-sig.tsx` --
a pure-SVG dual-panel React/TypeScript chart primitive that renders
the close in the top panel and a close-only smoothed Stochastic
signal line (SMA of raw %K) in the bottom panel, marking discrete
midline (default `50`) crossover events. This is the
midline-signal-line variant of the Stochastic family: it tracks
when the signal line (smoothed K) crosses the midline 50, flagging
the canonical bull / bear regime transitions analysts treat as
trend confirmation triggers.

The threshold is configurable so the midline can be tuned to the
bias of the underlying close series.

## Math anchor (bit-exact)

```
ll[i]   = min(close[i-n+1..i])
hh[i]   = max(close[i-n+1..i])
rawK[i] = 100 * (close - ll) / (hh - ll); 50 when hh==ll
sigK[i] = SMA(rawK, kSmoothing)
bullish : prev sigK <= 50 && cur sigK >  50
bearish : prev sigK >= 50 && cur sigK <  50
```

- **CONST close = K**: hh = ll = K every bar -> rawK = 50
  (degenerate midline default). SMA(50, 3) = 50 via the SMA
  `min === max` short-circuit. 50 == 50 -> regime `bullish`
  (sigK >= 50). 0 crosses. Verified across K = 0..1234.
- **LINEAR UP close = i**: hh = i, ll = i - 13, rawK = 100. SMA(100,
  3) = 100. 100 > 50 -> regime `bullish`. 0 crosses.
- **LINEAR DOWN close = -i**: hh = -i + 13, ll = -i, rawK = 0. SMA(0,
  3) = 0. 0 < 50 -> regime `bearish`. 0 crosses.

Strict inequality on `cur` (`> T` for bullish, `< T` for bearish)
means boundary equality (sigK == 50) never double-fires a cross.

## Accessibility

- Wrapping `<div role="region" aria-label="Stochastic Midline
  Signal Cross chart" aria-describedby="...">` with the sr-only
  description copy carrying the length / kSmoothing / threshold
  parameters plus a "smoothed K midline trigger" framing.
- `<svg role="img" aria-labelledby="..." aria-describedby="..."
  tabIndex={0}>` for keyboard focus.
- Each cross marker (both the signal-panel dot and the price-panel
  chevron overlay) carries `role="graphics-symbol"`, `tabIndex={0}`,
  and an `aria-label="<bullish|bearish> cross at <x>"` so screen
  readers can step through events.
- Legend buttons expose `aria-pressed` and toggle via mouse, focus,
  Enter, or Space.
