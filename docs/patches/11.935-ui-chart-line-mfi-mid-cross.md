# 11.935 -- UI: chart-line-mfi-mid-cross primitive (v1.11.953)

## Summary

Adds `ChartLineMfiMidCross`, a pure-SVG dual-panel chart that
renders the close in the top panel and a close-only Money Flow
Index (MFI) line in the bottom panel. Marks bullish (cross up
through midline 50) / bearish (cross down through midline 50)
volume-weighted momentum centerline regime trigger events.
Single-threshold cross variant of the MFI family that flags
the discrete MFI level 50 midline crossover distinct from the
canonical overbought (80) / oversold (20) extreme bands.
Continues the midline-cross family (RSI 50, Stochastic 50,
CCI 0, now MFI 50).

## Pipeline

For close-only data with default `length = 14`:

```
delta[i] = close[i] - close[i-1]
posFlow  = sum |delta| where delta > 0  (rolling length window)
negFlow  = sum |delta| where delta < 0  (rolling length window)
mfi[i]   = posFlow > 0 && negFlow > 0
             ? 100 - 100 / (1 + posFlow / negFlow)
             : posFlow > 0 ? 100
               : negFlow > 0 ? 0
                 : 50           (zero-flow neutral fallback)
bullish  : prev mfi <= 50 && cur mfi > 50  (momentum up)
bearish  : prev mfi >= 50 && cur mfi < 50  (momentum down)
```

Close-only adaptation: `|delta close|` acts both as the money-
flow magnitude (price * volume proxy) and the directional cue.
The zero-flow fallback returns 50 so a CONST series sits
exactly on the midline.

## Bit-exact anchor

- CONST `close = K` for any finite K -> delta = 0 every bar
  -> posFlow = negFlow = 0 -> mfi = 50 via the zero-flow
  neutral fallback. mfi = 50 sits on the threshold but the
  strict-inequality detector never fires. regime `bullish`
  (mfi >= 50). cross count = 0. Verified across K in {0, 1,
  42, 100, 1234}.
- LINEAR UP `close = i` (length 14) -> delta = +1 every bar
  -> posFlow > 0, negFlow = 0 -> mfi = 100 constant. 100 > 50,
  regime `bullish`. 0 crosses (mfi jumps from null to 100 in
  one bar so prev-null skips the strict-inequality detector).
- LINEAR DOWN `close = -i` -> delta = -1 every bar -> posFlow
  = 0, negFlow > 0 -> mfi = 0 constant. 0 < 50, regime
  `bearish`. 0 crosses.
- Balanced alternation `[10, 11, 10, 11, ...]` (length 14)
  -> posFlow = negFlow in each window -> mfi = 50, regime
  `bullish` (= 50 boundary), 0 crosses.
- Alternation followed by a large upward jump -> mfi spikes
  above 50, firing 1 bullish midline cross.
- Alternation followed by a large downward jump -> mfi
  crashes below 50, firing 1 bearish midline cross.

## Regime classifier

| mfi range            | regime      |
|----------------------|-------------|
| mfi >= 50            | bullish     |
| mfi < 50             | bearish     |
| null                 | none        |
