## KVO pipeline (Klinger)

```
typical[i]   = (high[i] + low[i] + close[i]) / 3
direction[i] = sign(typical[i] - typical[i-1])
vf[i]        = direction[i] * volume[i]
fastSma[i]   = SMA(vf, fastLength)
slowSma[i]   = SMA(vf, slowLength)
kvo[i]       = fastSma - slowSma
```

Mirrors the canonical KVO formula already used by
`chart-line-kvo-mid-cross-sig` (same volume-force / typical-
price / fast-minus-slow construction). Default tuning matches
Klinger's original recommendation: `fastLength = 34`,
`slowLength = 55`.

Warmup is `slowLength = 55`: vf is valid from index 1 (the
typical-price delta is undefined at index 0), and the slow
SMA needs `slowLength` valid vf samples to seed, putting the
first valid kvo at index `slowLength`.

## Divergence regime model

5-state classifier exactly like the other divergence primitives:

```
priceUp = close[i] > close[i - divergenceWindow]
kvoUp   = kvo[i]   > kvo[i - divergenceWindow]
state   = aligned-bullish    if priceUp && kvoUp
        | aligned-bearish    if !priceUp && !kvoUp
        | divergent-bullish  if !priceUp && kvoUp   (price down, KVO up)
        | divergent-bearish  if priceUp && !kvoUp   (price up, KVO down)
        | none               if either input is null
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
```

Defaults: `fastLength = 34`, `slowLength = 55`,
`divergenceWindow = 5`. Crosses suppress when either `prev` or
`cur` is `none`, so warmup transitions never fire spuriously.
Strict inequality guarantees boundary equality (KVO parked at
zero) falls into the aligned-bearish state instead of double-
firing.

## Bit-exact anchors

| input              | kvo (post-warmup)  | regime (post i=60)  | crosses |
| ------------------ | ------------------ | ------------------- | ------- |
| CONST HLC = K, vol=V | 0                | aligned-bearish     | 0       |
| LINEAR UP, vol=V   | 0 (SMAs collapse)  | divergent-bearish   | 0       |
| LINEAR DOWN, vol=V | 0 (SMAs collapse)  | aligned-bearish     | 0       |

CONST HLC = K verified across K = 0..1234 -- typical-price
delta = 0 every bar so direction = 0 and vf = 0; both fast and
slow SMAs of a flat-zero vf collapse to 0, giving kvo = 0.

LINEAR UP exercises a curious algebraic property: every bar has
delta typical = 1 -> direction = +1 -> vf = +V constant. The
SMA of a constant equals that constant, so fastSma = slowSma =
V and `kvo = V - V = 0` exactly. The KVO parks at the neutral
reading on a pure linear trend because the volume force has no
oscillation for the fast/slow SMAs to separate. priceUp = true,
kvoUp = false (0 === 0) -> regime `divergent-bearish` for the
full 60 valid samples on the 120-bar fixture.

LINEAR DOWN is the algebraic mirror: vf = -V constant ->
fastSma = slowSma = -V -> kvo = 0. priceUp = false, kvoUp =
false -> regime `aligned-bearish`.

Note: the constant-zero KVO on pure linear input is the
canonical signature of the SMA-of-constants identity, not a
real volume momentum signal. The divergence regimes are
correct algebraically and the cross count stays at 0 because
the divergent state is entered from `none`.
