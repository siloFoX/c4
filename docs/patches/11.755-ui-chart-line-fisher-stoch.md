# Patch 11.755 -- UI: chart-line-fisher-stoch primitive

Released as `v1.11.773` on 2026-05-26.

## Scope

Add `<ChartLineFisherStoch>` -- a pure-SVG two-panel chart
applying the inverse hyperbolic tangent (Fisher Transform) to the
normalised Stochastic value.

## Definition

Inputs:

- `data: ChartLineFisherStochPoint[]` where
  `ChartLineFisherStochPoint = { x; high; low; close }`. Bars
  with `high < low` are dropped.
- `period: number` -- the Stochastic lookback. Default `10`.
  Normalized to an integer `>= 2`.
- `threshold: number` -- the `+/-` peak-zone boundary. Default
  `1.5`. Normalized to a positive finite.

Algorithm (per bar `i >= period - 1`):

```
HH      = max(high over [i - period + 1, i])
LL      = min(low  over [i - period + 1, i])
stochK  = (close - LL) / (HH - LL)        (in [0, 1])
x       = clamp(2 * stochK - 1, -CLAMP, +CLAMP)
fisher  = 0.5 * ln((1 + x) / (1 - x))     (atanh(x))
```

`CLAMP = 0.999`. A bar with `HH == LL` (no range) is null. The
first `period - 1` bars are null.

## Bit-exact algebraic anchors

**CONST_FLAT** (`high == low == close == K`): `HH == LL == K`
-> no range -> bar null at every position.

**AT_MIDPOINT** (`high == 12, low == 8, close == 10`):
- `HH = 12, LL = 8`, range = 4.
- `stochK = (10 - 8) / (12 - 8) = 0.5` bit-exact (dyadic).
- `x = 2 * 0.5 - 1 = 0` bit-exact.
- `fisher = 0.5 * ln(1 / 1) = 0.5 * 0 = 0` bit-exact.

So `fisher = 0` at every defined bar.

**AT_HIGH** (`close == high`): `stochK = 1, x` clamps to `+CLAMP
-> fisher > 0`.

**AT_LOW** (`close == low`): `stochK = 0, x` clamps to `-CLAMP
-> fisher < 0`.

**Antisymmetry**: `AT_HIGH` and `AT_LOW` give opposite-sign
`fisher` of the same magnitude (`atanh(CLAMP) = -atanh(-CLAMP)`).
Tests use `toBeCloseTo(_, 10)`.

**Translation invariance**: shifting OHLC by `+k` shifts both
`HH` and `LL` by `+k`, so `stochK = (close + k - LL - k) / (HH +
k - LL - k) = stochK` unchanged -> `fisher` unchanged bit-exact.

## Acceptance

- CONST_FLAT identity: fisher null at every bar.
- AT_MIDPOINT anchor: stochK = 0.5, fisher = 0 bit-exact at
  every defined bar.
- AT_HIGH / AT_LOW: stochK = 1 / 0, fisher > 0 / < 0 with
  antisymmetric magnitude.
- Clamp: caps `+/- CLAMP` bit-exact.
- Translation invariance: `+1000` leaves stochK / fisher
  unchanged bit-exact.
- Layout / component / ARIA / data-section hooks all covered.
