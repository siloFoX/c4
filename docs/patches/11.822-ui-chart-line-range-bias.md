# 11.822 UI: chart-line-range-bias primitive (v1.11.840)

## Summary
Pure-SVG dual-panel React/TS primitive plotting a per-bar **Range
Bias** positional ratio: `(close - midpoint) / (high - low)`. The
ratio is bounded in `[-0.5, +0.5]` and indicates whether the bar
closed in the upper or lower half of its trading band -- a low-cost,
single-bar trend filter with no lookback dependency.

## Formula
```
midpoint[i] = (high[i] + low[i]) / 2
range[i]    = high[i] - low[i]
bias[i]     = range[i] > 0
               ? (close[i] - midpoint[i]) / range[i]
               : null
```
`bias[i]` is `null` only when `range <= 0` (degenerate `high == low`
bar) or any input is non-finite.

## Defaults
- `highThreshold = 0.3` (close above the band midpoint by >= 30%)
- `lowThreshold = -0.3`
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `bullish`: `bias >= highThreshold`
- `bearish`: `bias <= lowThreshold`
- `neutral`: in between
- `flat`: `bias == 0`
- `none`: `high == low` or non-finite

## Bit-exact testability anchors (dyadic-exact, per-bar)
- **close=high**: `(high - (h+l)/2) / (h-l) = ((h-l)/2) / (h-l) = 0.5`
  -- verified across `(h, l) in {(10, 2), (100, 50), (5, 1), (20,
  10)}`.
- **close=low**: mirror -> `bias = -0.5`.
- **close=midpoint**: numerator zero -> `bias = 0`.
- **high=low**: divide-by-zero guard -> `bias = null`.
- **close above high or below low**: bias extends past `[-0.5, +0.5]`
  -- e.g. `h=10, l=2, close=14 -> bias = 1` (verified). Useful when
  the close gaps outside the OHLC band.
- **Quarter-position**: `h=10, l=2, close=4 -> (4 - 6) / 8 = -0.25`
  (bit-exact dyadic).

## Design notes
- This is the simplest primitive in the push: a per-bar transform
  with no rolling state. The bit-exactness comes from the
  algebraic identity `(h - (h+l)/2) / (h - l) = 0.5` -- both the
  numerator `(h - l) / 2` and the denominator `(h - l)` are dyadic
  for any integer inputs, so the ratio collapses to exactly `0.5`.
- Bias axis is **fixed at [-0.5, +0.5]** even though values can
  exceed this range when the close gaps outside the OHLC band. This
  keeps the panel geometry consistent across data sets.
- Threshold lines dashed `#9333ea`, zero line dashed `#475569` with
  a different stroke pattern.
- Default colors: bias `#16a34a` (green; matches bullish marker),
  bullish marker `#16a34a`, bearish `#dc2626`, thresholds `#9333ea`.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
