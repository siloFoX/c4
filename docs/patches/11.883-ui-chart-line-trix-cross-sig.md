## Summary

Added `<ChartLineTrixCrossSig />` -- a pure-SVG, dual-panel
React/TS primitive that renders the close in the top panel and
overlays the TRIX (triple-smoothed EMA one-period rate of
change) with its EMA-smoothed signal in the bottom panel,
marking bullish / bearish cross trigger events. The trigger-
focused TRIX variant distinct from the base TRIX line: this
primitive isolates the actionable momentum trigger events when
TRIX crosses its signal and paints them as arrow overlays on
the price panel for direct charting-overlay use:

```
ema1[i]   = EMA(close, length)
ema2[i]   = EMA(ema1,  length)
ema3[i]   = EMA(ema2,  length)
TRIX[i]   = ema3[i-1] === 0
              ? null
              : 100 * (ema3[i] - ema3[i-1]) / ema3[i-1]
signal[i] = EMA(TRIX, signalLength)
bullish  : (TRIX - signal) crosses up    (prev <= 0, cur > 0)
bearish  : (TRIX - signal) crosses down  (prev >= 0, cur < 0)
```

Defaults: `length = 15` (canonical TRIX), `signalLength = 9`.
Regime classifier: `bullish` (TRIX > signal), `bearish` (TRIX
< signal), `neutral` (TRIX === signal), `none` (either side
null).

## Bit-exact anchor

- **CONST close = K (K > 0)**: every EMA in the triple chain
  collapses to K via the SMA-seeded `min === max` precision
  fix. ema3[i] === ema3[i-1] === K so TRIX = 100 * (K - K) / K
  = 0 every bar after the chained warmup at `3*length - 2`
  (each EMA seeds at length-1 in its own input stream).
  signal EMA of 0s = 0. TRIX === signal everywhere -> regime
  `neutral`, cross count = 0. Verified across K = 1..1234.
  CONST K = 0 triggers divide-by-zero guard -> TRIX = null ->
  regime `none`.
