# 11.925 -- UI: chart-line-cci-extreme-cross primitive (v1.11.943)

## Summary

Adds `ChartLineCciExtremeCross`, a pure-SVG dual-panel chart that
renders the close in the top panel and a close-only Commodity
Channel Index (CCI) line in the bottom panel. Marks bullish
(cross up through any positive band) and bearish (cross down
through any negative band) trigger entries at the canonical +/-
100 and +/- 200 extreme bands. Dual-band cross variant of the
CCI family that flags discrete level entry / exit events on the
same panel.

## Pipeline

For close-only data with default `length = 20`:

```
sma[i]  = SMA(close, length)
md[i]   = avg |close[i-n+1..i] - sma[i]|
cci[i]  = md > 0
            ? (close - sma) / (0.015 * md)
            : 0
bullishMild    : prev <= +100 && cur > +100
bullishExtreme : prev <= +200 && cur > +200
bearishMild    : prev >= -100 && cur < -100
bearishExtreme : prev >= -200 && cur < -200
```

When CCI moves through both bands in one bar (e.g. 0 -> 466.67),
both `bullishMild` and `bullishExtreme` fire at the same index.
Same for the symmetric bearish case.

## Bit-exact anchor

- CONST `close = K` for any finite K -> `sma = K`, all
  deviations are 0, mean deviation = 0, the 0/0 guard returns
  cci = 0. Result sits firmly between -100 and +100, regime is
  `neutral`, and the threshold is never crossed; cross count =
  0. Verified across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` (length 20) -> cci is exactly
  `9.5 / 0.075 = 126.667` from i = 19 onward, never moves --
  regime stays `bullish` with 0 crosses.
- LINEAR DOWN `close = -i` mirrors LINEAR UP to -126.667 ->
  regime `bearish` with 0 crosses.
- Single positive spike at the end of an otherwise constant
  window of length n -> cci = `n / 0.03` at the spike index
  (e.g. 14 / 0.03 = 466.67). This trigger fires both the
  `bullishMild` and `bullishExtreme` cross at the same bar.

## Regime classifier

| cci range                | regime           |
|--------------------------|------------------|
| cci >= +200              | bullishExtreme   |
| +100 <= cci < +200       | bullish          |
| -100 < cci < +100        | neutral          |
| -200 < cci <= -100       | bearish          |
| cci <= -200              | bearishExtreme   |
| null                     | none             |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- Five horizontal reference bands inside the oscillator panel
  (upperExtreme / upperMild / mid / lowerMild / lowerExtreme).
- Cross markers in oscillator panel + chevron overlays in price
  panel; each carries `role="graphics-symbol"`, `tabIndex=0`,
  and a discrete colour per kind.
- Default oscillator range `+/- 300` expands to wrap any
  observed CCI value (e.g. 466.67 spike) so the marker never
  clips off the panel.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter` /
  `Space` keyboard toggle parity.

## Defaults

- length = 20
- upperMild = +100, lowerMild = -100
- upperExtreme = +200, lowerExtreme = -200
- oscillator range = +/- 300 (auto-expands)
- width 720, height 460, padding 44, panelGap 12
