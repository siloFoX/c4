## Summary

Added `<ChartLineObvCross />` -- a pure-SVG dual-panel React/TS
primitive that renders a close line in the top panel and the
cumulative On Balance Volume (OBV) running total plus its
moving-average signal line in the bottom panel. Scatter markers
fire at every OBV-vs-signal crossover -- the canonical
accumulation / distribution regime flip:

- `up` cross   -> regime `accumulation`
- `down` cross -> regime `distribution`

Defaults: `signalLength = 20`, obvColor `#0ea5e9` (sky), signalColor
`#f97316` (orange), accumulationColor `#16a34a`, distributionColor
`#dc2626`, zero-reference line `#475569`.

## Formula

```
OBV[0] = 0
OBV[i] = OBV[i-1] + (close[i] > close[i-1] ?  volume[i]
                   : close[i] < close[i-1] ? -volume[i]
                   : 0)
signal = SMA(OBV, signalLength)
```

Cross detection uses the same prev-relation + strict-inequality
guard pattern as the other crossover primitives.

## Bit-exact anchors

The SMA helper carries the `min === max` window-constant precision
fix so a constant OBV window lands bit-exactly on its value.

- **CONST close = K, volume = V**: every step has close === prev,
  so the OBV delta is 0. OBV stays at 0 forever, signal SMA stays
  at 0. Relation is `equal` forever -> regime `neutral`, zero
  crosses fire.
- **LINEAR UP close = i+1, volume = V**: every step has
  close > prev, so `OBV[i] = i * V`. The signal SMA tracks the
  same arithmetic series shifted by `(n-1)/2` bars, giving the
  invariant `OBV - signal = V * (n-1) / 2 > 0`. Relation is
  `bullish` forever -> regime `accumulation`, zero crosses
  (jumps null -> bullish with no prev relation).
- **LINEAR DOWN close = N-i, volume = V**: mirror image, OBV
  decrements by V every step. `OBV - signal = -V * (n-1) / 2 < 0`
  permanently -> regime `distribution`, zero crosses.

The test asserts the diff invariant `OBV - signal === V*(n-1)/2`
across multiple `signalLength` choices to lock down the linearity
of the OBV / SMA pair.
