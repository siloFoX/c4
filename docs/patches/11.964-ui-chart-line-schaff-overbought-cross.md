# Patch 11.964 - UI: chart-line-schaff-overbought-cross primitive

Version: 1.11.982

## Summary

Add `web/src/components/ui/chart-line-schaff-overbought-cross.tsx` -- a
pure-SVG dual panel line chart primitive that visualizes price alongside
Doug Schaff's Trend Cycle (STC) oscillator, with explicit cross markers
and overlay arrows at every STC crossing of the canonical 75 overbought
line. Bullish (= entry, crossing UP through 75) and bearish (= exit,
crossing DOWN through 75). Shares the bit-exact STC pipeline with the
`-zero-cross` sibling but tracks the overbought threshold instead of
the centerline.

## Pipeline

Identical to `chart-line-schaff-zero-cross`. Threshold default flipped
to 75.

## Bit-exact anchors

| Pattern                  | STC (after warmup) | Regime  |
|--------------------------|--------------------|---------|
| CONST close = K          | 50 (seed)          | bearish |
| LINEAR UP close = i      | 50 (seed)          | bearish |
| LINEAR DOWN close = -i   | 50 (seed)          | bearish |

CONST anchor tested bit-exactly across K in `{0, 1, 42, 100, 1234}`
(regime bearish since 50 < 75 -- opposite from zero-cross where 50
sat on the boundary). LINEAR ramps lock MACD at a steady-state
constant, so the stochastic of a constant stays at the 50 seed --
also bearish. The interesting cross activity lives in transients.

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), STC panel below.
* STC panel uses a fixed `[0, 100]` range; threshold band sits at the
  75-line (closer to oscTop than midpoint).
* Cross markers drawn as circles on the STC panel at `(cx, cyOsc)`
  and as arrows on the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="STC Overbought Cross
  chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only
  text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
