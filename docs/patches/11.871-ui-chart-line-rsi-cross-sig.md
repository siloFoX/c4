## Summary

Added `<ChartLineRsiCrossSig />` -- a pure-SVG, dual-panel
React/TS primitive that renders the close on top and the Wilder
RSI plus its EMA-smoothed signal line on the bottom. Markers
fire at every RSI-vs-signal cross -- a momentum trigger
distinct from RSI overbought/oversold level crosses.

Cross events:

- `up` cross   -> regime `accelerating-up`
- `down` cross -> regime `accelerating-down`

Defaults: `rsiLength = 14`, `signalLength = 9`, rsiColor
`#0ea5e9` (sky), signalColor `#f97316` (orange), bullish marker
`#16a34a`, bearish marker `#dc2626`, midline `#475569`.

## Formula

```
gain[i] = max(0, close[i] - close[i-1])
loss[i] = max(0, close[i-1] - close[i])
avgGain = Wilder(gain, rsiLength)
avgLoss = Wilder(loss, rsiLength)
RSI     = (avgGain === 0 && avgLoss === 0) ? 50
        : avgLoss === 0                    ? 100
        : avgGain === 0                    ? 0
        : 100 - 100 / (1 + avgGain/avgLoss)
signal  = EMA(RSI, signalLength)
```

Cross detection uses the standard prev-relation + strict-
inequality guard.

## Bit-exact anchors

The Wilder helper carries the SMA-seeded `min === max` precision
fix; the EMA helper adds the `next = v === smoothed ? v : ...`
short-circuit, so constant inputs lock onto their values exactly.

- **CONST close = K**: every gain/loss = 0 -> RSI = 50 (neutral
  fallback). EMA(50) = 50. Relation `equal` forever, regime
  `aligned`, zero crosses.
- **LINEAR UP close = i + 1**: every gain = 1, loss = 0 -> RSI
  = 100. EMA(100) = 100. Zero crosses.
- **LINEAR DOWN close = N - i**: mirror -> RSI = 0, EMA(0) = 0.
  Zero crosses.

Verified across multiple K and `(rsiLength, signalLength)`
tuples.
