## Summary

New `ChartLineStiffness` UI primitive in
`web/src/components/ui/chart-line-stiffness.tsx` -- a pure-SVG two-panel
line chart with a **Katsanos Stiffness** indicator panel counting bars
above a lower volatility band. The top panel plots the close; the
bottom panel plots the stiffness inside a fixed 0..100 band with two
horizontal threshold lines.

## The computation

Markos Katsanos's Stiffness reads how "stiff" the price has been.
`ChartLineStiffnessPoint` carries `{x, value}`.

- `computeLineStiffnessSma(values, period)` -- the simple moving average
  over the window, null in the warm-up.
- `computeLineStiffnessStdev(values, period)` -- the population standard
  deviation over the same window.
- `computeLineStiffness(values, options)` -- derives the lower band
  `sma - factor * stdev`, flags each bar by whether the close sat above
  the band, sums the flags over the lookback window scaled to a 0..100
  percentage, and passes the raw count through an exponential moving
  average of `smoothPeriod` bars.
- `classifyLineStiffnessZone(stiffness, high, low)` -- marks a bar
  **stiff** at or above the high threshold, **mid** between thresholds,
  **loose** below the low threshold, **none** for null.

## Worked fixture

The whole pipeline is exact arithmetic except for the standard
deviation, but the sign of `close - lowerBand` is deterministic for
several patterns, so the stiffness anchors hit clean integers.

A **strictly rising close** (`[1..10]`, period 3) keeps every close
above the lower band: aboveFlag is 1 at every defined bar, the rolling
count is 3/3 = 1, and the stiffness pins to exactly 100. A **strictly
falling close** (`[10..1]`) keeps every close below the band:
aboveFlag is 0, the count is 0, and the stiffness pins to 0. A
**constant close** has zero stdev so the band collapses to the SMA;
`close > sma` is false at the equality, aboveFlag is 0 and the
stiffness is 0. All three are asserted with `toEqual` on the smoothed
stiffness array. The SMA helper is verified with a clean integer
window, the stdev helper with a constant window (`= 0`) and with
`Math.sqrt(2/3)` for `[1,2,3]`. The zone classifier is checked for
stiff (at and above 90), mid (between), loose (below 50) and none.

## API

`data` (a `ChartLineStiffnessPoint[]`), `period` (default 100), `factor`
(default 0.2), `smoothPeriod` (default 3), `highThreshold` (default 90),
`lowThreshold` (default 50), `pricePanelRatio` / `gap`,
sizing/format/a11y props, `strokeWidth` / `dotRadius`, the colour
props, the toggle flags (`showAxis` / `showGrid` / `showDots` default
false / `showStiffness` / `showThresholds` / `showMarkers` /
`showTooltip` / `showConfigBadge` / `showLegend` / `animate`),
controlled + uncontrolled `hiddenSeries` (keys `price` / `stiffness`),
`formatValue` / `formatX`, `onPointClick` / `onSeriesToggle`.

Exported helpers: `getLineStiffnessFinitePoints`,
`normalizeLineStiffnessPeriod`, `normalizeLineStiffnessFactor`,
`normalizeLineStiffnessThreshold`, `computeLineStiffnessSma`,
`computeLineStiffnessStdev`, `classifyLineStiffnessZone`,
`computeLineStiffness`, `runLineStiffness`, `computeLineStiffnessLayout`,
`describeLineStiffnessChart`, plus the
`DEFAULT_CHART_LINE_STIFFNESS_*` constants, `CHART_LINE_STIFFNESS_BOUND`
and the `ChartLineStiffnessZone` / `ChartLineStiffnessComputed` types.
Every exported name carries a `LineStiffness` infix so the `export *`
barrel line cannot collide.

## Accessibility

Root `<div role="region">` with `aria-label` + `aria-describedby`; SVG
`role="img"`; an off-screen
`data-section="chart-line-stiffness-aria-desc"` description span; the
close line, the stiffness line and every dot / marker are
`role="graphics-symbol" tabIndex={0}`. Every node carries a
`data-section`; the root mirrors `data-period`, `data-factor`,
`data-smooth-period`, `data-high-threshold`, `data-low-threshold`,
`data-stiffness-final`, `data-stiff-count`, `data-mid-count`,
`data-loose-count`, `data-total-points` and `data-animate`.
