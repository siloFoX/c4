# 11.761 chart-line-center-of-gravity (v1.11.779)

## Summary

Pure-SVG dual-panel React/TypeScript chart primitive with an
**Ehlers Center of Gravity (CG) oscillator** plotted beneath the
close. Added as the `<ChartLineCenterOfGravity />` UI primitive in
`web/src/components/ui/`.

## Algorithm

For each bar `i` and a length `L`:

```
num = sum_{j = 0..L - 1} (1 + j) * close[i - j]
den = sum_{j = 0..L - 1} close[i - j]
CG[i] = -num / den + (L + 1) / 2
```

The shift `(L + 1) / 2` recentres the centroid so a flat input
collapses to zero. When `den == 0` (e.g. an oscillating window
whose closes cancel) the bar is `null`. Bars before `i = L - 1`
are also `null` (warmup).

## Anchors

- **CONST_FLAT (close == K, K != 0) -> CG = 0 bit-exact**. With
  `close = K` over the window,
  `num = K * sum_{j = 0..L - 1} (1 + j) = K * L * (L + 1) / 2`
  and `den = K * L`, giving
  `-num / den = -(L + 1) / 2`. The shift cancels it exactly so
  `CG = 0`. Verified at `(K, L) = (1, 4), (2, 4), (7, 10),
  (100, 14), (-5, 4)`.
- **K == 0 -> null** (division by zero is detected and the bar is
  nulled).
- **Worked anchors**:
  - `close = [1, 2, 3, 4]`, `L = 4`:
    `num = 1*4 + 2*3 + 3*2 + 4*1 = 20`, `den = 10`, `CG = -2 +
    2.5 = 0.5` bit-exact.
  - `close = [4, 3, 2, 1]`, `L = 4`: `num = 30`, `den = 10`,
    `CG = -3 + 2.5 = -0.5` bit-exact.
  - `close = [1, 2, 3, 4, 5]`, `L = 5`: `CG = -7/3 + 3 = 2/3`
    (numerically exact to 12 decimal places).
- **Antisymmetry**: reversing the close series flips the sign of
  the CG output.
- **Scaling invariance**: multiplying the close series by a
  positive scalar leaves CG unchanged (numerically).
- **Zero-sum window**: `[1, -1, 1, -1]` over `L = 4` is
  singular -> `null`.
