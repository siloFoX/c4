# 11.572 -- web/src/components/ui/chart-line-ichimoku.tsx

Pure-SVG line chart with an **Ichimoku Kinko Hyo** overlay --
the "one glance equilibrium chart". On top of the raw price
line it draws the conversion line (Tenkan-sen), the base line
(Kijun-sen), the lagging span (Chikou) and the **kumo cloud**
formed between the two leading spans (Senkou Span A / B).

## The five Ichimoku lines

Every Ichimoku line is built from the same donchian-midpoint
formula -- `computeLineIchimokuMidline(values, period)` --
the `(highest + lowest) / 2` over a trailing window:

- **Tenkan-sen** (conversion): midline over the conversion
  period (default 9).
- **Kijun-sen** (base): midline over the base period
  (default 26).
- **Senkou Span A**: `(Tenkan + Kijun) / 2`, plotted
  `displacement` periods AHEAD (default 26).
- **Senkou Span B**: midline over the leading period
  (default 52), plotted `displacement` periods ahead.
- **Chikou** (lagging span): the price itself, plotted
  `displacement` periods BEHIND.

The forward / backward shift extends the x-axis: x positions
beyond the data are extrapolated from the average x-step, so
the cloud reaches into the "future" and the Chikou into the
"past".

The **kumo** is the band between Span A and Span B. A sample
is `bullish` when Span A is at or above Span B, `bearish`
otherwise; the cloud is drawn as per-segment polygons tinted
green (bullish) or orange (bearish) -- the canonical
two-tone cloud.

The component fixture (periods 2/3/4, displacement 2 on an
8-point series) produces a mixed cloud: five cloud samples
with bullish flags `[true, false, false, true, true]` -- 3
bullish, 2 bearish -- which exercises both tints.

## Distinct from neighbouring primitives

Ichimoku bundles five derived lines plus the two-tone cloud
into one overlay. No existing primitive computes the
donchian-midpoint family, the forward-displaced leading
spans or the lagging span. `chart-line-bollinger` /
`chart-line-band` draw a single statistical band;
`chart-line-moving-avg` overlays one smoothed line. The
Ichimoku cloud is a forward-projected, sign-tinted band
between two distinct leading spans.

All exported helpers carry a `LineIchimoku` infix so the
`export *` barrel line cannot collide.

## Public surface

- `<ChartLineIchimoku data={...} />` -- a single
  `ChartLineIchimokuPoint[]` (`{x, value}`).
- `conversionPeriod` / `basePeriod` / `leadingPeriod` /
  `displacement` -- the four Ichimoku periods (defaults
  9 / 26 / 52 / 26).
- Toggle flags: `showAxis`, `showGrid`, `showDots` (default
  false), `showCloud`, `showTenkan`, `showKijun`,
  `showChikou`, `showTooltip`, `showConfigBadge`,
  `showLegend`, `animate`.
- Controlled + uncontrolled visibility (`hiddenSeries` keys
  `price` / `tenkan` / `kijun` / `chikou` / `cloud`).
- Standard sizing + format + a11y props. `formatValue` /
  `formatX`.
- `onPointClick({point})` / `onSeriesToggle({seriesId,
  hidden})`.

## Pure helpers

- `getLineIchimokuFinitePoints(points)`.
- `normalizeLineIchimokuPeriod(period, fallback)`.
- `computeLineIchimokuMidline(values, period)` -- the
  donchian-midpoint window.
- `runLineIchimoku(points, options?)` -- the full Ichimoku
  computation (the five lines + the kumo).
- `computeLineIchimokuLayout({...})` -- projects every line
  and builds the per-segment cloud polygons.
- `describeLineIchimokuChart(data, options?)`.

## Tooltip / badge / legend

Hovering a price dot shows x, price, and the Tenkan and
Kijun values at that index -- the "one glance" read. Config
badge: `ICHI <conv>-<base>-<leading> cloud=<bull/bear/flat>`.
The legend has five toggle items (Price / Tenkan / Kijun /
Chikou / Kumo) plus a bullish-vs-bearish tally.
