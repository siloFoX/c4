# 11.521 -- web/src/components/ui/chart-line-rate.tsx

Pure-SVG multi-series line chart that **plots the per-interval rate
of change** (the discrete first derivative) of an input series.
Between each consecutive pair of finite samples sorted by x, the
chart emits a rate sample `rate = dy / dx` anchored at `'right'` /
`'left'` / `'midpoint'` mode. The result is rendered as a line with
a horizontal zero-rate baseline; sign-shaded regions paint the area
between the rate line and zero (green for positive growth, red for
negative). Segments that cross zero are split exactly at the
crossing point.

## Distinct from other transform / reference primitives

- `<ChartLineCumulative>` (11.520): plots the running **sum** of
  increments. Rate plots the **delta per interval**.
- `<ChartLineMovingAvg>` (11.513): windowed average of y. Rate is
  the slope between two y values.
- `<ChartLineTrend>` (11.512): global linear regression. Rate is
  the local slope per segment.
- `<ChartLineComparison>` (11.511): difference between two curves.
  Rate is the difference within one curve over x.
- `<ChartLineMinMax>` (11.517): markers at data extremes. Rate
  highlights segments by sign.
- `<ChartLineForecast>` (11.519): time-axis split. Rate is a
  transform, not a split.

The canonical "growth rate" / "velocity" / "first derivative" view
for momentum dashboards, traffic acceleration, balance-sheet delta
per period, etc.

## Public surface

- `<ChartLineRate series={...} mode="right" />`.
- Per-series `{id, label, data, color?, mode?}`. `data` entries are
  raw `{x, y}` samples; the chart computes the rates.
- `mode` chart-level default; per-series override always wins.
- Toggle flags: `showAxis`, `showGrid`, `showDots`, `showLegend`,
  `showTooltip`, `showSignFill`, `showZeroLine`, `animate`.
- `positiveColor` (`#16a34a`) / `negativeColor` (`#dc2626`) /
  `zeroColor` (`#94a3b8`) / `zeroDashArray` (`'4 3'`) /
  `fillOpacity` (`0.18`).
- Standard sizing + format + a11y props.
- `formatValue` / `formatX` / `formatRate` (sign-aware fallback when
  not supplied).
- Controlled + uncontrolled `hiddenSeries`.
- Callbacks: `onSampleClick({series, sample})`,
  `onRegionClick({series, region})`.

## Pure helpers

- `getLineRateDefaultColor(index)` -- 10-color palette cycle.
- `getLineRateFinitePoints(points)` -- finite-only filter.
- `computeRateOfChange(points, mode?)` -- canonical first-derivative
  helper. Returns `(n - 1)` samples for `n` finite inputs sorted by
  x ascending. Drops `dx === 0` segments (consecutive equal x).
  Output's `index` is position in the output array; `fromIndex` /
  `toIndex` reference the ORIGINAL input array (so callers can map
  back). Non-array / <2 finite samples -> `[]`.
- `findLineRateZeroCrossing(x1, rate1, x2, rate2)` -- crossing x via
  absolute-value proportion; returns `null` on same-sign,
  endpoint-on-zero, degenerate, or non-finite inputs.
- `computeLineRateStats(samples)` -- returns `{finiteCount, maxRate,
  minRate, averageRate, positiveCount, negativeCount, zeroCount,
  totalAbsoluteArea}`. `totalAbsoluteArea` is the trapezoid sum of
  `|rate|` between successive samples.
- `computeLineRateLayout({series, mode?, hiddenSeries?,
  showSignFill?, positiveColor?, negativeColor?, ...})` -- returns
  `{series, xTicks, yTicks, xMin, xMax, yMin, yMax, zeroY,
  innerWidth, innerHeight, totalSamples, visibleSeriesCount}`.
  Y range is always expanded to include `0`. Each layout series
  carries samples, regions, path, and stats. Sign-fill regions are
  closed `M ... L ... L ... Z` polygons between the rate line and
  the y=0 baseline.
- `describeLineRateChart(series, mode?, hidden?, formatValue?)` --
  builds the screen-reader description ("Rate-of-change line chart
  across N series. A: avg rate ..., max ..., min ..., K+ / K-").

## Anchor modes

- `'right'` (default): rate anchored at the later x in the pair.
- `'left'`: rate anchored at the earlier x.
- `'midpoint'`: rate anchored at the midpoint of the pair.

## ARIA + test selectors

- Root `role="region"` + `aria-label` + `aria-describedby`.
- SVG `role="img"`.
- Per-line path + per-dot are `role="graphics-symbol"` +
  `tabIndex={0}`; zero baseline is `role="graphics-symbol"` only.
  Dot aria-label includes rate.
- Every node carries `data-section="chart-line-rate-..."` for test
  selectors.
- Series groups expose `data-series-max-rate`,
  `data-series-min-rate`, `data-series-average-rate`,
  `data-series-positive-count`, `data-series-negative-count`,
  `data-series-zero-count`, `data-series-mode`,
  `data-series-color`, `data-series-finite-count`.
- Dots expose `data-rate-sign` (`'positive' | 'negative' | 'zero'`),
  `data-from-index`, `data-to-index`.
- Regions expose `data-region-is-positive`, `data-region-start-x`,
  `data-region-end-x`, `data-region-fill-color`, `data-region-area`.
- Root mirrors `data-mode`.

## Tooltip

Hovering a dot shows the series label, x, a coloured `rate:`
(positive or negative tinted), and a `mode: <kind> (from #i to
#j)` row indicating which original-array indices produced this
rate sample.
