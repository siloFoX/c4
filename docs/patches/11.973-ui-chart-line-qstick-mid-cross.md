## Summary

Adds the `ChartLineQstickMidCross` primitive
(`web/src/components/ui/chart-line-qstick-mid-cross.tsx`): a pure
SVG dual-panel chart that visualizes Tushar Chande's QStick
(close - open SMA) crossings through the 0 midline for candle body
momentum centerline regime transitions with bias coloring.
Companion to `chart-line-qstick-zero-cross` (TODO 11.952): shares
the same QStick pipeline (`SMA(close - open, length)`) and defaults
(length = 14, threshold = 0); reframes the ARIA / describe text
from "baseline regime transition" to "centerline regime transitions
with bias coloring" so it slots into the broader mid-cross family
(RSI / Stoch / CCI / MFI / Stoch-RSI / Williams %R / UO / CMO /
RMI / DPO).

## Pipeline

1. `body[i] = close[i] - open[i]`
2. `qstick[i] = SMA(body, length)` (with CONST short-circuit via
   `winMin === winMax`)
3. Regime classifier (strict `>=`):
   - `qstick >= 0` -> bullish (bodies turn green on net)
   - `qstick <  0` -> bearish (bodies turn red on net)
4. Cross detector emits a `bullish` cross when
   `qstick[i-1] <= 0 && qstick[i] > 0` and a `bearish` cross when
   `qstick[i-1] >= 0 && qstick[i] < 0`. Strict inequality on the
   current side ensures the boundary value 0 does not double-fire.

## Defaults

- `length = 14` (Chande's canonical window)
- `threshold = 0` (midline / centerline)
- Warmup boundary: bar `length - 1 = 13`, so the first 13 samples
  (30-bar run) are null and the remaining 17 are populated.

## Bit-exact anchors

- CONST `open = close = K` for all `K in {0, 1, 42, 100, 1234}` ->
  `body[i] === 0` and `qstick[i] === 0` literal (NOT `-0`; verified
  via `Object.is`) for every `i >= 13`. Regime classifies as
  bullish because 0 sits on the threshold (`qstick >= threshold`).
- CONST body = +B (open = i, close = i + B) -> body = B every bar,
  qstick = B constant -> bullish when B > 0, bearish when B < 0.
- Sign-flipping bodies (-B then +B) -> qstick transitions from -B
  to +B through zero over the smoothing window, firing exactly one
  bullish cross.
- CONST K full-run counters under defaults, 30 bars:
  `noneCount = 13`, `bullishCount = 17`, `bearishCount = 0`,
  `crosses.length = 0`.
- `pricePath` and `qstickPath` are byte-identical across repeated
  layout calls, with `.toFixed(2)` precision and `M ... L ...`
  command shapes. The qstick path uses a single `M` for post-warmup
  data.
- Cross detector boundary: strict inequality (`> T`, `< T`).
- Layout auto-fit: when qstick collapses to a single value (CONST
  case, qstick = 0), the panel uses the `threshold +/- 1` fallback
  (oscMin = -1, oscMax = 1).

## Rendering surface

- ARIA region with `aria-label="QStick Mid Cross chart"`, SVG with
  `role="img"`, hidden `<desc>` for screen readers that explicitly
  mentions "bias coloring" and "centerline regime transition".
- Sections (each with `data-section`): bands, axes, grid, price
  path, QStick path, cross markers (each with
  `role="graphics-symbol"` and `tabIndex={0}`), overlay chevrons
  on the price panel (green up for bullish, red down for bearish),
  hover lanes, tooltip, legend buttons, config badge.
- Legend toggles support both pointer (`click`) and keyboard
  (`Enter` / `Space`) activation, and respect the controlled
  `hiddenSeries` prop alongside the uncontrolled internal state.
- Motion-safe fade-in opt-out via `animate={false}`.
