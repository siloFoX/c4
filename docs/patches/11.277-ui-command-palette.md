# 11.277 -- Command palette primitive (v1.11.295)

Component-scope-only addition. New CommandPalette primitive +
useCommandPaletteShortcut hook + App-level Cmd+K wiring.

## Why

Power users navigating ten top-level views via the mouse / tab key
wanted a keyboard-first launcher. Most modern operator surfaces
(Linear, Raycast, claude.ai, VS Code) expose this as Cmd+K -- a
fuzzy-searchable command list with recent commands surfaced first.
Patch 11.277 ships the canonical implementation at the primitive
layer so any future surface (per-page palettes, scoped modal
palettes) reuses one ARIA + portal + recents contract.

## What changed

### `web/src/components/ui/command-palette.tsx` (new file)

- `CommandPalette` -- modal command launcher.
  - `open` / `onOpenChange(next)` controlled visibility.
  - `commands: Command[]` -- caller-supplied list of
    `{ id, label, group?, shortcut?, keywords?, disabled?, action }`.
  - `recentsKey?` (default `c4:command-palette:recents`) +
    `recentsCap?` (default 5) localStorage persistence with
    cross-tab `storage` event sync.
  - Tiny no-dependency fuzzy scorer that ranks across label +
    keywords + group.
  - Focus trap, ARIA combobox + listbox + option markup,
    aria-selected on active row, Esc / backdrop close.
  - Keyboard: ArrowDown / ArrowUp / Home / End / Enter / Esc.
  - `queueMicrotask`-deferred action so the modal closes (and
    focus restores) BEFORE the command fires.
- `useCommandPaletteShortcut(onToggle, opts?)` -- companion hook
  for global Cmd+K (mac) / Ctrl+K (others). Gated by default
  while focus is in input / textarea / contenteditable; pass
  `{ interceptInInputs: true }` to override.
- Re-exported from `web/src/components/ui/index.ts`.

### `web/src/App.tsx`

- Cmd+K mounts the palette via the hook + state pair.
- 13 commands: 10 navigation actions (Workers / History /
  Sessions / Meetings / Specialists / Wiki / Chat / Workflows /
  Autonomous / Features), Open Settings, Toggle Sidebar
  (Ctrl+B hint), Focus Search (targets
  `[data-section="search-bar"] input` or
  `input[type="search"]`), Logout.
