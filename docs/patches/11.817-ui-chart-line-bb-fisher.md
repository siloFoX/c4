# 11.817 UI: chart-line-bb-fisher primitive (v1.11.835)

## Summary
Pure-SVG dual-panel React/TS primitive that plots a **Bollinger Band
Fisher Transform** oscillator beneath the close. Each bar normalizes
its position within the Bollinger envelope to `[-1, +1]` and applies
`atanh` (the inverse hyperbolic tangent) to compress tails into a
smooth unbounded oscillator that highlights extremes.

## Formula
```
mean[i]       = SMA(close, length)[i]
stdDev[i]     = popStdDev(close, length)[i]
upper[i]      = mean[i] + sigma * stdDev[i]
lower[i]      = mean[i] - sigma * stdDev[i]
percentB[i]   = (upper[i] - lower[i] != 0)
                  ? (close[i] - lower[i]) / (upper[i] - lower[i])
                  : null
normalized[i] = clamp(2 * percentB[i] - 1, -clampLimit, +clampLimit)
fisher[i]     = Math.atanh(normalized[i])
```
`fisher[i]` is `null` during warmup (`i < length - 1`) or when the
band has zero width (degenerate `0/0`). The clamp avoids
`atanh(+/-1) = +/-Infinity` when the close breaches the band.

## Defaults
- `length = 20`, `sigma = 2`, `clampLimit = 0.999`
- `overbought = 1.5`, `oversold = -1.5`
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `overbought`: `fisher >= overbought`
- `oversold`: `fisher <= oversold`
- `neutral`: in between
- `none`: warmup / non-finite / collapsed band
- `crossed`: `'up'` entering overbought, `'down'` entering oversold.
  Markers fire on threshold transitions only.

## Bit-exact testability anchors
- **CONST close**: band collapses (`upper - lower = 0`) -> `percentB
  = null` -> `fisher = null` everywhere across `K in {1, 5, 100, -3}`
  and `length in {3, 4, 7, 10}`.
- **ALTERNATING [0, 1, 0, 1, ...]** with `length = 4`, `sigma = 2`:
  At bar 3: `mean = 0.5`, `stdDev = 0.5`, `upper = 1.5`,
  `lower = -0.5` (all dyadic-exact). With `close[3] = 1`:
  - `percentB = (1 - (-0.5)) / (1.5 - (-0.5)) = 1.5 / 2 = 0.75`
    (dyadic-exact).
  - `normalized = 2 * 0.75 - 1 = 0.5` (dyadic-exact).
  - `fisher = Math.atanh(0.5)` (deterministic IEEE 754 library
    value). The test asserts `toBe(Math.atanh(0.5))` so the anchor
    stays exact in any JS runtime.
  At bar 4 with `close[4] = 0`: `percentB = 0.25`, `normalized = -0.5`,
  `fisher = Math.atanh(-0.5)` (bit-exact mirror).

## Design notes
- The CONST case where the band has zero width is handled via an
  explicit `width <= 0` guard that returns `null` rather than
  propagating `NaN` from `0 / 0`.
- The `Math.atanh(0.5)` literal in the test ensures the bit-exact
  assertion holds across JS engines without depending on a specific
  IEEE 754 representation -- we just compare against the same library
  call the implementation uses.
- Fisher panel auto-scales but always includes the overbought and
  oversold thresholds in the visible range.
- Threshold lines dashed `#9333ea` (purple), zero line dashed
  `#475569` (different dash pattern).
- Default colors: Fisher line `#8b5cf6` (violet), bullish `#16a34a`,
  bearish `#dc2626`.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
