# 11.937 -- UI: chart-line-williams-r-mid-cross primitive (v1.11.955)

## Summary

Adds `ChartLineWilliamsRMidCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and a close-only
Williams %R line in the bottom panel. Marks bullish (cross up
through midline -50) / bearish (cross down through midline
-50) inverted oscillator centerline regime trigger events.
Single-threshold cross variant of the Williams %R family that
flags the discrete %R level -50 midline crossover distinct
from the canonical overbought (-20) / oversold (-80) extreme
bands. Continues the midline-cross family (RSI 50, Stochastic
50, CCI 0, MFI 50, Stochastic RSI 50, now Williams %R -50).

## Pipeline

For close-only data with default `length = 14`:

```
range[i] = max(close[i-n+1..i]) - min(close[i-n+1..i])
wr[i]    = range > 0
             ? -100 * (highest - close) / range
             : -50
bullish  : prev wr <= -50 && cur wr > -50   (momentum up)
bearish  : prev wr >= -50 && cur wr < -50   (momentum down)
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> range = 0 every bar
  -> wr = -50 via neutral fallback. wr = -50 sits on the
  threshold but the strict-inequality detector never fires.
  regime `bullish` (wr >= -50). cross count = 0. Verified
  across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` (length 14) -> close always at window
  high -> wr = 0 constant. 0 > -50, regime `bullish`. 0
  crosses (wr jumps from null to 0 so prev-null skips the
  strict-inequality detector).
- LINEAR DOWN `close = -i` -> close always at window low ->
  wr = -100 constant. -100 < -50, regime `bearish`. 0 crosses.
- Decline (14 bars) + sharp recovery -> wr starts at -100,
  rises strictly above -50 -> 1 bullish midline cross.
- Rise (14 bars) + sharp decline -> wr falls strictly below
  -50 -> 1 bearish midline cross.

## Regime classifier

| wr range            | regime      |
|---------------------|-------------|
| wr >= -50           | bullish     |
| wr < -50            | bearish     |
| null                | none        |
