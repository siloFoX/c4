## Summary

Added `<ChartLineVrocCross />` -- a pure-SVG, dual-panel React/TS
primitive that renders the close on top and the Volume Rate of
Change (VROC) plus its signal-line EMA on the bottom. Markers
fire at every VROC-vs-signal crossover -- the canonical volume
momentum regime shift.

Cross events:

- `up` cross   -> regime `expanding`
- `down` cross -> regime `contracting`

Defaults: `length = 12`, `signalLength = 6`, vrocColor `#0ea5e9`
(sky), signalColor `#f97316` (orange), bullish marker `#16a34a`,
bearish marker `#dc2626`, zero line `#475569`.

## Formula

```
VROC[i]   = volume[i - length] === 0 ? null
                                     : (volume[i] - volume[i - length])
                                       / volume[i - length] * 100
signal[i] = EMA(VROC, signalLength)
```

Cross detection uses the standard prev-relation + strict-
inequality guard.

## Bit-exact anchor

- **CONST volume = V (V > 0)**: every difference is 0 -> VROC =
  0 / V * 100 = 0. Signal = EMA(0) = 0. Relation `equal`
  forever, zero crosses. Verified across multiple V and length
  tuples.
- **CONST volume = V = 0**: divide-by-zero guard leaves VROC and
  signal null forever.

## Soft anchors

LINEAR UP volume yields VROC > 0 once defined (volume increase
between bars i-length and i is positive). LINEAR DOWN volume
yields VROC < 0. These are just sanity checks; specific cross
counts depend on signal lag and aren't anchored.
