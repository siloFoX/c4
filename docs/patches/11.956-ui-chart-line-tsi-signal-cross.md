# Patch 11.956 - UI: chart-line-tsi-signal-cross primitive

Version: 1.11.974

## Summary

Add `web/src/components/ui/chart-line-tsi-signal-cross.tsx` -- a pure-SVG
dual panel line chart primitive that visualizes price alongside William
Blau's True Strength Index AND its EMA signal line, with explicit cross
markers and overlay arrows at every TSI crossing of its signal line. A
signal-line cross fires earlier than a zero-line crossover when momentum
starts to turn -- a clearer entry cue at TSI lows (bullish trigger) and
exit cue at TSI highs (bearish trigger).

## Pipeline

1. Same TSI pipeline as `chart-line-tsi-zero-cross`:
   - `mom_i = close_i - close_{i-1}`, `abs_mom_i = |mom_i|`.
   - `sm1_mom = EMA(mom, long, SMA-seed)`,
     `sm2_mom = EMA(sm1_mom, short, SMA-seed)`.
   - `sm1_abs = EMA(abs_mom, long, SMA-seed)`,
     `sm2_abs = EMA(sm1_abs, short, SMA-seed)`.
   - `tsi = sm2_abs == 0 ? 0 : 100 * sm2_mom / sm2_abs`.
2. Signal line: `signal = EMA(tsi, signal_length, SMA-seed at i = long + short - 1)`.
3. `classifyLineTsiSignalCrossRegime(tsi, signal)`: null tsi or null
   signal -> none; `tsi >= signal` -> bullish; `tsi < signal` -> bearish.
4. `detectLineTsiSignalCrossCrosses(series, tsi, signal)`: strict
   `(prev_tsi - prev_signal) <= 0 && (cur_tsi - cur_signal) > 0` for
   bullish; symmetric for bearish. Boundary equality does not fire.

## Bit-exact anchors

| Pattern                  | TSI / signal (after warmup) | Regime  |
|--------------------------|-----------------------------|---------|
| CONST close = K          | 0 / 0                       | bullish |
| LINEAR UP close = i      | 100 / 100                   | bullish |
| LINEAR DOWN close = -i   | -100 / -100                 | bullish |

CONST anchor tested bit-exactly across K in `{0, 1, 42, 100, 1234}`.
LINEAR UP / DOWN saturate TSI to `+/- 100` constant, and signal chases
to the same value -- since `tsi >= signal` at equality the regime is
bullish in both cases. The interesting behaviour (and crosses) live in
transients (decline-then-rise, rise-then-decline patterns) where TSI
overshoots signal during the reversal.

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), TSI + signal
  panel below.
* Osc panel uses a fixed `[-100, 100]` range (TSI is bounded by
  construction) with the zero baseline at the midpoint.
* Zero band rendered as a horizontal rule at `y = zeroY` in the osc
  panel.
* Cross markers drawn as circles on the osc panel at `(cx, cyOsc)`
  (positioned at the TSI level at the cross index) and as arrows on
  the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="TSI Signal Cross chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only
  text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
