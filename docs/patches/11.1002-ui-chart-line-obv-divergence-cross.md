## Summary

Add `web/src/components/ui/chart-line-obv-divergence-cross.tsx`
-- close+volume On Balance Volume (OBV) cumulative divergence
detector marking discrete price-versus-OBV direction
disagreement events for accumulation / distribution warning.
Second volume-aware primitive in the chart-line family (after
KVO mid-cross-sig), and the first cumulative-running-total
divergence detector.

## Math anchor (bit-exact)

```
direction[i] = sign(close[i] - close[i-1])
obv[i]       = obv[i-1] + direction[i] * volume[i]; obv[0]=0
priceUp      = close[i] > close[i-window]
obvUp        = obv[i]   > obv[i-window]
five-state regime model with crosses suppressed when prev/cur is 'none'
```

- **CONST close=K, V=const**: direction=0 -> obv=0 constant
  -> aligned-bearish (0 crosses).
- **LINEAR UP close=i, V=1000**: direction=+1 -> obv=[0,V,2V,
  ...] -> priceUp=true, obvUp=true -> aligned-bullish (OBV
  correctly tracks the uptrend; unlike RSI/Stoch/CMO which
  saturate and read divergent on monotonic uptrends, OBV is a
  cumulative line that linearly accumulates). 0 crosses.
- **LINEAR DOWN close=-i, V=1000**: direction=-1 -> obv=
  [0,-V,-2V,...] -> aligned-bearish. 0 crosses.

For 50 bars: noneCount=5, post-warmup regime count=45 (no
length-based warmup, only divergenceWindow).
