# 11.932 -- UI: chart-line-rsi-mid-cross primitive (v1.11.950)

## Summary

Adds `ChartLineRsiMidCross`, a pure-SVG dual-panel chart that
renders the close in the top panel and a close-only Relative
Strength Index (RSI) line in the bottom panel. Marks bullish
(cross up through midline 50) / bearish (cross down through
midline 50) momentum centerline regime trigger events. Single-
threshold cross variant of the RSI family that flags the
discrete RSI level 50 midline crossover distinct from the
canonical overbought (70) / oversold (30) extreme bands.

## Pipeline

For close-only data with default `length = 14`:

```
delta[i] = close[i] - close[i-1]
gain[i]  = max(delta, 0)
loss[i]  = max(-delta, 0)
avgGain  = Wilder(gain, length)
avgLoss  = Wilder(loss, length)
rsi[i]   = avgGain > 0 && avgLoss > 0
             ? 100 - 100 / (1 + avgGain / avgLoss)
             : avgGain > 0 ? 100
               : avgLoss > 0 ? 0
                 : 50              (zero-flow neutral fallback)
bullish  : prev rsi <= 50 && cur rsi > 50   (momentum up)
bearish  : prev rsi >= 50 && cur rsi < 50   (momentum down)
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> delta = 0 every bar
  -> gain = loss = 0 -> avgGain = avgLoss = 0 -> rsi = 50
  via the zero-flow neutral fallback. rsi = 50 sits on the
  threshold but the strict-inequality detector never fires
  (cur > 50 is required for bullish, cur < 50 for bearish).
  regime `bullish` (rsi >= 50). cross count = 0. Verified
  across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` (length 14) -> delta = +1 every bar
  -> avgGain converges to 1, avgLoss = 0 -> rsi = 100
  constant once warm. regime `bullish`. 0 crosses (rsi jumps
  from null to 100 so prev-null skips the strict-inequality
  detector).
- LINEAR DOWN `close = -i` -> delta = -1 every bar -> avgGain
  = 0, avgLoss converges to 1 -> rsi = 0 constant. regime
  `bearish`. 0 crosses.
- Balanced alternation `[10, 11, 10, 11, ...]` (length 14)
  -> 7 gains + 7 losses each = 7 -> avgGain = avgLoss = 0.5
  -> rsi[14] = 50 exactly at seed; thereafter Wilder
  smoothing oscillates each step but stays bounded between
  40 and 60.
- Alternation followed by sustained rally / selloff -> Wilder
  pushes rsi strictly above / below 50, generating exactly
  one bullish / bearish midline cross.

## Regime classifier

| rsi range            | regime      |
|----------------------|-------------|
| rsi >= 50            | bullish     |
| rsi < 50             | bearish     |
| null                 | none        |

  / `Space` keyboard toggle parity.
