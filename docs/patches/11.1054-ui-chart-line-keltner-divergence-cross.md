# 11.1054 -- UI: chart-line-keltner-divergence-cross primitive (v1.11.1072)

## Scope

Add a pure-SVG dual-panel `ChartLineKeltnerDivergenceCross`
component that renders the close in the top panel and the
Keltner Channel (EMA midline plus upper / lower ATR-scaled
bands) in the bottom panel, marking price-vs-channel
direction disagreement events as volatility-reversal
warnings with bias coloring derived from the Keltner
midline slope at the trigger bar.

Chester Keltner's 1960 channel (popularised by Linda
Bradford Raschke's 1980s adaptation) brackets price with
an EMA-centered band of width `2 * ATR(period)`. The
midline is the EMA centerline of the volatility envelope.
This primitive compares the midline slope against price
direction to surface volatility-reversal warnings: when
price falls but the channel is still trending up, the
trend-following midline disagrees with the current bar's
direction, signalling a potential bullish reversal in the
volatility envelope; when price rises but the channel is
trending down, the inverse disagreement signals a bearish
reversal warning.

## Keltner divergence-cross pipeline

```
tr[i]        = max(high - low,
                   |high - close[i-1]|,
                   |low  - close[i-1]|)
atr[i]       = SMA(tr, period)
midline[i]   = EMA(close, period)   (SMA-seeded)
upperBand[i] = midline[i] + multiplier * atr[i]
lowerBand[i] = midline[i] - multiplier * atr[i]

priceUp     : close[i] > close[i-1]
priceDown   : close[i] < close[i-1]
midlineUp   : midline[i] > midline[i-1]
midlineDown : midline[i] < midline[i-1]

regime ->
  'aligned-bullish'   when priceUp   && midlineUp
  'aligned-bearish'   when priceDown && midlineDown
  'divergent-bullish' when priceDown && midlineUp
                       (selling pressure but channel
                        still trending up -- bullish
                        volatility-reversal warning)
  'divergent-bearish' when priceUp   && midlineDown
                       (rally but channel trending down
                        -- bearish volatility-reversal
                        warning)
  'none'              when either side flat or null

bullish (divergence trigger up) :
  prev regime !== 'divergent-bullish' &&
  cur regime === 'divergent-bullish'
bearish (divergence trigger down) :
  prev regime !== 'divergent-bearish' &&
  cur regime === 'divergent-bearish'

bias        : midline[i] vs midline[i-1] -> up / down /
              flat / none
```

Default `period = 20`, `multiplier = 2` -- modern Linda
Bradford Raschke tuning. Chester Keltner's original 1960
formulation used `period = 10`, `multiplier = 1`.

This primitive uses **SMA-based ATR** (matching the
chart-line-atr-breakout-cross / atr-divergence-cross /
supertrend-mid-cross-sig siblings) for bit-exact integer
values on linear input. The two ATR variants in the
codebase (SMA vs Wilder) are a documented family-level
tuning choice.

Warmup is `period = 20` for the default tuning: ATR and
the EMA midline both seed at i = period - 1 = 19, and
direction detection requires the previous bar's midline,
so the first regime classification lands at i = period =
20.

## Bit-exact anchors

All anchors use HLC input.

- **CONST band** `high = K + 1`, `low = K - 1`, `close
  = K`: tr = 2, atr = 2. SMA-seeded EMA on a constant
  input is exactly the constant. midline = K. Both price
  and midline are flat -> regime `none` for every valid
  bar. 0 crosses. Verified across K in {0, 1, 50, 200,
  1234}.
- **LINEAR UP** `high = i + 1`, `low = i - 1`, `close
  = i`: tr = max(2, 2, 0) = 2. atr = 2. SMA seed at i =
  19: mean of close[0..19] = 9.5. EMA recurrence on
  linear: midline[i] = i - 9.5 (matches the (period -
  1) / 2 centroid offset). upperBand = i - 5.5, lowerBand
  = i - 13.5. priceUp + midlineUp -> aligned-bullish for
  every valid bar. 0 crosses (no transition into
  divergent regime). This is the canonical EMA midline
  on a steady linear trend.
- **LINEAR DOWN** `high = -i + 1`, `low = -i - 1`,
  `close = -i`: mirror -> midline = -i + 9.5. priceDown
  + midlineDown -> aligned-bearish. 0 crosses.

All three anchors produce zero divergence crosses -- the
regime stays in `none` (CONST), `aligned-bullish` (LINEAR
UP), or `aligned-bearish` (LINEAR DOWN). Divergence
crosses only appear when price and midline directions
disagree, which requires curvature in the price series.

## Risk

- Pure-SVG, deterministic `.toFixed(2)`, no canvas, no
  chart lib dependency.
- 5-state regime classifier follows the divergence-cross
  family convention.
- SMA-based ATR (not Wilder) for bit-exact testability.
  Documented as a deliberate family-level choice. The
  Wilder ATR variant is used by
  chart-line-supertrend-divergence-cross v1.11.1062.
- The Keltner midline can change direction abruptly on
  trend flips (close ratcheting the EMA); the bands
  scale with ATR (volatility). On steady LINEAR input
  the midline produces a constant slope and zero
  divergence crosses, which is correct canonical
  behaviour.
- HLC input boundary filter requires `high >= low` --
  malformed bars are dropped at the input boundary.
- Complementary to the sibling divergence-cross
  primitives (HMA / EMA / SMA / WMA / TEMA / KAMA / VWAP
  / ichimoku / supertrend / williams / atr). Keltner
  adds the EMA + ATR envelope variant alongside the
  pure-MA, momentum-oscillator, volume-weighted, multi-
  timeframe-composite, ratcheting-stop, bounded-
  momentum, and volatility-magnitude siblings.
- Complementary to the sibling Bollinger Bands variant
  (would be a future primitive). Both share the
  midline-plus-volatility-bands shape, but Keltner uses
  ATR (true range) while Bollinger uses standard
  deviation.
