## Summary

Add `web/src/components/ui/chart-line-stc-oversold-cross.tsx` --
a pure-SVG dual-panel React/TypeScript chart primitive that
renders the close in the top panel and the close-only Schaff
Trend Cycle (STC) line in the bottom panel, marking discrete
bullish (STC crosses up through threshold 25, cycle oversold
trigger) / bearish (STC crosses down through threshold 25) STC
oversold-crossover events with bias coloring derived from the
STC slope at the trigger bar. Mirror of
`chart-line-stc-overbought-cross` (11.990) with the default
threshold flipped from 75 to the canonical oversold level 25.

## Math anchor (bit-exact)

```
macd      = SMA(close, fast) - SMA(close, slow)
k1        = stochastic(macd, cycleLength)  (degenerate = 50)
d1        = EMA(k1, factor)
k2        = stochastic(d1, cycleLength)    (degenerate = 50)
stc       = EMA(k2, factor)
bullish   : prev stc <= 25 && cur stc >  25
bearish   : prev stc >= 25 && cur stc <  25
bias      : sign(stc[i] - stc[i-1]) -- 'up' / 'down' / 'flat' / 'none'
```

- **CONST close = K**: macd = 0 -> cycle collapses through
  stochastic and EMA to stc = 50. 50 >= 25 -> regime `bullish`
  (opposite of overbought-cross's bearish at the same STC=50
  reading -- canonical mirror flip). 0 oversold crosses.
  Verified across K = 0..1234.
- **LINEAR UP close = i**: macd = 13.5 constant. cycle
  collapses to stc = 50. regime `bullish`. 0 crosses.
- **LINEAR DOWN close = -i**: macd = -13.5. stc = 50. regime
  `bullish`. 0 crosses.

For 80 bars at defaults (fastLength=23, slowLength=50,
cycleLength=10, factor=0.5):
- macd null for i = 0..48
- stc null for i = 0..66
- post-warmup regime count = 13 (i = 67..79)

Strict inequality on `cur` (`cur > 25` for bullish, `cur < 25`
for bearish) means boundary equality (stc == 25) never double-
fires.

Layout uses fixed oscMin=0 / oscMax=100 (STC canonical range).
