## Summary

`ChartLineKcPercentK` is a pure-SVG dual-panel React primitive that
plots the close on the top panel and a Keltner Channel %K oscillator
on the bottom panel. %K locates the close within the Keltner channel:

```
middle[i] = EMA(close, length)[i]
TR[i]     = max(high - low,
                |high - prevClose|,
                |low - prevClose|)       // TR[0] = high - low
ATR[i]    = SMA(TR, length)[i]
upper[i]  = middle[i] + multiplier * ATR[i]
lower[i]  = middle[i] - multiplier * ATR[i]
%K[i]     = (close[i] - lower[i]) / (upper[i] - lower[i])
```

Defaults: `length = 20`, `multiplier = 2`. When `ATR = 0` (the singular
"completely flat" case where every bar has `high = low = close`) %K is
emitted as `null`. The first `length - 1` bars are warmup nulls because
the ATR SMA needs `length` TR samples.

## Bit-exact anchors

The component is designed around two algebraic anchors that exercise
the entire pipeline (EMA, TR/ATR, channel math, ratio):

- **CONST_MID** -- `high = K + r`, `low = K - r`, `close = K` at every
  bar. Then `TR = 2r` at every bar (the first bar uses `high - low`,
  later bars still see `max(2r, r, r) = 2r`), so `ATR = 2r` after
  warmup, and `middle = EMA(K) = K` bit-exact. The channel evaluates
  to `upper = K + 4r`, `lower = K - 4r`, and:

      %K = (K - (K - 4r)) / ((K + 4r) - (K - 4r))
         = 4r / 8r
         = 0.5

  The numerator and denominator share the same IEEE 754 drift in the
  EMA branch, so the ratio simplifies to a literal `0.5` regardless of
  `K`, `r`, or multiplier. The test sweeps `(r, length, multiplier)` to
  prove this.

- **CONST_FLAT** -- `high = low = close = K`. Every TR is zero, every
  ATR is zero, the channel collapses to a point, and the component
  surfaces `null` rather than dividing by zero.

A handful of additional anchors fall out of the same algebra:

- `close = high` (constant) -> `%K = 0.5` (close stays at middle
  because `EMA(K + r) = K + r`).
- `close = low` (constant) -> `%K = 0.5` (symmetric argument).
- Translation invariance: shifting `H/L/close` by a constant `C` does
  not change `%K` (`EMA` and `ATR` both translate linearly).
