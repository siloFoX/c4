# Patch 11.909 -- UI: chart-line-mfi-cross-sig primitive

## Summary

Added `<ChartLineMfiCrossSig />` -- pure-SVG dual-panel React/TS
primitive that renders the close (top panel) with bullish /
bearish arrow overlays at every Money Flow Index vs EMA-
smoothed signal cross, and the MFI line alongside its signal
in the bottom panel on a fixed 0-100 oscillator with cross dot
markers + 20 / 50 / 80 reference bands. Signal-cross variant
of the MFI family that flags volume-weighted momentum trigger
events distinct from the canonical MFI overbought / oversold
readings.

## Algorithm

```
delta[i]    = close[i] - close[i-1]
posFlow[i]  = delta > 0 ? close[i] * delta : 0
negFlow[i]  = delta < 0 ? close[i] * -delta : 0
posSum[i]   = sum(posFlow, length) over last n bars
negSum[i]   = sum(negFlow, length) over last n bars
mfi[i]      = posSum + negSum <= 0
                ? 50                           (neutral fallback)
                : negSum <= 0
                    ? 100
                    : 100 - 100 / (1 + posSum/negSum)
signal[i]   = EMA(mfi, signalLength)
bullish    : (mfi - signal) crosses up   (prev <= 0, cur > 0)
bearish    : (mfi - signal) crosses down (prev >= 0, cur < 0)
```

Defaults `length = 14` (canonical MFI window), `signalLength
= 9`. Regime classifier `bullish` (mfi > signal), `bearish`
(mfi < signal), `neutral` (mfi === signal), `none` (either
null). Close-only adaptation uses `close[i] * delta` as the
proxy for raw money flow.

## Bit-exact anchor

CONST close = K: every delta = 0 -> posFlow = negFlow = 0 ->
posSum = negSum = 0 -> MFI uses the neutral fallback 50.
signal EMA of 50s = 50 via the `min === max` precision short-
circuit. mfi === signal -> regime `neutral`, cross count = 0.
Verified across K = 0, 1, 5, 17, 100, 1234.

LINEAR UP has bit-exact anchor: every delta > 0 means
negFlow = 0 everywhere -> negSum = 0 -> MFI = 100. LINEAR
DOWN mirror: posFlow = 0 -> posSum = 0 -> MFI = 0.

## Visual

- Two stacked panels with the osc panel pinned to 0-100.
- Top: close line + bullish/bearish polygon arrows at cross
  bars.
- Bottom: MFI + signal lines + cross dot markers + 20/50/80
  reference bands (oversold / mid / overbought).
- Tooltip lists x / close / mfi / signal / regime / counts /
  crosses.
- Config badge + togglable legend (close / MFI / signal).
- Cross marker is `role="graphics-symbol"` with `tabIndex={0}`.
