# Patch 11.947 - UI: chart-line-fisher-zero-cross primitive

Version: 1.11.965

## Summary

Add `web/src/components/ui/chart-line-fisher-zero-cross.tsx` -- a pure-SVG
dual panel line chart primitive that visualizes price alongside John Ehlers'
Fisher Transform panel, with explicit cross markers and overlay arrows at
every Fisher zero crossover. The Fisher Transform normalises a recursively
smoothed price-channel position to a Gaussian-like distribution by
applying an inverse hyperbolic tangent (`0.5 * ln((1 + x) / (1 - x))`),
so extreme momentum that would be heavily damped in a raw oscillator
becomes numerically separable from the neutral baseline -- making zero
crossovers a sharper trigger for momentum regime changes.

## Pipeline

1. Rolling channel position per bar:
   - `hi[i] = max(close[i-length+1..i])`
   - `lo[i] = min(close[i-length+1..i])`
   - `raw[i] = hi == lo ? 0 : 2 * (close - lo) / (hi - lo) - 1`
2. Recursively smoothed normalised series:
   - `x[i] = clamp(0.33 * raw + 0.67 * x[i-1], -0.999, 0.999)`
3. Fisher Transform with one-bar feedback:
   - `fisher[i] = 0.5 * ln((1 + x) / (1 - x)) + 0.5 * fisher[i-1]`
4. `classifyLineFisherZeroCrossRegime(fisher, T)`: null -> none,
   fisher >= T -> bullish, fisher < T -> bearish.
5. `detectLineFisherZeroCrossCrosses(series, fisher, T)`: strict
   `prev <= T && cur > T` for bullish, strict `prev >= T && cur < T` for
   bearish. Boundary equality does not fire.

The `0.999` clamp keeps `1 - x` strictly positive so `ln` stays finite,
even when an extended trend pins the channel position at its limit.

## Bit-exact anchors

| Pattern                  | Fisher (after warmup) | Regime  |
|--------------------------|-----------------------|---------|
| CONST close = K          | 0 (bit-exact)         | bullish |
| LINEAR UP (close = i)    | > 0, -> ~7.6 steady   | bullish |
| LINEAR DOWN (close = -i) | < 0, -> ~-7.6 steady  | bearish |

CONST anchors tested bit-exactly across K in `{0, 1, 42, 100, 1234}`.
LINEAR UP/DOWN anchors tested as sign-only over length=100 series with
the convergence bound `5 < |fisher_steady| < 10`.

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), Fisher panel below.
* Fisher panel auto-fits to the observed `[min, max]` range with 10%
  padding on each end, always including the threshold in the displayed
  range.
* Threshold band rendered as a horizontal rule at `y = thresholdY` in
  the Fisher panel.
* Cross markers drawn as circles on the Fisher panel at `(cx, cyOsc)`
  and as arrows on the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="Fisher Zero Cross chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only
  text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
