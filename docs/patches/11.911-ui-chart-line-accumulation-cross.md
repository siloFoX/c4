# Patch 11.911 -- UI: chart-line-accumulation-cross primitive

## Summary

Added `<ChartLineAccumulationCross />` -- pure-SVG dual-panel
React/TS primitive that renders the close (top panel) with
bullish / bearish arrow overlays at every Accumulation /
Distribution Line vs EMA-smoothed signal cross trigger, and
the close-only A/D Line + signal line in the bottom panel
with cross dot markers and a dashed zero baseline. Signal-
cross variant of the A/D Line family that flags accumulation
versus distribution regime change trigger events distinct
from the cumulative A/D Line trend.

## Algorithm

```
hi[i]     = max(close[i-b+1..i])               (close-only proxy)
lo[i]     = min(close[i-b+1..i])
mfm[i]    = hi - lo > 0
              ? (2*close - lo - hi) / (hi - lo)  (Money Flow Mult)
              : 0
mfv[i]    = mfm * close                         (volume proxy)
ad[i]     = ad[i-1] + mfv[i]                    (cumulative A/D)
signal[i] = EMA(ad, signalLength)
bullish  : (ad - signal) crosses up   (prev <= 0, cur > 0)
bearish  : (ad - signal) crosses down (prev >= 0, cur < 0)
```

Defaults `barLength = 2` (high / low proxy window from a 2-bar
window of closes), `signalLength = 9`. Regime classifier
`bullish` (ad > signal), `bearish` (ad < signal), `neutral`
(ad === signal), `none` (either null). Close-only adaptation:
high / low are rolling max / min of close; close is reused as
the volume proxy.

## Bit-exact anchor

CONST close = K: hi = lo = K every bar -> range = 0 -> mfm = 0
neutral fallback. mfv = 0 every bar so the cumulative ad = 0
constantly. signal EMA of 0s = 0 via the `min === max`
precision short-circuit. ad === signal -> regime `neutral`,
cross count = 0. Verified across K = 0, 1, 5, 17, 100, 1234.

LINEAR UP yields positive mfm values (close near recent high)
so the cumulative A/D grows positively. LINEAR DOWN is the
mirror (negative cumulative growth).
