## Summary

Add `web/src/components/ui/chart-line-mfi-divergence-cross.tsx`
-- HLCV Money Flow Index (MFI) divergence detector marking
discrete price-versus-MFI direction disagreement events for
volume-weighted reversal warning. MFI is the volume-weighted
RSI -- divergences carry more weight than pure-price RSI
divergences because the underlying momentum signal accounts
for both price direction AND traded volume.

## Math anchor (bit-exact)

```
typical[i]   = (high + low + close) / 3
moneyFlow    = typical * volume
posSum/negSum = sums over length of moneyFlow when typical rises/falls
mfi[i]       = 100 - 100 / (1 + posSum/negSum);
               50 when both 0, 100 when only positive, 0 when only negative
priceUp      = close[i] > close[i-window]
mfiUp        = mfi[i]   > mfi[i-window]
five-state regime model with crosses suppressed when prev/cur is 'none'
```

- **CONST H=L=C=K, V=const**: typical=K -> no flow -> mfi=50
  (degenerate neutral) -> aligned-bearish (0 crosses).
- **LINEAR UP H=L=C=i, V=const**: typical=i, only positive
  flow -> mfi=100 saturated -> divergent-bearish (canonical
  bearish divergence). 0 crosses.
- **LINEAR DOWN H=L=C=-i, V=const**: only negative flow ->
  mfi=0 saturated -> aligned-bearish. 0 crosses.

For 50 bars at defaults (length=14, divergenceWindow=5):
noneCount=19, post-warmup regime count=31.
