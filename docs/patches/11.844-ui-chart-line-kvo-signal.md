# Patch 11.844 -- ChartLineKvoSignal primitive (v1.11.862)

## Summary

Add `<ChartLineKvoSignal />`, a pure-SVG dual-panel chart with the
close on top and the Klinger Volume Oscillator signal line beneath.
Uses the simplified sign-based Volume Force formulation (per-bar
volume signed by typical-price direction) instead of Klinger's full
accumulator-based force, then passes it through MACD-style fast /
slow EMA spread plus the canonical signal-line EMA:

```
tp[i]      = (high[i] + low[i] + close[i]) / 3
sign[i]    = tp[i] > tp[i - 1] ?  1
           : tp[i] < tp[i - 1] ? -1
           :                      0   (and tp[0] -> sign 0)
vf[i]      = volume[i] * sign[i]
kvo[i]     = EMA(vf, fastLength)[i] - EMA(vf, slowLength)[i]
signal[i]  = EMA(kvo, signalLength)[i]
```

The EMA helper carries the `min === max` seed precision fix used in
the earlier MACD-style primitives so constant inputs collapse to the
exact constant without 1-ULP drift.

## Bit-exact anchors

- `CONST OHLC and volume = V`: typical price is constant, `sign = 0`
  every bar, so `vf = 0` from `i = 1`. Both EMAs collapse to 0,
  `kvo = 0`, signal EMA of 0 = 0. Verified across `(K, V, fast,
  slow, signal)` sweeps.
- `LINEAR UP high = low = close = i + 1, volume = V > 0`:
  `tp[i] = i + 1` rises strictly so `sign[i] = 1` for `i >= 1`;
  `vf[i] = V * 1 = V` constant. Both EMAs collapse to `V` via the
  min===max seed and CONST short-circuit, so `kvo = V - V = 0`,
  `signal = 0`. Verified across `(V, fast, slow, signal)` sweeps.
- `LINEAR DOWN`: symmetric, `vf = -V` constant, `kvo = -V - (-V) =
  0`, `signal = 0`.

## Surface

Dual-panel primitive surface:

- `ChartLineKvoSignalPoint = { x, high, low, close, volume }`
- `runLineKvoSignal(data, options?) -> Run` with
  `ok = series.length > slowLength + signalLength`.
- `computeLineKvoSignalLayout({ data, width?, height?, padding?, panelGap?, fastLength?, slowLength?, signalLength? }) -> Layout`.
  Bottom panel y-axis auto-fits but always includes 0.
- `describeLineKvoSignalChart(data, opts?) -> string` for the ARIA
  description.
- `<ChartLineKvoSignal />` `forwardRef<HTMLDivElement>` with three
  legend toggles (`price`, `kvo`, `signal`), config badge, zero-line
  overlay, markers on signal-line zero crosses, and a tooltip
  surfacing `close / vol / tp / vf / kvo / signal / zone / cross`.
