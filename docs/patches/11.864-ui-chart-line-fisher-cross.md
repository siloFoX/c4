## Summary

Added `<ChartLineFisherCross />` -- a pure-SVG, dual-panel
React/TS primitive that renders the close on top and John Ehlers'
Fisher Transform plus its signal (the one-bar-lagged Fisher) on
the bottom. The Fisher Transform gaussian-normalises price action
so extremes are emphasised. Markers fire at every Fisher-vs-
signal crossover.

Cross events:

- `up` cross   -> regime `accelerating-up`
- `down` cross -> regime `accelerating-down`

Defaults: `length = 10`, fisherColor `#0ea5e9` (sky), signalColor
`#f97316` (orange), bullish marker `#16a34a`, bearish marker
`#dc2626`, zero line `#475569`.

## Formula

```
hl2[i]        = (high[i] + low[i]) / 2
normalized[i] = (hi === lo)
                  ? 0.67 * normalized[i-1]
                  : 0.66 * ((hl2 - lo) / (hi - lo) - 0.5)
                    + 0.67 * normalized[i-1]
              clamped to [-0.999, 0.999]
fisher[i]     = 0.5 * ln((1 + normalized[i]) / (1 - normalized[i]))
                + 0.5 * fisher[i-1]
signal[i]     = fisher[i-1]
```

The +/-0.999 clamp avoids the singularity of the inverse
hyperbolic tangent at the boundaries. Cross detection uses the
standard prev-relation + strict-inequality guard.

## Bit-exact anchor

- **CONST h = l = close = K**: `hi === lo` is true on every bar,
  so the normalized update collapses to `0.67 * normalized[i-1]`.
  With `normalized[0] = 0`, the value stays at 0 forever, so
  `fisher = 0.5 * ln(1/1) + 0.5 * 0 = 0` and `signal = 0`.
  Relation `equal` forever, zero crosses. Verified across
  multiple K and length.
