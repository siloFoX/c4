## Summary

`ChartLineAlligator` is a pure-SVG single-panel React primitive
that overlays Bill Williams' Alligator on the close. The
Alligator is three smoothed moving averages (SMMA) of the median
price, each shifted forward by a separate offset:

```
median[i]   = (high[i] + low[i]) / 2
jaw[i]      = SMMA(median, jawPeriod)[i - jawShift]
teeth[i]    = SMMA(median, teethPeriod)[i - teethShift]
lips[i]     = SMMA(median, lipsPeriod)[i - lipsShift]
```

where the Smoothed Moving Average (Wilder smoothing) is:

```
SMMA[i] = (SMMA[i - 1] * (length - 1) + price[i]) / length
```

seeded with the SMA of the first `length` finite prices.

Defaults match Bill Williams' canonical configuration:
- `jawPeriod = 13, jawShift = 8`
- `teethPeriod = 8, teethShift = 5`
- `lipsPeriod = 5, lipsShift = 3`

The first valid bar for jaw is `i = jawPeriod + jawShift - 1 = 20`.

## Bit-exact anchor

For **CONST_FLAT** (`high = low = K`):

- `median = (K + K) / 2 = K` exactly.
- SMA seed of `length` constant `K` values is `K` exactly.
- The SMMA recurrence collapses to a tautology:

      SMMA[i] = (SMMA[i - 1] * (L - 1) + K) / L
              = (K * (L - 1) + K) / L
              = K * L / L
              = K   (bit-exact for any IEEE 754 representable K)

  The arithmetic stays exact because the numerator
  `K * (L - 1) + K = K * L` is closed under integer-coefficient
  combinations of `K`, and the divide-by-`L` cancels cleanly.

- After shifting, `jaw = teeth = lips = K` past warmup.

The integration sweep verifies this across `K` in
`{0, 1, 5, 100, -3, 7, -50}`.

## Zone classifier

- `eating` -- lines stacked monotonically: lips > teeth > jaw
  (strong uptrend) or lips < teeth < jaw (strong downtrend)
- `awake` -- lines stacked but with at least one inversion
  (transitional)
- `sleeping` -- at least two of the three lines coincide (no
  directional trend, e.g. CONST_FLAT)
- `none` -- any of the three is null (warmup)
