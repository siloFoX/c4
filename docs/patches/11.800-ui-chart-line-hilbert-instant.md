## Summary

`ChartLineHilbertInstant` is a pure-SVG dual-panel React primitive
that plots the close on the top panel and a Hilbert-Transform-
inspired Instantaneous Trendline panel beneath. The trendline is
the 4-bar weighted-moving-average smoothed close minus its lagged
value (the "InPhase smoothed close detrended by its lagged value"
from the dispatch):

```
smoothed[i] = (4 * close[i] + 3 * close[i - 1] +
               2 * close[i - 2] + close[i - 3]) / 10    (i >= 3)
instant[i]  = smoothed[i] - smoothed[i - lag]           (i >= 3 + lag)
```

Defaults: `lag = 6`. Bars before `i = 3 + lag` are warmup nulls.

The 4-bar WMA filter (with weights `[4, 3, 2, 1]` summing to 10)
acts as the InPhase-like low-pass smoother. Subtracting the
lagged value detrends the smoothed signal, leaving an oscillator
that reads positive when the trend is up, negative when down,
and zero when flat.

## Bit-exact anchors

- **CONST close** (`close = K`): the WMA weights sum to 10, so
  `smoothed = (4+3+2+1) * K / 10 = K` bit-exact at every bar.
  `instant = K - K = 0` bit-exact past warmup.
- **RAMP close** (`close[i] = a * i` for integer `a != 0`):

      smoothed[i] = (4 * a*i + 3 * a*(i-1) + 2 * a*(i-2) + a*(i-3)) / 10
                  = a * (10*i - 10) / 10
                  = a * (i - 1)                        (bit-exact)

  so `instant[i] = a*(i - 1) - a*(i - 1 - lag) = a * lag`
  bit-exact past warmup. The integration sweep verifies this
  across `a` in `{1, 2, 3, -1, -5}` and `lag` in `{2, 4, 6, 10}`.

The arithmetic stays exact because the numerator `10*i - 10` is
an exact integer multiple of 10, so the divide-by-10 produces an
exact integer (when `a` is integer). The smoothed series is
shifted by one bar but preserves slope, and the lagged
subtraction trivially returns `a * lag`.

## Zone classifier

The reading is bucketed by ratio to the maximum absolute value
observed in the current series:

- `strong-up` -- instant >= 0.5 * absMax
- `up` -- 0 < instant < 0.5 * absMax
- `flat` -- instant == 0
- `down` -- -0.5 * absMax < instant < 0
- `strong-down` -- instant <= -0.5 * absMax
- `none` -- instant is null (warmup)

Sane fallback when `absMax` is zero (CONST close): classify any
non-zero reading as `up` or `down` by sign.

## Out of scope

- No new dependencies.
- No canvas, no third-party chart libs.
- This primitive ships the simplified WMA-detrended variant.
  The full Ehlers Hilbert Transform (with InPhase + Quadrature
  decomposition and dominant-cycle period estimation) is a
  larger DSP component and is not in scope here.
