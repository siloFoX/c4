RSI / MACD divergence primitives.

## Math anchor (bit-exact)

```
cmo[i]   = 100 * (gainSum - lossSum) / (gainSum + lossSum); 0 degenerate
priceUp  = close[i] > close[i-window]
cmoUp    = cmo[i]   > cmo[i-window]
states: aligned-bullish, aligned-bearish, divergent-bullish, divergent-bearish, none
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
(suppressed when prev or cur state == 'none')
```

- **CONST close = K**: cmo=0 -> priceUp=false, cmoUp=false ->
  aligned-bearish. 0 crosses.
- **LINEAR UP close = i**: cmo=100 (saturated) -> priceUp=true,
  cmoUp=false -> divergent-bearish (price still rising while
  CMO saturated -- waning momentum). 0 crosses (prev=none).
- **LINEAR DOWN close = -i**: cmo=-100 -> priceUp=false,
  cmoUp=false -> aligned-bearish. 0 crosses.

For 50 bars: noneCount=19, post-warmup regime count=31.
