## Summary

Add a new `<ChartStreamgraph>` UI primitive in
`web/src/components/ui/chart-streamgraph.tsx` -- pure-SVG
streamgraph (stacked-area variant with selectable
baseline) with smooth flow curves. Distinct from
`<ChartStream>` (11.453) which is the simple centered
variant -- ChartStreamgraph adds first-class **baseline
algorithms** (`silhouette`, `wiggle`, `expand`, `zero`),
**curve interpolation choice** (`cardinal`, `catmullRom`,
`linear`, `step`), an explicit x-axis label track with
`xLabels` + `formatXLabel`, and a `tension` parameter
for the cardinal curve so an adopter can dial smoothness
between piecewise-linear and full-Bezier.

## Behaviour

- Renders `<div role="region">` wrapping
  `<svg role="img">` with the same `aria-label`.
- Baseline algorithms (`computeStreamgraphBaseline`):
  - `silhouette` (default): every sample is centred
    around `-total/2`, producing the canonical
    streamgraph silhouette growing symmetrically up +
    down.
  - `wiggle`: minimises layer slope by walking the
    layers per sample (the streamgraph minimum-
    wiggle algorithm). Layers with all-zero totals
    inherit the previous offset so the baseline
    never jumps.
  - `expand`: 100%-stacked. Each sample's values
    rescale to `[0, 1]`; baseline stays at 0.
  - `zero`: traditional flat-bottom stacked area.
- Curve interpolation (`buildStreamgraphCurve`):
  - `cardinal` (default) uses tension to control
    smoothness (0 -> Catmull-Rom; 1 -> linear).
  - `catmullRom` is the tension=0 special case.
  - `linear` paints L commands between vertices.
  - `step` paints square corners between vertices.
- Non-positive / non-finite raw values clamp to 0 so
  a series never inverts the stack.
- Layers tile cleanly: layer N's `topValue` always
  equals layer N+1's `baseValue` at every sample.
- Hidden series (controlled or uncontrolled) drop
  from the layout entirely; the remaining series
  reproportion across the stack.
- Hover/focus on a layer dims every other layer to
  opacity 0.3 (focused layer dominates) and opens a
  tooltip with the series label, sum of raw values
  across all samples, and the sample count.
- `xLabels` (length = max series data length) drives
  the x-axis tick labels. For more than 8 samples
  the primitive auto-thins to ~8 tick positions
  (preserving the first + last index) so labels
  never overlap.
- Mount animation:
  `motion-safe:animate-fade-in` per layer.
  `animate=false` removes the class.
- Default ARIA description: `"Streamgraph (<baseline>)
  with N series across M samples. Total <sum>, peak
  total <peak>."` Empty / all-hidden returns
  `"No data"`; `ariaDescription` overrides.

## Out of scope

- Negative values stacked below the baseline (host
  normalises; values clamp to 0).
- Smooth animated transitions between baseline /
  curve choices (snap v1; mount fade only).
- Per-layer order control (host owns array; first
  series renders at the bottom of the stack).
- Continuous time-axis ticks: v1 emits one label
  per sample index (auto-thinned).
