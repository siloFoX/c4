local signal disorder.

## Indicator math

Shannon entropy measures the disorder of a probability
distribution.

- `computeLineEntropyHistogram` -- bin a window of values into
  an equal-width histogram. The window range `[min, max]` is
  split into `bins` buckets; each value is counted into its
  bucket (the maximum lands in the last bucket); a flat window
  collapses into the first bucket. The `probabilities` are the
  counts divided by the window length.

- `computeLineEntropyValue` -- the Shannon entropy of a
  distribution, in bits:

```
H = - SUM[p > 0] p * log2(p)
```

  Zero-probability bins are skipped (the `0*log(0)=0`
  convention).

- `computeLineEntropy` -- the rolling estimate. For each bar the
  trailing window of `period` values is binned into `bins`
  buckets and the Shannon entropy of that distribution is taken.
  `entropy` is the raw entropy in bits; `normalized` divides it
  by `log2(bins)` -- the entropy of a uniform distribution -- so
  it lands in `[0, 1]`. A normalized entropy near 1 marks a
  disordered (noisy) window, near 0 an ordered (concentrated)
  window. Bars before the window is full are null.

### Worked fixture

Shannon entropy is exact when every probability is a power of
`1/2`: `log2` of an exact power of two is an exact integer.

- A two-bin even split `[0.5, 0.5]` has entropy `1` bit.
- Four equal outcomes `[0.25, 0.25, 0.25, 0.25]` have entropy
  `2` bits.
- A flat series collapses every window into one bin, entropy
  `0` -- perfectly ordered.
- A strict ramp with `period 4` and `4` bins puts one value in
  each bin, entropy `2` bits, normalized `2/log2(4) = 1` --
  maximally disordered.
- A two-level series fills two bins evenly, entropy `1` bit,
  normalized `1/log2(4) = 0.5`.

## API

- `getLineEntropyFinitePoints(points)` -- drop points with a
  non-finite `x` or `value`.
- `normalizeLineEntropyPeriod(period, fallback)` -- coerce to an
  integer of at least 2.
- `normalizeLineEntropyBins(bins, fallback)` -- coerce to an
  integer of at least 2 (so `log2(bins)` stays above zero).
- `computeLineEntropyHistogram(window, bins)` -- the
  `{ counts, probabilities }` histogram of a window.
- `computeLineEntropyValue(probabilities)` -- the Shannon
  entropy of a distribution, in bits.
- `computeLineEntropy(values, period, bins)` -- the
  `{ entropy, normalized }` rolling series.
- `runLineEntropy(points, options?)` -- sort finite points by
  `x`, run the pipeline, return `{ series, period, bins,
  entropy, normalized, samples, normalizedFinal, normalizedMin,
  normalizedMax, disorderedCount, orderedCount, ok }`. Each
  sample is classified `disordered` / `ordered` / `neutral`.
  `ok` is false for fewer than two points.
- `computeLineEntropyLayout(options)` -- stack the price panel
  above an entropy panel with a fixed `[0, 1]` y-domain; project
  the price and entropy paths plus markers and the 0.5 reference
  line.
- `describeLineEntropyChart(data, options?)` -- ARIA
  description.

## Component

`ChartLineEntropy` is a `forwardRef` region (`role="region"`)
with an ARIA-linked description, a `role="img"` SVG, two stacked
panels (price on top, entropy below) with their own axis ticks
and grid, panel labels, a config badge showing the period and
bins, a two-series legend (Price / Entropy) with toggle, a
hover/focus tooltip exposing the value, raw entropy, normalized
entropy and class, and a `motion-safe:animate-fade-in` entrance.
It consumes `{ x, value }` points. The entropy panel has a fixed
`[0, 1]` y-domain with a dashed 0.5 reference line and
class-coloured markers (red disordered, green ordered, grey
neutral). Every node carries a `data-section` attribute.

## Distinctness

- **vs chart-line-hurst** -- both are fixed `[0, 1]` gauge
  panels with a 0.5 reference. The Hurst exponent measures the
  long-term memory of the series (rescaled range analysis); the
  Shannon entropy measures the disorder of the local amplitude
  distribution (histogram).
- **vs the oscillator panels (roofing / trendflex / reflex /
  pmo)** -- those draw a zero-centred oscillator. The entropy
  panel is a fixed unit-interval gauge.
- **vs the band primitives** -- those draw a band around the
  price. The entropy panel is a disorder gauge.
