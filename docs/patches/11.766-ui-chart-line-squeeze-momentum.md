# 11.766 chart-line-squeeze-momentum (v1.11.784)

## Summary

Pure-SVG dual-panel React/TypeScript chart primitive with a
**LazyBear Squeeze Momentum** oscillator panel beneath the
close. The oscillator is the linear regression of
`close - avg(avg(highest, lowest), sma(close))` over the
lookback window. Added as the `<ChartLineSqueezeMomentum />`
UI primitive in `web/src/components/ui/`.

## Algorithm

For each bar with lookback `n` (default 20):

```
midDonchian = (highest(high, n) + lowest(low, n)) / 2
smaClose    = sma(close, n)
base        = (midDonchian + smaClose) / 2
src[i]      = close[i] - base[i]
val[i]      = linreg(src[i - n + 1..i], n, 0)
```

`linreg(y, n, 0)` is the linear-regression value evaluated at
`x = n - 1` (the last bar of the window):

```
meanX = (n - 1) / 2
slope = sum((x - meanX) * (y - meanY)) / sum((x - meanX)^2)
val   = meanY + slope * (n - 1) / 2
```

Bars before `i = 2 * (n - 1)` are `null` (warmup: the linreg
window requires n consecutive `src` values, and `src` itself
needs an n-bar window).

## Anchors

- **CONST_FLAT (high = low = close = K)**: `midDonchian = K`,
  `smaClose = K`, `base = K`, so `src = K - K = 0` at every
  finite bar. Linear regression of zeros is zero, so `val = 0`
  bit-exact past the warmup window. Verified across
  `(K, length) in {0, 1, 5, 50, -10, 100} x {3, 5, 10, 20}`.
- **`linreg` of constants**: `linreg([K, K, ..., K]) = K`
  bit-exact (mean = K, slope = 0).
- **`linreg` of zeros**: `linreg([0, 0, ..., 0]) = 0` bit-exact.
- **`linreg` of perfectly linear ramp `[0, 1, 2, ..., n - 1]`**:
  returns `n - 1` bit-exact (matches the last point).
- **Translation invariance**: shifting `(high, low, close)` by
  the same constant `C` leaves the momentum unchanged (within
  numerical tolerance).
