# 11.819 UI: chart-line-atr-ratio primitive (v1.11.837)

## Summary
Pure-SVG dual-panel React/TS primitive plotting a **short-over-long
ATR ratio** as a volatility regime indicator. Ratio > 1 signals
expanding volatility (recent moves are larger than the long-run
average); ratio < 1 signals contracting volatility.

## Formula
```
tr[i]       = max(high - low, |high - prevClose|, |low - prevClose|)
shortAtr[i] = SMA(tr, shortLength)[i]
longAtr[i]  = SMA(tr, longLength)[i]
ratio[i]    = longAtr[i] > 0 ? shortAtr[i] / longAtr[i] : null
```
`ratio[i]` is `null` during warmup or when the long ATR is zero
(divide-by-zero guard).

## Defaults
- `shortLength = 5`, `longLength = 20`
- `highThreshold = 1.2`, `lowThreshold = 0.8`
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `expanding`: `ratio >= highThreshold`
- `contracting`: `ratio < lowThreshold`
- `normal`: in between
- `flat`: `ratio == 0`
- `none`: warmup / non-finite
- `crossed`: `'up'` (enter expansion), `'down'` (enter contraction).
  Markers fire only on threshold transitions.

## Bit-exact testability anchors
- **CONST h=l=close=K**: `tr = 0` -> both ATRs = 0 -> divide-by-zero
  guard -> `ratio = null` everywhere. Verified across `K in {0, 1, 5,
  100, -3}` and `(shortLength, longLength) in {(3, 5), (5, 8)}`.
- **LINEAR close=k+1 with high=low=close**: `tr[0] = 0`, `tr[i >= 1] =
  1`. Once `tr[0]` falls out of the longer window (at `i = longLength`)
  both ATRs evaluate to `1` exactly, so `ratio = 1` (integer-exact).
  Verified for `(shortLength, longLength) in {(3, 5), (5, 8), (5,
  10)}`.

## Design notes
- The LINEAR anchor is intentionally simple: with `tr` saturating at
  `1` everywhere post-warmup, both ATRs converge to the same `1`,
  making the ratio exactly `1`. The "post-warmup" qualifier matters:
  `tr[0] = 0` slightly drags the longer ATR during initial bars, so
  the anchor only holds at `i >= longLength`.
- Ratio y-axis starts at `0` (ATR ratios are non-negative) and
  includes `max(highThreshold * 1.25, 1.5)` so the high-zone reference
  stays visible.
- Midline at `ratio = 1` drawn as a dashed reference (different dash
  pattern from thresholds).
- Default colors: ratio line `#ea580c` (orange), thresholds `#9333ea`
  (purple), midline `#475569`.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
