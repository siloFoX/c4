# 11.939 -- UI: chart-line-trix-zero-cross primitive (v1.11.957)

## Summary

Adds `ChartLineTrixZeroCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and the close-only
TRIX line in the bottom panel. Marks bullish (cross up
through zero) / bearish (cross down through zero) triple-
smoothed momentum baseline regime transition events. Zero-
line cross variant of the TRIX family that flags the discrete
TRIX crossing of the zero baseline.

## Pipeline

For close-only data with default `length = 15`:

```
ema1[i]  = EMA(close, length)
ema2[i]  = EMA(ema1, length)
ema3[i]  = EMA(ema2, length)
trix[i]  = ema3[i-1] != 0
             ? (ema3[i] - ema3[i-1]) / |ema3[i-1]| * 100
             : 0
bullish  : prev trix <= 0 && cur trix > 0
bearish  : prev trix >= 0 && cur trix < 0
```

Total warmup is `3 * length - 2` = 43 bars by default.

## Bit-exact anchor

- CONST `close = K` for any finite K -> ema1 = ema2 = ema3 =
  K every bar -> `ema3[i] - ema3[i-1] = 0` -> trix = 0
  (even when K = 0, the ema3 = 0 guard returns 0 rather than
  NaN). trix = 0 sits on the threshold but the strict-
  inequality detector never fires. regime `bullish` (trix >=
  0). cross count = 0. Verified across K in {0, 1, 42, 100,
  1234}.
- LINEAR UP `close = i + 100` -> at steady state each EMA
  adds `(length - 1) / 2` units of lag so ema3 increases by
  ~1 per bar. trix ~ 100 / ema3 -> positive but decreasing
  toward 0 as i grows. regime `bullish`, 0 crosses.
- LINEAR DOWN `close = 200 - i` -> ema3 decreases by ~1 per
  bar -> trix ~ -100 / |ema3| -> negative. regime `bearish`,
  0 crosses.
- Decline followed by sustained rise -> TRIX transitions
  from negative to positive -> 1 bullish zero-line cross.
- Rise followed by sustained decline -> 1 bearish cross.

## Regime classifier

| trix range            | regime      |
|-----------------------|-------------|
| trix >= 0             | bullish     |
| trix < 0              | bearish     |
| null                  | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- One horizontal reference band inside the oscillator panel
  (threshold at zero, the baseline).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind.
- Oscillator range auto-fits to observed TRIX values with
  10% padding above and below; TRIX is unbounded.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.

## Defaults

- length = 15
- threshold = 0
- oscillator range = auto-fit (TRIX is unbounded)
- width 720, height 460, padding 44, panelGap 12
