## Summary

`ChartLineDonchianWidth` is a pure-SVG dual-panel React primitive
companion to `<ChartLineDonchianMid>` (TODO 11.784). It plots the
close on the top panel and the Donchian channel width on the bottom
panel:

```
upper[i] = max(high, [i - length + 1, i])
lower[i] = min(low,  [i - length + 1, i])
width[i] = upper[i] - lower[i]     (>= 0)
```

Defaults: `length = 20`. Bars `0 .. length - 2` are warmup
(`width = null`).

## Bit-exact anchors

- **CONST_FLAT** (`high = low = close = K`): every window's max and
  min equal `K`, so `width = K - K = 0` bit-exact past warmup. The
  integration test sweeps `K` across `{0, 1, 5, 100, -3}`.
- **CONST_BAR** (constant `high = H`, `low = L`): `upper = H`,
  `lower = L`, so `width = H - L` -- an integer-friendly anchor past
  warmup. The integration test sweeps `(H, L)` across several pairs
  (including negative L) and varies `length`.
- Ramped highs / lows (`high[i] = i + 2`, `low[i] = i - 2`, length 5):
  `max(high, [i-4..i]) = i + 2`, `min(low, [i-4..i]) = i - 6`, so
  `width = (i + 2) - (i - 6) = 8` constant. Integer-arithmetic check.

## Zone classifier

The bottom-panel reading is bucketed by ratio to the maximum width
observed in the current series:

- `wide` -- width >= 0.75 * widthMaxSeen
- `normal` -- 0.25 * widthMaxSeen <= width < 0.75 * widthMaxSeen
- `narrow` -- 0 < width < 0.25 * widthMaxSeen
- `flat` -- width == 0 (singular)
- `none` -- width is null (warmup)

When `widthMaxSeen` is zero (the whole series is flat) the classifier
falls back to `normal` for non-zero widths and `flat` for zero -- this
keeps the bucket buckets defined under the singular case.
