## Summary

New `ChartLineKlinger` UI primitive at
`web/src/components/ui/chart-line-klinger.tsx` -- a pure-SVG line
chart with a Klinger Volume Oscillator panel and signal line. The top
panel renders the close price line; the bottom panel renders the
oscillator line and a dashed signal line on a symmetric y-axis with a
dashed zero centerline and sign-coloured markers.

## Indicator math

Stephen Klinger's Volume Oscillator, with a fast period (default 34),
a slow period (default 55) and a signal period (default 13):

```
hlc[i]   = (high + low + close) / 3
trend[i] = hlc[i] > hlc[i-1] ? +1 : -1     (trend[0] seeded +1)
dm[i]    = high - low                      (daily measurement)
cm[i]    = trend unchanged ? cm[i-1] + dm[i] : dm[i-1] + dm[i]
vf[i]    = volume * |2 * (dm/cm - 1)| * trend * 100     (vf[0] = 0)
KVO      = EMA(vf, fastPeriod) - EMA(vf, slowPeriod)
signal   = EMA(KVO, signalPeriod)
```

`computeLineKlingerVf` builds the volume force; the cumulative
measurement resets to `dm[i-1] + dm[i]` whenever the trend flips.
`computeLineKlingerEma` is an SMA-seeded EMA. The oscillator -- the
fast EMA minus the slow EMA of the volume force -- swings around
zero, and the signal line is itself an EMA of the oscillator.

### Worked fixture

The typical price alternates direction so the trend flips every bar
and the daily/cumulative ratio settles at `0.5`; `dm` is a constant
`6`; `fast = 1`, `slow = 3`, `signal = 3`:

```
hlc   = [13,7,19,5,23,3]   trend = [1,-1,1,-1,1,-1]   dm = [6,...]
cm    = [6,12,12,12,12,12]
vf    = volume * trend * 100 = [0,-100,400,-100,200,-100]
kvo   = EMA(vf,1) - EMA(vf,3) = [.,.,300,-100,100,-100]
signal = EMA(kvo,3) = [.,.,.,.,100,0]
```

Two readings above zero and two below.

## API

- `getLineKlingerFinitePoints(points)` -- drop points with a
  non-finite `x`, `high`, `low`, `close` or `volume`.
- `normalizeLineKlingerPeriod(period, fallback)` -- positive
  integer; non-finite / sub-1 falls back, fractional floors.
- `computeLineKlingerVf(highs, lows, closes, volumes)` -- returns
  `{ hlc, trend, dm, cm, vf }`.
- `computeLineKlingerEma(src, period)` -- an SMA-seeded EMA.
- `computeLineKlinger(highs, lows, closes, volumes, fastPeriod,
  slowPeriod, signalPeriod)` -- returns `{ vf, fastEma, slowEma, kvo,
  signal }`.
- `runLineKlinger(points, options?)` -- sort finite points by `x`,
  compute the oscillator, return `{ series, fastPeriod, slowPeriod,
  signalPeriod, vf, kvo, signal, samples, kvoFinal, signalFinal,
  kvoMin, kvoMax, positiveCount, negativeCount, ok }`. Each sample
  carries a `positive` / `negative` / `zero` sign. `ok` is false for
  fewer than two points.
- `computeLineKlingerLayout(options)` -- stack a price panel above an
  oscillator panel with a symmetric y-bound. Returns panels, ticks,
  the price path, the oscillator path, the signal path, sign-coloured
  markers, and the zero centerline `y`.
- `describeLineKlingerChart(data, options?)` -- ARIA description.

## Component

`ChartLineKlinger` is a `forwardRef` region (`role="region"`) with an
ARIA-linked description, a `role="img"` SVG, panel labels, a config
badge (EMA and signal periods), a three-series legend (Price / KVO /
Signal) with toggle, a dashed zero centerline, the oscillator line, a
dashed signal line, markers coloured by sign, a hover/focus tooltip
exposing the close, volume, volume force, oscillator, signal, and
sign, and a `motion-safe:animate-fade-in` entrance. Every node
carries a `data-section` attribute.

## Distinctness

- **vs chart-line-chaikin-osc** -- the Chaikin Oscillator is the MACD
  of the *cumulative* Accumulation/Distribution Line. The Klinger
  builds a different base series first: a *volume force* that
  combines a trend-resetting cumulative measurement with the daily
  range and volume. Only then does it take the EMA difference.
- **vs chart-line-macd / chart-line-ppo** -- those take the EMA
  difference of the *price*. The Klinger takes it of the volume
  force, so it tracks volume conviction rather than price momentum.
- **vs chart-line-obv / chart-line-adl** -- those are cumulative
  volume totals. The Klinger is an unbounded EMA-difference
  oscillator with a signal line.
