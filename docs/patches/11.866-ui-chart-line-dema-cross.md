## Summary

Added `<ChartLineDemaCross />` -- a pure-SVG, dual-panel React/TS
primitive that overlays the close with the Double Exponential
Moving Average (DEMA) in the top panel and renders the
`close - DEMA` deviation in the bottom panel. Markers fire at
every close-vs-DEMA crossover -- the canonical double-smoothed
trend regime event.

Cross events:

- `up` cross   -> regime `trending-up`
- `down` cross -> regime `trending-down`

Defaults: `length = 14`, demaColor `#7c3aed` (violet),
deviationColor `#0ea5e9` (sky), bullish marker `#16a34a`,
bearish marker `#dc2626`, zero line `#475569`.

## Formula

```
ema1[i] = EMA(close, length)
ema2[i] = EMA(ema1,  length)
DEMA[i] = 2 * ema1[i] - ema2[i]
```

The double-smoothing followed by the `2x - x` lag-cancellation
produces a tighter-tracking trend line than a plain EMA while
preserving smoothness. Cross detection uses the standard
prev-relation + strict-inequality guard.

## Bit-exact anchor

- **CONST close = K**: every EMA in the chain collapses to K
  via the SMA-seeded `min === max` precision fix. `DEMA = 2K -
  K = K`, deviation = 0, relation `equal` forever, regime
  `aligned`, zero crosses. Verified across multiple K and length
  tuples.
