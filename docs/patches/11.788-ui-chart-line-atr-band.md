## Summary

`ChartLineAtrBand` is a pure-SVG single-panel React primitive that
overlays an ATR-based band envelope on the close. The midline is an
SMA of the close, with bands offset by `multiplier * ATR`:

```
middle[i] = SMA(close, length)[i]
TR[i]     = max(high - low, |high - prevClose|, |low - prevClose|)
ATR[i]    = SMA(TR, length)[i]
upper[i]  = middle[i] + multiplier * ATR[i]
lower[i]  = middle[i] - multiplier * ATR[i]
```

Defaults: `length = 20`, `multiplier = 2`. Bars before `i = length - 1`
are warmup nulls.

The component renders close, middle, upper, and lower as four series
on the same Y-axis, with an optional filled polygon between upper and
lower. Each series can be hidden via the legend.

## Bit-exact anchors

- **CONST_FLAT** (`high = low = close = K`): every TR is zero, ATR is
  zero, both SMAs collapse to `K`, and `upper = lower = middle = K`
  exactly past warmup. The integration test sweeps `K` across
  positive, zero, and negative values.

- **CONST_BAR with close at midpoint** (constant `high = K + r`,
  `low = K - r`, `close = K`):
  - Every TR is `max(2r, r, r) = 2r` (every bar after bar 0 has
    `prevC = K`, so `|H - K| = r` and `|L - K| = r`).
  - ATR is `2r` (SMA of constant `2r`).
  - `middle = SMA(K) = K`.
  - Therefore `upper = K + multiplier * 2r = K + 2*multiplier*r` and
    `lower = K - 2*multiplier*r` -- an integer-friendly anchor past
    warmup. The integration sweep verifies this across `(K, r, mult,
    length)` combos.

## Zone classifier

- `breakout-up` -- close >= upper
- `above` -- middle < close < upper
- `at` -- close == middle
- `below` -- lower < close < middle
- `breakout-down` -- close <= lower
- `none` -- any of middle / upper / lower is null

For the singular `upper == lower` (zero-width band) case the
classifier collapses to the above / at / below comparison so a
CONST_FLAT bar reads `at` (close = middle) rather than firing both
`breakout-up` and `breakout-down`.
