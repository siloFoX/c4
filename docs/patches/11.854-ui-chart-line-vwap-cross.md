## Summary

Added `<ChartLineVwapCross />` -- a pure-SVG dual-panel React/TS
primitive that overlays the anchored VWAP on the close in the top
panel and renders the `close - VWAP` deviation in the bottom
panel. Markers fire whenever close crosses VWAP -- the canonical
session bias transition. Anchor reset is supported via a per-point
`anchor` flag: when `anchor === true`, the cumulative
`sum(typical * volume)` and `sum(volume)` reset at that bar so the
next session starts fresh.

Cross events:

- `up`   -> regime `bullish` (close newly exceeds VWAP)
- `down` -> regime `bearish` (close newly drops below VWAP)

Anchor reset bars never emit a cross (the prev relation is
discarded across a reset), and the renderer breaks the VWAP /
deviation paths into separate session subpaths.

## Formula

```
typical[i] = (high[i] + low[i] + close[i]) / 3
anchor[i] = true  -> reset cumulative sums at i
VWAP[i]   = sum(typical*volume)[from last anchor .. i] /
            sum(volume)[from last anchor .. i]
deviation = close - VWAP
```

## Bit-exact anchors

- **CONST h = l = close = K, volume = V**: typical = K, VWAP = K
  on every bar. `deviation = 0` forever, relation `equal`, zero
  crosses.
- **LINEAR UP h = l = close = i+1, volume = V**: typical = i+1,
  `VWAP[i] = (i + 2) / 2`. `deviation = i / 2`. Bar 0 has
  `close == VWAP` (no cross because prev is null), and bar 1
  strictly crosses upward (prev equal -> cur bullish), so exactly
  one `up` event fires. Bars 2+ stay bullish.
- **LINEAR DOWN h = l = close = N-i, volume = V**: mirror image.
  Exactly one `down` cross fires at i=1.
- **Multi-session**: two LINEAR UP segments stitched with
  `anchor = true` on the first bar of the second segment yield
  exactly two `up` crosses (one per session, suppressed on the
  reset bar itself).

The tests assert each `deviation[i]` value bit-exactly (the i=0
case lands at +0 thanks to the `posZero` normalizer).
