## Summary

New `ChartLineSupertrend` UI primitive at
`web/src/components/ui/chart-line-supertrend.tsx` -- a pure-SVG line
chart with a SuperTrend ATR-based trailing-stop band that flips with
trend direction. The price line and the SuperTrend line share one
panel; the SuperTrend renders as colour-coded segments (green support
below price in an uptrend, red resistance above price in a downtrend)
with amber flip markers at the bars where the trend reverses.

## Indicator math

### True range and ATR

`computeLineSupertrendTrueRanges(values)` is the absolute period-over-
period change `|v[i] - v[i-1]|` (index 0 null).
`computeLineSupertrendATR(values, period)` is Welles Wilder's ATR:
simple-mean seed at index `period`, then Wilder smoothing
`(prev*(period-1) + tr)/period`.

### SuperTrend

`computeLineSupertrend(values, period, multiplier)` builds the
indicator. The ATR-scaled basic bands are:

```
basicUpper = value + multiplier * ATR
basicLower = value - multiplier * ATR
```

The final bands trail -- the upper band only ratchets down, the lower
band only ratchets up, unless price breaks through:

```
finalUpper[i] = (basicUpper[i] < finalUpper[i-1] || value[i-1] > finalUpper[i-1])
                  ? basicUpper[i] : finalUpper[i-1]
finalLower[i] = (basicLower[i] > finalLower[i-1] || value[i-1] < finalLower[i-1])
                  ? basicLower[i] : finalLower[i-1]
```

The SuperTrend line is the final lower band in an uptrend (support
below price) and the final upper band in a downtrend (resistance above
price); the trend flips side whenever price crosses the active band.

### Worked fixture

`values = [10, 11, 13, 12, 9, 8]`, `period = 2`, `multiplier = 1`:

```
true range  = [.,1,2,1,3,1]
ATR         = [.,.,1.5,   1.25,  2.125,  1.5625]
basicUpper  = [.,.,14.5,  13.25, 11.125, 9.5625]
basicLower  = [.,.,11.5,  10.75, 6.875,  6.4375]
finalUpper  = [.,.,14.5,  13.25, 11.125, 9.5625]
finalLower  = [.,.,11.5,  11.5,  11.5,   6.4375]
supertrend  = [.,.,11.5,  11.5,  11.125, 9.5625]
direction   = [.,.,up,    up,    down,   down]
flip        = [F, F, F,   F,     T,      F]
```

The trend opens up (value rises 11 -> 13) and the line trails the
lower band; at index 4 price 9 breaks below the lower band 11.5, the
trend flips to down and the line jumps to the upper band.
`supertrendFinal = 9.5625`, `directionFinal = down`, `flipCount = 1`.

## API

- `getLineSupertrendFinitePoints(points)` -- drop points with a
  non-finite `x` or `value`.
- `normalizeLineSupertrendPeriod(period, fallback)` -- positive
  integer; non-finite / sub-1 falls back, fractional floors.
- `normalizeLineSupertrendMultiplier(multiplier, fallback)` --
  positive number; non-finite / non-positive falls back, fractional
  multipliers kept as-is.
- `computeLineSupertrendTrueRanges(values)` -- per-period true range.
- `computeLineSupertrendATR(values, period)` -- Wilder ATR.
- `computeLineSupertrend(values, period, multiplier)` -- the core
  `{ atr, basicUpper, basicLower, finalUpper, finalLower, supertrend,
  direction, flip }` arrays.
- `runLineSupertrend(points, options?)` -- sort finite points by `x`,
  compute the core, return `{ series, period, multiplier, samples,
  supertrendFinal, directionFinal, flipCount, ok }`. `ok` is false for
  fewer than two points.
- `computeLineSupertrendLayout(options)` -- project the price line,
  split the SuperTrend into one segment per contiguous trend run, emit
  per-point markers (flip points flagged). Returns the panel, ticks,
  the price path, segments, and markers.
- `describeLineSupertrendChart(data, options?)` -- ARIA description.

## Component

`ChartLineSupertrend` is a `forwardRef` region (`role="region"`) with
an ARIA-linked description, a `role="img"` SVG, a config badge (period
+ multiplier), a three-series legend (Price / Uptrend / Downtrend)
with toggle, colour-coded SuperTrend segments, per-point markers
(amber for flip bars), a hover tooltip exposing the value, ATR,
SuperTrend, and direction, and a `motion-safe:animate-fade-in`
entrance. Every node carries a `data-section` attribute.

## Distinctness

- **vs chart-line-keltner** -- Keltner draws two persistent ATR bands
  around an EMA centre as a static channel. The SuperTrend is a single
  line that is only ever one band at a time and jumps sides at a flip.
- **vs chart-line-parabolic-sar** -- the Parabolic SAR trailing stop
  follows a parabolic acceleration formula; the SuperTrend stop is set
  by ATR-scaled bands.
- **vs chart-line-atr** -- chart-line-atr plots the raw ATR in a
  sub-panel; chart-line-supertrend uses the ATR only to size the
  trailing band and overlays the resulting line on the price.
