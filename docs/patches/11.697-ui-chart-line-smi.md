## Summary

New `ChartLineSmi` UI primitive in
`web/src/components/ui/chart-line-smi.tsx` -- a pure-SVG two-panel line
chart with a **Stochastic Momentum Index** panel from the double-smoothed
distance to the range midpoint. The top panel plots the price; the bottom
panel plots the SMI.

## The computation

The Stochastic Momentum Index (William Blau) refines the stochastic
oscillator by measuring the close against the **midpoint** of the recent
high-low range rather than its low, and double-smoothing the result. It
needs high, low and close per bar, so `ChartLineSmiPoint` carries
`{x, high, low, close}`.

- `computeLineSmiRaw(bars, qPeriod)` -- per bar, the distance
  `D = close - (highestHigh + lowestLow) / 2` over the trailing window,
  and the range `highestHigh - lowestLow`.
- `computeLineSmiEma(values, period)` -- an EMA seeded from the first
  finite value.
- `computeLineSmi(bars, qPeriod, smoothPeriod1, smoothPeriod2)` -- D and
  the range are each double-smoothed by two EMAs, then
  `SMI = 100 * emaD2 / (emaR2 / 2)`. A zero double-smoothed range yields
  null.

The SMI runs around zero in roughly [-100, 100] -- positive when the close
leads the range midpoint. Each defined bar is classified against the
thresholds: **overbought** above the upper, **oversold** below the lower,
**neutral** between.

## Worked fixture

The double EMA is transcendental in general, but two conditions make the
fixture exact: hold the **range constant** and pick **smoothing period 3**
so the EMA alpha is exactly `2/4 = 0.5`.

The fixture is 10 bars that all share high 30 and low 10, so over any
3-bar window the range is the constant 20 and the midpoint is the constant
20. The close steps `20 -> 28 -> 12`, so `D = close - 20` is
`[.,.,8,8,8,8,-8,-8,-8,-8]`. With alpha 0.5 the double EMA stays dyadic,
and the range being constant collapses `SMI` to `10 * emaD2` -- the whole
series is exact: `[.,.,80,80,80,80,40,0,-30,-50]`, asserted with
`toEqual`. The zones cover overbought, neutral and oversold. The EMA
helper is checked with a dyadic alpha (`[4,8,16]` -> `[4,6,11]`), and a
constant series with the close at the high pins the SMI at 100. A flat bar
(high equals low) gives a zero range and a null SMI.

## API

`data` (a `ChartLineSmiPoint[]`), `qPeriod` (default 10), `smoothPeriod1`
/ `smoothPeriod2` (default 3), `upperThreshold` (default 40),
`lowerThreshold` (default -40), `pricePanelRatio` / `gap`,
sizing/format/a11y props, `strokeWidth` / `dotRadius`, the colour props,
the toggle flags (`showAxis` / `showGrid` / `showDots` default false /
`showSmi` / `showZeroLine` / `showThresholds` / `showMarkers` /
`showTooltip` / `showConfigBadge` / `showLegend` / `animate`), controlled +
uncontrolled `hiddenSeries` (keys `price` / `smi`), `formatValue` /
`formatX`, `onPointClick` / `onSeriesToggle`.

Exported helpers: `getLineSmiFinitePoints`, `normalizeLineSmiPeriod`,
`computeLineSmiEma`, `computeLineSmiRaw`, `computeLineSmi`,
`classifyLineSmiZone`, `runLineSmi`, `computeLineSmiLayout`,
`describeLineSmiChart`, plus the `DEFAULT_CHART_LINE_SMI_*` constants and
the `ChartLineSmiZone` type. Every exported name carries a `LineSmi` infix
so the `export *` barrel line cannot collide.

## Accessibility

Root `<div role="region">` with `aria-label` + `aria-describedby`; SVG
`role="img"`; an off-screen `data-section="chart-line-smi-aria-desc"`
description span; the price line, the SMI line and every dot / marker are
`role="graphics-symbol" tabIndex={0}`. Every node carries a
`data-section`; the root mirrors `data-q-period`, `data-smooth-period1`,
`data-smooth-period2`, `data-upper-threshold`, `data-lower-threshold`,
`data-smi-final`, `data-overbought-count`, `data-oversold-count`,
`data-neutral-count`, `data-total-points` and `data-animate`.
