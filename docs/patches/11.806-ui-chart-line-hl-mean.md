# 11.806 UI: chart-line-hl-mean primitive (v1.11.824)

## Summary
Pure-SVG **single-panel** React/TS primitive that overlays a "High Low
Mean" line on the close. The mean is the rolling SMA of each bar's
midpoint `(high + low) / 2` across the lookback. The close and the
mean share the same y-axis; markers fire on close-vs-mean crosses.

## Formula
```
midpoint[i] = (high[i] + low[i]) / 2
hlMean[i]   = SMA(midpoint, length)[i]
zone[i]     = close[i] > hlMean[i] ? 'above'
            : close[i] < hlMean[i] ? 'below'
            : 'at'
```
`hlMean[i]` is `null` during warmup (`i < length - 1`) or when any
input value is non-finite.

## Defaults
- `length = 14`
- `width = 720`, `height = 380`, `padding = 44`

## Zones and crosses
- `above`: `close > hlMean`
- `below`: `close < hlMean`
- `at`: `close == hlMean`
- `none`: warmup / non-finite
- `crossed`: `'up'` when prev zone was `below`/`at` and current is
  `above`; `'down'` is the mirror. Markers fire only on cross bars and
  are anchored to the close on the cross bar.

## Bit-exact testability anchors
- **CONST high=low=close=K**: midpoint = K, SMA = K, `close == hlMean`
  -> zone `'at'` at every valid bar with no crosses. Verified across
  `K in {0, 1, 5, 100, -3}` and `length in {3, 4, 7, 10}`.
- **ASYMMETRIC CONST high=10, low=2, close=8**: midpoint = 6 every bar,
  SMA = 6 (any length), close=8 > 6 -> zone `'above'` at every valid
  bar. Mirror with close=4 -> zone `'below'`. Both bit-exact.

## Design notes
- Single panel keeps `close` and `hlMean` on the same y-axis: the user
  can read the relationship directly without re-mapping the scale.
- Marker `cy` anchors to the close on the cross bar (not the mean) so
  the dot is plotted on the price line, matching the visual semantics
  of a "price crossing the mean."
- Default colors: price `#2563eb` (blue), mean `#f97316` (orange),
  bullish `#16a34a` (green), bearish `#dc2626` (red).
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` /
  `?? 'none'` and defensive guards.
