# Patch 11.835 -- ChartLineMomentumPct primitive (v1.11.853)

## Summary

Add `<ChartLineMomentumPct />`, a pure-SVG dual-panel line chart
with a Momentum-Percent oscillator beneath a price line. Computes
the rate of change of close over the lookback, expressed as a
percentage of the prior close:

```
momentum[i] = close[i - length] === 0
                ? null
                : (close[i] - close[i - length])
                    / close[i - length] * 100
```

- Warmup: `i < length` -> `null`.
- `null` whenever `close[i - length]` is zero (divide-by-zero guard).
- `-0` normalized to `+0` on every arithmetic result.

This is algebraically the standard Rate-of-Change indicator,
re-skinned with a momentum-style naming to match
`chart-line-momentum-divergence` (11.807) which compares the
absolute-units Momentum and percentage-units ROC.

## Defaults

| Knob              | Value     |
|-------------------|-----------|
| length            | 10        |
| bullishThreshold  | 0         |
| bearishThreshold  | 0         |
| momentumColor     | `#ec4899` (pink, distinct from prior pink/magenta primitives) |

## Bit-exact anchors

- `CONST close = K`, `K != 0`: `close[i] - close[i - L] = 0`, so
  `momentum = 0 / K * 100 = 0` bit-exact post-warmup. Verified
  across `(K, length)` sweeps.
- `CONST close = 0`: divide-by-zero guard returns `null`.
- `DOUBLED step`: `close = K` for `i < L`, `close = 2K` for
  `i >= L`. For `L <= i < 2L` the lookback bar is `K` and the
  current bar is `2K`, giving
  `momentum = (2K - K) / K * 100 = 100` bit-exact for any nonzero
  `K`. For `i >= 2L` both bars sit in the second segment, so the
  delta is 0 and `momentum = 0`. Verified across `(K, length)`
  sweeps.
- `LINEAR UP close = i + 1` at `i = L`: `close[L] - close[0] = L`,
  `close[0] = 1`, so `momentum = L / 1 * 100 = L * 100` (bit-exact,
  integer arithmetic). Used as a concrete-value spot-check at
  `L = 4` -> 400.

## Surface

Standard primitive surface:

- `ChartLineMomentumPctPoint = { x, close }`
- `runLineMomentumPct(data, options?) -> Run` with
  `ok = series.length > length`.
- `computeLineMomentumPctLayout({ data, width?, height?, padding?, panelGap?, length?, bullishThreshold?, bearishThreshold? }) -> Layout`.
  Momentum axis auto-fits but always includes 0.
- `describeLineMomentumPctChart(data, opts?) -> string` for the ARIA
  description.
- `<ChartLineMomentumPct />` `forwardRef<HTMLDivElement>` with legend
  toggle (`price`, `momentum`), config badge, optional threshold
  band + zero-line overlay, markers on cross events, and a tooltip
  surfacing `close / prior / delta / momentum / zone / cross`.
