# Patch 11.754 -- UI: chart-line-adx-di-minus primitive

Released as `v1.11.772` on 2026-05-26.

## Scope

Add `<ChartLineAdxDiMinus>` -- a pure-SVG two-panel chart with
the Wilder **-DI** directional indicator (SMA-smoothed variant)
reading the smoothed negative directional movement over the
true range, scaled to `100`. Mirror of the v1.11.771 +DI
primitive.

## Definition

Inputs:

- `data: ChartLineAdxDiMinusPoint[]` where
  `ChartLineAdxDiMinusPoint = { x; high; low; close }`. Bars
  with `high < low` are dropped.
- `period: number` -- the SMA length. Default `14`. Normalized
  to an integer `>= 2`.
- `threshold: number` -- the bear boundary; strong starts at
  `2 * threshold`. Default `25`. Normalized to a finite in
  `(0, 100]`.

Algorithm (per bar `i >= 1`):

```
upMove   = high[i] - high[i - 1]
downMove = low[i - 1] - low[i]
minusDM  = (downMove > upMove && downMove > 0) ? downMove : 0
TR       = max(high - low, abs(high - prevClose),
               abs(low  - prevClose))

minusS = SMA(minusDM, period)
trS    = SMA(TR, period)

-DI[i] = 100 * minusS / trS
```

A bar is null when `trS == 0`.

## Bit-exact algebraic anchors

**CONST_FLAT** (`high == low == close == K`): `TR = 0` -> bar
null.

**RISING** (`high == low == close == i + 10`): `upMove = +1,
downMove = -1` -> `minusDM = 0`. `TR = 1`. `minusS = 0, trS = 1`
-> `-DI = 0` bit-exact.

**FALLING** (`high == low == close == 19 - i`): `upMove = -1,
downMove = +1` -> `minusDM = 1`. `TR = 1`. `minusS = 1, trS = 1`
-> `-DI = 100` bit-exact.

**FALLING_HALF** (`low == close == 19 - i, high == 21 - i`):
`upMove = (21 - i) - (21 - (i - 1)) = -1, downMove = (19 - (i -
1)) - (19 - i) = 1` -> `minusDM = 1`. `TR`: `hl = 2, hc = |21 -
i - (19 - (i - 1))| = 1, lc = |19 - i - (19 - (i - 1))| = 1`.
`TR = max(2, 1, 1) = 2`. `minusS = 1, trS = 2` -> `-DI = 100 *
1 / 2 = 50` bit-exact.

**Translation invariance**: shifting OHLC by `+k` leaves all
diffs unchanged -> -DI bit-exact unchanged.

## Component

`forwardRef<HTMLDivElement, ChartLineAdxDiMinusProps>` with
displayName. Renders top panel (close line) and bottom panel
(-DI line + zone-coloured markers + zero reference line +
dashed `threshold` and `2 * threshold` reference lines),
`arps` design-system tokens, ARIA region / img / graphics-
symbol roles, and `data-section` hooks throughout.
