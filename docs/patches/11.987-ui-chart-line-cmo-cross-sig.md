## Summary

Add `web/src/components/ui/chart-line-cmo-cross-sig.tsx` -- a
pure-SVG dual-panel React/TypeScript chart primitive that
renders the close in the top panel and the close-only Chande
Momentum Oscillator (CMO) plus its smoothed signal line (SMA of
CMO) in the bottom panel, marking discrete CMO-over-signal-line
crossover events. Signal-line variant of the CMO family that
flags the discrete CMO crossing of its own SMA-smoothed signal
line -- the canonical trigger analysts use to filter raw-CMO
whipsaws and confirm momentum shifts.

## Math anchor (bit-exact)

```
gainSum[i] = sum of positive close deltas in [i-length+1..i]
lossSum[i] = sum of absolute negative close deltas in window
cmo[i]     = 100 * (gainSum - lossSum) / (gainSum + lossSum);
              0 when both sums collapse to 0 (degenerate)
signal[i]  = SMA(cmo, kSmoothing)
bullish    : prev cmo <= prev signal && cur cmo >  cur signal
bearish    : prev cmo >= prev signal && cur cmo <  cur signal
```

- **CONST close = K**: deltas = 0 -> gainSum = lossSum = 0 ->
  cmo = 0 (degenerate). signal = SMA(0, 3) = 0 via the SMA
  `min === max` short-circuit. cmo == signal == 0 -> regime
  `bullish` (cmo >= signal at boundary). 0 crosses. Verified
  across K = 0..1234.
- **LINEAR UP close = i**: deltas = +1, gainSum = length,
  lossSum = 0 -> cmo = 100. signal = SMA(100, 3) = 100. cmo
  == signal == 100 -> regime `bullish`. 0 crosses.
- **LINEAR DOWN close = -i**: deltas = -1, gainSum = 0, lossSum
  = length -> cmo = -100. signal = -100. cmo == signal == -100
  -> regime `bullish` (CMO sits on its own SMA, so by the
  boundary-inclusive classifier it reads bullish even at the
  saturated bear extreme). 0 crosses.

For 50 bars at defaults (length=14, kSmoothing=3):
- cmo null for i = 0..13 (14 entries)
- signal null for i = 0..15 (16 entries)
- post-warmup regime count = 34 (i = 16..49)

Strict inequality on `cur` (`cur > signal` for bullish, `cur <
signal` for bearish) means boundary equality (cmo == signal)
never double-fires.

Layout uses fixed oscMin=-100 / oscMax=100 (CMO canonical
symmetric range) with the zero reference.

## Accessibility

- Wrapping `<div role="region" aria-label="CMO Signal Cross
  chart" aria-describedby="...">` with the sr-only description
  copy carrying the length / kSmoothing parameters plus a
  "smoothed-CMO trigger" framing.
- `<svg role="img" aria-labelledby="..." aria-describedby="..."
  tabIndex={0}>` for keyboard focus.
- Each cross marker (both the CMO-panel dot and the price-panel
  chevron overlay) carries `role="graphics-symbol"`,
  `tabIndex={0}`, and an `aria-label="<bullish|bearish> cross at
  <x>"` so screen readers can step through events.
- Legend buttons (3 series: price, cmo, signal) expose
  `aria-pressed` and toggle via mouse, focus, Enter, or Space.
