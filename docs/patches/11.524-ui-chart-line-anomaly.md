# 11.524 -- web/src/components/ui/chart-line-anomaly.tsx

Pure-SVG multi-series line chart that **detects statistical outliers
via z-score** and highlights them with larger coloured markers. For
each series the chart computes the mean and sample standard deviation
of finite y values, then z-scores every point and flags `|z| >=
threshold` (default `2`) as either `'high'` (z >= threshold) or
`'low'` (z <= -threshold). High and low anomalies use distinct
colours so the type of outlier reads at a glance. A faint horizontal
band marks the "normal range" `[mean - t*stddev, mean + t*stddev]`.

## Distinct from other anomaly / extreme primitives

- `<ChartLineMinMax>` (11.517): markers at the **data-derived
  extremes** (always two markers regardless of distribution).
  Anomaly highlights points whose **statistical distance** from the
  mean exceeds a threshold (could be 0 or many).
- `<ChartLinePercentile>` (11.514): percentile bands derived from
  **multiple samples per x**. Anomaly works on a single-sample-per-x
  series and uses parametric (Gaussian) thresholds.
- `<ChartLineMovingAvg>` (11.513): windowed smoothing. Anomaly
  catches spikes against a global mean.
- `<ChartLineConfidence>` (11.502): per-point bounds supplied
  externally. Anomaly derives bounds from the data.
- `<ChartLineTrend>` (11.512): regression line.
- `<ChartLineBand>` (11.516), `<ChartLineThreshold>` (11.503):
  external horizontal references.

The canonical "spike detection" / "outlier flag" / "alert when
metric crosses 3-sigma" visualisation.

## Public surface

- `<ChartLineAnomaly series={...} threshold={2} />`.
- Per-series `{id, label, data, color?, highAnomalyColor?,
  lowAnomalyColor?, threshold?}`. `data` entries are raw `{x, y}`.
- `threshold` is the z-score cutoff. Per-series override always
  wins; non-finite or `< 0` falls back to the chart-level default,
  and chart-level non-finite falls back to `2`.
- `|z| === threshold` is **inclusive** -- equal-to-threshold counts
  as an anomaly.
- Toggle flags: `showAxis`, `showGrid`, `showDots`, `showLegend`,
  `showTooltip`, `showBand`, `showMeanLine`, `showBoundLines`,
  `animate`.
- `meanDashArray` (`'4 3'`), `boundDashArray` (`'2 3'`), `meanColor`
  (`'#64748b'`), `bandColor` (`'#94a3b8'`).
- Standard sizing + format + a11y props.
- `formatValue` / `formatX` / `formatZ` (sign-aware fallback).
- Controlled + uncontrolled `hiddenSeries`.
- `onPointClick({series, point})`.

## Pure helpers

- `getLineAnomalyDefaultColor(index)` -- 10-color palette cycle.
- `getLineAnomalyFinitePoints(points)` -- finite-only filter.
- `computeLineAnomalyStats(points)` -- returns `{finiteCount, mean,
  stddev, ok}`. Uses **sample standard deviation** (Bessel
  correction: `sqrt(SS / (n - 1))`). `ok=true` only when `n >= 2`
  AND `stddev > 0` -- constant / single-sample series get `ok=false`
  (no meaningful z-score is computable).
- `computeLineAnomalyZScore(y, mean, stddev)` -- `(y - mean) /
  stddev`. Returns `0` on non-finite inputs or `stddev <= 0`.
- `classifyLineAnomalyDirection(zScore, threshold?)` -- returns
  `'high' | 'low' | 'normal'`. Inclusive at `|z| === threshold`.
  Non-finite z -> `'normal'`. Non-finite / negative threshold falls
  back to the default.
- `computeLineAnomalyLayout({series, threshold?, hiddenSeries?,
  showBand?, ...})` -- returns `{series, xTicks, yTicks, xMin, xMax,
  yMin, yMax, innerWidth, innerHeight, totalPoints, totalAnomalies,
  visibleSeriesCount}`. Each layout series carries `{id, label,
  index (stable original), color, highColor, lowColor, threshold,
  stats, points, path, meanY, upperY, lowerY, bandPath, bandValid,
  anomalyCount, highCount, lowCount, finiteCount, totalCount}`.
  Per-point `{index, x, y, px, py, zScore, direction, isAnomaly}`.
  Y range is expanded to include the upper/lower bounds when the
  band is shown.
- `describeLineAnomalyChart(series, threshold?, hidden?,
  formatValue?)` -- builds the screen-reader description with
  per-series mean, stddev, and anomaly counts split by direction.

## ARIA + test selectors

- Root `role="region"` + `aria-label` + `aria-describedby`.
- SVG `role="img"`.
- Line paths + dots are `role="graphics-symbol"` + `tabIndex={0}`;
  band rectangle is `role="graphics-symbol"`. Dot aria-label
  includes z and anomaly direction.
- Every node carries `data-section="chart-line-anomaly-..."` for
  test selectors.
- Series groups expose `data-series-mean`, `data-series-stddev`,
  `data-series-threshold`, `data-series-stats-ok`,
  `data-series-anomaly-count`, `data-series-high-count`,
  `data-series-low-count`, `data-series-finite-count`,
  `data-series-color`.
- Dots expose `data-z-score`, `data-direction`, `data-is-anomaly`.
- Root mirrors `data-threshold`, `data-total-anomalies`.

## Tooltip

Hovering a dot shows the series label, x, y, and a coloured `z:`
row indicating the z-score; appends ` (<direction> anomaly)` when
the point is an outlier.
