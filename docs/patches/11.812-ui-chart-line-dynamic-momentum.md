# 11.812 UI: chart-line-dynamic-momentum primitive (v1.11.830)

## Summary
Pure-SVG dual-panel React/TS primitive that plots a **Dynamic Momentum
Index** oscillator beneath the close. The lookback adapts per bar to
the ratio of short-window volatility over long-window volatility -- so
the indicator shortens during turbulent regimes and lengthens during
calm ones, then samples the close-to-close percent change over the
adaptive window.

## Formula
```
shortStd[i]  = popStdDev(close, shortVolLength)[i]
longStd[i]   = SMA(shortStd, longVolLength)[i]
ratio[i]     = longStd[i] > 0 ? shortStd[i] / longStd[i] : 1
dynLength[i] = clamp(round(baseLength / ratio[i]),
                    minLength, maxLength)
dmi[i]       = (close[i] - close[i - dynLength[i]])
              / close[i - dynLength[i]] * 100
```
`dmi[i]` is `null` during warmup (when either stdDev channel is
`null`, or when `i - dynLength[i] < 0`) and when
`close[i - dynLength[i]] === 0` (divide-by-zero guard).

## Defaults
- `baseLength = 14`
- `shortVolLength = 5`
- `longVolLength = 10`
- `minLength = 5`, `maxLength = 30`
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `positive`: `dmi > 0`
- `negative`: `dmi < 0`
- `zero`: `dmi == 0`
- `none`: warmup / non-finite
- `crossed`: `'up'` when prev was `<= 0` and current is `> 0`, mirror
  for `'down'`. Markers fire only on cross bars.

## Bit-exact testability anchors
- **CONST close** (`close = K`, `K != 0`): `shortStd = 0`,
  `longStd = 0` -> ratio falls back to `1`, `dynLength = baseLength`
  (clamped), and `momentum = (K - K) / K * 100 = 0` -> `dmi = 0`
  (bit-exact). Verified across `K in {1, 5, 100, -3}` in the
  integration sweep.
- **CONST close=0** -> `close[i - dynLength] = 0` -> `dmi = null`
  everywhere (divide-by-zero guard).
- Clamp guarantee: every defined `dynLength` is within
  `[minLength, maxLength]` regardless of input.

## Design notes
- The ratio fallback of `1` when `longStd === 0` keeps the CONST
  anchor bit-exact -- otherwise `0/0` would propagate as NaN and the
  indicator would be `null` for flat-input fixtures.
- Clamp `[minLength, maxLength]` prevents pathological windows from
  blowing up dynLength to the entire history; `Math.round` keeps the
  result an integer.
- Default colors: DMI line `#7c3aed` (violet), bullish `#16a34a`,
  bearish `#dc2626`, zero line `#475569` dashed.
- DMI auto-scales y-axis but always includes zero.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
