## KAMA pipeline (Kaufman)

```
direction[i]  = close[i] - close[i - efficiencyPeriod]
volatility[i] = sum_{j=i-eP+1..i} |close[j] - close[j-1]|
er[i]         = |direction| / volatility; 0 when volatility===0
fastSc        = 2 / (fastPeriod + 1)
slowSc        = 2 / (slowPeriod + 1)
sc[i]         = (er * (fastSc - slowSc) + slowSc) ** 2
kama[i]       = kama[i-1] + sc[i] * (close[i] - kama[i-1])
signal[i]     = SMA(kama, signalLength)
```

The smoothing constant adapts to recent efficiency: in a
clean trend (`er = 1`) the SC approaches the fast smoothing
weight; in a choppy market (`er = 0`) it approaches the slow
weight. This makes KAMA more responsive in directional
markets and more damped in choppy conditions -- a volatility-
adaptive moving average.

Defaults: `efficiencyPeriod = 10`, `fastPeriod = 2`,
`slowPeriod = 30`, `signalLength = 3` (Kaufman canonical).

Warmup is `efficiencyPeriod + signalLength - 2 = 11` for the
default tuning: KAMA SMA-seeds at `efficiencyPeriod - 1 = 9`,
then the signal SMA needs another `signalLength - 1 = 2`
valid KAMA samples.

## Bit-exact anchors

| input              | kama (steady-state) | signal (steady-state) | kama - signal | regime  | crosses |
| ------------------ | ------------------- | --------------------- | ------------- | ------- | ------- |
| CONST close = K    | K                   | K                     | 0             | bullish | 0       |
| LINEAR UP, close=i | i - 1.25            | i - 2.25              | +1            | bullish | 0       |
| LINEAR DOWN, close=-i | -i + 1.25        | -i + 2.25             | -1            | bearish | 0       |

CONST K verified across K = 0..1234 -- direction = 0,
volatility = 0, so er hits the degenerate fallback and
returns 0. SC collapses to `slowSc^2`, but `close[i] - kama
= K - K = 0`, so kama stays at K forever. signal = K. kama
=== signal -> regime `bullish` (>=).

LINEAR UP exercises the KAMA steady-state algebra: er = 1
(direction = `+eP`, volatility = `eP`, ratio = 1), so SC =
`(2/3)^2 = 4/9`. The recurrence `kama[i] = kama[i-1] + (4/9)
* (i - kama[i-1])` has a steady-state solution of `kama[i] =
i - 1.25` (1 = (4/9)*(1 + L) -> L = 5/4 = 1.25). signal SMA
of length 3 introduces another bar of lag, so signal = `i -
2.25`. kama - signal = `+1` constant in steady state.

LINEAR DOWN is the algebraic mirror: er = 1, sc = 4/9, kama
= `-i + 1.25`, signal = `-i + 2.25`, kama - signal = `-1`.

During the SMA-seed convergence (i = 9..~30), KAMA is
catching up from the initial SMA value (`4.5` for LINEAR UP
seeded over `close[0..9]`). The convergence is monotonic
(KAMA climbs faster than signal SMA), so the regime stays
`bullish` throughout the entire valid range -- not just in
steady state. Tests assert regime at every valid bar and
algebraic identity (kama - signal = +/- 1) at `i >= 50`
(steady state).
