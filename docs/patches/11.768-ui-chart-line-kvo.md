# 11.768 chart-line-kvo (v1.11.786)

## Summary

Pure-SVG dual-panel React/TypeScript chart primitive with a
**Klinger Volume Oscillator** (KVO) panel beneath the close.
The KVO is the difference between two EMAs of the signed
volume force. Added as the `<ChartLineKvo />` UI primitive in
`web/src/components/ui/`.

## Algorithm

For each bar:

```
T[i]      = (high[i] + low[i] + close[i]) / 3   (typical price)
trend[i]  = +1 if T[i] > T[i - 1]
           -1 if T[i] < T[i - 1]
           trend[i - 1] otherwise
dm[i]     = high[i] - low[i]
cm[i]     = cm[i - 1] + dm[i] when trend[i] == trend[i - 1]
           dm[i - 1] + dm[i] otherwise
vf[i]     = volume[i] * |2 * (dm[i] / cm[i]) - 1| * trend[i] * 100
KVO[i]    = EMA(vf, fast)[i] - EMA(vf, slow)[i]
```

Defaults: `fast = 34`, `slow = 55`. The EMA uses
`alpha = 2 / (length + 1)` and seeds with the first value. The
first bar has no prior reference: `trend[0] = 0`, `vf[0] = 0`.
When `cm == 0` (singular) `vf = 0` for that bar.

## Anchors

- **ZERO_VOLUME (volume == 0 at every bar)**: `vf = 0` at every
  bar regardless of price action, both EMAs of zero are zero,
  so `KVO = 0` bit-exact at every bar. Verified across
  `(fast, slow) in {3, 5, 13, 34} x {10, 20, 55}` (valid combos
  where fast < slow).
- **CONST_OHLC (open = high = low = close = K, volume != 0)**:
  T is constant -> trend = 0 after seed; dm = 0; vf = 0;
  KVO = 0 bit-exact. Verified across `(fast, slow) in
  {3, 5, 13} x {10, 20}`.
- **EMA helper**: EMA of zeros is zero bit-exact; EMA of
  constant 2 (dyadic-friendly) is 2 bit-exact.
- **Non-trivial data**: rising followed by falling produces
  non-zero KVO at some bar (sanity).
