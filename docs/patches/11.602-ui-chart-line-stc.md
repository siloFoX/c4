## Summary

New `ChartLineStc` UI primitive at
`web/src/components/ui/chart-line-stc.tsx` -- a pure-SVG line chart
with a Schaff Trend Cycle oscillator panel on a 0 to 100 scale. The
top panel renders the price line; the bottom panel renders the STC
line on a fixed 0-100 axis with shaded overbought and oversold zones,
dashed threshold lines, and markers coloured by zone.

## Indicator math

Doug Schaff's Trend Cycle, with a fast EMA period (default 23), a
slow EMA period (default 50), a cycle period (default 10), and a
smoothing factor (default 0.5):

```
MACD = EMA(values, fast) - EMA(values, slow)
%K1  = StochK(MACD, cycle)         scale to 0-100 within the window range
%D1  = Smooth(%K1, factor)         out[i] = out[i-1] + factor*(src[i]-out[i-1])
%K2  = StochK(%D1, cycle)
STC  = Smooth(%K2, factor)
```

`computeLineStcStochK(src, cycle)` scales each value to its position
within a cycle-length high-low window; a flat window (zero range)
reads the neutral 50, and results are clamped to [0, 100].
`computeLineStcSmooth(src, factor)` is the Schaff modified-EMA,
seeded with the first defined value. Applying the
stochastic-then-smooth pair twice to the MACD line yields a fast
oscillator bounded to 0-100.

### Worked fixture

`values = [4,4,4,12,12,12,4,4,4,12]`, `fast 1`, `slow 3`, `cycle 2`,
`factor 0.5`:

```
macd = fastEMA - slowEMA = [.,.,0,4,2,1,-3.5,-1.75,-0.875,3.5625]
k1   = StochK(macd, 2)   = [.,.,.,100,0,0,0,100,100,100]
d1   = Smooth(k1, 0.5)   = [.,.,.,100,50,25,12.5,56.25,78.125,89.0625]
k2   = StochK(d1, 2)     = [.,.,.,.,0,0,0,100,100,100]
stc  = Smooth(k2, 0.5)   = [.,.,.,.,0,0,0,50,75,87.5]
```

With thresholds 75/25: two overbought readings (75, 87.5) and three
oversold (0, 0, 0).

## API

- `getLineStcFinitePoints(points)` -- drop points with a non-finite
  `x` or `value`.
- `normalizeLineStcPeriod(period, fallback)` -- positive integer;
  non-finite / sub-1 falls back, fractional floors.
- `computeLineStcEma(src, period)` -- the null-tolerant EMA.
- `computeLineStcMacd(values, fastPeriod, slowPeriod)` -- the MACD
  line.
- `computeLineStcStochK(src, cyclePeriod)` -- the raw stochastic %K
  (0-100, flat window reads 50).
- `computeLineStcSmooth(src, factor)` -- the Schaff modified-EMA
  smoother.
- `computeLineStc(values, fastPeriod, slowPeriod, cyclePeriod,
  factor)` -- returns `{ macd, k1, d1, k2, stc }`.
- `runLineStc(points, options?)` -- sort finite points by `x`, compute
  the MACD and STC, return `{ series, fastPeriod, slowPeriod,
  cyclePeriod, factor, upperThreshold, lowerThreshold, macd, stc,
  samples, stcFinal, stcMin, stcMax, overboughtCount, oversoldCount,
  ok }`. Each sample carries an `overbought` / `oversold` / `neutral`
  zone. `ok` is false for fewer than two points.
- `computeLineStcLayout(options)` -- stack a price panel above an STC
  panel whose y-axis is fixed 0-100. Returns panels, ticks, the price
  path, the STC path, zone-coloured markers, the overbought and
  oversold zone rects, and the threshold-line `y` positions.
- `describeLineStcChart(data, options?)` -- ARIA description.

## Component

`ChartLineStc` is a `forwardRef` region (`role="region"`) with an
ARIA-linked description, a `role="img"` SVG, panel labels, a config
badge (MACD and cycle periods), a two-series legend (Price / STC)
with toggle, shaded overbought and oversold zones, dashed threshold
lines, the STC line, markers coloured by zone, a hover/focus tooltip
exposing the value, MACD, STC, and zone, and a
`motion-safe:animate-fade-in` entrance. Every node carries a
`data-section` attribute.

## Distinctness

- **vs chart-line-macd** -- MACD plots the *raw, unbounded* EMA
  difference. The STC takes that MACD line as its *input* and runs it
  through a double stochastic, producing a *bounded 0-100* oscillator
  that responds far faster than MACD itself.
- **vs chart-line-stochastic** -- the stochastic oscillator runs a
  single stochastic transform on the *price*. The STC runs *two*
  stochastic transforms, each followed by a modified-EMA smoothing,
  and runs them on the *MACD line* rather than price.
- **vs chart-line-ppo / chart-line-trix** -- those stay in the EMA /
  percentage family. The STC is the only chart-line primitive that
  composes MACD with a double stochastic cycle.
