# Patch 11.899 -- UI: chart-line-macd-hist-cross primitive

## Summary

Added `<ChartLineMacdHistCross />` -- pure-SVG dual-panel React
/TS primitive that renders the close (top panel) with bullish /
bearish arrow overlays at every MACD histogram zero crossover
trigger, and the MACD histogram (`macd - signal`) bottom panel
as colored bars centered on a zero baseline. Histogram-cross
variant of the MACD family that flags momentum acceleration
trigger events distinct from the MACD line / signal cross
primitives.

## Algorithm

```
ema_fast[i] = EMA(close, fastLength)
ema_slow[i] = EMA(close, slowLength)
macd[i]    = ema_fast[i] - ema_slow[i]
signal[i]  = EMA(macd, signalLength)
hist[i]    = macd[i] - signal[i]
bullish   : prev hist <= 0 && cur hist > 0
bearish   : prev hist >= 0 && cur hist < 0
```

Defaults `fastLength = 12`, `slowLength = 26`, `signalLength =
9` (canonical MACD). Regime classifier `bullish` (hist > 0),
`bearish` (hist < 0), `neutral` (hist === 0), `none` (hist
null).

## Bit-exact anchor

CONST close = K: EMA(K) collapses to K via the `min === max`
precision short-circuit, so `ema_fast = ema_slow = K`,
`macd = 0`, `signal = EMA(0, 9) = 0`, `histogram = 0` every
settled bar. Regime is `neutral`, cross count is exactly 0.
Verified across K = 0, 1, 5, 17, 100, 1234.
