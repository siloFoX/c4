# Patch 11.946 - UI: chart-line-uo-zero-cross primitive

Version: 1.11.964

## Summary

Add `web/src/components/ui/chart-line-uo-zero-cross.tsx` -- a pure-SVG dual
panel line chart primitive that visualizes price alongside Larry Williams'
Ultimate Oscillator (UO) panel, with explicit cross markers and overlay
arrows at every UO midline-50 crossover. UO blends buying-pressure /
true-range ratios across three windows (short / mid / long, default
7 / 14 / 28) so the indicator smooths multi-timeframe momentum into a
single bounded line on [0, 100]. Midline crossovers mark multi-timeframe
momentum centerline regime transitions: cross-up = bullish (buying
pressure dominates across the windows), cross-down = bearish.

## Pipeline

1. `applyLineUoZeroCrossPressure(close)`:
   - `delta_i = close_i - close_{i-1}`
   - `bp_i = max(0, delta_i)` (buying pressure)
   - `tr_i = abs(delta_i)` (close-only true range proxy)
   - bp[0] / tr[0] = null (no prior close).
2. `computeLineUoZeroCross(series, {short, mid, long})`:
   - `avg_n_i = sum(bp[i-n+1..i]) / sum(tr[i-n+1..i])`
   - When `sumTr === 0`, the ratio falls back to `0.5` (neutral).
   - `uo_i = 100 * (4 * avg_short + 2 * avg_mid + avg_long) / 7`
   - `uo[i < long] = null` (warmup gap).
3. `classifyLineUoZeroCrossRegime(uo, T)`: null -> none, uo >= T -> bullish,
   uo < T -> bearish.
4. `detectLineUoZeroCrossCrosses(series, uo, T)`: strict
   `prev <= T && cur > T` for bullish, strict `prev >= T && cur < T` for
   bearish. Boundary equality does not fire.

## Bit-exact anchors

| Pattern                  | UO (after warmup)    | Regime  |
|--------------------------|----------------------|---------|
| CONST close = K          | 50 (0 / 0 -> 0.5)    | bullish |
| LINEAR UP (close = i)    | 100                  | bullish |
| LINEAR DOWN (close = -i) | 0                    | bearish |

The CONST anchor is tested across K in `{0, 1, 42, 100, 1234}` -- all
must produce `uo === 50` bit-exactly.

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), UO panel below.
* UO panel uses a fixed `[0, 100]` range (UO is bounded by construction)
  so the threshold band sits exactly at the midpoint when `threshold = 50`.
* Threshold band rendered as a horizontal rule at `y = thresholdY` in the
  UO panel.
* Cross markers drawn as circles on the UO panel at `(cx, cyOsc)` and as
  arrows on the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="UO Zero Cross chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
