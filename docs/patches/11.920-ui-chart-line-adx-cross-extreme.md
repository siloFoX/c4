# Patch 11.920 -- UI: chart-line-adx-cross-extreme primitive

## Summary

Added `<ChartLineAdxCrossExtreme />` -- pure-SVG dual-panel
React/TS primitive that renders the close (top panel) with
bullish (strong trend entry) / bearish (weak trend exit) arrow
overlays at every ADX extreme threshold cross trigger, and the
close-only ADX line (bottom panel) on a fixed 0-100 oscillator
with 20 / 50 / 40 reference bands. Threshold-cross variant of
the ADX family that separates trend strength entry / exit
events from the ADX magnitude alone.

## Algorithm

```
TR[i]      = |close[i] - close[i-1]|              (close-only)
plusDM[i]  = max(delta close, 0)
minusDM[i] = max(-delta close, 0)
ATR[i]     = Wilder smooth of TR over length
plusDM_s   = Wilder smooth of plusDM
minusDM_s  = Wilder smooth of minusDM
plusDI     = ATR > 0 ? 100 * plusDM_s / ATR : 0
minusDI    = ATR > 0 ? 100 * minusDM_s / ATR : 0
dx[i]      = sum > 0 ? 100 * |plusDI - minusDI| / sum : 0
adx[i]     = Wilder smooth of dx over length
bullish   : prev <= upper && cur > upper  (strong trend entry)
bearish   : prev >= lower && cur < lower  (weak trend exit)
```

Defaults `length = 14` (canonical Wilder ADX window),
`upperThreshold = 40`, `lowerThreshold = 20`. Regime classifier
`bullish` (ADX >= upper, strong), `bearish` (ADX < lower,
weak), `neutral` (lower <= ADX < upper, moderate), `none`
(ADX null).

## Bit-exact anchor

CONST close = K: every delta = 0 -> TR = +DM = -DM = 0 ->
ATR = 0 -> +DI = -DI = 0 via zero-ATR guard -> DX = 0 via 0/0
guard -> ADX = 0 via Wilder short-circuit. 0 < 20 so regime
is `bearish` (weak). 0 never crosses above 40 and the bearish
cross condition `prev >= 20 && cur < 20` is never satisfied
(ADX starts already below 20 the moment it becomes non-null),
so cross count = 0. Verified across K = 0, 1, 5, 17, 100,
1234.

LINEAR UP / DOWN both yield ADX approaching 100 (pure trend),
which is above the upper threshold of 40 -> regime `bullish`
(strong directional movement, regardless of direction).

## Visual

- Two stacked panels with the osc panel pinned to 0-100.
- Top: close line + bullish (strong) / bearish (weak) polygon
  arrows at threshold cross bars.
- Bottom: ADX line + cross dot markers + 20 / 50 / 40
  reference bands (weak / mid / strong).
- Tooltip lists x / close / ADX / regime / counts / crosses.
- Config badge + togglable legend (close / ADX).
- Cross marker is `role="graphics-symbol"` with `tabIndex={0}`.
