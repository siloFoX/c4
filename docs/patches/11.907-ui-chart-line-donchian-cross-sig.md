# Patch 11.907 -- UI: chart-line-donchian-cross-sig primitive

## Summary

Added `<ChartLineDonchianCrossSig />` -- pure-SVG dual-panel
React/TS primitive that renders the close + upper / middle /
lower Donchian Channel (top panel) with bullish / bearish
arrow overlays at every close vs middle line cross trigger,
and a `close - middle` deviation oscillator (bottom panel)
with cross dot markers and a dashed zero baseline. Signal-
cross variant of the Donchian Channel family that flags high
/ low range regime transition triggers distinct from envelope
breakouts.

## Algorithm

```
upper[i] = max(close[i-n+1..i])
lower[i] = min(close[i-n+1..i])
mid[i]   = (upper + lower) / 2
diff[i]  = close[i] - mid[i]
bullish : (close - mid) crosses up   (prev <= 0, cur > 0)
bearish : (close - mid) crosses down (prev >= 0, cur < 0)
```

Default `length = 20`. Regime classifier `bullish` (close >
mid), `bearish` (close < mid), `neutral` (close === mid),
`none` (mid null).

## Bit-exact anchor

CONST close = K: upper = lower = K every bar; mid = K; close
- mid = 0 every bar. Regime is `neutral`, cross count = 0.
Verified across K = 0, 1, 5, 17, 100, 1234.

LINEAR UP with step 1 also has bit-exact anchors: every
settled bar i >= 19 has upper = i, lower = i - 19, mid = i -
9.5, close = i, so diff = 9.5 exactly. Regime is `bullish`,
0 crosses (diff is constant after warmup).
