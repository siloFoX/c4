# Patch 11.948 - UI: chart-line-rmi-zero-cross primitive

Version: 1.11.966

## Summary

Add `web/src/components/ui/chart-line-rmi-zero-cross.tsx` -- a pure-SVG dual
panel line chart primitive that visualizes price alongside Roger Altman's
Relative Momentum Index (RMI) panel, with explicit cross markers and
overlay arrows at every RMI midline-50 crossover. RMI is RSI with a
lookback period: instead of comparing the current close to the previous
close, it compares to the close `lookback` bars ago and Wilder-smooths
the resulting up/down moves over a `length` window. The lookback turns
RSI into a momentum-aware oscillator -- the indicator reads positive
only when sustained price gains over the lookback window dominate.

## Pipeline

1. Momentum step:
   - `mom_i = close_i - close_{i - lookback}` (i >= lookback)
   - `gain_i = max(0,  mom_i)`
   - `loss_i = max(0, -mom_i)`
2. Wilder-smoothed averages with SMA seed at `i = lookback + length - 1`:
   - Seed: SMA of `gain[lookback..lookback + length - 1]`.
   - Recursion: `avg[i] = (avg[i-1] * (length-1) + cur) / length`.
3. RMI:
   - `avg_loss == 0 && avg_gain == 0` -> 50 (neutral fallback)
   - `avg_loss == 0 && avg_gain > 0`  -> 100
   - `avg_gain == 0 && avg_loss > 0`  -> 0
   - else: `100 - 100 / (1 + avg_gain / avg_loss)`
4. `classifyLineRmiZeroCrossRegime(rmi, T)`: null -> none, rmi >= T ->
   bullish, rmi < T -> bearish.
5. `detectLineRmiZeroCrossCrosses(series, rmi, T)`: strict
   `prev <= T && cur > T` for bullish, strict `prev >= T && cur < T`
   for bearish. Boundary equality does not fire.

## Bit-exact anchors

| Pattern                  | RMI (after warmup) | Regime  |
|--------------------------|--------------------|---------|
| CONST close = K          | 50 (both-zero -> neutral) | bullish |
| LINEAR UP (close = i)    | 100                | bullish |
| LINEAR DOWN (close = -i) | 0                  | bearish |

CONST anchors tested bit-exactly across K in `{0, 1, 42, 100, 1234}`.
LINEAR UP/DOWN reach the saturation limits because `mom == +lookback`
or `mom == -lookback` constantly, leaving the unused side at zero.

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), RMI panel below.
* RMI panel uses a fixed `[0, 100]` range (RMI is bounded by
  construction) so the threshold band sits exactly at the midpoint when
  `threshold = 50`.
* Threshold band rendered as a horizontal rule at `y = thresholdY` in
  the RMI panel.
* Cross markers drawn as circles on the RMI panel at `(cx, cyOsc)` and
  as arrows on the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="RMI Zero Cross chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only
  text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
