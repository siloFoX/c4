# 11.522 -- web/src/components/ui/chart-line-normalize.tsx

Pure-SVG multi-series line chart that **rebases each series to a
common index basis** (default `100`) so series with different y
scales become directly comparable. The canonical "indexed
performance" view used in finance / economics dashboards.

Three rebasing modes:

- `'first'` (default): every series starts at `indexBase` from its
  first finite sample.
- `'value'`: rebase using the y interpolated at a specific x value.
- `'min'`: rebase so the **minimum** finite y maps to `indexBase`.

Each layout point carries the raw `y`, the normalised value, and the
signed `percentChange` from baseline. The legend shows the final %
change, the tooltip surfaces raw + normalised + percent, and the
y range is auto-expanded to include the `indexBase` baseline so the
reference line is always visible.

## Distinct from other transform / reference primitives

- `<ChartLineCumulative>` (11.520): running sum transform.
  Normalize is a **rebase** (rescaling), not an accumulation.
- `<ChartLineRate>` (11.521): first-derivative transform.
- `<ChartLineMovingAvg>` (11.513): windowed average.
- `<ChartLineTrend>` (11.512): regression. Normalise preserves the
  full shape.
- `<ChartLineComparison>` (11.511): between two curves. Normalise
  rebases all series independently.
- `<ChartLineBaseline>` (11.510), `<ChartLineTarget>` (11.518):
  per-point classification or comparison. Normalize is a rescale.

## Public surface

- `<ChartLineNormalize series={...} indexBase={100} mode="first" />`.
- Per-series `{id, label, data, color?, mode?, referenceX?}`. `data`
  entries are raw `{x, y}` samples; the chart rebases them.
- Per-series `mode` and `referenceX` override the chart-level
  defaults.
- Toggle flags: `showAxis`, `showGrid`, `showDots`, `showLegend`,
  `showTooltip`, `showBaseline`, `showBaselineLabel`, `animate`.
- `baselineDashArray` (`'4 3'`), `baselineColor` (`'#64748b'`).
- Standard sizing + format + a11y props.
- `formatValue` / `formatX` / `formatPercent` (sign-aware fallback).
- Controlled + uncontrolled `hiddenSeries`.
- `onPointClick({series, point})`.

## Pure helpers

- `getLineNormalizeDefaultColor(index)` -- 10-color palette cycle.
- `getLineNormalizeFinitePoints(points)` -- finite-only filter.
- `resolveLineNormalizeYAtX(points, referenceX)` -- linearly
  interpolates y at a given x. Clamps to endpoints when outside the
  range. Non-array / non-finite x -> `null`.
- `findLineNormalizeReference(points, mode, referenceX?)` -- returns
  the reference y for normalization, or `null` when none is
  derivable (empty input, missing referenceX for 'value' mode, etc).
  `'first'` uses the first sample in x-sorted order; `'min'` uses
  the minimum; `'value'` uses the interpolated y at `referenceX`.
- `normalizeLineNormalizeSeries(points, mode, indexBase,
  referenceX?)` -- returns `{normalized, reference, ok}`. Each
  output `y` is `(raw_y / reference) * indexBase`. **`reference ===
  0`** or `null` returns `{normalized: [], reference, ok: false}`
  (impossible to rebase). Non-finite `indexBase` falls back to
  `100`.
- `computeLineNormalizeStats(points, reference, ok, indexBase)` --
  returns `{reference, ok, finalNormalized, finalPercentChange,
  maxNormalized, minNormalized, maxPercentChange, minPercentChange,
  finiteCount}`.
- `computeLineNormalizeLayout({series, mode?, indexBase?,
  referenceX?, hiddenSeries?, ...})` -- returns `{series, xTicks,
  yTicks, xMin, xMax, yMin, yMax, baselineY, indexBase,
  innerWidth, innerHeight, totalPoints, visibleSeriesCount}`. Each
  layout series carries `{id, label, index (stable original),
  color, mode, points, path, stats, reference, referenceOk,
  finiteCount, totalCount}`. Per-point `{index, x, y, normalized,
  percentChange, px, py}` (y is the raw value, normalized is the
  rebased value).
- `describeLineNormalizeChart(series, indexBase?, mode?, hidden?,
  formatValue?, referenceX?)` -- builds the screen-reader
  description.

- Root `role="region"` + `aria-label` + `aria-describedby`.
- SVG `role="img"`.
- Per-line path + per-dot are `role="graphics-symbol"` +
  `tabIndex={0}`; baseline is `role="graphics-symbol"`. Dot
  aria-label includes raw + normalised + percent from base.
- Every node carries `data-section="chart-line-normalize-..."` for
  test selectors.
- Series groups expose `data-series-mode`, `data-series-reference`,
  `data-series-reference-ok`, `data-series-final-normalized`,
  `data-series-final-percent`, `data-series-max-normalized`,
  `data-series-min-normalized`, `data-series-color`,
  `data-series-finite-count`.
- Dots expose `data-normalized`, `data-percent`.
- Root mirrors `data-mode`, `data-index-base`, `data-reference-x`.

## Tooltip

Hovering a dot shows the series label, x, `raw:`, `index:` (bold),
and a coloured `<+/->P%` row indicating percent change from
baseline.
