# 11.807 UI: chart-line-momentum-divergence primitive (v1.11.825)

## Summary
Pure-SVG dual-panel React/TS primitive that plots a **Momentum
Divergence** oscillator beneath the close. The divergence highlights
when the close's percent change (ROC) deviates from its absolute
change (momentum) across the lookback -- a signal that price velocity
differs meaningfully from price magnitude.

## Formula
```
ROC[i]        = (close[i] - close[i - length]) / close[i - length] * 100
momentum[i]   = close[i] - close[i - length]
divergence[i] = ROC[i] - momentum[i]
```
`divergence[i]` is `null` during warmup (`i < length`) and when
`close[i - length] === 0` (ROC is undefined) or any input is non-finite.

## Defaults
- `length = 14`
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `positive`: `divergence > 0`
- `negative`: `divergence < 0`
- `zero`: `divergence == 0`
- `none`: warmup / non-finite
- `crossed`: `'up'` when prev value was `<= 0` and current is `> 0`,
  mirror for `'down'`. Markers fire only on cross bars.

## Bit-exact testability anchors
- **CONST close** (`close = K`, `K != 0`): `ROC = 0` and `momentum = 0`
  -> `divergence = 0` (bit-exact). Verified across `K in {1, 5, 100, -3}`
  and `length in {3, 4, 7, 10}`. `-0 -> +0` normalization in both the
  ROC and momentum helpers keeps negative-K CONST tests
  `Object.is`-bit-exact.
- **ZERO close**: `close[i - length] = 0` -> ROC null -> divergence
  null everywhere.
- **GEOMETRIC close = 2^k**:
  - `ROC(L) = (2^L - 1) * 100` constant integer.
  - `momentum(L)[i] = 2^(i - L) * (2^L - 1)` varies with i.
  - For L=4, divergence at i=L = (2^4 - 1) * 100 - (2^4 - 1) * 1 =
    `15 * 99 = 1485` (bit-exact integer).
  - At i=L+1: divergence = 1500 - 30 = `1470` (bit-exact).

## Design notes
- The ROC-minus-momentum form deliberately mixes units (percent vs
  absolute) -- this is what makes it a "divergence" indicator rather
  than a normalized oscillator. The bit-exact GEOMETRIC anchor takes
  advantage of the unit mismatch: ROC stays constant while momentum
  scales geometrically, so divergence flips sign once `2^(i - L)`
  exceeds 100.
- Divergence y-axis auto-scales but always includes the zero line.
- `-0 -> +0` normalization in both ROC and momentum helpers keeps
  CONST tests `Object.is`-bit-exact regardless of K's sign.
- Default colors: divergence line `#06b6d4` (cyan, distinct from prior
  primitives), bullish `#16a34a`, bearish `#dc2626`, zero line
  `#475569` dashed.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
