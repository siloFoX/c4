## Indicator

The Chandelier Exit (Chuck LeBeau) is a volatility-based trailing
stop -- the stop "hangs" from the recent extreme like a chandelier
hangs from a ceiling. On a close-only series the high and low of
each bar collapse onto the close, matching the existing
`chart-line-atr` convention.

- `computeLineChandelierTrueRanges(closes)` -- the bar-to-bar true
  range, the absolute close-to-close change `|close[i] -
  close[i-1]|`; index 0 is null.
- `computeLineChandelierAtr(closes, period)` -- Welles Wilder's
  Average True Range: the first value is the simple mean of the
  first `period` true ranges, then Wilder smoothing `(prev *
  (period - 1) + tr) / period`.
- `computeLineChandelierRollingMax` / `...RollingMin` -- the
  highest and lowest close of the trailing `period` bars.
- `computeLineChandelierExit(closes, period, multiplier)` -- the
  Chandelier Exit pair `{ long, short }`. The long exit is
  `highestClose - ATR * multiplier`, the short exit is
  `lowestClose + ATR * multiplier`. This is the basic
  (non-ratcheting) form: each bar is recomputed from the current
  rolling extreme and ATR, which keeps it distinct from the
  position-state flip of `chart-line-supertrend`.
- `runLineChandelier` -- sorts the finite points by x, runs the
  pipeline, and returns per-bar samples classified `above` /
  `below` / `inside` the exit channel, the zone counts and the
  final readings; `ok` is false for fewer than two points.
- `computeLineChandelierLayout` -- overlays the price line and the
  two dashed exit lines on a single panel, with one marker per
  classified bar coloured by zone.
- `describeLineChandelierChart` -- an off-screen sentence summary
  for assistive technology.

## Testability

The Chandelier Exit chains an ATR (recursive Wilder smoothing)
with rolling extremes. The fixture sidesteps the recursion: every
close steps by exactly +/-10, so every close-to-close true range
is a constant 10 and Wilder's ATR -- a weighted blend of a
constant -- settles at exactly 10. With an integer ATR and integer
extremes the long and short exits are exact integers, asserted
with `toEqual`. The price ramps up, crashes, then ramps again, so
the rolling max/min move non-trivially and all three zones (above
/ below / inside) appear. A separate dyadic fixture checks the
Wilder recurrence after the seed.
