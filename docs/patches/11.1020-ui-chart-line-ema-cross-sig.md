Add a pure-SVG dual-panel `ChartLineEmaCrossSig` component
that renders the close in the top panel and the Exponential
Moving Average (EMA) plus its smoothed SMA signal line in the
bottom panel, marking bullish (EMA crosses up through signal,
trend trigger up) / bearish (EMA crosses down through signal,

## EMA-over-signal pipeline

```
ema[i]    = SMA-seeded EMA(close, period)
signal[i] = SMA(ema, signalLength)
bullish   : prev ema <= prev signal && cur ema > cur signal
bearish   : prev ema >= prev signal && cur ema < cur signal
regime    : bullish (ema >= signal), bearish (ema < signal)
bias      : up/down/flat/none from ema[i] vs ema[i-1]
```

SMA-seeded EMA preserves steady-state behavior exactly at the
seed bar -- on LINEAR UP close=i the EMA settles at `i - (L
- 1)/2` from the seed onward, and the signal SMA lags the EMA
by `(signalLength - 1)/2`.

Defaults: `period = 14`, `signalLength = 3`. Bias-coloured
cross markers match the convention used by
`chart-line-hma-cross-sig`,
`chart-line-macd-zero-cross-sig`, and
`chart-line-stc-divergence-cross`.

Warmup is `period + signalLength - 2 = 15` for the default
tuning: SMA-seeded EMA becomes valid at `period - 1 = 13`,
then the signal SMA adds another `signalLength - 1 = 2` bars.

## Bit-exact anchors

| input              | ema (post-warmup) | signal (post-warmup) | ema - signal | regime  | crosses |
| ------------------ | ----------------- | -------------------- | ------------ | ------- | ------- |
| CONST close = K    | K                 | K                    | 0            | bullish | 0       |
| LINEAR UP, close=i | i - 6.5           | i - 7.5              | +1           | bullish | 0       |
| LINEAR DOWN, close=-i | -i + 6.5       | -i + 7.5             | -1           | bearish | 0       |

CONST K verified across K = 0..1234 -- SMA-seeded EMA on a
constant collapses to K exactly, then signal = SMA(K, 3) = K.
ema === signal -> regime `bullish` (inclusive `>=`).

LINEAR UP exercises the steady-state EMA lag algebra: for
period = 14, ema settles at `i - (14-1)/2 = i - 6.5`. The SMA
signal of length 3 introduces an additional `(3-1)/2 = 1` bar
lag, giving signal = `i - 7.5`. ema - signal = `+1`
constant -> regime `bullish`.

LINEAR DOWN is the algebraic mirror: ema = `-i + 6.5`,
signal = `-i + 7.5`, ema - signal = `-1` -> regime `bearish`.

The cross detector is exercised in dedicated unit tests for
`detectLineEmaCrossSigCrosses`, which checks bullish / bearish
trigger semantics, null-handling, and no double-firing on
consecutive bars above the signal.
