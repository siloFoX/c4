# Patch 11.904 -- UI: chart-line-adx-di-cross primitive

## Summary

Added `<ChartLineAdxDiCross />` -- pure-SVG dual-panel React/TS
primitive that renders the close (top panel) with bullish /
bearish arrow overlays at every Wilder +DI vs -DI cross trigger,
and the +DI / -DI lines (bottom panel) on a fixed 0-100
oscillator with cross dot markers. DI-cross variant of the
ADX family that flags directional movement regime shift events
distinct from the base ADX trend strength reading.

## Algorithm

```
TR[i]      = |close[i] - close[i-1]|            (close-only)
plusDM[i]  = max(close[i] - close[i-1], 0)
minusDM[i] = max(close[i-1] - close[i], 0)
ATR[i]     = Wilder smooth of TR
plusDM_s   = Wilder smooth of plusDM
minusDM_s  = Wilder smooth of minusDM
plusDI     = ATR > 0 ? 100 * plusDM_s / ATR : 0
minusDI    = ATR > 0 ? 100 * minusDM_s / ATR : 0
bullish   : (plusDI - minusDI) crosses up   (prev <= 0, cur > 0)
bearish   : (plusDI - minusDI) crosses down (prev >= 0, cur < 0)
```

Default `length = 14` (canonical Wilder ADX/DI window). Regime
classifier `bullish` (+DI > -DI), `bearish` (+DI < -DI),
`neutral` (+DI === -DI), `none` (either null).

## Bit-exact anchor

CONST close = K: every delta close = 0 -> TR = +DM = -DM = 0
every bar -> ATR = 0 -> +DI = -DI = 0 via the zero-ATR short-
circuit. diff = 0 -> regime `neutral`, cross count = 0.
Verified across K = 0, 1, 5, 17, 100, 1234.

LINEAR UP also has bit-exact anchors: every delta close = 1 so
plusDM = 1, minusDM = 0, TR = 1 each bar -> ATR = 1, plusDM_s =
1, minusDM_s = 0 -> +DI = 100, -DI = 0, regime `bullish`.
LINEAR DOWN is the mirror image.

## Visual

- Two stacked panels, ARPS design system colors via tokens.
- Top: close line + bullish/bearish polygon arrows at cross
  bars.
- Bottom: +DI (green) + -DI (red) lines + cross dot markers +
  dashed 50 reference line.
- Tooltip lists x / close / +DI / -DI / regime / counts /
  crosses.
- Config badge + togglable legend (close / +DI / -DI).
- Cross marker is `role="graphics-symbol"` with `tabIndex={0}`.
