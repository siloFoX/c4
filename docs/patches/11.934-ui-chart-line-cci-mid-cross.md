# 11.934 -- UI: chart-line-cci-mid-cross primitive (v1.11.952)

## Summary

Adds `ChartLineCciMidCross`, a pure-SVG dual-panel chart that
renders the close in the top panel and a close-only Commodity
Channel Index (CCI) line in the bottom panel. Marks bullish
(cross up through midline 0) / bearish (cross down through
midline 0) momentum centerline regime trigger events. Single-
threshold cross variant of the CCI family that flags the
discrete CCI zero-line crossover distinct from the +/- 100
overbought / oversold extreme bands. Completes the midline-
cross family (RSI, Stochastic, CCI).

## Pipeline

For close-only data with default `length = 20`:

```
sma[i]  = SMA(close, length)
md[i]   = avg |close[i-n+1..i] - sma[i]|
cci[i]  = md > 0
            ? (close - sma) / (0.015 * md)
            : 0
bullish : prev cci <= 0 && cur cci > 0   (momentum up)
bearish : prev cci >= 0 && cur cci < 0   (momentum down)
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> sma = K, md = 0 ->
  0/0 guard returns cci = 0. cci = 0 sits on the threshold
  but the strict-inequality detector never fires. regime
  `bullish` (cci >= 0). cross count = 0. Verified across K
  in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` (length 20) -> cci = 126.667
  constant once warm. 126.667 > 0, regime `bullish`. 0
  crosses.
- LINEAR DOWN `close = -i` -> cci = -126.667. -126.667 < 0,
  regime `bearish`. 0 crosses.
- Decline (20 bars) followed by sharp rise -> cci transitions
  from negative to positive, firing 1 bullish midline cross.
- Rise (20 bars) followed by sharp decline -> 1 bearish
  midline cross.
- Custom thresholds: the CCI midline is conventionally 0 but
  the prop accepts any finite value (positive or negative)
  since CCI is unbounded.

## Regime classifier

| cci range            | regime      |
|----------------------|-------------|
| cci >= 0             | bullish     |
| cci < 0              | bearish     |
| null                 | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- One horizontal reference band inside the oscillator panel
  (threshold at 0, the zero-line).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind.
- Default oscillator range +/- 300 expands to wrap any
  observed CCI value so the line never clips off the panel.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.

## Defaults

- threshold = 0
- oscillator range = +/- 300 (auto-expands)
