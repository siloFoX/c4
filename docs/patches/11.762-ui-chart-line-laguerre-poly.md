# 11.762 chart-line-laguerre-poly (v1.11.780)

## Summary

Pure-SVG single-panel React/TypeScript chart primitive with an
**Ehlers Laguerre polynomial** smoother overlay. The filter
applies a four-stage cascaded gamma recurrence to the close.
Added as the `<ChartLineLaguerrePoly />` UI primitive in
`web/src/components/ui/`.

## Algorithm

With damping `gamma in [0, 1]`:

```
L0[i] = (1 - gamma) * close[i] + gamma * L0[i - 1]
L1[i] = -gamma * L0[i] + L0[i - 1] + gamma * L1[i - 1]
L2[i] = -gamma * L1[i] + L1[i - 1] + gamma * L2[i - 1]
L3[i] = -gamma * L2[i] + L2[i - 1] + gamma * L3[i - 1]
Laguerre[i] = (L0[i] + 2 * L1[i] + 2 * L2[i] + L3[i]) / 6
```

Seeds at `i = 0`: `L0 = L1 = L2 = L3 = close[0]`, so
`Laguerre[0] = close[0]` exactly. A non-finite close nulls the
bar and resets the cascade.

## Anchors

- **`K = 0` with any `gamma`**: all stages remain zero so
  `Laguerre = 0` bit-exact at every bar (verified across
  `gamma in {0.1, 0.5, 0.7, 0.8, 0.9, 0.99}`).
- **`gamma = 0` collapses to a (1, 2, 2, 1) / 6 polynomial
  weighted average**:
  `Lk[i] = L_{k - 1}[i - 1]` so
  `Laguerre[i] = (close[i] + 2 * close[i - 1] +
  2 * close[i - 2] + close[i - 3]) / 6` for `i >= 3`. Worked
  anchor on `[10, 20, 30, 40, 50]`:
  `Laguerre[3] = (40 + 60 + 40 + 10) / 6 = 25` bit-exact, and
  `Laguerre[4] = (50 + 80 + 60 + 20) / 6 = 35` bit-exact.
- **`gamma = 1` freezes the cascade at the seed**: every stage
  retains `close[0]`, so `Laguerre = close[0]` from i = 0
  onwards.
- **`CONST_FLAT (close == K)` at `gamma = 0` yields K** at every
  bar bit-exact (the polynomial average of a constant is the
  constant); at `gamma in (0, 1)` it settles to K within
  numerical tolerance.
- **Hand-computed step**: with seeds `L0=L1=L2=L3=0`, `gamma =
  0.5`, `close = [0, 1]` -> `L0 = 0.5`, `L1 = -0.25`, `L2 =
  0.125`, `L3 = -0.0625`, `Laguerre[1] = 0.1875 / 6 = 0.03125`.
- **Polynomial weights**: `(1 + 2 + 2 + 1) / 6 = 1` exactly in
  IEEE 754.
