## QStick pipeline (Chande)

```
body[i]   = close[i] - open[i]
qstick[i] = SMA(body[i-length+1..i], length)
```

QStick is the rolling average of candle bodies: positive when
closes have tended to finish above their opens over the
window (buying pressure), negative when they tend to finish
below (selling pressure). Mirrors the canonical formula used
by `chart-line-qstick-cross-sig`,
`chart-line-qstick-overbought-cross`, and
`chart-line-qstick-oversold-cross`.

Warmup is `length - 1`: the rolling SMA of bodies needs
`length` valid OHLC bars to seed.

## Divergence regime model

5-state classifier exactly like the other divergence primitives:

```
priceUp  = close[i]  > close[i - divergenceWindow]
qstickUp = qstick[i] > qstick[i - divergenceWindow]
state    = aligned-bullish    if priceUp && qstickUp
         | aligned-bearish    if !priceUp && !qstickUp
         | divergent-bullish  if !priceUp && qstickUp   (price down, QStick up)
         | divergent-bearish  if priceUp && !qstickUp   (price up, QStick down)
         | none               if either input is null
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
```

Defaults: `length = 14` (Chande's canonical QStick window),
`divergenceWindow = 5`. Crosses suppress when either `prev` or
`cur` is `none`, so warmup transitions never fire spuriously.
Strict inequality guarantees boundary equality (QStick
saturated) falls into the aligned-bearish state instead of
double-firing.

## Bit-exact anchors

| input                                | qstick (post-warmup) | regime (post i=18)  | crosses |
| ------------------------------------ | -------------------- | ------------------- | ------- |
| CONST OHLC = K                       | 0                    | aligned-bearish     | 0       |
| LINEAR UP (open=i-0.5, close=i+0.5)  | +1 (saturated)       | divergent-bearish   | 0       |
| LINEAR DOWN (open=-i+0.5, close=-i-0.5) | -1 (saturated)    | aligned-bearish     | 0       |

CONST OHLC = K verified across K = 0..1234 -- body = close -
open = K - K = 0 for every bar, so qstick collapses to 0 and
priceUp = false (close[i] === close[i-5]).

LINEAR UP exercises the constant-body case: body =
(i + 0.5) - (i - 0.5) = +1 every bar, so qstick = 1 from
i=13 onward. priceUp = true, qstickUp = false (1 === 1) ->
regime `divergent-bearish` for the full 62 valid samples on
the 80-bar fixture. The cross count stays at 0 because the
regime is entered from `none`.

LINEAR DOWN is the algebraic mirror: body =
(-i - 0.5) - (-i + 0.5) = -1, so qstick = -1. priceUp =
false, qstickUp = false -> regime `aligned-bearish`.
