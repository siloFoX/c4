## Summary

Add a new `<ChartParallel>` UI primitive in
`web/src/components/ui/chart-parallel.tsx` -- pure-SVG
parallel coordinates plot. Each dimension becomes a
vertical axis; each row becomes a polyline crossing every
axis at the value for that dimension. Drag on an axis to
brush a y-range -- rows whose values fall within every
active brush stay visible; the rest fade out. Hovering a
row highlights it and dims the others. Brushes are fully
controllable via `brushes` + `onBrushesChange` for
host-driven filter state. Double-click an axis to clear
its brush.

## API

```ts
interface ChartParallelDimension {
  id: string;
  label: string;
  min?: number;
  max?: number;
  format?: (v: number) => string;
}

interface ChartParallelRow {
  id: string;
  label?: string;
  values: Record<string, number>;
  color?: string;
  category?: string;
}

interface ChartParallelBrush {
  dimensionId: string;
  min: number;
  max: number;
}

interface ChartParallelProps {
  dimensions: readonly ChartParallelDimension[];
  rows: readonly ChartParallelRow[];
  width?: number;                  // default 640
  height?: number;                 // default 320
  padding?: number;                // default 36
  brushes?: readonly ChartParallelBrush[];
  defaultBrushes?: readonly ChartParallelBrush[];
  onBrushesChange?: (brushes: ChartParallelBrush[]) => void;
  showAxisLabels?: boolean;        // default true
  showAxisTicks?: boolean;         // default true
  showTooltip?: boolean;           // default true
  highlightOnHover?: boolean;      // default true
  animate?: boolean;               // default true
  className?: string;
  ariaLabel?: string;              // default 'Parallel coordinates plot'
  ariaDescription?: string;
  defaultLineColor?: string;       // default '#2563eb'
  lineOpacity?: number;            // default 0.45
  strokeWidth?: number;            // default 1.2
  tickCount?: number;              // default 5
  onRowClick?: (args: {
    row: ChartParallelRow;
    index: number;
  }) => void;
}
```

## Behaviour

- Renders a `<div role="region">` wrapping `<svg
  role="img">` carrying the same `aria-label`.
- Axes: one vertical axis per dimension, evenly
  spaced across the inner width. Each axis has its
  own `(min, max)` derived from
  `getParallelDimensionBounds` -- explicit
  `dimension.min` / `dimension.max` win; otherwise
  the data range is used. Collapsed range expands
  `+/- 0.5`; empty / non-finite falls back to
  `(0, 1)`.
- Rows: each row becomes a polyline that crosses
  every axis at the projected y for that dimension's
  value. Rows missing a dimension value snap to that
  axis's centre so the polyline stays continuous.
- Brushing: clicking + dragging within an axis's
  16-px hit-rect creates / replaces the brush for
  that dimension. While dragging, the brush emits
  `onBrushesChange` on every move so host state stays
  in sync. Brushes are AND-combined: a row must pass
  every active brush to remain unfiltered.
- Filtered rows render with reduced opacity (0.08)
  rather than disappearing so adopters can still see
  the overall shape of the data.
- Hover highlight: hovering a row raises its
  opacity to ~0.95 and dims every other row to
  `<= 0.18`. `highlightOnHover=false` removes the
  dim.
- Brush clear: double-click anywhere on an axis hit
  rect to clear that dimension's brush.
- Tooltip: hover opens a tooltip with the row
  label and per-dimension `<label>: <value>` rows.
  Filtered rows show a `"filtered out"` hint.
- Mount animation:
  `motion-safe:animate-fade-in` per row group on
  mount. `animate=false` removes the class.
- Default ARIA description: `"Parallel coordinates
  plot with D dimensions and R rows[ (F after
  brush)]."` -- the optional suffix only appears
  when at least one brush is active.

## Pure helpers (exported)

```ts
DEFAULT_CHART_PARALLEL_WIDTH         = 640
DEFAULT_CHART_PARALLEL_HEIGHT        = 320
DEFAULT_CHART_PARALLEL_PADDING       = 36
DEFAULT_CHART_PARALLEL_TICK_COUNT    = 5
DEFAULT_CHART_PARALLEL_LINE_OPACITY  = 0.45
DEFAULT_CHART_PARALLEL_STROKE_WIDTH  = 1.2
DEFAULT_CHART_PARALLEL_LINE_COLOR    = '#2563eb'

getParallelDimensionBounds(dim, rows): { min, max }
isRowInBrushes(row, brushes): boolean
applyBrushFilter(rows, brushes): ChartParallelRow[]
buildParallelRowPath(points): string
getParallelTicks(min, max, count?): number[]
describeParallelChart(dimensions, rows, brushes?): string
```

- `getParallelDimensionBounds` honours
  `dim.min` / `dim.max` first; otherwise scans row
  values; falls back to `(0, 1)`; expands collapsed
  range.
- `isRowInBrushes` AND-combines every brush; rows
  with non-finite values for any brushed dimension
  fail the check.
- `applyBrushFilter` is the canonical filter pass;
  no-brushes returns a shallow copy.
- `buildParallelRowPath` emits a Move + L segments
  in order.
- `describeParallelChart` produces the default ARIA
  summary; the brush suffix is appended when
  brushes is non-empty.
