# 11.809 UI: chart-line-adv-decline primitive (v1.11.827)

## Summary
Pure-SVG dual-panel React/TS primitive that plots the **Advance Decline
line** -- a classic market-breadth oscillator -- beneath a price/index
close. The line is the rolling window sum of per-bar net breadth
`advances - declines` across the lookback.

## Formula
```
net[i]    = advances[i] - declines[i]
adLine[i] = sum_{k = i - length + 1 .. i} net[k]
```
`adLine[i]` is `null` during warmup (`i < length - 1`) or when any
input is non-finite.

## Input shape
```ts
interface ChartLineAdvDeclinePoint {
  x: number;
  close: number;
  advances: number;
  declines: number;
}
```

## Defaults
- `length = 14`
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `positive`: `adLine > 0` (advancers dominate the window)
- `negative`: `adLine < 0` (decliners dominate)
- `zero`: `adLine == 0` (balanced)
- `none`: warmup / non-finite
- `crossed`: `'up'` when prev value was `<= 0` and current is `> 0`;
  `'down'` is the mirror. Markers fire on cross bars only.

## Bit-exact testability anchors
- **CONST advances=A, declines=D**: `net = A - D` at every bar, so the
  rolling sum is `(A - D) * length` at every valid bar (integer-exact
  in IEEE 754 within `Number.MAX_SAFE_INTEGER`). Verified across
  `A in {0, 5, 100}`, `B in {0, 3, 50}`, and `length in {3, 4, 7, 10}`
  in the integration sweep.
- **EQUAL** (`A = D`): line collapses to exactly zero -> zone `'zero'`
  at every valid bar, no crosses.
- **`A > D`**: line is positive `(A - D) * length` -> zone `'positive'`.
- **`A < D`**: line is negative -> zone `'negative'`.
- `-0 -> +0` normalization applied to both `net` and `rollingSum` to
  keep all CONST tests `Object.is`-bit-exact.

## Design notes
- The "rolling window sum" approach treats Advance Decline as a
  bounded-window oscillator rather than the classic unbounded
  cumulative line. The user can simulate the unbounded line by
  passing `length` equal to the data length.
- AD axis auto-scales but always includes the zero line so the
  reference is visually consistent.
- Default colors: A/D line `#d97706` (amber), bullish `#16a34a`,
  bearish `#dc2626`, zero line `#475569` dashed.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` /
  `?? 0` and defensive guards on every window access.
