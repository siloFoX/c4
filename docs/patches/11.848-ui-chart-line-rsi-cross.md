## Summary

Added `<ChartLineRsiCross />` -- a pure-SVG, dual-panel React/TS
primitive that renders a close line in the top panel and a
short-vs-long Wilder RSI crossover oscillator in the bottom panel.
Scatter markers light up at every momentum-regime cross. `up`
means the short RSI newly exceeded the long RSI; `down` means it
newly fell below. Defaults: `shortLength = 7`, `longLength = 21`
(quick-vs-positional pair), shortColor `#fb7185` (rose), longColor
`#1d4ed8` (blue), bullish marker `#16a34a`, bearish marker
`#dc2626`. Midline drawn at RSI = 50 for visual reference.

## Formula

```
gain[i] = max(0, close[i] - close[i - 1])
loss[i] = max(0, close[i - 1] - close[i])
avgGain = Wilder(gain, n);  avgLoss = Wilder(loss, n)
RSI(n)  = avgLoss === 0 && avgGain === 0 ? 50
        : avgLoss === 0                  ? 100
        : avgGain === 0                  ? 0
        : 100 - 100 / (1 + avgGain / avgLoss)
```

`shortRsi` is the RSI evaluated at `shortLength`; `longRsi` is the
RSI evaluated at `longLength`. Cross events flag the bar where the
strict-inequality between the two newly flips (prev relation guard
+ strict-inequality on current side keeps repeated samples from
double-firing).

## Bit-exact anchors

The implementation reuses the SMA-seeded Wilder helper with the
`min === max` precision fix from 11.831 TRIX so that all three
degenerate inputs land bit-exactly without any floating-point drift:

- **CONST close = K**: every gain = 0 and every loss = 0 ->
  avgGain = avgLoss = 0 -> RSI = 50 (neutral fallback) for both
  periods. The relation stays equal so zero cross events fire.
- **LINEAR UP close = i + 1**: every change is +1; gain = 1
  constant, loss = 0. Wilder of a constant 1 seed equals 1 exactly,
  so avgGain = 1 and avgLoss = 0 -> RSI = 100 for both periods.
  Zero crosses.
- **LINEAR DOWN close = N - i**: gain = 0, loss = 1 -> RSI = 0
  for both periods. Zero crosses.
