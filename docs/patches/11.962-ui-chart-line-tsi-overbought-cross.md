# Patch 11.962 - UI: chart-line-tsi-overbought-cross primitive

Version: 1.11.980

## Summary

Add `web/src/components/ui/chart-line-tsi-overbought-cross.tsx` -- a
pure-SVG dual panel line chart primitive that visualizes price alongside
William Blau's True Strength Index (TSI) panel, with explicit cross
markers and overlay arrows at every TSI crossing of the canonical +25
overbought line. Bullish (= entry, crossing UP through 25 = entering
overbought) and bearish (= exit, crossing DOWN through 25 = leaving
overbought). Shares the bit-exact TSI pipeline with the `-zero-cross`
sibling but tracks the overbought threshold instead of the centerline.

## Pipeline

Identical to `chart-line-tsi-zero-cross`. Threshold default flipped to
25.

## Bit-exact anchors

| Pattern                  | TSI (after warmup) | Regime  |
|--------------------------|--------------------|---------|
| CONST close = K          | 0 (0/0 -> 0)       | bearish |
| LINEAR UP close = i      | 100                | bullish |
| LINEAR DOWN close = -i   | -100               | bearish |

CONST anchor tested bit-exactly across K in `{0, 1, 42, 100, 1234}`
(regime bearish since 0 < 25 -- opposite side from the zero-cross
variant where 0 sat on the boundary). LINEAR UP reaches +100 (bullish,
well above 25). LINEAR DOWN reaches -100 (bearish, well below 25).

Cross events fire during transients:
* decline-then-rise: tsi transitions from -100 through 25 going up ->
  bullish entry through 25.
* rise-then-decline: tsi transitions from 100 through 25 going down ->
  bearish exit through 25.

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), TSI panel below.
* TSI panel uses a fixed `[-100, 100]` range; threshold band sits above
  the midpoint (= 0) at the +25 line.
* Cross markers drawn as circles on the TSI panel at `(cx, cyOsc)` and
  as arrows on the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="TSI Overbought Cross chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only
  text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
