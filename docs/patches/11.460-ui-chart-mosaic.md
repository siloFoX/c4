## Summary

Add a new `<ChartMosaic>` UI primitive in
`web/src/components/ui/chart-mosaic.tsx` -- pure-SVG
mosaic plot for 2D contingency tables. Each column's
width is proportional to its marginal total; within each
column, row cells stack with heights proportional to the
conditional counts. Cell fill encodes the standardized
chi-square residual `(O - E) / sqrt(E)` -- blue for
positive (more than expected), red for negative (less
than expected). Hovering a cell shows count, expected
count, residual, and percent of total. Per-cell click
handler.

## Behaviour

- Renders a `<div role="region">` wrapping `<svg
  role="img">` carrying the same `aria-label`.
- Totals: `getMosaicTotals` ignores non-finite and
  non-positive counts. Row, column, and grand totals
  drive the geometry.
- Layout: `buildMosaicLayout` walks `columnOrder` and
  `rowOrder`. Each column width =
  `(colTotal / grand) * innerWidth`; within a column,
  each cell height =
  `(count / colTotal) * innerHeight`. Empty cells
  collapse to zero width / height but still appear in
  the layout list so hover lookup remains stable.
- Empty columns (zero marginal total) are skipped
  entirely so they don't consume horizontal space.
- Expected count: `getMosaicExpectedCount(R, C, N) =
  (R * C) / N`. Returns 0 for non-positive grand
  total.
- Residual: `getMosaicResidual(O, E) = (O - E) /
  sqrt(E)`. Returns 0 when expected is non-positive
  or inputs are non-finite.
- Fill colour: `getMosaicResidualColor` interpolates
  from `neutralColor` toward `positiveColor` for
  positive residuals and toward `negativeColor` for
  negative residuals. The interpolation ratio is
  `|residual| / clamp` clamped to `[0, 1]` so very
  large residuals don't saturate the palette beyond
  the configured ceiling.
- Labels: in-cell count label renders when the cell
  is at least 32px wide and 14px tall so cramped
  cells stay readable.
- Row + column axis labels render outside the chart;
  each is individually suppressible via
  `showRowLabels` / `showColumnLabels`.
- Residual legend: small three-swatch row in the
  bottom-right (`negativeColor` / `neutralColor` /
  `positiveColor`) with a "residual" caption.
- Hover / focus: thickens the cell border, opens a
  tooltip with row x column, count, expected,
  residual, and percent of total.
- Mount animation:
  `motion-safe:animate-fade-in` per cell group.
  `animate=false` removes the class.
- Default ARIA description: `"Mosaic plot with R rows
  and C columns. Grand total <N>."` --
  `ariaDescription` overrides; empty / all-zero input
  produces `"No data"`.

## Pure helpers (exported)

```ts
DEFAULT_CHART_MOSAIC_WIDTH           = 540
DEFAULT_CHART_MOSAIC_HEIGHT          = 320
DEFAULT_CHART_MOSAIC_PADDING         = 36
DEFAULT_CHART_MOSAIC_CELL_GAP        = 1
DEFAULT_CHART_MOSAIC_POSITIVE_COLOR  = '#1d4ed8'
DEFAULT_CHART_MOSAIC_NEGATIVE_COLOR  = '#b91c1c'
DEFAULT_CHART_MOSAIC_NEUTRAL_COLOR   = '#f1f5f9'
DEFAULT_CHART_MOSAIC_RESIDUAL_CLAMP  = 3

getMosaicFirstSeen(cells, field): string[]
getMosaicTotals(cells): { rows, columns, grand }
getMosaicExpectedCount(R, C, N): number
getMosaicResidual(O, E): number
buildMosaicLayout(cells, rows, cols, innerW, innerH,
                  originX, originY, gap): MosaicLayoutCell[]
getMosaicResidualColor(residual, posColor, negColor,
                       neutralColor, clamp?): string
describeMosaicChart(cells, formatCount?): string
```

- `getMosaicFirstSeen` is the canonical first-seen
  order helper for either field. Used as a fallback
  when `rowOrder` / `columnOrder` are omitted.
- `getMosaicTotals` is the single-pass marginal +
  grand sum helper. Non-finite / non-positive counts
  are silently dropped.
- `getMosaicExpectedCount` returns 0 for non-positive
  grand total so downstream residual stays defined.
- `getMosaicResidual` is the standardized Pearson
  residual; sign reflects over- vs under-
  representation.
- `buildMosaicLayout` is the canonical layout pass.
  Empty columns are skipped entirely; empty rows
  remain in the layout list with zero geometry so
  hover lookup stays stable.
- `getMosaicResidualColor` interpolates via the
  shared `interpolateColor` from `<ChartFunnel>`;
  `|residual| > clamp` collapses to the clamp colour
  (no over-saturation).
- `describeMosaicChart` produces the default ARIA
  summary; all-zero input returns `"No data"`.

  non-positive grand -> 0, non-finite -> 0.
