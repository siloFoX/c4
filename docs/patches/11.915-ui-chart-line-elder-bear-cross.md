# Patch 11.915 -- UI: chart-line-elder-bear-cross primitive

## Summary

Added `<ChartLineElderBearCross />` -- pure-SVG dual-panel
React/TS primitive that renders the close (top panel) with
bullish / bearish arrow overlays at every Elder Bear Power
zero crossover, and the close-only Bear Power line (bottom
panel) with cross dot markers and a dashed zero baseline.
Zero-cross variant of the Elder Ray family that flags seller
strength regime transition events distinct from the absolute
Bear Power magnitude. Mirror of 11.914 `chart-line-elder-bull-
cross` using the rolling low instead of the rolling high.

## Algorithm

```
lowProxy[i] = min(close[i-h+1..i])                (close-only)
ema[i]      = EMA(close, length)
bear[i]     = lowProxy - ema
bullish    : bear crosses up   (prev <= 0, cur > 0)
bearish    : bear crosses down (prev >= 0, cur < 0)
```

Defaults `length = 13` (canonical Elder Ray EMA window),
`lowProxyLength = 2` (close-only low proxy that synthesizes
the recent low from a 2-bar window of closes). Regime
classifier `bullish` (bear > 0 meaning weak bears), `bearish`
(bear < 0 meaning strong bears), `neutral` (bear === 0),
`none` (bear null).

## Bit-exact anchor

CONST close = K: lowProxy = K every bar; EMA(K) = K via the
`min === max` precision short-circuit; bear = K - K = 0.
Regime is `neutral`, cross count = 0. Verified across K = 0,
1, 5, 17, 100, 1234.

LINEAR DOWN yields bear < 0 every bar (lowProxy = current
close, EMA lags above current). LINEAR UP yields bear > 0
because the rolling low remains above the lagging EMA.
