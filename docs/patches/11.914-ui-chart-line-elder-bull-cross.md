# Patch 11.914 -- UI: chart-line-elder-bull-cross primitive

## Summary

Added `<ChartLineElderBullCross />` -- pure-SVG dual-panel
React/TS primitive that renders the close (top panel) with
bullish / bearish arrow overlays at every Elder Bull Power
zero crossover, and the close-only Bull Power line (bottom
panel) with cross dot markers and a dashed zero baseline. Zero-
cross variant of the Elder Ray family that flags buyer
strength regime transition events distinct from the absolute
Bull Power magnitude.

## Algorithm

```
highProxy[i] = max(close[i-h+1..i])               (close-only)
ema[i]       = EMA(close, length)
bull[i]      = highProxy - ema
bullish     : bull crosses up   (prev <= 0, cur > 0)
bearish     : bull crosses down (prev >= 0, cur < 0)
```

Defaults `length = 13` (canonical Elder Ray EMA window),
`highProxyLength = 2` (close-only high proxy that synthesizes
the recent high from a 2-bar window of closes). Regime
classifier `bullish` (bull > 0), `bearish` (bull < 0),
`neutral` (bull === 0), `none` (bull null).

## Bit-exact anchor

CONST close = K: highProxy = K every bar; EMA(K) = K via the
`min === max` precision short-circuit; bull = K - K = 0.
Regime is `neutral`, cross count = 0. Verified across K = 0,
1, 5, 17, 100, 1234.

LINEAR UP yields bull > 0 every bar (highProxy = current
close, EMA lags). LINEAR DOWN yields bull < 0 every bar
(highProxy = previous close, EMA lags above current).
