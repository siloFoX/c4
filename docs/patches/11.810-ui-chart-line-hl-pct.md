# 11.810 UI: chart-line-hl-pct primitive (v1.11.828)

## Summary
Pure-SVG dual-panel React/TS primitive that plots the **High Low
percent of price** oscillator beneath the close. Each bar's range
`(high - low)` is scaled to a percentage of `|midpoint|`, then
smoothed with a rolling SMA across the lookback. Markers fire on
volatility-threshold crossings.

## Formula
```
midpoint[i] = (high[i] + low[i]) / 2
raw[i]      = (high[i] - low[i]) / |midpoint[i]| * 100
hlPct[i]    = SMA(raw, length)[i]
```
`raw[i]` is `null` when `midpoint == 0` (flat-at-zero bar);
`hlPct[i]` is `null` during warmup (`i < length - 1`) or when any
raw value in the window is `null`.

## Defaults
- `length = 14`
- `volatilityThreshold = 5` (5% range relative to midpoint)
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `high`: `hlPct >= volatilityThreshold`
- `low`: `0 < hlPct < volatilityThreshold`
- `flat`: `hlPct == 0`
- `none`: warmup / non-finite
- `crossed`: `'up'` when prev value was `< threshold` and current is
  `>= threshold`; mirror for `'down'`. Markers fire only on cross bars.

## Bit-exact testability anchors
- **CONST high=low=K** (`K != 0`): `raw = 0` at every bar, SMA(0) = 0
  -> `hlPct = 0` (bit-exact). Verified across `K in {1, 5, 100, -3}`
  and `length in {3, 4, 7, 10}`.
- **CONST high=12, low=8**: midpoint = 10, `raw = 4 / 10 * 100 = 40`
  (integer dyadic arithmetic), SMA = 40 -> `hlPct = 40` (bit-exact).
  Verified across the length sweep.
- **CONST high=11, low=9**: midpoint = 10, `raw = 2 / 10 * 100 = 20`,
  SMA = 20 (bit-exact).
- **CONST high=low=0**: midpoint = 0 -> raw = null -> hlPct = null
  everywhere (divide-by-zero guard).
- **Negative midpoint**: `Math.abs(midpoint)` keeps `raw` non-negative
  -- verified with `h=-8, l=-12` -> midpoint = -10, raw = 40.

## Design notes
- `Math.abs(midpoint)` guards against negative-midpoint sign-flip:
  the indicator should report the *magnitude* of the bar range
  relative to its price level, not its sign.
- HL% panel y-axis is **0-based with auto max**: `hlPctMax =
  max(observed, threshold * 1.25)` so the threshold line stays
  visible even when the data is all-low-volatility.
- Threshold line dashed `#475569`, distinct from grid.
- Markers anchored to the hlPct value on the cross bar.
- Default colors: HL% line `#db2777` (pink), bullish `#16a34a`,
  bearish `#dc2626`, threshold `#475569`.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
