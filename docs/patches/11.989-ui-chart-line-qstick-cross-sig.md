## Summary

Add `web/src/components/ui/chart-line-qstick-cross-sig.tsx` --
a pure-SVG dual-panel React/TypeScript chart primitive that
renders the close in the top panel and the QStick (SMA of close
- open candle body) plus its smoothed signal line in the bottom
panel, marking discrete QStick-over-signal crossover events.
Signal-line variant of the QStick family that flags the discrete
QStick crossing of its own SMA-smoothed signal line -- the
canonical trigger Tushar Chande uses to filter raw-QStick
whipsaws and confirm candle-body momentum shifts.

QStick differs from the rest of the family in requiring true
OHLC-shaped input -- the data interface is `{ x, open, close }`
because the body = close - open is the underlying measurement.

## Math anchor (bit-exact)

```
body[i]    = close[i] - open[i]
qstick[i]  = SMA(body, length)
signal[i]  = SMA(qstick, kSmoothing)
bullish    : prev qstick <= prev signal && cur qstick >  cur signal
bearish    : prev qstick >= prev signal && cur qstick <  cur signal
```

- **CONST open == close == K**: body = 0 -> qstick = 0 constant
  (SMA `min === max` short-circuit). signal = SMA(0, 3) = 0.
  qstick == signal == 0 -> regime `bullish` (boundary
  inclusive). 0 crosses. Verified across K = 0..1234.
- **LINEAR UP body = +1** (open[i] = i - 1, close[i] = i): body
  = 1 every bar -> qstick = 1 constant. signal = 1. qstick ==
  signal == 1 -> regime `bullish`. 0 crosses.
- **LINEAR DOWN body = -1** (open[i] = i + 1, close[i] = i):
  body = -1 every bar -> qstick = -1. signal = -1. qstick ==
  signal == -1 -> regime `bullish` (boundary inclusive at the
  saturated bear extreme). 0 crosses.

For 50 bars at defaults (length=8, kSmoothing=3):
- qstick null for i = 0..6 (7 entries)
- signal null for i = 0..8 (9 entries)
- post-warmup regime count = 41 (i = 9..49)

Strict inequality on `cur` (`cur > signal` for bullish, `cur <
signal` for bearish) means boundary equality (qstick == signal)
never double-fires.

Layout uses symmetric range around 0 (oscMin = -1.1 * span,
oscMax = +1.1 * span where span = max(|qsMin|, |qsMax|))
combining both QStick and signal extrema, with fallback to [-1,
1] when all values collapse to exactly 0.

## Accessibility

- Wrapping `<div role="region" aria-label="QStick Signal Cross
  chart" aria-describedby="...">` with the sr-only description
  copy carrying the length / kSmoothing parameters plus a
  "smoothed candle-body momentum trigger" framing.
- `<svg role="img" aria-labelledby="..." aria-describedby="..."
  tabIndex={0}>` for keyboard focus.
- Each cross marker (both the QStick-panel dot and the price-
  panel chevron overlay) carries `role="graphics-symbol"`,
  `tabIndex={0}`, and an `aria-label="<bullish|bearish> cross at
  <x>"` so screen readers can step through events.
- Legend buttons (3 series: price, qstick, signal) expose
  `aria-pressed` and toggle via mouse, focus, Enter, or Space.
