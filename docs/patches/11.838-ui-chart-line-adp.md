# Patch 11.838 -- ChartLineAdp primitive (v1.11.856)

## Summary

Add `<ChartLineAdp />`, a pure-SVG dual-panel line chart with an
Advance-Decline Percent breadth oscillator beneath a price line. The
oscillator aggregates advancing and declining issues across a
fixed-bar lookback and scales the result by the rolling total issues:

```
rollAdv[i] = sum(advances[i - length + 1 .. i])
rollDec[i] = sum(declines[i - length + 1 .. i])
rollTot[i] = sum(total[i    - length + 1 .. i])
adp[i]     = rollTot[i] === 0
               ? null
               : (rollAdv[i] - rollDec[i]) / rollTot[i] * 100
```

- `total[i]` falls back to `advances[i] + declines[i]` when not
  supplied via `totalIssues`.
- Warmup: `i < length - 1` -> `null`.
- `null` whenever the rolling total is zero (divide-by-zero guard).
- `-0` normalized to `+0` on every arithmetic result.

Distinct from the rolling-sum-of-difference primitive 11.809
`chart-line-adv-decline` (which returns absolute counts) -- this
primitive normalizes by total issues to produce a `[-100, 100]`
oscillator.

## Defaults

| Knob              | Value     |
|-------------------|-----------|
| length            | 10        |
| bullishThreshold  | +30       |
| bearishThreshold  | -30       |
| adpColor          | `#f43f5e` (rose) |

Thresholds at +/-30 match typical breadth-indicator conventions
(NYSE McClellan-style oscillators flag market breadth above +30 and
below -30 as extreme).

## Bit-exact anchors

The integration sweep iterates `(A, D, T, length)` and asserts:

- `CONST adv = A, dec = D, total = T, T != 0`: rolling sums become
  `L*A, L*D, L*T`. The `L` factor cancels exactly in the division,
  so `adp = (A - D) / T * 100`. Verified bit-exact for the dyadic
  ratio combinations:
  - `(100, 0, 100) -> 100`
  - `(0, 100, 100) -> -100`
  - `(50, 50, 100) -> 0`
  - `(75, 25, 100) -> 50`
  - `(25, 75, 100) -> -50`
  - `(200, 0, 200) -> 100`
  - `(50, 0, 100) -> 50`
  - `(0, 50, 100) -> -50`
- `CONST total = 0`: divide-by-zero guard returns `null`.

The `L` cancellation works in IEEE 754 because division of two
integer-representable quantities by the same `L` gives identical
quotient as the unscaled version, and the chosen ratios reduce to
exact dyadics or simple integers.

## Surface

Standard primitive surface:

- `ChartLineAdpPoint = { x, close, advances, declines, totalIssues? }`
- `runLineAdp(data, options?) -> Run` with
  `ok = series.length >= length`.
- `computeLineAdpLayout({ data, width?, height?, padding?, panelGap?, length?, bullishThreshold?, bearishThreshold? }) -> Layout`.
  ADP axis fixed to `[-100, 100]`.
- `describeLineAdpChart(data, opts?) -> string` for the ARIA
  description.
- `<ChartLineAdp />` `forwardRef<HTMLDivElement>` with legend toggle
  (`price`, `adp`), config badge, threshold + zero-line overlays,
  markers on cross events, and a tooltip surfacing
  `close / adv / dec / total / rollAdv / rollDec / rollTot / adp /
  zone`.
