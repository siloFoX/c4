# 11.926 -- UI: chart-line-adx-strength-cross primitive (v1.11.944)

## Summary

Adds `ChartLineAdxStrengthCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and a close-only
Average Directional Index (ADX) line in the bottom panel.
Marks mild (cross up / down through 25) and strong (cross up
/ down through 50) trend-strength threshold transitions. Dual-
band cross variant of the ADX family that flags discrete ADX
level `25` and `50` entry / exit events on the same panel.

## Pipeline

For close-only data with default `length = 14`:

```
tr[i]  = |close[i] - close[i-1]|
up[i]  = close[i] - close[i-1]
dn[i]  = close[i-1] - close[i]
+DM[i] = up > dn && up > 0 ? up : 0
-DM[i] = dn > up && dn > 0 ? dn : 0
+DI    = 100 * Wilder(+DM, n) / Wilder(tr, n)   (0 when tr = 0)
-DI    = 100 * Wilder(-DM, n) / Wilder(tr, n)   (0 when tr = 0)
DX     = 100 * |+DI - -DI| / (+DI + -DI)        (0 when sum = 0)
ADX    = Wilder(DX, n)
mildEnter   : prev <= 25 && cur > 25
strongEnter : prev <= 50 && cur > 50
mildExit    : prev >= 25 && cur < 25
strongExit  : prev >= 50 && cur < 50
```

When ADX moves through both bands in one bar (e.g. 10 -> 80),
both `mildEnter` and `strongEnter` fire at the same index.
Same for the symmetric exit case.

## Bit-exact anchor

- CONST `close = K` for any finite K -> delta = 0 every bar ->
  tr = +DM = -DM = 0 -> Wilder smoothing of zeros = 0 -> +DI =
  -DI = 0 via the 0/0 short-circuit -> DX = 0/0 -> Wilder(DX)
  = 0 -> adx = 0 sits firmly below 25, regime is `weak`, cross
  count = 0. Verified across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` (length 14) -> all deltas = +1 -> +DM =
  1, -DM = 0, tr = 1 -> Wilder seeds at 1 / 0 / 1 -> +DI = 100,
  -DI = 0, DX = 100 -> adx converges to 100 at index 27, stable
  at 100 for the tail. regime `veryStrong`, 0 crosses (adx
  jumps from null to 100 in one bar so prev-null skips the
  detector).
- LINEAR DOWN `close = -i` mirrors LINEAR UP -> adx = 100,
  regime `veryStrong`.
- Alternating `close = [10, 11, 10, 11, ...]` (no directional
  bias) -> +DI and -DI balance at 50/50 -> DX -> 0 -> adx
  trends low; stays well below the mild threshold (< 25) and
  remains in the `weak` regime over a long window.

## Regime classifier

| adx range            | regime      |
|----------------------|-------------|
| adx >= 50            | veryStrong  |
| 25 <= adx < 50       | strong      |
| adx < 25             | weak        |
| null                 | none        |
