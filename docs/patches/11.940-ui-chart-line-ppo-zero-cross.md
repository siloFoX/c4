# 11.940 -- UI: chart-line-ppo-zero-cross primitive (v1.11.958)

## Summary

Adds `ChartLinePpoZeroCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and the close-only
Percentage Price Oscillator (PPO) line in the bottom panel.
Marks bullish (cross up through zero) / bearish (cross down
through zero) momentum baseline regime transition events.
Zero-line cross variant of the PPO family that flags the
discrete PPO crossing of the zero baseline. PPO is the
percentage version of MACD, scale-invariant for cross-
instrument comparison.

## Pipeline

For close-only data with defaults `fastLength = 12`,
`slowLength = 26`:

```
fastEma  = EMA(close, fastLength)
slowEma  = EMA(close, slowLength)
ppo[i]   = slowEma != 0
             ? (fastEma - slowEma) / |slowEma| * 100
             : 0
bullish  : prev ppo <= 0 && cur ppo > 0
bearish  : prev ppo >= 0 && cur ppo < 0
```

PPO differs from MACD by normalizing the difference by the
slow EMA magnitude, producing a scale-invariant percent
that allows direct comparison across instruments at
different price levels.

## Bit-exact anchor

- CONST `close = K` for any finite K -> fastEma = slowEma =
  K -> numerator = 0 -> ppo = 0 (even when K = 0 the slowEma
  = 0 guard returns 0 instead of NaN). ppo = 0 sits on the
  threshold but the strict-inequality detector never fires.
  regime `bullish` (ppo >= 0). cross count = 0. Verified
  across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i + 100` -> at steady state fastEma -
  slowEma = (slowLength - fastLength) / 2 = 7. slowEma grows
  with i so ppo = 7 / |slowEma| * 100 stays positive but
  decreases toward 0 as i grows. regime `bullish`. 0 crosses.
- LINEAR DOWN `close = 200 - i` -> ppo converges to a
  negative value with magnitude shrinking as i grows. regime
  `bearish`. 0 crosses.
- Decline followed by sustained rise -> PPO transitions
  from negative to positive -> 1 bullish zero-line cross.
- Rise followed by sustained decline -> 1 bearish cross.

## Regime classifier

| ppo range            | regime      |
|----------------------|-------------|
| ppo >= 0             | bullish     |
| ppo < 0              | bearish     |
| null                 | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- One horizontal reference band inside the oscillator panel
  (threshold at zero, the baseline).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind.
- Oscillator range auto-fits to observed PPO values with
  10% padding above and below; PPO is unbounded.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.

## Defaults

- fastLength = 12
- slowLength = 26
- threshold = 0
- oscillator range = auto-fit (PPO is unbounded)
- width 720, height 460, padding 44, panelGap 12
