## Stoch RSI pipeline (Chande / Kroll)

```
rsi[i]      = Wilder RSI(close, length); 50 when
              avgGain + avgLoss === 0 (degenerate)
stochRsi[i] = 100 * (rsi[i] - LLV(rsi, stochLength)) /
              (HHV - LLV); 50 when HHV === LLV (degenerate)
```

Mirrors the canonical Stoch RSI formula already used by
`chart-line-stoch-rsi-mid-cross-sig` (RSI followed by a
stochastic). For the divergence variant the raw stochRsi line
is used directly without the additional %K SMA smoothing the
mid-cross-sig variant applies.

Warmup is `length + stochLength - 1 = 27` for the default
tuning: Wilder RSI seeds at `length = 14` (SMA-seed of
gain/loss arrays starting at index 1), then the stochastic
needs another `stochLength - 1 = 13` valid RSI samples.

Defaults: `length = 14`, `stochLength = 14`, `divergenceWindow
= 5`.

## Divergence regime model

5-state classifier exactly like the other divergence primitives:

```
priceUp    = close[i]    > close[i - divergenceWindow]
stochRsiUp = stochRsi[i] > stochRsi[i - divergenceWindow]
state      = aligned-bullish    if priceUp && stochRsiUp
           | aligned-bearish    if !priceUp && !stochRsiUp
           | divergent-bullish  if !priceUp && stochRsiUp
           | divergent-bearish  if priceUp && !stochRsiUp
           | none               if either input is null
```

Crosses suppress when either `prev` or `cur` is `none`, so
warmup transitions never fire spuriously. Strict inequality
guarantees boundary equality (stochRsi parked at 50) falls
into the aligned-bearish state instead of double-firing.

## Bit-exact anchors

| input              | rsi (post-warmup) | stochRsi (post-warmup)  | regime (post i=32)  | crosses |
| ------------------ | ----------------- | ----------------------- | ------------------- | ------- |
| CONST close = K    | 50 (degenerate)   | 50 (degenerate cascade) | aligned-bearish     | 0       |
| LINEAR UP, close=i | 100 (saturated)   | 50 (rsi flat -> stoch degenerate) | divergent-bearish | 0       |
| LINEAR DOWN, close=-i | 0 (saturated)  | 50 (rsi flat -> stoch degenerate) | aligned-bearish   | 0       |

CONST K verified across K = 0..1234 -- delta close = 0 every
bar, so gain = loss = 0 and the Wilder RSI degenerate
fallback returns 50. Then the rolling stochastic over a flat
RSI = 50 window has hi === lo and returns 50 via the second
degenerate fallback. priceUp = false (close[i] ===
close[i-5]), stochRsiUp = false (50 === 50) -> regime
`aligned-bearish`.

LINEAR UP exercises the saturated-RSI case: delta = +1 every
bar, loss = 0, RSI = 100 saturated. The window of constant 100
again triggers the stochastic degenerate fallback (hi === lo
=== 100) -> stochRsi = 50. priceUp = true, stochRsiUp = false
-> regime `divergent-bearish` for the full 48 valid samples
on the 80-bar fixture.

LINEAR DOWN: delta = -1, gain = 0, RSI = 0, window of zeros ->
stochRsi = 50 (degenerate). priceUp = false, stochRsiUp =
false -> regime `aligned-bearish`.

Note: the stochRsi parking at the neutral 50 on pure linear
input is the algebraic signature of nested degenerate
fallbacks (RSI saturates -> RSI window is flat -> stochastic
returns midline). The divergence regime classification is
correct, and the 0-cross anchor count reflects that the
divergent state is entered from `none` (prev = none is never
a valid cross precursor).
