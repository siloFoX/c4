## Summary

Add a new `<ChartSunburst>` UI primitive in
`web/src/components/ui/chart-sunburst.tsx` -- pure-SVG
**sunburst** (radial hierarchical) chart with **click-to-
zoom into branches**. Each ring is one level of the
hierarchy; each arc encodes its share of its parent.
Clicking any arc makes it the new focus -- its subtree
expands to fill the full 2π and every other branch is
hidden. Clicking the center disc zooms back out one
level toward the root.

## API

```ts
interface ChartSunburstNode {
  id: string;
  label: string;
  value?: number;
  color?: string;

interface ChartSunburstProps {
  root: ChartSunburstNode | null;

  defaultFocusPath?: readonly string[];

  showLabels?: boolean;      // default true
  showTooltip?: boolean;     // default true
```

## Behaviour

- Renders `<div role="region">` wrapping
  `<svg role="img">` with the same `aria-label`.
- Value resolution (`getSunburstNodeValue`): leaves use
  their explicit `value`; non-leaf nodes prefer the sum
  of their children's values (an explicit parent
  `value` is treated as a fallback only when child
  totals collapse to 0). Childless nodes without a
  value contribute 0.
- Layout (`computeSunburstLayout`):
  - Each level of the focus subtree gets its own ring;
    ring count = `maxDepth - focusDepth + 1`.
  - Ring width = `(outerRadius - centerRadius) /
    ringCount`.
  - Inner ring renders the focus node spanning the
    full 2π; child rings stack outward; each child's
    angular span is proportional to its share of the
    parent.
  - Outside the focus subtree, no arcs render; the
    layout still recurses to assign angles for the
    sub-tree relative to the visible parent.
- Click-to-zoom (controlled + uncontrolled):
  - Uncontrolled: `defaultFocusPath` seeds initial
    focus; arc click updates internal focus state.
  - Controlled: `focusPath` always wins; arc click
    fires `onFocusPathChange(nextPath)` and
    `onArcClick({node, becameFocus})` so a parent
    can persist the next focus.
  - Center disc click zooms out by one level (the
    last element of the focus path is dropped). If
    already at the root, the click is a no-op.
- Center disc shows the focus label + focus value;
  has `role="button"` + `tabIndex={0}` so keyboard
  users can zoom out. `showCenterLabel=false` removes
  the entire center group.
- Arc labels paint at the centroid (radius = midpoint
  of inner+outer) when the arc's angular span >=
  `labelMinArc` (default 0.18 rad) AND the ring is at
  least 14 px tall. `showLabels=false` suppresses all
  arc labels.
- Hover/focus on an arc opens a tooltip with the
  full breadcrumb path (`a / b / a1`), the formatted
  value, the share against the focus value, and the
  share against the overall total. `showTooltip=false`
  suppresses.
- Mount animation:
  `motion-safe:animate-fade-in` per arc group.
  `animate=false` removes the class.
- Default ARIA description: `"Sunburst chart with N
  nodes across L levels, total <total>. Focused on
  <focus.label> (depth D)."` Null root / zero total
  -> `"No data"`.

## Pure helpers (exported)

```ts
DEFAULT_CHART_SUNBURST_WIDTH          = 400
DEFAULT_CHART_SUNBURST_HEIGHT         = 400
DEFAULT_CHART_SUNBURST_PADDING        = 24
DEFAULT_CHART_SUNBURST_CENTER_RADIUS  = 36
DEFAULT_CHART_SUNBURST_FILL_OPACITY   = 0.85
DEFAULT_CHART_SUNBURST_LABEL_MIN_ARC  = 0.18
DEFAULT_CHART_SUNBURST_PALETTE        = 10-color palette

getSunburstDefaultColor(index): string
polarToCartesian(cx, cy, r, angle): { x, y }
getSunburstNodeValue(node): number
flattenSunburstHierarchy(root): ChartSunburstFlatNode[]
getSunburstMaxDepth(flat): number
computeSunburstLayout(input): {
  arcs, flat, focusNode, focusValue, maxDepth, ringWidth
}
describeSunburstChart(root, focusPath, formatValue?): string
```

- `flattenSunburstHierarchy` returns one entry per
  node with `{ id, label, depth, parentId, path,
  value, isLeaf, originalIndex, color? }`.
- `computeSunburstLayout` returns the arcs array
  (one arc per visible node), the flat tree, the
  resolved focus node, the focus value, the
  observed max depth (relative to the focus), and
  the computed ring width. Empty arcs are returned
  for null root, non-positive outer radius, or
  zero focus value.

## ARIA + data attributes

- Root: `<div role="region">` + `aria-label`.
- SVG: `role="img"` + same `aria-label`.
- sr-only span:
  `data-section="chart-sunburst-aria-desc"`.
- Each arc path: `role="graphics-symbol"` +
  `tabIndex={0}`. aria-label = `"<label>: <value>
  (<percent>)"`.
- Center disc: `role="button"` + `tabIndex={0}`.
  aria-label = `"Zoom out from <focus>"` (or
  `"Focused on <focus>"` when at root).
- `data-section` on every node:
  chart-sunburst / -canvas / -aria-desc / -svg /
  -arcs / -arc / -path / -arc-label / -center /
  -center-disc / -center-label / -center-value /
  -tooltip / -tooltip-label / -tooltip-value /
  -tooltip-share / -tooltip-global.
- Root mirrors `data-node-count`,
  `data-arc-count`, `data-focus-id`,
  `data-focus-depth`, `data-max-depth`,
  `data-animate`.
- Arc groups mirror `data-arc-id`,
  `data-arc-depth`, `data-arc-parent`,
  `data-arc-value`, `data-arc-share`,
  `data-arc-global-share`, `data-arc-color`,
  `data-arc-is-focus`, `data-hovered`.
