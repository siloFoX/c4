# 11.816 UI: chart-line-recombining-kc primitive (v1.11.834)

## Summary
Pure-SVG **single-panel** React/TS primitive that overlays the close
with a recombining Keltner Channel envelope. Mirror of 11.811
recombining-bb but with **EMA** replacing SMA and **ATR** replacing
standard deviation. On every bar a standard band of `EMA +/- multiplier
* ATR` is computed, but the upper/lower lines **collapse to the EMA
mean** on bars where the close touches or breaches the band
(band-touch reset trigger).

## Formula
```
mean[i]      = EMA(close, length)[i]  (SMA-seeded at i = length - 1)
tr[i]        = max(high - low, |high - prevClose|, |low - prevClose|)
atr[i]       = SMA(tr, atrLength)[i]
fullUpper[i] = mean[i] + multiplier * atr[i]
fullLower[i] = mean[i] - multiplier * atr[i]
recombine[i] = close[i] >= fullUpper[i] || close[i] <= fullLower[i]
upper[i]     = recombine[i] ? mean[i] : fullUpper[i]
lower[i]     = recombine[i] ? mean[i] : fullLower[i]
```
Warmup ends at the later of the EMA seed (`i = length - 1`) and the
ATR fill (`i = atrLength - 1`); markers anchor on the mean at
recombine bars.

## Defaults
- `length = 20` (EMA period)
- `atrLength = 10`
- `multiplier = 2`
- `width = 720`, `height = 400`, `padding = 44`

## Zones
- `recombine`: trigger fired this bar
- `above-mid`: `close > mean` (no recombine)
- `below-mid`: `close < mean`
- `at-mid`: `close == mean`
- `none`: warmup / non-finite

## Bit-exact testability anchors
- **CONST high=low=close=K**: `tr = 0` everywhere -> `atr = 0`; EMA of
  a constant is the constant (SMA seed = K, EMA recurrence preserves
  the value via the **CONST short-circuit**: `next = v === prev ? v :
  alpha * v + (1 - alpha) * prev` -- avoids 1-ULP drift when `alpha =
  2 / (length + 1)` is non-dyadic, e.g. `1/3` at `length = 5`). So
  `mean = K`, `fullUpper = fullLower = K`, and the `>=` trigger fires
  every valid bar. All four series read exactly `K`. Verified across
  `K in {0, 1, 5, 100, -3}`, `length in {3, 5, 7, 10}`, and
  `multiplier in {0, 1, 2, 3}` in the integration sweep.
- **`multiplier = 0`**: `fullUpper = fullLower = mean`. The `close ==
  mean` case satisfies `close >= fullUpper` -> recombine fires.
  Verified.

## Design notes
- **CONST short-circuit on EMA**: the explicit `alpha * v + (1 - alpha)
  * prev` arithmetic drifts by 1 ULP for non-dyadic alpha values (e.g.
  `1/3` at `length = 5`). A `v === prev ? v : ...` guard preserves
  the bit-exact CONST anchor without changing any other behavior.
- `>=` / `<=` triggers (rather than strict) ensure that the CONST case
  (`close == fullUpper == fullLower == mean`) is flagged as recombine,
  providing a consistent bit-exact anchor.
- Mean line dashed `#475569` to distinguish from band lines.
- Default colors: price `#2563eb`, band `#a855f7` (purple, distinct
  from BB's `#f59e0b`), recombine marker `#dc2626`, mean `#475569`
  dashed.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` /
  `?? false` and defensive guards.
