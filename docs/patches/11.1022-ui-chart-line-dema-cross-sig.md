## DEMA pipeline (Mulloy)

```
ema1[i]   = SMA-seeded EMA(close, period)
ema2[i]   = SMA-seeded EMA(ema1, period)
dema[i]   = 2 * ema1 - ema2
signal[i] = SMA(dema, signalLength)
```

The double-EMA combination cancels lag exactly for linear
input via the algebraic identity:

```
2 * (i - (L-1)/2) - (i - (L-1)) = i
```

So DEMA on LINEAR UP close=i settles at exactly `dema = i`
at the SMA seed bar `2 * (period - 1)` and stays at that
zero-lag value -- the defining feature of the DEMA
construction (the lighter-weight sibling of TEMA's
3-EMA cascade).

Defaults: `period = 14`, `signalLength = 3`. Bias-coloured
cross markers match the convention used by
`chart-line-tema-cross-sig` / `chart-line-ema-cross-sig`.

Warmup is `2 * (period - 1) + signalLength - 1 = 28` for the
default tuning: ema1 SMA-seeds at `period - 1 = 13`, ema2
needs another `period - 1 = 13` valid ema1 samples, then the
signal SMA adds another `signalLength - 1 = 2` bars.

## Bit-exact anchors

| input              | dema (post-warmup) | signal (post-warmup) | dema - signal | regime  | crosses |
| ------------------ | ------------------ | -------------------- | ------------- | ------- | ------- |
| CONST close = K    | K                  | K                    | 0             | bullish | 0       |
| LINEAR UP, close=i | i (zero lag)       | i - 1                | +1            | bullish | 0       |
| LINEAR DOWN, close=-i | -i              | -i + 1               | -1            | bearish | 0       |

CONST K verified across K = 0..1234 -- the double-EMA chain
collapses to K at each layer (`2K - K = K`); signal = SMA(K,
3) = K. dema === signal -> regime `bullish` (>=).

LINEAR UP exercises the Mulloy zero-lag identity directly:
ema1 = `i - 6.5`, ema2 = `i - 13`. dema = `2(i - 6.5) - (i -
13) = 2i - 13 - i + 13 = i`. signal SMA introduces
`(signalLength - 1)/2 = 1` bar of lag, so signal = `i - 1`.
dema - signal = `+1` constant -> regime `bullish`.

LINEAR DOWN is the algebraic mirror: dema = `-i`, signal =
`-i + 1`, dema - signal = `-1` -> regime `bearish`.

DEMA's zero-lag property has a shorter warmup than TEMA
(`2(L-1) = 26` vs `3(L-1) = 39`), making it useful when the
data is short or the trader prefers a less lag-corrected
trigger.
