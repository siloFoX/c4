## Summary

Adds `ChartLineMfiOverboughtCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and a close-only Money
Flow Index (MFI) line in the bottom panel. Marks bullish (entry
into overbought) / bearish (exit from overbought) trigger
events at the canonical 80 overbought threshold. Single-
threshold cross variant of the MFI family that flags discrete
MFI level-80 entry / exit events distinct from the canonical
MFI / signal crossover.

## Pipeline

For close-only data with default `length = 14`:

```
delta[i] = close[i] - close[i-1]
posFlow  = sum |delta| where delta > 0  (rolling length window)
negFlow  = sum |delta| where delta < 0  (rolling length window)
mfi[i]   = posFlow > 0 && negFlow > 0
             ? 100 - 100 / (1 + posFlow / negFlow)
             : posFlow > 0 ? 100
               : negFlow > 0 ? 0
                 : 50          (zero-flow neutral fallback)
bullish  : prev mfi <= 80 && cur mfi > 80   (entry)
bearish  : prev mfi >= 80 && cur mfi < 80   (exit)
```

Close-only adaptation: with no high / low / volume available,
`|delta close|` acts both as the money-flow magnitude (price *
volume proxy) and as the directional cue. The zero-flow fallback
returns 50 so a CONST series never crosses any extreme
threshold.

## Bit-exact anchor

- CONST `close = K` for any finite K -> delta = 0 every bar
  -> posFlow = negFlow = 0 -> mfi = 50 via the zero-flow
  neutral fallback. 50 < 80, regime is `neutral` and the
  threshold is never crossed. cross count = 0. Verified
  across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` (length 14) -> delta = +1 every bar
  -> posFlow > 0, negFlow = 0 -> mfi = 100 constant once
  warm. regime `overbought`. 0 crosses (mfi jumps from null
  to 100 in one bar so prev-null skips the strict-inequality
  detector).
- LINEAR DOWN `close = -i` mirrors LINEAR UP -> mfi = 0
  constant, regime `neutral` (0 < 80), 0 crosses.
- Balanced alternation `close = [10, 11, 10, 11, ...]` (length
  14) -> posFlow = negFlow in each window -> mfi = 50,
  regime `neutral`, 0 crosses.
- Alternation followed by a single large upward jump -> the
  +90 delta pushes mfi above 80 (bullish entry), then mfi
  falls back to 50 once the +90 delta exits the rolling
  window (bearish exit) -> at least 2 crosses.

## Regime classifier

| mfi range            | regime      |
|----------------------|-------------|
| mfi >= 80            | overbought  |
| mfi < 80             | neutral     |
| null                 | none        |

  / `Space` keyboard toggle parity (2 series: price, MFI).
