# 11.763 chart-line-ad-line (v1.11.781)

## Summary

Pure-SVG dual-panel React/TypeScript chart primitive with a
**Chaikin Accumulation / Distribution (AD) line** plotted
beneath the close. The AD line cumulates the **Close Location
Value** times volume across every bar. Added as the
`<ChartLineAdLine />` UI primitive in `web/src/components/ui/`.

## Algorithm

For each bar:

```
CLV[i] = ((close - low) - (high - close)) / (high - low)
       = (2 * close - high - low) / (high - low)
AD[i]  = AD[i - 1] + CLV[i] * volume[i]
```

When `high == low` the bar is flat and the CLV is undefined; the
implementation treats it as zero so the bar contributes no flow.
Non-finite fields null the bar entirely (the cumulative carries
through unchanged across the null).

## Anchors

- **Close at high, high > low**:
  `CLV = (high - low) / (high - low) = 1` bit-exact, so the bar
  adds `+volume[i]` to the cumulative.
  Worked anchor: five `close-at-high` bars at `volume = 100`
  yield cumulative `100, 200, 300, 400, 500` bit-exact.
- **Close at low, high > low**:
  `CLV = -1` bit-exact, so the bar subtracts `volume[i]`.
  Five bars yield `-100, -200, -300, -400, -500` bit-exact.
- **Close at midpoint**: `CLV = 0` bit-exact, so the cumulative
  stays at zero.
- **Flat bar (high == low)**: `CLV = 0`, so the cumulative is
  unchanged.
- **Zero volume**: flow is zero, so the cumulative is unchanged.
- **Quarter-high anchor**: `high = 10, low = 2, close = 8`
  yields `CLV = (6 - 2) / 8 = 0.5` bit-exact.
- **Worked AD step**: `high = 8, low = 0, close = 2, volume = 10`
  -> `CLV = -0.5` (dyadic), `flow = -5` bit-exact.
- **Sawtooth**: alternating close-at-high then close-at-low at
  the same `(high, low, volume)` produces a sawtooth cumulative
  whose net is zero bit-exact.
- **Scaling**: scaling `(high, low, close)` by the same factor
  K > 0 leaves CLV unchanged (within numerical tolerance).
