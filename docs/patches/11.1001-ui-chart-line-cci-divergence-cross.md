## Math anchor (bit-exact)

```
sma[i]      = SMA(close, length)
meanDev[i]  = mean of |close[j] - sma[i]| in window
cci[i]      = (close[i] - sma[i]) / (0.015 * meanDev[i]);
              0 when meanDev == 0 (degenerate)
priceUp     = close[i] > close[i-window]
cciUp       = cci[i]   > cci[i-window]
five-state regime model with crosses suppressed when prev/cur is 'none'
```

- **CONST close = K**: sma=K, meanDev=0 -> cci=0 (degenerate)
  -> aligned-bearish (0 crosses).
- **LINEAR UP close = i**: sma = i - 9.5, meanDev = 5 -> cci =
  9.5 / (0.015 * 5) = 126.6... constant. cci[i] === cci[i-5]
  bit-for-bit at steady state -> cciUp=false, priceUp=true ->
  divergent-bearish (canonical bearish divergence). 0 crosses.
- **LINEAR DOWN close = -i**: cci = -126.6... constant ->
  aligned-bearish. 0 crosses.

For 50 bars at defaults (length=20, divergenceWindow=5):
noneCount=24, post-warmup regime count=26.
