# 11.767 chart-line-mass-index-signal (v1.11.785)

## Summary

Pure-SVG dual-panel React/TypeScript chart primitive with a
**Mass Index** oscillator panel beneath the close. The Mass
Index (Donald Dorsey, 1992) detects reversal bulges by summing
the ratio of `EMA(high - low)` to `EMA(EMA(high - low))` across
a lookback window. Added as the `<ChartLineMassIndexSignal />`
UI primitive in `web/src/components/ui/`.

## Algorithm

For each bar with EMA length `emaLength` and lookback
`lookback` (defaults 9 and 25):

```
range[i] = high[i] - low[i]
ema1[i]  = EMA(range, emaLength)
ema2[i]  = EMA(ema1,  emaLength)
ratio[i] = ema1[i] / ema2[i]
MI[i]    = sum(ratio[i - lookback + 1..i])
```

The EMA is seeded with the first value and uses
`alpha = 2 / (length + 1)`. Bars before `i = lookback - 1` are
`null` (warmup). When `ema2 == 0` (singular: flat range)
`ratio` is `null` and any window touching such a bar yields
`null`.

A reversal-bulge classifier (default threshold 27) tags bars
whose Mass Index reaches or exceeds the threshold.

## Anchors

- **CONST_HL (high - low == K, K != 0)**:
  `range = K` constant; both EMAs decay onto the same trajectory
  (subject to identical ULP drift), so `ratio = ema1 / ema2 = 1`
  bit-exact at every bar. The cumulative `MI = lookback * 1 =
  lookback` **bit-exact** past the warmup. Verified across
  `(range, emaLength, lookback) in
  {1, 2, 5, 10, 100} x {5, 9, 20} x {10, 15, 25}` (45 combos):
  every one yields `MI = lookback` exactly.
- **EMA of dyadic constant (K = 2, length 9)**: `EMA = 2` at
  every bar bit-exact (the IEEE 754 sum `0.2 * 2 + 0.8 * 2`
  rounds back to 2 exactly).
- **EMA of constant K = 7**: stays at K within numerical
  tolerance (12 decimal places); the round-trip `0.2 * 7 +
  0.8 * 7` does not round back to 7 exactly.
- **FLAT_HL (high == low, range == 0)**: ema1 = ema2 = 0,
  ratio = 0 / 0 -> `null`, `MI = null` for every bar.
- **Threshold classification**: at default threshold 27,
  CONST_HL yields `MI = 25 < 27 -> normal`; with threshold 20,
  `MI = 25 >= 20 -> reversal`.
