## Summary

Added `<ChartLineCciCrossSig />` -- a pure-SVG, dual-panel
React/TS primitive that renders the close in the top panel and
overlays the Commodity Channel Index (CCI) with its EMA-smoothed
signal in the bottom panel, marking bullish / bearish cross
trigger events. The trigger-focused CCI variant that separates
the *momentum trigger events* from the base CCI level: this
primitive isolates the cross events when CCI crosses its signal
and also paints them as arrow overlays on the price panel for
direct charting-overlay use:

```
tp[i]      = close[i]                       (single-series TP)
sma[i]     = SMA(tp, length)
meanDev[i] = (1 / length) *
             sum_{k=0..length-1} |tp[i-k] - sma[i]|
CCI[i]     = meanDev === 0
               ? 0
               : (tp[i] - sma[i]) / (0.015 * meanDev[i])
signal[i]  = EMA(CCI, signalLength)
bullish   : (CCI - signal) crosses up    (prev <= 0, cur > 0)
bearish   : (CCI - signal) crosses down  (prev >= 0, cur < 0)
```

Defaults: `length = 20` (canonical CCI), `signalLength = 9`.
Regime classifier: `bullish` (CCI > signal), `bearish` (CCI <
signal), `neutral` (CCI === signal), `none` (either side null).

## Bit-exact anchor

- **CONST close = K (any K)**: every TP === SMA so the
  numerator is 0; meanDev = 0 -> the guard returns CCI = 0
  every bar after warmup. signal EMA of 0s = 0. CCI === signal
  everywhere -> regime `neutral`, cross count = 0. Verified
  across K = 0..1234.
