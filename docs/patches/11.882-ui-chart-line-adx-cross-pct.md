## Summary

Added `<ChartLineAdxCrossPct />` -- a pure-SVG, dual-panel
React/TS primitive that renders the close in the top panel and
overlays the simplified single-series Average Directional Index
(ADX), its EMA-smoothed signal, and the `adxPct = (ADX -
signal) / 100 * 100` deviation channel in the bottom panel.
The percent treatment surfaces the *relative trend strength
momentum* (how stretched ADX is above or below its signal,
scaled by the 0..100 ADX range) for cross-instrument trend
strength comparison.

Single-series simplified ADX using

```
TR[i]     = i === 0 ? 0 : |close[i] - close[i-1]|
+DM[i]    = i === 0 ? 0
                   : close[i] > close[i-1]
                       ? close[i] - close[i-1] : 0
-DM[i]    = i === 0 ? 0
                   : close[i] < close[i-1]
                       ? close[i-1] - close[i] : 0
trS[i]    = Wilder smooth of TR
pDmS[i]   = Wilder smooth of +DM
nDmS[i]   = Wilder smooth of -DM
+DI[i]    = trS === 0 ? 0 : 100 * pDmS / trS
-DI[i]    = trS === 0 ? 0 : 100 * nDmS / trS
DX[i]     = (+DI + -DI) === 0
              ? 0
              : 100 * |+DI - -DI| / (+DI + -DI)
ADX[i]    = Wilder smooth of DX over length
signal[i] = EMA(ADX, signalLength)
adxPct[i] = (ADX - signal) / 100 * 100
```

Defaults: `length = 14` (canonical Wilder), `signalLength = 9`.
Regime classifier: `above` (adxPct > 0), `below` (< 0),
`at` (= 0), `none` (null).

## Bit-exact anchors (three)

- **CONST close = K (any K)**: every TR = +DM = -DM = 0 ->
  trS = pDmS = nDmS = 0 -> +DI = -DI = 0 -> DX = 0 -> ADX = 0
  via Wilder short-circuit -> signal EMA of 0s = 0 -> adxPct =
  0 every bar after warmup. Verified across K = 0..1234.
- **LINEAR UP step = 1**: TR = 1, +DM = 1, -DM = 0 every bar.
  trS = 1, pDmS = 1, nDmS = 0 via Wilder short-circuit. +DI =
  100, -DI = 0. DX = 100 * 100 / 100 = 100. ADX = 100. signal
  EMA of 100s = 100. adxPct = 0. Verified across start values.
- **LINEAR DOWN step = -1**: symmetric, +DM = 0, -DM = 1. +DI
  = 0, -DI = 100. DX = 100. ADX = 100. signal = 100. adxPct =
  0. Verified across start values.

## Relationship to chart-line-rsi-cross-pct (11.878) and 11.879/11.880

| primitive | indicator | scale-by |
| --- | --- | --- |
| `chart-line-rsi-cross-pct` (11.878) | RSI (0..100) | RSI range |
| `chart-line-mfi-cross-pct` (11.879) | MFI (0..100) | MFI range |
| `chart-line-atr-cross-pct` (11.880) | ATR (price units) | close |
| `chart-line-adx-cross-pct` (11.882) | ADX (0..100) | ADX range |

ADX joins RSI and MFI as a 0..100-bounded indicator that uses
its own range as the natural normalisation scale.
