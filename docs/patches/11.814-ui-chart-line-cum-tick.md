# 11.814 UI: chart-line-cum-tick primitive (v1.11.832)

## Summary
Pure-SVG dual-panel React/TS primitive plotting the **Cumulative Tick**
oscillator beneath the close. Each bar contributes a `+1` / `-1` / `0`
tick depending on its close relative to the prior close; the rolling
window sums these directions across the lookback.

## Formula
```
direction[i] = sign(close[i] - close[i - 1])
cumTick[i]   = sum_{k = i - length + 1 .. i} direction[k]
```
`cumTick[i]` is `null` during warmup (`i < length`) and whenever a
close in the window is non-finite. The first bar has no direction
(no prior close).

## Defaults
- `length = 14`
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `positive`: `cumTick > 0`
- `negative`: `cumTick < 0`
- `zero`: `cumTick == 0`
- `none`: warmup / non-finite
- `crossed`: `'up'` when prev `<= 0` and current `> 0`, mirror for
  `'down'`. Markers fire only on cross bars.

## Bit-exact testability anchors
- **CONST close** (`close = K`): every direction = 0, sum = 0 ->
  `cumTick = 0` (bit-exact). Verified across `K in {0, 1, 5, 100, -3}`
  and `length in {3, 4, 7, 10}`.
- **MONOTONIC UP** (`close[k] = k + 1`): every direction = +1, sum =
  +length -> `cumTick = +length` (bit-exact). Verified for `length in
  {3, 4, 7, 10}`.
- **MONOTONIC DOWN** (`close[k] = N - k`): mirror -> `cumTick =
  -length` (bit-exact).
- **ALTERNATING [10, 11, 10, 11, ...]** with **even** length: ups and
  downs balance exactly -> `cumTick = 0` (bit-exact).

All values are integer-exact in IEEE 754 within
`Number.MAX_SAFE_INTEGER`.

## Design notes
- This primitive is the close-derived twin of `chart-line-adv-decline`
  (11.809): same rolling-window sum structure but with a single-bar
  derived direction signal instead of explicit advances/declines
  counts. The bit-exact `+/- length` anchors come from MONO inputs
  exactly because every direction is `+/- 1`.
- y-axis default seeded at `[-length, +length]` (the theoretical
  bound) so the panel reads consistently across data magnitudes.
- Default colors: cumTick line `#f97316` (orange), bullish `#16a34a`,
  bearish `#dc2626`, zero line `#475569` dashed.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
