# 11.1037 -- UI: chart-line-vwap-divergence-cross primitive (v1.11.1055)

## Scope

Add a pure-SVG dual-panel `ChartLineVwapDivergenceCross`
component that renders the close in the top panel and the
rolling Volume Weighted Average Price (VWAP) in the bottom
panel, flagging price-vs-VWAP direction-disagreement events
as institutional flow reversal warnings with bias coloring
derived from the VWAP slope at the trigger bar. Uses the
canonical 5-state divergence regime model (aligned-bullish
/ aligned-bearish / divergent-bullish / divergent-bearish
/ none).

VWAP is the institutional-flow proxy: weighting each bar's
typical price by its volume captures the average price paid
for the most active trades. Price diverging from VWAP
signals retail action pulling away from the institutional
average -- a classic mean-reversion warning, complementary
to the close-only MA divergence-cross siblings (SMA, EMA,
HMA) which ignore volume entirely.

## VWAP divergence pipeline

```
typical[i]  = (high[i] + low[i] + close[i]) / 3
vwap[i]     = sum(typical[j] * volume[j], j = i - period
                  + 1 .. i)
              / sum(volume[j], j = i - period + 1 .. i)
              ; null when the volume sum is 0

priceUp     : close[i] > close[i-1]
priceDown   : close[i] < close[i-1]
vwapUp      : vwap[i]  > vwap[i-1]
vwapDown    : vwap[i]  < vwap[i-1]

regime ->
  'aligned-bullish'   when priceUp   && vwapUp
  'aligned-bearish'   when priceDown && vwapDown
  'divergent-bullish' when priceDown && vwapUp
  'divergent-bearish' when priceUp   && vwapDown
  'none'              when either side flat or null

bullish (divergence trigger up) :
  prev regime !== 'divergent-bullish' &&
  cur regime === 'divergent-bullish'
bearish (divergence trigger down) :
  prev regime !== 'divergent-bearish' &&
  cur regime === 'divergent-bearish'

bias        : vwap[i] vs vwap[i-1] -> up / down / flat /
              none
```

Warmup is `period - 1 = 13` for the default tuning.
Divergence detection requires the previous VWAP + close,
so the first regime classification lands at i = 14.

## Bit-exact anchors

All anchors use HLCV input with constant volume `V = 1` so
the rolling VWAP reduces to `SMA(typical, period)`. The
typical-price computation `(high + low + close) / 3` is
chosen so the LINEAR anchors yield clean integers.

- **CONST band** `high = K + 1`, `low = K - 1`, `close
  = K`, `volume = 1`: typical = ((K + 1) + (K - 1) + K)
  / 3 = K. vwap = SMA(K, period) = K. Both close and VWAP
  are flat -> regime `none` for every bar. 0 divergence
  crosses. Verified across K in {0, 1, 50, 200, 1234}.
- **LINEAR UP** `high = i + 1`, `low = i - 1`, `close
  = i`, `volume = 1`: typical = ((i + 1) + (i - 1) + i)
  / 3 = i. vwap = SMA(typical, period) = i - 6.5. Both
  close and VWAP up -> regime `aligned-bullish`. 0
  crosses.
- **LINEAR DOWN** `high = -i + 1`, `low = -i - 1`,
  `close = -i`, `volume = 1`: mirror -> typical = -i,
  vwap = -i + 6.5. Both close and VWAP falling -> regime
  `aligned-bearish`. 0 crosses.

For varying volume the divergence between SMA and VWAP
becomes meaningful -- a synthetic test verifies that bars
with heavier volume pull the VWAP toward their price more
strongly than the SMA would.

## Risk

- Pure-SVG, deterministic `.toFixed(2)`, no canvas, no
  chart lib dependency.
- 5-state regime classifier follows the
  chart-line-*-divergence-cross family convention; the
  `bullish` / `bearish` cross *kinds* still follow the
  sibling cross-sig convention (line direction).
- Rolling VWAP uses a typical-price-weighted-by-volume
  computation. The volume sum guard ensures a zero-volume
  window (or any window with floating-point NaN volume)
  returns null instead of producing Infinity or NaN.
- Negative volume is filtered at the input boundary --
  the typical accounting convention treats volume as
  non-negative.
- Among the chart-line-*-divergence-cross family, VWAP is
  the only sibling that incorporates volume. It is
  complementary to the close-only MA divergence-cross
  siblings (SMA, EMA, HMA), not redundant -- a stock
  that diverges from its SMA but not its VWAP suggests
  retail-driven action without institutional confirmation;
  a stock that diverges from its VWAP but not its SMA
  suggests volume-driven mean reversion.
- Uses a distinct VWAP ink colour (`#0d9488`, teal) so it
  does not collide with SMA/EMA/HMA divergence-cross
  primitives' purple (`#7c3aed`) when multiple
  divergence-cross primitives ship in the same dashboard.
