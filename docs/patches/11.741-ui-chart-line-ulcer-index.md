# Patch 11.741 -- UI: chart-line-ulcer-index primitive

Released as `v1.11.759` on 2026-05-26.

## Scope

Add `<ChartLineUlcerIndex>` -- a pure-SVG two-panel chart with a
Peter Martin **Ulcer Index** panel reading the root-mean-square of
the percentage drawdowns against the running peak inside the
lookback window.

## Definition

Inputs:

- `data: ChartLineUlcerIndexPoint[]` where `ChartLineUlcerIndexPoint
  = { x; close }`. Bars with non-finite `x` / `close` are dropped.
- `period: number` -- the lookback. Default `14`. Normalized to an
  integer `>= 2`.
- `threshold: number` -- the low/medium boundary; high starts at
  `2 * threshold`. Default `5`. Normalized to a positive finite.

Algorithm (per bar `i >= period - 1`):

```
For j in [i - period + 1, i]:
  peak[j]  = max(close over [i - period + 1, j])
  ddPct[j] = 100 * (close[j] - peak[j]) / peak[j]
ulcer[i]   = sqrt(mean of ddPct[j]^2 over j)
```

The first `period - 1` bars are null. A non-finite or non-positive
close anywhere in the window also nulls the bar.

## Bit-exact algebraic anchors

**CONST_FLAT** (`close = K`): every bar's peak inside the window is
`K`, so every drawdown is `100 * (K - K) / K = 0` exactly. Sum of
squares is `0`, mean is `0`, `sqrt(0) = 0` exactly.

**Monotone rising** (`close[j+1] > close[j]`): for each `j`, the
peak over `[i - period + 1, j]` equals `close[j]` (the latest is the
new max). So `ddPct[j] = 0` exactly at every bar. `UI = 0` exactly.

**DRAWDOWN_25** (`close = [12, 6, 12, 12]` period 4 at bar 3):

- `j = 0`: peak = 12, ddPct = 0
- `j = 1`: peak = 12, ddPct = `100 * (6 - 12) / 12 = -50`, sq = 2500
- `j = 2`: peak = 12, ddPct = 0
- `j = 3`: peak = 12, ddPct = 0
- Sum of squares = 2500, mean = `2500 / 4 = 625 = 25^2`
- `UI[3] = sqrt(625) = 25` bit-exact

Scaled variant (`[4, 2, 4, 4]`): same -50% drawdown structure, same
`UI = 25` bit-exact.

**Scale invariance**: multiplying every close by a positive
constant leaves the percentage drawdowns unchanged (they are
ratios), so the UI is unchanged bit-exact.

## Component

`forwardRef<HTMLDivElement, ChartLineUlcerIndexProps>` with
displayName. Renders:

- A region-role outer `<div data-section="chart-line-ulcer-index">`.
- An `<svg role="img">` with an `aria-describedby` description.
- Top panel: close line, optional dots.
- Bottom panel: Ulcer line, zone-coloured markers (low / medium /
  high), zero reference line, dashed threshold reference line.
- Legend toggle for `price` / `ulcer`, plus `showUlcer` and an
  `onPointClick` handler.
- ARS design-system tokens for colour and motion (`motion-safe:
  animate-fade-in`).
