## Summary

Added `<ChartLineAtrCrossSig />` -- a pure-SVG, dual-panel
React/TS primitive that renders the close in the top panel and
overlays the simplified single-series Average True Range (ATR)
with its EMA-smoothed signal in the bottom panel, marking
bullish / bearish cross trigger events. Trigger-focused
companion to 11.880 `chart-line-atr-cross-pct` (which publishes
the close-scaled deviation magnitude). This primitive isolates
the actionable *volatility regime triggers* and paints them as
arrow overlays on the price panel for direct charting-overlay
use:

```
TR[i]     = i === 0 ? 0 : |close[i] - close[i-1]|
ATR[i]    = Wilder smooth of TR, length
signal[i] = EMA(ATR, signalLength)
bullish  : (ATR - signal) crosses up    (prev <= 0, cur > 0)
bearish  : (ATR - signal) crosses down  (prev >= 0, cur < 0)
```

Defaults: `length = 14` (canonical), `signalLength = 9`.
Regime classifier: `bullish` (ATR > signal), `bearish` (ATR <
signal), `neutral` (ATR === signal), `none` (either side null).

## Bit-exact anchors (three)

- **CONST close = K (any K)**: every TR = 0 -> ATR = 0 ->
  signal EMA of 0s = 0. ATR === signal everywhere -> regime
  `neutral`, cross count = 0. Verified for K = 0..1234.
- **LINEAR UP step = 1**: every TR = 1 (after the first bar)
  -> ATR = 1 via the Wilder short-circuit -> signal EMA of 1s
  = 1. ATR === signal -> regime `neutral`, cross count = 0.
  Verified across multiple start values.
- **LINEAR DOWN step = -1**: same magnitudes, TR = 1 every
  bar -> ATR = 1 -> signal = 1 -> regime `neutral`, cross
  count = 0. Verified across multiple start values.

## Difference vs chart-line-atr-cross-pct (11.880)

| primitive | bottom panel | overlays | use-case |
| --- | --- | --- | --- |
| `chart-line-atr-cross-pct` (11.880) | ATR + signal + (ATR-signal)/close*100 | none | volatility momentum magnitude scaled to price |
| `chart-line-atr-cross-sig` (11.881) | ATR + signal + cross dots | bullish/bearish arrows on price | actionable volatility regime triggers |

11.880 publishes the close-scaled deviation magnitude as a
continuous channel for cross-instrument comparison. 11.881 adds
discrete bullish / bearish cross trigger events with dual
rendering (oscillator dots and price-panel arrow overlays) so
the regime shifts can be acted on directly.
