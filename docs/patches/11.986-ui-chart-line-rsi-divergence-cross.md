## Summary

Add `web/src/components/ui/chart-line-rsi-divergence-cross.tsx`
-- a pure-SVG dual-panel React/TypeScript chart primitive that
renders the close in the top panel and a close-only Wilder RSI
line in the bottom panel, marking discrete bullish (price down +
RSI up, potential bottom reversal warning) / bearish (price up
+ RSI down, potential top reversal warning) divergence cross
events. RSI counterpart to `chart-line-macd-divergence-cross`
(11.985): instead of tracking a single threshold crossover, it
compares the direction of price and RSI over a configurable
look-back window and flags every transition into a disagreement
state.

## Math anchor (bit-exact)

```
rsi[i]   = WilderRSI(close, length)
priceUp  = close[i] > close[i-window]
rsiUp    = rsi[i]   > rsi[i-window]
state
  aligned-bullish    : priceUp && rsiUp
  aligned-bearish    : !priceUp && !rsiUp
  divergent-bullish  : !priceUp && rsiUp   (price down, RSI up)
  divergent-bearish  : priceUp && !rsiUp   (price up, RSI down)
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
(crosses suppressed when prev or cur state == 'none')
```

- **CONST close = K**: deltas = 0 -> avgGain == avgLoss == 0 ->
  RSI degenerate = 50. priceUp = false (close[i] === close[i-5]),
  rsiUp = false (50 === 50) -> regime `aligned-bearish`. 0
  divergence crosses. Verified across K = 0..1234.
- **LINEAR UP close = i**: deltas = +1 -> avgLoss = 0 -> RSI =
  100 constant. priceUp = true (close[i] > close[i-5]), rsiUp =
  false (100 === 100) -> regime `divergent-bearish` (price still
  rising while RSI has saturated at 100 -- waning relative-
  strength momentum, canonical bearish divergence). 0 crosses
  because divergent state is entered from `none` (prev == 'none'
  is never a valid cross precursor).
- **LINEAR DOWN close = -i**: RSI = 0 constant. priceUp = false,
  rsiUp = false -> regime `aligned-bearish`. 0 crosses.

For 60 bars at defaults (length=14, divergenceWindow=5):
- noneCount = 19 (i = 0..18 where rsi[i-5] is not yet valid)
- second-half regime count = 41 (i = 19..59)

Layout uses fixed oscMin=0 / oscMax=100 (RSI canonical range)
with midline 50 reference.

## Accessibility

- Wrapping `<div role="region" aria-label="RSI Divergence Cross
  chart" aria-describedby="...">` with the sr-only description
  copy carrying the length / divergenceWindow parameters plus
  a "reversal warning" framing.
- `<svg role="img" aria-labelledby="..." aria-describedby="..."
  tabIndex={0}>` for keyboard focus.
- Each divergence marker (both the RSI-panel dot and the
  price-panel chevron overlay) carries `role="graphics-symbol"`,
  `tabIndex={0}`, and an `aria-label="<bullish|bearish>
  divergence at <x>"` so screen readers can step through events.
- Legend buttons expose `aria-pressed` and toggle via mouse,
  focus, Enter, or Space.
