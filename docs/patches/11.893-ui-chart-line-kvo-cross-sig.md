## Summary

Added `<ChartLineKvoCrossSig />` -- a pure-SVG, dual-panel
React/TS primitive that renders the close in the top panel and
overlays the Klinger Volume Oscillator (KVO) over its EMA-
smoothed signal line in the bottom panel, marking bullish /
bearish cross trigger events for *volume force trigger events*.
Explicit cross-sig family naming companion to 11.886
chart-line-kvo-cross; this primitive publishes the actionable
signal cross events alongside MACD, ATR, RSI, MFI, OBV, CCI,
TRIX, StochRSI, ROC, and the rest of the cross-sig family with
the canonical price-panel arrow overlays.

```
trend[i]    = i === 0 ? 0
                     : close[i] > close[i-1] ? 1
                     : close[i] < close[i-1] ? -1
                     : 0
volForce[i] = volume[i] * trend[i]
emaFast[i]  = EMA(volForce, fastLength)
emaSlow[i]  = EMA(volForce, slowLength)
KVO[i]      = emaFast[i] - emaSlow[i]
signal[i]   = EMA(KVO, signalLength)
bullish    : (KVO - signal) crosses up    (prev <= 0, cur > 0)
bearish    : (KVO - signal) crosses down  (prev >= 0, cur < 0)
```

Defaults: `fastLength = 34`, `slowLength = 55`, `signalLength =
13` (canonical KVO). Regime classifier: `bullish` (KVO >
signal), `bearish` (KVO < signal), `neutral` (KVO === signal),
`none` (either side null).

## Bit-exact anchors (three)

- **CONST {close = K, volume = V}**: every trend = 0 ->
  volForce = 0 -> emaFast = emaSlow = 0 -> KVO = 0 every bar
  -> signal EMA of 0s = 0. KVO === signal -> regime `neutral`,
  cross count = 0. Verified across multiple K and V tuples.
- **LINEAR UP step > 0 with V > 0**: trend = 1 every bar,
  volForce = V (constant). emaFast and emaSlow both collapse
  to V via the `min === max` precision fix -> KVO = V - V =
  0 -> signal = 0 -> regime `neutral`, cross count = 0.
- **LINEAR DOWN step < 0 with V > 0**: trend = -1 every bar,
  volForce = -V (constant). emaFast = emaSlow = -V -> KVO =
  -V - (-V) = 0 -> signal = 0 -> regime `neutral`, cross
  count = 0.

All three anchors hit `KVO = signal = 0` bit-exact.

## Relationship to chart-line-kvo-cross (11.886)

| primitive | naming | use-case |
| --- | --- | --- |
| `chart-line-kvo-cross` (11.886) | generic cross | KVO with EMA signal crossings |
| `chart-line-kvo-cross-sig` (11.893) | explicit cross-sig family | same KVO + EMA signal, family-consistent naming |

Both publish identical KVO over signal crossings. 11.893 adds
explicit cross-sig family naming so KVO participates in the
trigger-focused family alongside MACD, ATR, RSI, MFI, OBV, CCI,
TRIX, StochRSI, ROC, ehlers-fisher, and others.
