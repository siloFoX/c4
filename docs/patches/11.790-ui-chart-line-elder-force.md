## Summary

`ChartLineElderForce` is a pure-SVG dual-panel React primitive
that plots the close on the top panel and the Elder Force Index
(EFI) oscillator on the bottom panel. EFI weights price change by
volume and smooths the result with an EMA:

```
raw[i] = (close[i] - close[i - 1]) * volume[i]   (i >= 1)
EFI[i] = EMA(raw, length)[i]
```

Defaults: `length = 13` (Elder's classic period). Bar 0 has no
prior close so `raw[0]` is `null` and the EMA is seeded at the
first finite `raw` (bar 1). `-0` from `0 * negative_volume` is
normalized to `+0`.

## Bit-exact anchor

For **CONST close** (`close = K` constant): every
`close[i] - close[i - 1]` is exactly zero, so every `raw[i]` is
zero (regardless of `volume[i]`, including zero or negative
volume). EMA of zero stays at zero in IEEE 754 (`alpha * 0 +
(1 - alpha) * 0 = 0`), so `EFI = 0` exactly past bar 0. The
integration sweep verifies this across many `K`, `V` (including
zero and negative), and `length`.

This anchor demonstrates that the volume weighting cannot "leak"
in: when price is flat the force is zero regardless of how the
volume moves. The `-0 -> +0` normalization keeps the equality
exact for negative volumes.

## Zone classifier

The bottom-panel reading is bucketed by ratio to the maximum
absolute EFI observed in the current series:

- `strong-up` -- EFI >= 0.5 * efiAbsMaxSeen (positive half)
- `above` -- 0 < EFI < 0.5 * efiAbsMaxSeen
- `at` -- EFI == 0
- `below` -- -0.5 * efiAbsMaxSeen < EFI < 0
- `strong-down` -- EFI <= -0.5 * efiAbsMaxSeen
- `none` -- EFI is null (bar 0 only, or invalid input)

When `efiAbsMaxSeen` is zero (all-zero series, e.g. CONST close)
the classifier falls back to `above` for positive non-zero
readings and `below` for negative, so the buckets remain defined.

## Out of scope

- No new dependencies.
- No canvas, no third-party chart libs.
- This primitive ships the EMA-smoothed variant. The raw
  unsmoothed Force (`length = 1`) is reachable via the same
  pipeline (`computeLineElderForce(series, { length: 1 })`); a
  dedicated component for it is not yet shipped.
