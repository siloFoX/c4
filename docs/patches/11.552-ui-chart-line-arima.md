# 11.552 -- web/src/components/ui/chart-line-arima.tsx

Pure-SVG line chart with a **simple ARIMA(1,0,0) / AR(1)
one-step prediction overlay**. The raw observations render
as a dimmed reference line; the fitted AR(1) one-step
prediction renders as a solid coloured overlay; an optional
dashed forecast segment extends one step beyond the data.

## Math

ARIMA(1, 0, 0) is a pure first-order autoregressive model
(AR order 1, zero differencing, zero moving-average terms):

```
y_t = c + phi * y_{t-1} + epsilon_t
```

where `c` is the intercept, `phi` is the lag-1
autoregressive coefficient, and `epsilon_t` is white noise.

### Fitting (conditional least squares)

The model is fitted by **conditional ordinary least
squares**: regress `y_t` on `y_{t-1}` over the pairs
`(y_{t-1}, y_t)` for `t = 1 .. N-1`.

```
xbar = mean(y_0 .. y_{N-2})     (the lagged predictor)
ybar = mean(y_1 .. y_{N-1})     (the response)

phi  = sum (x_i - xbar)(y_i - ybar) / sum (x_i - xbar)^2
c    = ybar - phi * xbar
```

This is the standard estimator used as the starting point
of Box-Jenkins ARIMA fitting (and what an OLS regression of
a series on its own lag-1 produces).

Degenerate cases:
- `N < 2` -> `phi = 0`, `c = mean` (a flat prediction at the
  series mean).
- A constant series (zero lagged variance) -> `phi = 0`,
  `c = mean`.

### One-step prediction

```
yhat_t = c + phi * y_{t-1}    for t = 1 .. N-1
```

Index 0 has no prior value, so its prediction is null. The
residual `e_t = y_t - yhat_t` is the one-step prediction
error; RMSE is `sqrt(mean(e_t^2))` over the valid
predictions.

### Forecast

A single one-step-ahead forecast beyond the last
observation: `yhat_N = c + phi * y_{N-1}`, placed at
`x_{N-1} + step` where `step` is the mean x-spacing.

### Stationarity

The AR(1) stationarity condition is `|phi| < 1`. A fit with
`|phi| >= 1` is flagged `non-stationary` -- the model still
produces predictions but a long-horizon forecast would
diverge. A perfectly linear series fits to `phi = 1` (a unit
root), which is the boundary non-stationary case; a
mean-reverting / decaying series fits to `|phi| < 1`.

### Verified properties (by test)

- **Linear series `[1,2,3,4,5]`** -> `phi = 1`, `c = 1`
  exactly. Predictions reproduce the series exactly, all
  residuals 0, RMSE 0. (`phi = 1` is flagged
  non-stationary.)
- **Constant series** -> `phi = 0`, `c = mean`, predictions
  flat at the mean.
- **Geometric decay `y_t = 0.5 * y_{t-1}`** -> `phi = 0.5`,
  `c = 0` recovered exactly; flagged stationary.
- **Explosive series `y_t = 2 * y_{t-1}`** -> `phi = 2`,
  flagged non-stationary.

## Distinct from related primitives

- `<ChartLineForecast>`: renders **pre-computed** forecast
  data (a `forecastFrom` split index plus optional
  `yLower`/`yUpper` bands). It fits no model. AR(1) FITS the
  model from the data and computes the prediction itself.
- `<ChartLineEwma>` (11.539): exponential smoothing. EWMA is
  equivalent to an ARIMA(0,1,1); AR(1) is ARIMA(1,0,0) -- a
  different model class (autoregressive vs. moving-average
  on differences).
- `<ChartLineKalman>` (11.538): Bayesian state-space filter
  with process/measurement noise. AR(1) is a closed-form
  least-squares regression with no noise model.
- `<ChartLineAutocorrelation>` (11.551): the ACF diagnostic.
  ACF at lag 1 is closely related to `phi`, but ACF is a
  *diagnostic plot* whereas AR(1) is the *fitted model* and
  prediction.
- `<ChartLineTrend>` (11.512): global linear regression of
  `y` on `x`. AR(1) regresses `y_t` on `y_{t-1}` (the
  previous VALUE, not the x position).
- `<ChartLineLoess>` (11.549) / `<ChartLineSavgol>`
  (11.547) / `<ChartLineMovingAvg>` (11.513): smoothers that
  filter noise. AR(1) is a *predictive* model -- each point
  is predicted purely from the one before it.

The simplest member of the ARIMA family, AR(1) is the
canonical introduction to autoregressive time-series
modelling and the natural baseline a one-step forecast is
measured against.

- `<ChartLineArima series={...} />`.
- `series: ChartLineArimaSeries[]` --
  `{id, label, data, color?}`. The AR(1) coefficients are

  `residualPosColor`, `residualNegColor`.

  `showForecast`, `showResidualSticks`, `animate`.
- Controlled + uncontrolled visibility.
- Standard sizing + format + a11y props.
- `formatValue` / `formatX` / `formatCoefficient`.
- `onPointClick({series, point})`.

## Pure helpers

- `getLineArimaDefaultColor(index)`.
- `getLineArimaFinitePoints(points)`.
- `classifyLineArimaResidualSign(residual)`.
- `fitLineArimaAR1(values)` -- conditional OLS fit;
  returns `{phi, intercept, mean, stationary}`.
- `predictLineArimaOneStep(values, fit)` -- one-step
  predictions; index 0 is null; non-finite prior -> null.
- `forecastLineArimaNext(values, fit)` -- the single
  forecast beyond the data; null for an empty series or a
  non-finite last value.
- `runLineArima(points)` -- canonical pipeline. Sorts
  ascending, drops non-finite, fits the model, computes
  predictions + residuals + RMSE + forecast.
- `computeLineArimaLayout({...})` -- projects raw +
  predicted paths + residual segments + forecast point.
- `describeLineArimaChart(series, options?)`.

## Tooltip

Hovering a dot shows the series label, x, raw value, bold
predicted value (or n/a at index 0), residual (with `+`/`-`
prefix), and the config `phi=<phi>, c=<intercept>`.
