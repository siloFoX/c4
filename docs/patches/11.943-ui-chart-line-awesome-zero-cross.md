# 11.943 -- UI: chart-line-awesome-zero-cross primitive (v1.11.961)

## Summary

Adds `ChartLineAwesomeZeroCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and the close-only
Awesome Oscillator (AO) line in the bottom panel. Marks
bullish (cross up through zero) / bearish (cross down through
zero) 5-over-34 SMA midpoint baseline regime transition
events. Zero-line cross variant of the Awesome Oscillator
family that flags the discrete AO crossing of the zero
baseline.

## Pipeline

For close-only data with defaults `fastLength = 5`,
`slowLength = 34`:

```
fastSma  = SMA(close, fastLength)
slowSma  = SMA(close, slowLength)
ao[i]    = fastSma[i] - slowSma[i]
bullish  : prev ao <= 0 && cur ao > 0
bearish  : prev ao >= 0 && cur ao < 0
```

Close-only adaptation: with no high / low available, the
close itself is used as the median price proxy (the
traditional AO uses `(high + low) / 2`).

## Bit-exact anchor

- CONST `close = K` for any finite K -> fastSma = slowSma =
  K every bar via the SMA `min === max` short-circuit -> ao
  = 0. ao = 0 sits on the threshold but the strict-
  inequality detector never fires. regime `bullish` (ao >=
  0). cross count = 0. Verified across K in {0, 1, 42, 100,
  1234}.
- LINEAR UP `close = i` -> at steady state SMA(close, n) =
  i - (n-1)/2 so ao = (i - 2) - (i - 16.5) = 14.5 constant.
  14.5 > 0, regime `bullish`. 0 crosses.
- LINEAR DOWN `close = -i` -> ao = -14.5 constant. -14.5 <
  0, regime `bearish`. 0 crosses.
- Decline followed by sustained rise -> AO transitions from
  negative to positive -> 1 bullish zero-line cross.
- Rise followed by sustained decline -> 1 bearish cross.

## Regime classifier

| ao range             | regime      |
|----------------------|-------------|
| ao >= 0              | bullish     |
| ao < 0               | bearish     |
| null                 | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- One horizontal reference band inside the oscillator panel
  (threshold at zero, the baseline).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind.
- Oscillator range auto-fits to observed AO values with
  10% padding above and below; AO is unbounded.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.

## Defaults

- fastLength = 5
- slowLength = 34
- threshold = 0
- oscillator range = auto-fit (AO is unbounded)
- width 720, height 460, padding 44, panelGap 12
