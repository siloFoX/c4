## TEMA pipeline (Mulloy)

```
ema1[i]   = SMA-seeded EMA(close, period)
ema2[i]   = SMA-seeded EMA(ema1, period)
ema3[i]   = SMA-seeded EMA(ema2, period)
tema[i]   = 3 * ema1 - 3 * ema2 + ema3
signal[i] = SMA(tema, signalLength)
```

The triple-EMA combination cancels lag exactly for linear
input via the algebraic identity:

```
3 * (i - (L-1)/2) - 3 * (i - (L-1)) + (i - 3*(L-1)/2) = i
```

So TEMA on LINEAR UP close=i settles at exactly `tema = i`
at the SMA seed bar `3 * (period - 1)` and stays at that
zero-lag value forever -- the defining feature of the TEMA
construction.

Defaults: `period = 14`, `signalLength = 3`. Bias-coloured
cross markers match the convention used by
`chart-line-ema-cross-sig` / `chart-line-hma-cross-sig`.

Warmup is `3 * (period - 1) + signalLength - 1 = 41` for the
default tuning: each nested EMA SMA-seed adds `period - 1 =
13` bars, then the signal SMA adds another `signalLength - 1
= 2` bars.

## Bit-exact anchors

| input              | tema (post-warmup) | signal (post-warmup) | tema - signal | regime  | crosses |
| ------------------ | ------------------ | -------------------- | ------------- | ------- | ------- |
| CONST close = K    | K                  | K                    | 0             | bullish | 0       |
| LINEAR UP, close=i | i (zero lag)       | i - 1                | +1            | bullish | 0       |
| LINEAR DOWN, close=-i | -i              | -i + 1               | -1            | bearish | 0       |

CONST K verified across K = 0..1234 -- the triple-EMA chain
collapses to K at each layer (`3K - 3K + K = K`); signal =
SMA(K, 3) = K. tema === signal -> regime `bullish` (>=).

LINEAR UP exercises the Mulloy zero-lag identity directly:
ema1 = `i - 6.5`, ema2 = `i - 13`, ema3 = `i - 19.5`. tema
= `3(i - 6.5) - 3(i - 13) + (i - 19.5) = 3i - 19.5 - 3i +
39 + i - 19.5 = i`. signal SMA introduces `(signalLength -
1)/2 = 1` bar of lag, so signal = `i - 1`. tema - signal =
`+1` constant -> regime `bullish`.

LINEAR DOWN is the algebraic mirror: tema = `-i`, signal =
`-i + 1`, tema - signal = `-1` -> regime `bearish`.

The TEMA zero-lag property is verified directly in
`applyLineTemaCrossSigTema` tests (LINEAR UP returns TEMA = i
exactly).
