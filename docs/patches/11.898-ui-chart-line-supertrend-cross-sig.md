# Patch 11.898 -- UI: chart-line-supertrend-cross-sig primitive

## Summary

Added `<ChartLineSupertrendCrossSig />` -- pure-SVG dual-panel
React/TS primitive that renders the close (top panel) with
bullish / bearish arrow overlays at every Supertrend / signal
cross trigger event, and the Supertrend trailing band over its
EMA-smoothed signal line (bottom panel) with bullish / bearish
dot markers at every cross. Signal-cross variant of the
Supertrend family that separates the smoothed trend trigger
events from the discrete Supertrend flip line published by
11.859 chart-line-supertrend-cross.

## Algorithm

```
TR[i]      = i == 0 ? 0 : |close[i] - close[i-1]|
ATR[i]     = Wilder smooth of TR over length
upperBand  = close + factor * ATR
lowerBand  = close - factor * ATR
ST[i]      = sticky trend flip on band cross
signal[i]  = EMA(ST, signalLength)
bullish   : (ST - signal) crosses up
bearish   : (ST - signal) crosses down
```

Defaults `length = 10`, `factor = 3` (canonical Supertrend),
`signalLength = 9`. Regime classifier `bullish` / `bearish` /
`neutral` / `none`.

## Bit-exact anchor

CONST close = K (K > 0): TR = 0 every bar -> ATR = 0 ->
upperBand = lowerBand = K -> Supertrend = K every bar. The
signal EMA collapses every value to K via the `min === max`
precision short-circuit, so ST == signal on every settled bar,
regime is `neutral`, and the cross count is exactly 0. Verified
across K = 1, 5, 17, 100, 1234.
