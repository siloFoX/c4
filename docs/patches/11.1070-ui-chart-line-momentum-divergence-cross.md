# 11.1070 -- UI: chart-line-momentum-divergence-cross primitive (v1.11.1088)

## Scope

Add a pure-SVG dual-panel `ChartLineMomentumDivergenceCross`
component that renders the close in the top panel and the
Momentum oscillator in the bottom panel, marking bullish
(price down while momentum up -- bullish reversal warning;
downward price exhaustion while momentum recovers) /
bearish (price up while momentum down -- bearish reversal
warning; upward price exhaustion while momentum decays)
price-vs-momentum direction disagreement (divergence)
trigger events with bias coloring derived from the
momentum slope at the divergence-entry bar.

The classical Momentum oscillator measures absolute price
change over the lookback period: M[i] = close[i] -
close[i-period]. When price and momentum disagree on
direction, the trend is either decelerating (divergent-
bearish: price still rising but at slower pace, momentum
already falling) or accelerating against itself (divergent-
bullish: price still falling but momentum already turning
up). These divergences are classical reversal warning
signals -- the price has not reversed yet but the
underlying momentum has.

## Momentum divergence-cross pipeline

```
M[i]  = close[i] - close[i - period]   (Momentum)

priceUp     = close[i]  > close[i-1]
priceDown   = close[i]  < close[i-1]
momUp       = M[i]      > M[i-1]
momDown     = M[i]      < M[i-1]

regime :
  'aligned-bullish'   when priceUp   && momUp
  'aligned-bearish'   when priceDown && momDown
  'divergent-bullish' when priceDown && momUp
                      (bullish reversal warning)
  'divergent-bearish' when priceUp   && momDown
                      (bearish reversal warning)
  'none'              otherwise (null or any flat)

bullish cross :
  prev regime !== 'divergent-bullish' &&
  cur regime === 'divergent-bullish'
bearish cross :
  prev regime !== 'divergent-bearish' &&
  cur regime === 'divergent-bearish'

bias : M[i] vs M[i-1] -> up/down/flat/none
```

Default `period = 10` -- canonical Momentum oscillator
tuning.

## Sibling family (divergence-cross family)

- **chart-line-atr-divergence-cross v1.11.1046** -- price
  vs ATR (volatility-direction divergence)
- **chart-line-keltner-divergence-cross v1.11.1054** --
  price vs Keltner mid
- **chart-line-bollinger-divergence-cross v1.11.1055** --
  price vs Bollinger mid
- **chart-line-donchian-divergence-cross v1.11.1059** --
  price vs Donchian mid
- **chart-line-momentum-divergence-cross v1.11.1088**
  (this primitive) -- price vs Momentum oscillator
  (the canonical reversal warning signal)

All siblings share the same 5-state regime classifier and
the same cross-on-divergent-entry detector pattern. They
differ in WHICH indicator's direction is compared to
price direction:
- ATR: volatility direction (expanding vs contracting)
- Keltner/Bollinger/Donchian: mid-line direction
- Momentum (this primitive): raw momentum direction (the
  most direct reversal-warning measure)

Distinct from sibling chart-line-momentum-mid-cross-sig
v1.11.1080 (also from the momentum family): that one
compares Momentum vs its zero-line midpoint (level cross),
while this primitive compares Momentum DIRECTION against
PRICE DIRECTION (divergence detector).

Warmup is `period = 10` for the default tuning: first
momentum value at i = period (M[period] = close[period] -
close[0]). Cross detection needs the previous bar's
regime, so the first potential cross lands at i = period
+ 1 = 11.

## Bit-exact anchors

All anchors use single-close input.

- **CONST** `close = K`: M[i] = K - K = 0 (constant from
  i = period). Price delta = 0 (flat), momentum delta =
  0 (flat). Both priceUp/Down and momUp/Down are false
  at every bar. regime `none` throughout. 0 crosses.
  Verified across K in {0, 1, 50, 200, 1234}.
- **LINEAR UP** `close = i`: M[i] = i - (i-period) =
  period (constant from i = period). Price delta = +1
  (priceUp = true). Momentum delta = 0 (flat). momUp =
  false. regime `none` throughout post-warmup. 0
  crosses.
- **LINEAR DOWN** `close = -i`: M[i] = -period
  (constant). Price delta = -1 (priceDown), momentum
  delta = 0. regime `none`. 0 crosses.
- **QUADRATIC UP** `close = i*i`: M[i] = period*(2*i -
  period) -- monotonically increasing. Price delta = 2*i
  - 1 (always > 0, priceUp). Momentum delta = period*2
  (constant, momUp). regime `aligned-bullish`
  throughout. 0 crosses (no transition into divergent
  state).
- **SQRT UP** `close = sqrt(i+1)` -- monotonically
  increasing but slope decreases. Price delta > 0
  (priceUp). Momentum delta < 0 eventually (momentum
  decays as the rate slows). regime
  `divergent-bearish`. At least one bearish cross
  fires.

All three linear/const anchors produce 0 crosses (regime
stays in `none` or aligned states). The QUADRATIC UP
anchor verifies the aligned-bullish path; the SQRT UP
anchor verifies the divergent-bearish path with at least
one cross.

## Risk

- Pure-SVG, deterministic `.toFixed(2)`, no canvas, no
  chart lib dependency.
- 5-state regime classifier follows the divergence-cross
  family convention exactly (aligned-bullish /
  aligned-bearish / divergent-bullish / divergent-bearish
  / none).
- Strict inequalities (priceUp = `>`, etc.) -- exact
  ties produce regime `none` rather than ambiguous
  state. Matches the family convention so CONST input
  cleanly returns 0 crosses.
- The Momentum oscillator is the SIMPLEST divergence
  detector in the family -- no smoothing, no
  normalisation. This makes it the most reactive (every
  bar's price change shows up in M[i] - M[i-period+1]
  comparison) and also the noisiest (small fluctuations
  fire crosses). Pair with a smoothing filter
  externally for production use.
- The QUADRATIC UP anchor verifies that aligned
  acceleration is NOT flagged as divergence (regime
  stays aligned-bullish; 0 crosses). The SQRT UP anchor
  verifies that DECELERATING acceleration IS flagged
  (regime divergent-bearish; at least one cross).
- Zero-visibility guarantee in layout ensures the
  momentum sign is always readable; even on a series
  that never crosses zero, the zero line renders at
  oscBottom or oscTop.
- Distinct from chart-line-momentum-mid-cross-sig
  v1.11.1080 (Momentum vs zero level cross -- absolute
  momentum sign change) and chart-line-momentum (the
  raw Momentum oscillator). This primitive specifically
  measures price-vs-momentum DIRECTION disagreement,
  which is the canonical reversal-warning signal in
  technical analysis.
