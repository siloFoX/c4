## Summary

Add a new `<ChartRadialArea>` UI primitive in
`web/src/components/ui/chart-radial-area.tsx` -- pure-SVG
**radial area** chart variant where each series is a
**closed polygon with a filled interior** in polar
coordinates. Two stack modes:

1. **overlay** (default): every series fills from the
   centre (innerRadius) out to its value-radius;
   overlapping fills use `fillOpacity` so multiple
   series stay legible on top of each other.
2. **stacked**: the cumulative sum of all visible
   series stacks radially; each layer paints an annular
   band between the previous cumulative and its own
   cumulative -- so the outermost layer's outer edge
   reaches the cumulative total. Peak total maps to
   `outerRadius`.

Two curve styles: `linear` (default; straight polygon
edges between samples) and `cardinal` (closed cubic-
bezier smoothing -- the classic D3 radial area look).

Distinct from `<ChartRadialLine>` (11.490) which is
line-emphasized for cyclic/spiral data (no
stacked mode), `<ChartRadar>` (11.445) which is
multi-axis polygon for categorical dimensions,
`<ChartPolarArea>` (11.478) which uses equal-angle
wedges with radius encoding value, and
`<ChartRadialStackedBar>` (11.489) which stacks
*angularly* within each category ring.

## Behaviour

- Renders `<div role="region">` wrapping
  `<svg role="img">` with the same `aria-label`.
- Layout (`computeRadialAreaLayout`):
  - `radiusFor(value)` = `innerRadius + (value /
    maxValue) * (outerRadius - innerRadius)`,
    clamped to `[0, 1]`.
  - **overlay**: every layer's `innerRadius` is the
    chart's inner radius; `outerRadius` is
    `radiusFor(value)`.
  - **stacked**: per-sample cumulative sums are
    computed left-to-right across visible series.
    Each layer occupies the annulus from
    `radiusFor(cumulativeBefore)` to
    `radiusFor(cumulativeAfter)`. Hidden series
    pause the cumulative cursor.
  - Non-finite / non-positive values clamp to 0
    so the layer never inverts.
- Path building (`buildRadialAreaClosedPath`):
  - For overlay (inner ring collapses to a point),
    emits a single closed M..Z ring for each
    series.
  - For stacked (distinct inner ring), emits two
    closed rings (outer + reversed inner) with
    `fillRule="evenodd"` so the SVG correctly
    paints only the annulus.
  - `linear` uses straight L commands between
    samples; `cardinal` uses Catmull-Rom-to-cubic-
    Bezier on the closed loop.
- Axis spokes + axis labels: one per `cycleLength`
  step around the perimeter; labels default to
  sample index; `axisLabels` prop + `formatAxis`
  rewrite.
- Rings: one concentric dashed ring per non-zero
  tick value.
- Hidden series (controlled / uncontrolled) drop
  from the layout; cumulative cursor pauses for
  hidden layers in stacked mode.
- Hover/focus on a series group sets the hovered
  id, dims every other series to opacity 0.4, and
  opens a tooltip with the series label, the sum
  of finite sample values, and the sample count.
- Mount animation:
  `motion-safe:animate-fade-in` per series group.
  `animate=false` removes the class.
- Default ARIA description: `"Radial area chart
  (<stackMode>) with N series, cycle length L,
  peak value <peak>."` Empty / all-hidden / non-
  positive cycle -> `"No data"`.
