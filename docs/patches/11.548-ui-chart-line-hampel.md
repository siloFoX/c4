# 11.548 -- web/src/components/ui/chart-line-hampel.tsx

Pure-SVG line chart with a **Hampel** robust outlier filter
(median + MAD) and an **original-vs-filtered** overlay. For
each index, the filter:

1. Builds a centered window of length `W` (default 7, odd).
2. Computes the window median `m` and the median absolute
   deviation `MAD = median(|y - m|)`.
3. Scales MAD by `c` (default 1.4826 -- the canonical
   normality-consistency factor that makes `c * MAD` an
   unbiased estimator of sigma under Gaussian noise).
4. Flags index `i` as an outlier when
   `|y[i] - m| > n * c * MAD` (default `n = 3`).
5. Replaces outliers with the window median; passes
   non-outliers through unchanged.

Edge windows are clamped to in-bounds samples (no leading /
trailing nulls; the filter operates on shorter windows at the
edges).

## Why robust (and distinct from related primitives)

The Hampel filter is built on robust statistics (median + MAD)
with a 50% breakdown point. That makes it detect outliers
**even when other outliers are present in the same window** --
unlike methods that estimate the spread with mean + stddev,
where the outliers themselves inflate the stddev and hide
each other.

- `<ChartLineAnomaly>` (11.524): z-score based on rolling
  mean + stddev. NON-robust -- a cluster of outliers
  inflates stddev and hides them.
- `<ChartLineControl>` (11.534): global SPC mean +/- k*sigma.
  Non-robust + non-local.
- `<ChartLineChangepoint>` (11.546): regime shift detector,
  finds PERSISTENT shifts. Hampel finds isolated outliers.
- `<ChartLineSavgol>` (11.547): polynomial smoothing, no
  outlier flag. Hampel replaces flagged points with median;
  SG smooths through everything.
- `<ChartLineMovingAvg>` (11.513): SMA. Non-robust -- a
  single outlier moves the average.
- `<ChartLineKalman>` (11.538): Bayesian filter,
  statistical not robust.
- `<ChartLineBollinger>` (11.537): rolling SMA + envelope,
  non-robust.

The canonical robust outlier rejector used in sensor signal
processing, financial cleanup, and any context where the
question is "drop the spikes so the underlying signal is
visible".

## Public surface

- `<ChartLineHampel series={...} />`.
- `series: ChartLineHampelSeries[]` -- `{id, label, data,
  color?, windowLength?, nSigmas?, madScale?}`. Per-series
  overrides always beat chart-level.
- `windowLength` (default 7; clamped to odd >= 3; even
  inputs rounded up).
- `nSigmas` (default 3; clamped to >= 0 with 0 allowed for
  "flag every non-median sample").
- `madScale` (default 1.4826; clamped to > 0).
- `outlierColor` -- color for the outlier markers + tooltip.
- Toggle flags: `showAxis`, `showGrid`, `showDots`,
  `showLegend`, `showTooltip`, `showOutlierBadge`,
  `showRaw`, `showOutlierMarkers`, `showReplacementSticks`,
  `animate`.
- Controlled + uncontrolled visibility.
- Standard sizing + format + a11y props.
- `formatValue` / `formatX` / `formatScore`.
- `onPointClick({series, point})` /
  `onOutlierClick({series, point})` /
  `onSeriesToggle({series, hidden})`.

## Pure helpers

- `getLineHampelDefaultColor` / `getLineHampelFinitePoints`.
- `normaliseLineHampelWindow` (clamps to odd >= 3; even
  rounded up).
- `normaliseLineHampelNSigmas` (clamps to >= 0).
- `normaliseLineHampelMadScale` (clamps to > 0).
- `computeHampelMedian(values)` -- robust median; drops
  non-finite; empty -> NaN; verified at 50% breakdown
  point.
- `computeHampelMad(values, centerMedian?)` -- canonical
  MAD; accepts pre-computed center; verified at MAD([1..5])
  = 1 and MAD([1..5], center=1) = 2.
- `applyLineHampel(values, options?)` -- canonical filter;
  returns `{filtered, outlierIndices, medians, mads,
  deviations, scores, windowLength, nSigmas, madScale}`.
  Edge windows clamped to in-bounds samples.
- `runLineHampel(points, options?)` -- canonical pipeline;
  sorts ascending; drops non-finite; per-sample
  `{index, x, raw, filtered, windowMedian, windowMad,
  deviation, score, isOutlier}`.
- `computeLineHampelLayout` -- projects raw + filtered
  paths + outlier markers + replacement sticks.
- `describeLineHampelChart(series, options?)` -- screen-
  reader description.

## Tooltip

Hovering a dot shows label (with `OUTLIER` suffix when
flagged), x, raw, bold filtered, window median, window MAD,
score in sigma units, and the SG-style config row
`W=<w>, n=<n>, c=<c>`.
