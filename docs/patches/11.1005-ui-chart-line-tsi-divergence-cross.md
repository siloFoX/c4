## TSI pipeline (matches Blau 1991)

```
delta[i]    = close[i] - close[i-1] for i >= 1 (null at i=0)
absDelta[i] = |delta[i]|
innerDelta  = SMA-seeded EMA(delta, lengthR=25)
outerDelta  = SMA-seeded EMA(innerDelta, lengthS=13)
innerAbs    = SMA-seeded EMA(absDelta, lengthR=25)
outerAbs    = SMA-seeded EMA(innerAbs, lengthS=13)
tsi[i]      = 100 * outerDelta[i] / outerAbs[i]
            = 0 when outerAbs[i] === 0 (degenerate fallback)
```

The SMA-seeded EMA helper is exported as
`applyLineTsiDivergenceCrossSmaSeededEma`: the first `length`
contiguous valid samples seed the recurrence with their
arithmetic mean, after which `ema[i] = alpha * v + (1 - alpha) *
ema[i-1]` with `alpha = 2 / (length + 1)`. Any null gap aborts
further output -- consistent with how the other primitives treat
warmup boundaries.

Warmup length is `lengthR + lengthS - 1 = 37`: the inner EMA
becomes valid at index `lengthR = 25` (first delta at index 1
plus `lengthR - 1`), and the outer EMA needs another `lengthS -
1` valid inner samples to seed.

## Divergence regime model

5-state classifier exactly like CMO / RSI / Stoch divergence:

```
priceUp = close[i] > close[i - divergenceWindow]
tsiUp   = tsi[i]   > tsi[i - divergenceWindow]
state   = aligned-bullish    if priceUp && tsiUp
        | aligned-bearish    if !priceUp && !tsiUp
        | divergent-bullish  if !priceUp && tsiUp   (price down, TSI up)
        | divergent-bearish  if priceUp && !tsiUp   (price up, TSI down)
        | none               if either input is null
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
```

Crosses suppress when either `prev` or `cur` is `none`, so
warmup transitions never fire spuriously. Strict inequality
guarantees boundary equality (price flat or TSI saturated)
falls into the aligned-bearish state instead of double-firing.

Defaults: `lengthR = 25`, `lengthS = 13`, `divergenceWindow = 5`.

## Bit-exact anchors

| input            | tsi (post-warmup)        | regime (post i=42)  | crosses |
| ---------------- | ------------------------ | ------------------- | ------- |
| CONST close = K  | 0 (degenerate fallback)  | aligned-bearish     | 0       |
| LINEAR UP, close=i | +100 (saturated)       | divergent-bearish   | 0       |
| LINEAR DOWN, close=-i | -100 (saturated)    | aligned-bearish     | 0       |

CONST K verified across K = 0..1234 via parameterised test --
`delta = 0` collapses both numerator and denominator EMA chains to
0, the degenerate fallback returns +0, and `priceUp = false`
because `close[i] === close[i - 5]` (strict `>`). LINEAR UP saturates
TSI at +100 from index 37 onward; price is still rising (`close[i]
> close[i-5]`) but TSI is no longer rising (`100 === 100`), which
yields the canonical bearish divergence regime continuously for 38
samples on the 80-bar test fixture without ever transitioning out
of `divergent-bearish` -- so the cross count stays at 0 because
the regime is entered from `none` (prev = none is never a valid
cross precursor).
