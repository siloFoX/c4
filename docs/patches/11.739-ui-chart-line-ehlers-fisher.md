# Patch 11.739 -- UI: chart-line-ehlers-fisher primitive

Released as `v1.11.757` on 2026-05-26.

## Scope

Add `<ChartLineEhlersFisher>` -- a pure-SVG two-panel chart with an
**Ehlers Fisher Transform** panel mapping the normalised price into a
Gaussian-shaped distribution via the inverse hyperbolic tangent.

## Definition

Inputs:

- `data: ChartLineEhlersFisherPoint[]` where
  `ChartLineEhlersFisherPoint = { x; high; low }`. Bars with `high <
  low` are dropped.
- `period: number` -- the lookback. Default `9`. Normalized to an
  integer `>= 2`.
- `threshold: number` -- the +/- overbought / oversold band. Default
  `1.5`.

Algorithm (per bar `i >= period - 1`):

```
hl2[i] = (high[i] + low[i]) / 2
HH     = max(hl2 over [i-period+1 .. i])
LL     = min(hl2 over [i-period+1 .. i])
ratio  = HH == LL ? 0 : (hl2 - LL) / (HH - LL) - 0.5
v[i]   = clamp(0.66 * ratio + 0.67 * v[i-1], -0.999, +0.999)
fisher[i] = 0.5 * ln((1 + v[i]) / (1 - v[i])) + 0.5 * fisher[i-1]
```

`v` and `fisher` both seed at `0` for the recurrence (so warm-up
bars carry no contribution). The first `period - 1` bars are null on
both tracks.

## Bit-exact algebraic anchor

The clean anchor is the **constant-series identity**. If `high ==
low == K` for all bars:

- `hl2 = K`, `HH = LL = K`, `HH - LL = 0`, the degenerate branch sets
  `ratio = 0`.
- `v[i] = 0.66 * 0 + 0.67 * v[i-1]`. With the seed `v = 0` this stays
  at exactly `0`.
- `clamp(0) = 0` (the open interval contains zero).
- `fisher[i] = 0.5 * ln(1 / 1) + 0.5 * fisher[i-1] = 0 + 0 = 0`
  exactly.

So `CONST_FLAT` reads `norm[i] = 0` and `fisher[i] = 0` bit-exact at
every defined bar. This identity is the test backbone.

The window-bound identities are also bit-exact on integer fixtures:

- `RISING` (`high = low = j + 10`): `hh[i] = i + 10`, `ll[i] = i -
  period + 1 + 10`, both exact integers.
- `FALLING` (`high = low = 19 - j`): `hh[i] = 19 - (i - period + 1)`,
  `ll[i] = 19 - i`.

The Fisher recurrence on non-constant series passes through `ln`, so
those bars are checked with `toBeCloseTo(_, 10)` (RISING / FALLING
antisymmetry, translation invariance).

## Component

`forwardRef<HTMLDivElement, ChartLineEhlersFisherProps>` with
displayName. Renders:

- A region-role outer `<div data-section="chart-line-ehlers-fisher">`.
- An `<svg role="img">` with an `aria-describedby` description and a
  `<title>` / `<desc>` pair.
- Top panel: HL2 price line (`data-section="chart-line-ehlers-fisher-price-path"`)
  plus optional circle dots.
- Bottom panel: Fisher line
  (`data-section="chart-line-ehlers-fisher-line"`), zone-coloured
  markers (`data-section="chart-line-ehlers-fisher-marker"`,
  `data-zone` attribute), a zero line, and the +/- threshold
  dashed lines.
- Legend toggle (`data-section="chart-line-ehlers-fisher-legend-item"`)
  for the `price` and `fisher` series, plus a `showFisher` prop and
  an `onPointClick` handler.
- ARS design-system tokens for colour and motion (`motion-safe:
  animate-fade-in`).
