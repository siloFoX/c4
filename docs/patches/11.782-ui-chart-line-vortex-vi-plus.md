## Summary

`ChartLineVortexViPlus` is a pure-SVG dual-panel React primitive that
plots the close on the top panel and a Vortex Indicator +VI oscillator
on the bottom panel. +VI measures the positive vortex movement as a
fraction of the total true range over the lookback:

```
VM+[i] = |high[i] - low[i - 1]|       (i >= 1; null at i = 0)
TR[i]  = max(high[i] - low[i],
             |high[i] - close[i - 1]|,
             |low[i] - close[i - 1]|)  (bar 0 falls back to high-low)
+VI[i] = sum(VM+, [i - length + 1, i]) /
         sum(TR,  [i - length + 1, i])
```

Defaults: `length = 14`. Bar 0 has no prior low, so VM+[0] is `null`
and the rolling sum sets +VI to `null` for bars `0 .. length - 1` (the
window must lie entirely within bars where VM+ is defined). When the
TR sum is zero (singular: completely flat bars) +VI is `null`.

## Bit-exact anchor

For **CONST_BAR** (high `H`, low `L`, close `C` constant at every bar,
with `H > L` and `L <= C <= H`):

- `VM+[i] = |H - L| = H - L` for every `i >= 1`.
- `TR[i] = max(H - L, |H - C|, |L - C|) = H - L` for every `i >= 1`,
  because both `|H - C|` and `|L - C|` are bounded above by `H - L`
  when `C` lies in `[L, H]`.
- Therefore for any bar `i >= length` the rolling sums both equal
  `length * (H - L)`, so:

      +VI = (length * (H - L)) / (length * (H - L)) = 1

  **bit-exact** -- numerator and denominator share the same finite
  integer accumulation and trivially cancel.

For **CONST_FLAT** (`H = L = C`): every TR is zero, the rolling sum is
zero, and the component emits `null` rather than dividing by zero.
