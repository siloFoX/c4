# Patch 11.828 -- ChartLineVolumeRatio primitive (v1.11.846)

## Summary

Add `<ChartLineVolumeRatio />`, a pure-SVG dual-panel line chart that
renders a Volume Ratio oscillator beneath a price line. Volume Ratio
is the current bar's volume divided by the rolling-mean volume across
the lookback:

```
avgVol[i] = mean(volume[i - length + 1 .. i])
ratio[i]  = avgVol[i] === 0 ? null : volume[i] / avgVol[i]
```

- Warmup: `i < length - 1` is `null`.
- `null` whenever `avgVol = 0` (divide-by-zero guard).
- `-0` normalized to `+0` on every arithmetic result.
- Bounded below at zero (volume non-negative); unbounded above (a
  single spike can dwarf the rolling mean by orders of magnitude).

## Defaults

| Knob          | Value     |
|---------------|-----------|
| length        | 14        |
| highThreshold | 1.5       |
| lowThreshold  | 0.5       |
| ratioColor    | `#f59e0b` (amber) |

The amber color differentiates the ratio oscillator from the teal
net-volume primitive (11.827) shipped just before this one.

## Bit-exact anchors

The integration sweep iterates `(K, length, V)` and asserts:

- `CONST volume = V > 0`: `avgVol = V`, `ratio = V / V = 1` bit-exact
  for every post-warmup index. Identity division in IEEE 754 is
  exact regardless of `V`.
- `CONST volume = 0`: `avgVol = 0` -> divide-by-zero guard returns
  `null`.
- `LINEAR UP volume = i + 1` at `i = L - 1`:
  `avgVol = (L + 1) / 2`, `ratio = L / ((L + 1) / 2) = 2L / (L + 1)`.
  For `L in {3, 7, 15}` the constant is dyadic (1.5, 1.75, 1.875)
  and exact in IEEE 754 -- explicitly tested per L.

## Surface

Standard primitive surface:

- `ChartLineVolumeRatioPoint = { x, close, volume }`
- `runLineVolumeRatio(data, options?) -> Run` with `ok = series.length
  >= length`.
- `computeLineVolumeRatioLayout({ data, width?, height?, padding?, panelGap?, length?, highThreshold?, lowThreshold? }) -> Layout`.
  Ratio axis auto-fits but always starts at 0 and includes both
  threshold lines.
- `describeLineVolumeRatioChart(data, opts?) -> string` for the ARIA
  description.
- `<ChartLineVolumeRatio />` `forwardRef<HTMLDivElement>` with legend
  toggle (`price`, `ratio`), config badge, threshold + midline
  overlays (midline at `1.0` parity), markers on cross events, and
  a tooltip surfacing `close / vol / avgVol / ratio / zone / cross`.
