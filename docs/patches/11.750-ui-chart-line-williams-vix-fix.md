# Patch 11.750 -- UI: chart-line-williams-vix-fix primitive

Released as `v1.11.768` on 2026-05-26.

## Scope

Add `<ChartLineWilliamsVixFix>` -- a pure-SVG two-panel chart with
a Larry Williams **VIX Fix** panel reading the spike of the bar low
relative to the highest close across the lookback.

## Definition

Inputs:

- `data: ChartLineWilliamsVixFixPoint[]` where
  `ChartLineWilliamsVixFixPoint = { x; low; close }`. Bars with
  non-finite fields are dropped.
- `period: number` -- the lookback. Default `22`. Normalized to
  an integer `>= 1`.
- `threshold: number` -- the elevated-zone boundary; spike starts
  at `2 * threshold`. Default `16`. Normalized to a positive
  finite.

Algorithm (per bar `i >= period - 1`):

```
hh[i]  = max(close over [i - period + 1, i])
WVF[i] = 100 * (hh[i] - low[i]) / hh[i]
```

A bar is null when:

- It is in the warm-up region (`i < period - 1`).
- Any close inside the window is non-finite.
- The bar low is non-finite.
- The highest close `hh` is non-positive.

## Bit-exact algebraic anchors

**CONST_FLAT** (`low == close == K`): `hh = K`, so `WVF = 100 *
(K - K) / K = 0` bit-exact at every defined bar.

**RISING low == close** (`low == close == i + 10`): `hh = i + 10
= low`, so `WVF = 0` bit-exact at every defined bar.

**SPIKE_50** (`close == 20` constant, last bar `low = 10`): `hh
= 20, low = 10`. `WVF = 100 * (20 - 10) / 20 = 100 * 10 / 20 =
50` bit-exact (integer division by power of two).

**SPIKE_25** (`close == 20` constant, last bar `low = 15`): `hh
= 20, low = 15`. `WVF = 100 * 5 / 20 = 25` bit-exact (integer
division by power of two).

**Scale invariance**: multiplying both close and low by a
positive integer `s` scales `hh - low` and `hh` by `s`, so the
ratio is unchanged bit-exact.

## Component

`forwardRef<HTMLDivElement, ChartLineWilliamsVixFixProps>` with
displayName. Renders:

- A region-role outer `<div data-section="chart-line-williams-vix-fix">`.
- An `<svg role="img">` with an `aria-describedby` description.
- Top panel: close line, optional dots.
- Bottom panel: WVF line, zone-coloured markers (spike /
  elevated / normal / zero / none), zero reference line, dashed
  `threshold` reference line.
- Legend toggle for `price` / `wvf`, plus a `showWvf` prop and an
  `onPointClick` handler.
- ARS design-system tokens for colour and motion (`motion-safe:
  animate-fade-in`).

## Acceptance

- CONST_FLAT identity: WVF = 0 bit-exact.
- RISING (low == close) identity: WVF = 0 bit-exact.
- SPIKE_50: WVF = 50 bit-exact.
- SPIKE_25: WVF = 25 bit-exact.
- Non-positive `hh` / non-finite close in the window nulls the
  bar.
- Scale invariance: `*100` leaves WVF unchanged.
- WVF bounded `[0, 100]` on the wave fixture.
- Layout / component / ARIA / data-section hooks all covered.
