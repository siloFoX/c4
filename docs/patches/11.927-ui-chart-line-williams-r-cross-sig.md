# 11.927 -- UI: chart-line-williams-r-cross-sig primitive (v1.11.945)

## Summary

Adds `ChartLineWilliamsRCrossSig`, a pure-SVG dual-panel chart
that renders the close in the top panel and a close-only
Williams %R line plus an SMA signal line in the bottom panel,
marking bullish (cross up through signal) / bearish (cross
down through signal) trigger events. Each cross also carries a
`zone` tag (`overbought` / `oversold` / `neutral`) computed
from the %R value at the cross bar, surfacing overbought /
oversold trigger events via the regime classifier.

## Pipeline

For close-only data with default `length = 14`,
`signalLength = 3`:

```
range[i] = max(close[i-n+1..i]) - min(close[i-n+1..i])
wr[i]    = range > 0
             ? -100 * (highest - close) / range
             : -50
signal[i]= SMA(wr, signalLength)
bullish  : prev wr <= prev signal && cur wr > cur signal
bearish  : prev wr >= prev signal && cur wr < cur signal
zone     : 'overbought' if wr >= -20
           'oversold'   if wr <= -80
           'neutral'    otherwise
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> range = 0 every bar
  -> wr = -50 (neutral fallback). SMA of -50s = -50 via the
  `min === max` short-circuit. wr === signal = -50 every bar,
  the strict-inequality cross detector never fires, regime is
  `neutral`. cross count = 0. Verified across K in {0, 1, 42,
  100, 1234}.
- LINEAR UP `close = i` (length 14) -> close always sits at
  the window high so `(highest - close) = 0` -> wr = 0
  constant. signal = SMA(0, 3) = 0. wr === signal = 0, regime
  `overbought`, 0 crosses.
- LINEAR DOWN `close = -i` mirrors LINEAR UP -> wr = -100
  constant, signal = -100, regime `oversold`, 0 crosses.
- Recovery from oversold (close sweeps high after a long
  decline) -> wr surges from deeply oversold (< -80) toward 0,
  crosses signal upward, generating at least one zone-tagged
  cross.

## Regime classifier

| wr range            | regime      |
|---------------------|-------------|
| wr >= -20           | overbought  |
| -80 < wr < -20      | neutral     |
| wr <= -80           | oversold    |
| null                | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- Three horizontal reference bands inside the oscillator
  panel (overbought at -20, mid at -50, oversold at -80).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, `data-zone`, and a discrete colour per kind.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity (3 series: price, %R,
  signal).
