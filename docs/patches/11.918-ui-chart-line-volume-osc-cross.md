# Patch 11.918 -- UI: chart-line-volume-osc-cross primitive

## Summary

Added `<ChartLineVolumeOscCross />` -- pure-SVG dual-panel
React/TS primitive that renders the close (top panel) with
bullish / bearish arrow overlays at every Volume Oscillator
zero crossover, and the close-only Volume Oscillator line
(bottom panel) with cross dot markers and a dashed zero
baseline. Zero-cross variant of the Volume Oscillator family
that flags short over long volume MA regime transition events
distinct from the absolute volume delta magnitude.

## Algorithm

```
shortMa[i]  = EMA(close, shortLength)            (close-only volume proxy)
longMa[i]   = EMA(close, longLength)
volOsc[i]   = longMa > 0
              ? ((shortMa - longMa) / longMa) * 100
              : 0
bullish    : volOsc crosses up   (prev <= 0, cur > 0)
bearish    : volOsc crosses down (prev >= 0, cur < 0)
```

Defaults `shortLength = 5`, `longLength = 10` (canonical
Volume Oscillator parameters). Regime classifier `bullish`
(volOsc > 0 meaning short MA above long MA), `bearish` (volOsc
< 0), `neutral` (volOsc === 0), `none` (volOsc null).

## Bit-exact anchor

CONST close = K (K > 0): EMA(K) collapses to K for both
lengths via the `min === max` precision short-circuit; short
- long = 0; volOsc = 0 / K * 100 = 0. Regime `neutral`,
cross count = 0. CONST K = 0 yields the divide-by-zero guard
at every bar (volOsc = 0). Verified across K = 0, 1, 5, 17,
100, 1234.

LINEAR UP yields volOsc > 0 (faster short EMA catches up to
the rising series faster than the slower long EMA) -> regime
`bullish`. LINEAR DOWN yields volOsc < 0.
