# 11.777 chart-line-tsi-signal (v1.11.795)

## Summary

Pure-SVG dual-panel React/TypeScript chart primitive with
William Blau's **True Strength Index (TSI)** and its EMA signal
line beneath the close. Added as the `<ChartLineTsiSignal />`
UI primitive in `web/src/components/ui/`.

## Algorithm

For each bar with `longLength` (default 25), `shortLength`
(default 13), and `signalLength` (default 13):

```
mom[i]     = close[i] - close[i - 1]
ema1Mom[i] = EMA(mom, longLength)[i]
ema2Mom[i] = EMA(ema1Mom, shortLength)[i]
ema1Abs[i] = EMA(|mom|, longLength)[i]
ema2Abs[i] = EMA(ema1Abs, shortLength)[i]
TSI[i]     = 100 * ema2Mom[i] / ema2Abs[i]
Signal[i]  = EMA(TSI, signalLength)[i]
```

Bar 0 has no momentum (null). When `ema2Abs == 0` (singular:
completely flat close) `TSI` is `null`.

A configurable `overboughtThreshold` / `oversoldThreshold`
(defaults `+25` / `-25`) classifies bars as overbought, oversold,
or neutral.

## Anchors

- **RISING_BY_S (close[i] = c0 + S * i, S > 0)**: mom = +S
  every bar past the seed, |mom| = +S. Both EMA chains operate
  on the same constant series, so `ema2Mom == ema2Abs` exactly
  in IEEE 754, yielding `TSI = +100` **bit-exact** past the
  first momentum bar. Signal EMA of constant 100 (dyadic
  representable) is 100 bit-exact too. Verified across
  `S in {0.25, 0.5, 1, 2, 5, 10, 100}`.
- **FALLING_BY_S**: mom = -S, |mom| = +S. The EMAs differ only
  by sign, so `TSI = -100` and `Signal = -100` **bit-exact**.
  Verified across the same S set.
- **CONST_FLAT (close == K)**: mom = 0 -> ema2Abs = 0 ->
  singular -> `TSI = null` at every bar.
- **Bounded**: `|TSI| <= 100` for any finite input (since
  `|ema(mom)| <= ema(|mom|)`).
