## Summary

Added `<ChartLineStochCrossPct />` -- a pure-SVG, dual-panel
React/TS primitive that renders the close in the top panel and
overlays Stochastic %K, %D, and the `%K - %D` percent deviation
in the bottom panel. The "cross percent" treatment surfaces the
*momentum magnitude* (how stretched K is above or below D)
separate from the sign-based K-over-D crossings rendered by
other Stochastic primitives in the family:

```
highest[i]  = max(close[i - kLength + 1 .. i])
lowest[i]   = min(close[i - kLength + 1 .. i])
rawK[i]     = highest === lowest
                ? null
                : (close[i] - lowest) / (highest - lowest) * 100
K[i]        = SMA(rawK, slowKLength)
D[i]        = SMA(K, dLength)
stochPct[i] = K[i] - D[i]
```

Defaults: `kLength = 14`, `slowKLength = 3`, `dLength = 3`.
Regime classifier: `above` (stochPct > 0), `below` (< 0),
`at` (= 0), `none` (null).

## Bit-exact anchors

- **LINEAR UP step = 1 (close[i] = start + i)**: highest =
  close[i] = i + start, lowest = close[i - k + 1] = i - k + 1 +
  start, denominator = k - 1, numerator = close[i] - lowest = k
  - 1. So rawK = 100 every bar after warmup. K = SMA(100) = 100,
  D = SMA(100) = 100, stochPct = 100 - 100 = 0. Regime `at`
  everywhere after the chained SMA warmup at `k - 1 + slow - 1 +
  d - 1`. Verified across multiple start values.
- **CONST close = K**: highest === lowest = K so the divide-by-
  zero guard yields rawK = null -> K = null -> D = null ->
  stochPct = null. Regime `none` everywhere.

## Difference vs other Stoch primitives

| primitive | bottom panel | use-case |
| --- | --- | --- |
| `chart-line-stoch-cross` (existing) | %K + %D + cross dots | trigger events |
| `chart-line-stoch-rsi-cross` (existing) | StochRSI K + D + cross dots | momentum oscillator regime |
| `chart-line-stoch-cross-pct` (11.877) | %K + %D + (K - D) magnitude | momentum magnitude separate from crossings |

The cross-pct variant surfaces the *magnitude* of K's stretch
above or below D as a continuous channel, distinct from the
discrete sign-based crossover events the other Stoch primitives
emit.
