# 11.930 -- UI: chart-line-cci-overbought-cross primitive (v1.11.948)

## Summary

Adds `ChartLineCciOverboughtCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and a close-only
Commodity Channel Index (CCI) line in the bottom panel. Marks
bullish (entry into overbought) / bearish (exit from
overbought) trigger events at the canonical +100 overbought
threshold. Single-threshold cross variant of the CCI family
that flags discrete CCI level +100 entry / exit events
distinct from the +/- 100 / +/- 200 dual-band crossover of
`chart-line-cci-extreme-cross` (11.925).

## Pipeline

For close-only data with default `length = 20`:

```
sma[i]  = SMA(close, length)
md[i]   = avg |close[i-n+1..i] - sma[i]|
cci[i]  = md > 0
            ? (close - sma) / (0.015 * md)
            : 0
bullish : prev <= +100 && cur > +100  (entry into overbought)
bearish : prev >= +100 && cur < +100  (exit from overbought)
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> sma = K, all
  deviations are 0, mean deviation = 0, the 0/0 guard returns
  cci = 0. cci = 0 < 100, regime is `neutral` and the
  threshold is never crossed. cross count = 0. Verified
  across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` (length 20) -> cci is exactly 9.5 /
  0.075 = 126.667 from i = 19 onward, never moves -- 126.667
  > 100, regime `overbought`, 0 crosses (cci jumps from null
  to 126.667 so prev-null skips the strict-inequality
  detector).
- LINEAR DOWN `close = -i` -> cci = -126.667, -126.667 < 100,
  regime `neutral`, 0 crosses.
- Single positive spike at the end of an otherwise constant
  window of length 14 -> cci = 14 / 0.03 = 466.67 at the
  spike index, which crosses the +100 band in one bar and
  fires exactly 1 bullish entry cross.

## Regime classifier

| cci range            | regime      |
|----------------------|-------------|
| cci >= +100          | overbought  |
| cci < +100           | neutral     |
| null                 | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- Two horizontal reference bands inside the oscillator panel
  (threshold at +100, mid at 0).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind.
- Default oscillator range `+/- 300` expands to wrap any
  observed CCI value (e.g. 466.67 spike) so the marker never
  clips off the panel.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.

## Defaults

- length = 20
- threshold = +100
- oscillator range = +/- 300 (auto-expands)
- width 720, height 460, padding 44, panelGap 12
