# 11.941 -- UI: chart-line-roc-zero-cross primitive (v1.11.959)

## Summary

Adds `ChartLineRocZeroCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and the close-only
Rate of Change (ROC) line in the bottom panel. Marks bullish
(cross up through zero) / bearish (cross down through zero)
momentum baseline regime transition events at the zero line.
Zero-line cross variant of the ROC family that flags the
discrete ROC crossing of the zero baseline.

## Pipeline

For close-only data with default `length = 12`:

```
roc[i] = close[i-length] != 0
           ? (close[i] - close[i-length]) / |close[i-length]| * 100
           : 0
bullish : prev roc <= 0 && cur roc > 0   (momentum up)
bearish : prev roc >= 0 && cur roc < 0   (momentum down)
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> close[i] = close[i-
  length] = K -> numerator = 0 -> roc = 0 (even when K = 0
  the close[i-length] = 0 guard returns 0 rather than NaN).
  roc = 0 sits on the threshold but the strict-inequality
  detector never fires. regime `bullish` (roc >= 0). cross
  count = 0. Verified across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i + 100` -> roc at i=12 = (112 - 100) /
  100 * 100 = 12; positive but decreasing toward 0 as i
  grows. regime `bullish`. 0 crosses.
- LINEAR DOWN `close = 200 - i` -> roc at i=12 = (188 - 200)
  / 200 * 100 = -6; negative with magnitude shrinking.
  regime `bearish`. 0 crosses.
- Decline followed by sustained rise -> ROC transitions from
  negative to positive -> 1 bullish zero-line cross.
- Rise followed by sustained decline -> 1 bearish cross.

## Regime classifier

| roc range            | regime      |
|----------------------|-------------|
| roc >= 0             | bullish     |
| roc < 0              | bearish     |
| null                 | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- One horizontal reference band inside the oscillator panel
  (threshold at zero, the baseline).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind.
- Oscillator range auto-fits to observed ROC values with
  10% padding above and below; ROC is unbounded.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.

## Defaults

- length = 12
- threshold = 0
- oscillator range = auto-fit (ROC is unbounded)
- width 720, height 460, padding 44, panelGap 12
