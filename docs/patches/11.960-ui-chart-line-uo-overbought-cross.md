# Patch 11.960 - UI: chart-line-uo-overbought-cross primitive

Version: 1.11.978

## Summary

Add `web/src/components/ui/chart-line-uo-overbought-cross.tsx` -- a
pure-SVG dual panel line chart primitive that visualizes price alongside
Larry Williams' Ultimate Oscillator (UO) panel, with explicit cross
markers and overlay arrows at every UO crossing of the canonical 70
overbought line. Bullish (= entry, crossing UP through 70 = entering
overbought) and bearish (= exit, crossing DOWN through 70 = leaving
overbought). Shares the bit-exact UO pipeline with the `-zero-cross`
/ `-mid-cross` siblings; the only difference is the default threshold
(70) and the regime mapping (CONST UO=50 sits BELOW 70, so all CONST
bars are bearish).

## Pipeline

Identical to `chart-line-uo-mid-cross`. Threshold default flipped to
70.

## Bit-exact anchors

| Pattern                  | UO (after warmup)    | Regime  |
|--------------------------|----------------------|---------|
| CONST close = K          | 50 (0 / 0 -> 0.5)    | bearish |
| LINEAR UP (close = i)    | 100                  | bullish |
| LINEAR DOWN (close = -i) | 0                    | bearish |

The CONST anchor is tested across K in `{0, 1, 42, 100, 1234}` -- all
must produce `uo === 50` bit-exactly (regime bearish since 50 < 70).
LINEAR UP saturates at 100 (regime bullish at the upper saturation,
above 70). LINEAR DOWN saturates at 0 (regime bearish, below 70).

Cross events fire during transients:
* decline-then-rise: uo transitions from 0 through 70 going up ->
  bullish entry through 70.
* rise-then-decline: uo transitions from 100 through 70 going down ->
  bearish exit through 70.

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), UO panel below.
* UO panel uses a fixed `[0, 100]` range so the threshold band sits at
  the 70-line (closer to oscTop than midpoint).
* Cross markers drawn as circles on the UO panel at `(cx, cyOsc)` and
  as arrows on the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="UO Overbought Cross chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only
  text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
