# Patch 11.901 -- UI: chart-line-aroon-cross primitive

## Summary

Added `<ChartLineAroonCross />` -- pure-SVG dual-panel React/TS
primitive that renders the close (top panel) with bullish /
bearish arrow overlays at every Aroon Up vs Aroon Down cross
trigger, and the Aroon Up / Aroon Down lines (bottom panel) on
a fixed 0-100 oscillator with cross dot markers. Crossover
variant of the Aroon family that flags trend regime transition
events at the high or low extremes of the Aroon range.

## Algorithm

```
barsSinceHigh[i] = bars since highest close in last (n+1)
barsSinceLow[i]  = bars since lowest close in last (n+1)
aroonUp[i]       = ((n - barsSinceHigh) / n) * 100
aroonDown[i]     = ((n - barsSinceLow) / n) * 100
bullish         : (up - down) crosses up   (prev <= 0, cur > 0)
bearish         : (up - down) crosses down (prev >= 0, cur < 0)
```

Tie break: the most recent bar wins, so a CONST series yields
barsSinceHigh = barsSinceLow = 0 on every settled bar. Defaults
`length = 14`. Regime classifier `bullish` (up > down),
`bearish` (up < down), `neutral` (up === down), `none` (either
null).

## Bit-exact anchor

CONST close = K: every close equals K, so the most recent bar
is both the highest and the lowest. Therefore barsSinceHigh =
barsSinceLow = 0, Aroon Up = Aroon Down = 100, diff = 0, regime
`neutral`, cross count = 0. Verified across K = 0, 1, 5, 17,
100, 1234.

LINEAR UP also has a bit-exact anchor: every settled bar
satisfies highest = current and lowest = bar (i - n), so Aroon
Up = 100, Aroon Down = 0, regime `bullish`. LINEAR DOWN is the
mirror image.

## Visual

- Two stacked panels with the osc panel pinned to 0-100.
- Top: close line + bullish/bearish polygon arrows at cross
  bars.
- Bottom: Aroon Up (green) + Aroon Down (red) lines + cross
  dot markers + 30/50/70 reference bands.
- Tooltip lists x / close / up / down / regime / counts /
  crosses.
- Config badge + togglable legend (close / Aroon Up / Aroon
  Down).
- Cross marker is `role="graphics-symbol"` with `tabIndex={0}`.
