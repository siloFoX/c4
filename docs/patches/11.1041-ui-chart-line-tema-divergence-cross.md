# 11.1041 -- UI: chart-line-tema-divergence-cross primitive (v1.11.1059)

## Scope

Add a pure-SVG dual-panel `ChartLineTemaDivergenceCross`
component that renders the close in the top panel and the
Triple Exponential Moving Average (TEMA) in the bottom
panel, flagging price-vs-TEMA direction-disagreement
events as responsive trend reversal warnings with bias
coloring derived from the TEMA slope at the trigger bar.
Uses the canonical 5-state divergence regime model
(aligned-bullish / aligned-bearish / divergent-bullish /
divergent-bearish / none).

TEMA is Patrick Mulloy's zero-lag triple EMA composite
(1994): `3 * EMA1 - 3 * EMA2 + EMA3` where each EMA layer
applies the same period. The triple-stacked combination
cancels the EMA centroid lag exactly on linear input, so
TEMA tracks the close with zero lag -- the divergence
detector here surfaces only genuine direction
disagreements caused by sub-linear price patterns, no
lag-driven false positives.

## TEMA divergence pipeline

```
ema1[i]     = EMA(close, period)
ema2[i]     = EMA(ema1,  period)
ema3[i]     = EMA(ema2,  period)
tema[i]     = 3 * ema1[i] - 3 * ema2[i] + ema3[i]

priceUp     : close[i] > close[i-1]
priceDown   : close[i] < close[i-1]
temaUp      : tema[i]  > tema[i-1]
temaDown    : tema[i]  < tema[i-1]

regime ->
  'aligned-bullish'   when priceUp   && temaUp
  'aligned-bearish'   when priceDown && temaDown
  'divergent-bullish' when priceDown && temaUp
  'divergent-bearish' when priceUp   && temaDown
  'none'              when either side flat or null

bullish (divergence trigger up) :
  prev regime !== 'divergent-bullish' &&
  cur regime === 'divergent-bullish'
bearish (divergence trigger down) :
  prev regime !== 'divergent-bearish' &&
  cur regime === 'divergent-bearish'

bias        : tema[i] vs tema[i-1] -> up / down / flat /
              none
```

Default `period = 14`. Each EMA layer uses SMA seeding (
seed at index `period - 1` with `SMA(values, period)`),
which keeps the steady-state lag on linear input exactly
equal to `(period - 1) / 2` from the seed bar onwards.
Stacked through the Mulloy combiner the three lags cancel,
giving TEMA = close on linear input.

Warmup is `3 * (period - 1) = 39` for the default tuning:
ema1 seeds at `period - 1 = 13`, ema2 at `2 * (period - 1)
= 26`, ema3 at `3 * (period - 1) = 39`. Divergence
detection requires the previous TEMA + close, so the first
regime classification lands at i = 40.

## Bit-exact anchors

- **CONST close = K**: each EMA seeds at K -> ema1 = ema2
  = ema3 = K -> TEMA = `3K - 3K + K = K`. Both close and
  TEMA are flat -> regime `none` for every bar. 0
  divergence crosses. Verified across K in {0, 1, 50, 200,
  1234}.
- **LINEAR UP close = i**: SMA-seeded EMA on linear input
  settles at `i - (period - 1) / 2 = i - 6.5`. Each
  subsequent layer adds another `(period - 1) / 2 = 6.5`
  bars of lag: ema1 = i - 6.5, ema2 = i - 13, ema3 = i -
  19.5. The Mulloy identity gives TEMA = `3 * (i - 6.5) -
  3 * (i - 13) + (i - 19.5) = 3i - 19.5 - 3i + 39 + i -
  19.5 = i + (39 - 19.5 - 19.5) = i + 0 = i` -- zero lag,
  exact match to close. close[i] - close[i-1] = +1 (up);
  TEMA[i] - TEMA[i-1] = +1 (up). Both up -> regime
  `aligned-bullish`. 0 divergence crosses.
- **LINEAR DOWN close = -i**: mirror -> TEMA = -i. Both
  close and TEMA falling -> regime `aligned-bearish`. 0
  divergence crosses.

The clean zero-lag identity on LINEAR input is the textbook
Patrick Mulloy 1994 property and the same identity the
`chart-line-tema-cross-sig` sibling exploits.

## Risk

- Pure-SVG, deterministic `.toFixed(2)`, no canvas, no
  chart lib dependency.
- 5-state regime classifier follows the
  chart-line-*-divergence-cross family convention; the
  `bullish` / `bearish` cross *kinds* still follow the
  sibling cross-sig convention (line direction).
- TEMA's zero-lag identity makes the LINEAR anchors
  exceptionally clean -- but on noisy real-world data the
  triple-stacked EMA amplifies single-bar volatility,
  surfacing more divergence triggers than the SMA / EMA /
  WMA / HMA siblings. TEMA is the most sensitive variant
  in the MA-divergence-cross family.
- Cross detection fires only on transition INTO a
  divergent state (not while staying in one), preventing
  double-fire on consecutive bars in the same regime.
- Warmup is `3 * (period - 1) = 39` bars -- 3x the
  SMA / EMA / WMA warmup. Callers feeding short data
  series may want to use a shorter period to reduce the
  warmup boundary (e.g. period = 7 -> warmup = 18).
- EMA helper uses SMA seeding (not first-value seeding)
  so the steady-state lag on linear input is exactly
  `(period - 1) / 2` from the seed bar onwards, making
  the Mulloy zero-lag identity bit-exact rather than
  asymptotic.
