## Summary

Added `<ChartLineStochCross />` -- a pure-SVG dual-panel React/TS
primitive that renders a close line in the top panel and the
Stochastic %K and %D lines in the bottom panel, with crossover
markers tagged by the overbought / oversold zone at the cross bar.
Strong trigger events surface the actionable signals:

- `oversoldTrigger`   -- `up` cross while %K <= oversoldLevel
- `overboughtTrigger` -- `down` cross while %K >= overboughtLevel
- `neutral`           -- cross outside the extremes

Defaults: `kLength = 14`, `dLength = 3`, `overboughtLevel = 80`,
`oversoldLevel = 20`, kColor `#0ea5e9` (sky), dColor `#f97316`
(orange), overbought line `#dc2626`, oversold line `#16a34a`,
trigger markers green / red / slate.

## Formula

```
lowestLow   = min(low[i-kLen+1 .. i])
highestHigh = max(high[i-kLen+1 .. i])
range       = highestHigh - lowestLow
%K = range > 0 ? 100 * (close - lowestLow) / range : null
%D = SMA(%K, dLength)
```

Cross detection uses a prev-relation + strict-inequality guard
(`prev <= prev_d && cur > cur_d` is an up cross). The trigger
type is `oversold` when an `up` cross lands at %K <= oversoldLevel,
`overbought` when a `down` cross lands at %K >= overboughtLevel,
otherwise `neutral`.

## Bit-exact anchors

The SMA helper carries the `min === max` window-constant precision
fix so a constant %K window lands bit-exactly on its value.

- **CONST h = l = close = K**: highestHigh = lowestLow = K, so
  range = 0 -> `%K = null` forever. `%D = SMA(null) = null`. Zero
  crosses fire and the zone is `none` for every bar.
- **LINEAR UP h = l = close = i+1**: highestHigh - lowestLow =
  kLen-1, close - lowestLow = kLen-1, so `%K = 100` once the
  window is full. `%D = SMA(100) = 100`. Relation stays `equal`,
  zero crosses fire; %K settles in the `overbought` zone but no
  trigger lands because there's no cross.
- **LINEAR DOWN h = l = close = N-i**: lowestLow = N-i,
  highestHigh = N-i+kLen-1, close - lowestLow = 0, so `%K = 0`
  once seeded. `%D = 0`. Mirror image -- zero crosses, settles in
  `oversold` zone.
