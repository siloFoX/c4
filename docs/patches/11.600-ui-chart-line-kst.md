## Summary

New `ChartLineKst` UI primitive at
`web/src/components/ui/chart-line-kst.tsx` -- a pure-SVG line chart
with a Know Sure Thing summed rate-of-change momentum oscillator panel
and signal line. The top panel renders the price line; the bottom
panel renders the KST line and a dashed signal line on a symmetric
y-axis with a dashed zero centerline and sign-coloured markers.

## Indicator math

Martin Pring's Know Sure Thing, with four ROC periods (default
10/15/20/30), four SMA periods (default 10/10/10/15), and a signal
period (default 9):

```
ROC_k    = 100 * (value[i] - value[i-rocK]) / value[i-rocK]
RCMA_k   = SMA(ROC_k, smaK)
KST      = RCMA1*1 + RCMA2*2 + RCMA3*3 + RCMA4*4
signal   = SMA(KST, signalPeriod)
```

The four legs are summed at the fixed ascending weights `1/2/3/4` --
the slower, more reliable legs carry the most weight. The KST
oscillates around zero; a cross of the KST and its signal line marks
a momentum shift.

`computeLineKstRoc(values, period)` is the percentage rate of change
(leading nulls before the lookback, zero on a zero base, negative-zero
normalised). `computeLineKstSma(src, period)` is a simple moving
average that tolerates leading `null` placeholders.

### Worked fixture

`values = [100,100,100,100,200,200,200,200]`, ROC `1/2/3/4`, SMA
`1/1/1/1` (identity), signal `2`:

```
rcma1 = [null,0,0,0,100,0,0,0]
rcma2 = [null,null,0,0,100,100,0,0]
rcma3 = [null,null,null,0,100,100,100,0]
rcma4 = [null,null,null,null,100,100,100,100]
kst   = sum(rcma_k * weight_k) = [null,null,null,null,1000,900,700,400]
signal = SMA(kst, 2)           = [null,null,null,null,null,950,800,550]
```

At index 7 only the slowest leg (rcma4, weight 4) is non-zero, so
`kst[7] = 100 * 4 = 400` -- the weighting is visible in the
descending `1000 -> 900 -> 700 -> 400`.

## API

- `getLineKstFinitePoints(points)` -- drop points with a non-finite
  `x` or `value`.
- `normalizeLineKstPeriod(period, fallback)` -- positive integer;
  non-finite / sub-1 falls back, fractional floors.
- `computeLineKstRoc(values, period)` -- the percentage ROC series.
- `computeLineKstSma(src, period)` -- the null-tolerant SMA series.
- `computeLineKst(values, config)` -- takes a `LineKstConfig` (the
  nine resolved periods) and returns
  `{ rcma1, rcma2, rcma3, rcma4, kst, signal }`.
- `runLineKst(points, options?)` -- sort finite points by `x`, compute
  the legs and the KST, return `{ series, config, rcma1..rcma4, kst,
  signal, samples, kstFinal, signalFinal, kstMin, kstMax,
  positiveCount, negativeCount, ok }`. Each sample carries a
  `positive` / `negative` / `zero` sign from the KST. `ok` is false
  for fewer than two points.
- `computeLineKstLayout(options)` -- stack a price panel above a KST
  panel with a symmetric y-bound. Returns panels, ticks, the price
  path, the KST path, the signal path, sign-coloured markers, and the
  zero centerline `y`.
- `describeLineKstChart(data, options?)` -- ARIA description.

## Component

`ChartLineKst` is a `forwardRef` region (`role="region"`) with an
ARIA-linked description, a `role="img"` SVG, panel labels, a config
badge (the four ROC periods and the signal period), a three-series
legend (Price / KST / Signal) with toggle, a dashed zero centerline,
the KST line, a dashed signal line, markers coloured by sign, a
hover/focus tooltip exposing the value, KST, signal, and sign, and a
`motion-safe:animate-fade-in` entrance. Every node carries a
`data-section` attribute.

## Distinctness

- **vs chart-line-coppock** -- the Coppock Curve sums *two* ROC legs
  and smooths the *total* with one weighted moving average. The KST
  smooths *each of four* ROC legs *separately* with its own SMA, then
  sums them at distinct ascending weights, and adds a signal line --
  none of which the Coppock Curve does.
- **vs chart-line-trix** -- TRIX is the rate of change of a triple
  EMA. The KST never triple-smooths; it is a weighted sum of
  SMA-smoothed ROCs.
- **vs chart-line-ppo** -- the PPO is a percentage of two EMAs of the
  price. The KST has no EMA and no percentage-of-EMA step; it is the
  weighted four-leg ROC sum.
