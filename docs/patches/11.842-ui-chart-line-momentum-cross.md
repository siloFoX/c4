# Patch 11.842 -- ChartLineMomentumCross primitive (v1.11.860)

## Summary

Add `<ChartLineMomentumCross />`, a pure-SVG dual-panel chart that
plots an absolute-units Momentum oscillator beneath a price line,
with scatter markers placed at every zero-line crossover event:

```
momentum[i] = close[i] - close[i - length]
up cross    = prev momentum <= 0  AND  curr momentum > 0
down cross  = prev momentum >= 0  AND  curr momentum < 0
```

- Warmup: `i < length` -> `null`.
- Output unit is price absolute (not percentage); the
  percent-normalized variant lives in 11.835 chart-line-momentum-pct.
- The cross detector treats `prev = 0, curr > 0` as an up-cross and
  `prev > 0, curr = 0` as a non-cross (strict requirement on the
  current side).

## Defaults

| Knob              | Value     |
|-------------------|-----------|
| length            | 10        |
| priceColor        | `#2563eb` |
| momentumColor     | `#d946ef` (fuchsia) |
| bullishColor      | `#16a34a` (green) for up-cross markers |
| bearishColor      | `#dc2626` (red) for down-cross markers |
| markerRadius      | 5         |

Fuchsia momentum keeps the bottom panel visually separate from the
neighbour pct-style oscillators (pink `#ec4899` of 11.835 and
violet `#8b5cf6` of 11.836).

## Bit-exact anchors

- `CONST close = K`: `momentum = 0` for every post-warmup bar. The
  relation never strictly flips, so `upCrossCount = downCrossCount =
  0`. Verified across `(K, length)` sweeps.
- `LINEAR UP close = i + 1`: `momentum = L > 0` constant for every
  post-warmup bar (integer subtraction of consecutive linear values
  is exact). Zero crosses.
- `LINEAR DOWN close = N - i`: `momentum = -L < 0` constant. Zero
  crosses.
- `STEP close = K1 for i < N, K2 for i >= N` with `K1 != K2` and
  `N >= length`: between `i = length` and `i = N - 1` momentum is
  zero (both bars in segment one). At `i = N`, momentum jumps to
  `K2 - K1` and a single up-cross (if `K2 > K1`) or down-cross (if
  `K2 < K1`) fires. When the lookback bar finally enters segment
  two (`i > N + length - 1`), momentum returns to zero without a
  second cross (strict `curr > 0` / `curr < 0` rules out the
  transition back).

## Surface

Dual-panel primitive surface:

- `ChartLineMomentumCrossPoint = { x, close }`
- `runLineMomentumCross(data, options?) -> Run` with
  `ok = series.length > length`.
- `computeLineMomentumCrossLayout({ data, width?, height?, padding?, panelGap?, length? }) -> Layout`.
  Bottom panel y-axis auto-fits but always includes 0.
- `describeLineMomentumCrossChart(data, opts?) -> string` for the
  ARIA description.
- `<ChartLineMomentumCross />` `forwardRef<HTMLDivElement>` with two
  legend toggles (`price`, `momentum`), config badge with live
  cross counts, scatter markers at each crossover event, and a
  tooltip surfacing `close / prior / momentum / zone / cross`. Live
  cross counts also exposed as `data-up-cross-count` /
  `data-down-cross-count` attributes for selector-driven assertions.
