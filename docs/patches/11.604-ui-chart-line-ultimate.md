## Summary

New `ChartLineUltimate` UI primitive at
`web/src/components/ui/chart-line-ultimate.tsx` -- a pure-SVG line
chart with an Ultimate Oscillator panel blending three lookback
windows. The top panel renders the close price line; the bottom
panel renders the oscillator line on a fixed 0-100 axis with shaded
overbought and oversold zones, dashed threshold lines, and markers
coloured by zone.

This primitive consumes `{ x, high, low, close }` points.

## Indicator math

Larry Williams' Ultimate Oscillator, with three lookback periods
(defaults 7 / 14 / 28):

```
prevClose  = close[i-1]
BP[i]      = close[i] - min(low[i], prevClose)            buying pressure
TR[i]      = max(high[i], prevClose) - min(low[i], prevClose)   true range
Avg_p[i]   = sum(BP over p) / sum(TR over p)
UO[i]      = 100 * (4*Avg_short + 2*Avg_medium + 1*Avg_long) / 7
```

`computeLineUltimateBpTr` derives buying pressure and true range;
index 0 has no prior close so both read null.
`computeLineUltimateAvg` sums buying pressure and true range over a
lookback window and divides them -- a zero total true range reads
the neutral 0.5. The three ratios are blended at the fixed
descending weights 4/2/1 (the shortest window carries the most
weight) and scaled to 0-100.

Because `0 <= BP <= TR` for every bar, each `Avg` lies in `[0, 1]` and
the weighted blend stays bounded to `[0, 100]`.

### Worked fixture

`high = [12,16,16,20,20,24,22]`, `low = [8,8,10,10,12,14,12]`,
`close = [10,14,12,18,16,20,14]`, lookbacks `1/2/3`:

```
bp = close - min(low, prevClose)              = [null,6,2,8,4,6,2]
tr = max(high, prevClose) - min(low, prevClose) = [null,8,6,10,8,10,10]
uo = 100 * (4*Avg1 + 2*Avg2 + Avg3) / 7        = [.,.,.,73.10,55.95,59.34,28.98]
```

With thresholds 70/30: one overbought reading (73.10) and one
oversold (28.98).

## API

- `getLineUltimateFinitePoints(points)` -- drop points with a
  non-finite `x`, `high`, `low` or `close`.
- `normalizeLineUltimatePeriod(period, fallback)` -- positive
  integer; non-finite / sub-1 falls back, fractional floors.
- `computeLineUltimateBpTr(highs, lows, closes)` -- returns
  `{ bp, tr }`.
- `computeLineUltimateAvg(bp, tr, period)` -- the buying-pressure to
  true-range ratio over the window (zero-range window reads 0.5).
- `computeLineUltimate(highs, lows, closes, shortPeriod,
  mediumPeriod, longPeriod)` -- returns `{ bp, tr, avg1, avg2, avg3,
  uo }`.
- `runLineUltimate(points, options?)` -- sort finite points by `x`,
  compute the oscillator, return `{ series, shortPeriod,
  mediumPeriod, longPeriod, upperThreshold, lowerThreshold, bp, tr,
  uo, samples, uoFinal, uoMin, uoMax, overboughtCount,
  oversoldCount, ok }`. Each sample carries an `overbought` /
  `oversold` / `neutral` zone. `ok` is false for fewer than two
  points.
- `computeLineUltimateLayout(options)` -- stack a price panel above
  an oscillator panel whose y-axis is fixed 0-100. Returns panels,
  ticks, the price path, the oscillator path, zone-coloured markers,
  the zone rects, and the threshold-line `y` positions.
- `describeLineUltimateChart(data, options?)` -- ARIA description.

## Component

`ChartLineUltimate` is a `forwardRef` region (`role="region"`) with
an ARIA-linked description, a `role="img"` SVG, panel labels, a
config badge (the three lookback periods), a two-series legend
(Price / Ultimate) with toggle, shaded overbought and oversold
zones, dashed threshold lines, the oscillator line, markers coloured
by zone, a hover/focus tooltip exposing the high, low, close,
oscillator value, and zone, and a `motion-safe:animate-fade-in`
entrance. Every node carries a `data-section` attribute.

## Distinctness

- **vs chart-line-rsi / chart-line-stochastic** -- those are
  *single-window* oscillators (one gain/loss ratio or one range
  position). The Ultimate Oscillator blends *three* windows at the
  fixed weights 4/2/1, and it is built from *buying pressure* and
  *true range* rather than gains/losses or range position.
- **vs chart-line-stc** -- the Schaff Trend Cycle is a double
  stochastic of the MACD line. The Ultimate Oscillator never touches
  MACD or a stochastic; it is a weighted blend of three BP/TR ratios.
- **vs chart-line-williams-r** -- both are Larry Williams indicators,
  but Williams %R is a single-window range-position reading. The
  Ultimate Oscillator is the three-window buying-pressure blend.
