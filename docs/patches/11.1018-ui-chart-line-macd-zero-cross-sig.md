component that renders the close in the top panel and the

## MACD pipeline (Appel)

```
ema_fast   = SMA-seeded EMA(close, fastLength=12)
ema_slow   = SMA-seeded EMA(close, slowLength=26)
macd[i]    = ema_fast - ema_slow
signal[i]  = SMA-seeded EMA(macd, signalLength=9)
bullish    : prev signal <= 0 && cur signal >  0
bearish    : prev signal >= 0 && cur signal <  0
regime     : bullish (signal >= 0), bearish (signal < 0)
bias       : up/down/flat/none from signal[i] vs signal[i-1]
```

Warmup is `slowLength + signalLength - 2 = 33` for the
default tuning: ema_slow seeds at `slowLength - 1 = 25`, then
the signal SMA-seeded EMA needs another `signalLength - 1 =
8` valid macd samples. SMA-seeded EMA preserves steady-state
exactly at the seed bar for linear input, so for LINEAR UP
ema_fast = `i - 5.5` and ema_slow = `i - 12.5` from their
respective seed indices, yielding macd = `+7` constant from
i=25.

Defaults: `fastLength = 12`, `slowLength = 26`, `signalLength
= 9`, `threshold = 0`. Bias-coloured cross markers mirror the
convention already used by `chart-line-adx-mid-cross-sig` /
`chart-line-stc-divergence-cross`.

## Bit-exact anchors

| input              | macd (post-warmup) | signal (post-warmup) | regime  | bias post-warmup | crosses |
| ------------------ | ------------------ | -------------------- | ------- | ---------------- | ------- |
| CONST close = K    | 0                  | 0                    | bullish | flat (after WARMUP+1) | 0 |
| LINEAR UP, close=i | +7                 | +7                   | bullish | mostly flat (FP drift)| 0 |
| LINEAR DOWN, close=-i | -7              | -7                   | bearish | mostly flat (FP drift)| 0 |

CONST K verified across K = 0..1234 -- ema_fast and ema_slow
both collapse to K via the SMA-seed-and-constant-EMA identity,
so macd = K - K = 0 exactly, and the signal SMA of zeros is 0
exactly. Bias is `flat` from the second valid signal sample
onward (`signal[WARMUP] === signal[WARMUP+1] === 0`).

LINEAR UP / DOWN exercise the steady-state EMA algebra: macd
settles at `+7` / `-7` via the lag formula `fast lag - slow lag
= (fastLength - 1)/2 - (slowLength - 1)/2 = 5.5 - 12.5 = -7`
(direction depends on sign of price slope). signal SMA-seeded
EMA of a near-constant macd settles at ~7 / ~-7, with tiny
floating-point drift from the EMA recurrence. The drift
produces a small number of `up` / `down` bias samples
interleaved with the dominant `flat` bias; the test asserts
the sum `upBias + downBias + flatBias = (validSamples - 1)`
rather than the exact split, since the FP noise is
deterministic per platform but not bit-exact across compilers.
regime stays `bullish` / `bearish` and 0 zero-line crossings
fire because signal never changes sign.

## Risk

Zero risk -- net-new component, no modifications to existing
files except the barrel append and the four manifest bumps.
SMA-seeded EMA helper is locally scoped. The component is
companion to the existing MACD divergence primitive
(`chart-line-macd-divergence-cross`, 11.985) but uses the
mid-cross-sig pattern (zero-line trigger with signal SMA)
instead of the divergence-regime classifier.
