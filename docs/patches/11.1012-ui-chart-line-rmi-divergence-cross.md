# 11.1012 -- UI: chart-line-rmi-divergence-cross primitive (v1.11.1030)

## Scope

Add a pure-SVG dual-panel `ChartLineRmiDivergenceCross`
component that renders the close in the top panel and the
Roger Altman Relative Momentum Index (RMI) line in the bottom
panel, marking discrete price-versus-RMI direction
disagreement events as bullish / bearish divergence cross
transitions. Divergence variant of the RSI-with-lookback
family; pairs the canonical Altman RMI (RSI computed over a
multi-bar momentum lookback) with the same 5-state regime
model already used by the other divergence primitives.

## RMI pipeline (Altman)

```
mom[i]      = close[i] - close[i - momentumLength]
gain[i]     = max(mom[i], 0)
loss[i]     = max(-mom[i], 0)
avgGain     = SMA-seeded Wilder RMA over `length`
avgLoss     = SMA-seeded Wilder RMA over `length`
rmi[i]      = 100 * avgGain / (avgGain + avgLoss);
              50 when (avgGain + avgLoss) === 0 (degenerate)
```

When `momentumLength = 1` the RMI reduces to the standard
Wilder RSI. The `momentumLength` lookback smooths out single-
bar noise and is the indicator's defining feature.

Warmup is `momentumLength + length - 1 = 17` for the default
tuning: the momentum lookback needs `momentumLength = 4` bars
of history (gain/loss valid from index 4), then the Wilder
RMA seed needs another `length - 1 = 13` valid samples.

## Divergence regime model

5-state classifier exactly like the other divergence primitives:

```
priceUp = close[i] > close[i - divergenceWindow]
rmiUp   = rmi[i]   > rmi[i - divergenceWindow]
state   = aligned-bullish    if priceUp && rmiUp
        | aligned-bearish    if !priceUp && !rmiUp
        | divergent-bullish  if !priceUp && rmiUp   (price down, RMI up)
        | divergent-bearish  if priceUp && !rmiUp   (price up, RMI down)
        | none               if either input is null
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
```

Defaults: `length = 14`, `momentumLength = 4`,
`divergenceWindow = 5`. Crosses suppress when either `prev` or
`cur` is `none`, so warmup transitions never fire spuriously.
Strict inequality guarantees boundary equality (RMI
saturated) falls into the aligned-bearish state instead of
double-firing.

## Bit-exact anchors

| input              | rmi (post-warmup)      | regime (post i=22)  | crosses |
| ------------------ | ---------------------- | ------------------- | ------- |
| CONST close = K    | 50 (degenerate fallback) | aligned-bearish   | 0       |
| LINEAR UP, close=i | 100 (saturated)        | divergent-bearish   | 0       |
| LINEAR DOWN, close=-i | 0 (saturated)       | aligned-bearish     | 0       |

CONST K verified across K = 0..1234 -- mom = 0 for every bar
once the lookback is in range, so gain = loss = 0; both
Wilder RMA chains collapse to 0; the (avgGain + avgLoss) ===
0 degenerate fallback returns 50.

LINEAR UP exercises the saturated-trend case: mom =
`close[i] - close[i - 4] = +4` for every bar, so gain = 4
constant, loss = 0. The Wilder RMA seeds at avgGain = 4,
avgLoss = 0, and `rmi = 100 * 4 / (4 + 0) = 100` saturated.
priceUp = true, rmiUp = false (100 === 100) -> regime
`divergent-bearish` for the full 38 valid samples on the
60-bar fixture.

LINEAR DOWN is the algebraic mirror: mom = -4 constant ->
gain = 0, loss = 4 -> avgGain = 0, avgLoss = 4 -> rmi =
0 saturated. priceUp = false, rmiUp = false -> regime
`aligned-bearish`.
