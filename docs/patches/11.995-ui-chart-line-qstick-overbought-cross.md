## Summary

Add `web/src/components/ui/chart-line-qstick-overbought-cross.tsx`
-- a pure-SVG dual-panel React/TypeScript chart primitive that
renders the close in the top panel and the QStick (SMA of close
- open candle body) line in the bottom panel, marking discrete
bullish (QStick crosses up through overbought threshold +0.5,
candle body momentum overbought trigger) / bearish (QStick
crosses down through overbought threshold +0.5) QStick
overbought-crossover events. Overbought-threshold variant of
the QStick family: uses a fixed positive threshold to flag
sustained bullish-body strength.

## Math anchor (bit-exact)

```
body[i]    = close[i] - open[i]
qstick[i]  = SMA(body, length)
bullish    : prev qstick <= 0.5 && cur qstick >  0.5
bearish    : prev qstick >= 0.5 && cur qstick <  0.5
```

- **CONST open == close == K**: body = 0 -> qstick = 0
  constant. 0 < 0.5 -> regime `bearish` (not overbought).
  0 crosses. Verified across K = 0..1234.
- **CONST body = +1** (open=i-1, close=i): qstick = 1 constant.
  1 >= 0.5 -> regime `bullish` (in overbought zone). 0 crosses.
- **CONST body = -1** (open=i+1, close=i): qstick = -1
  constant. -1 < 0.5 -> regime `bearish`. 0 crosses.

For 50 bars at defaults (length=8, threshold=0.5):
- qstick null for i = 0..6 (7 entries)
- post-warmup regime count = 43 (i = 7..49)

Strict inequality on `cur` (`cur > 0.5` for bullish, `cur < 0.5`
for bearish) means boundary equality (qstick == 0.5) never
double-fires.

Layout auto-fits oscMin / oscMax with 10% padding and expands
to include the threshold band when QStick values collapse to
one side.
