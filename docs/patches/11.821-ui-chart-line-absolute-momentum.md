# 11.821 UI: chart-line-absolute-momentum primitive (v1.11.839)

## Summary
Pure-SVG dual-panel React/TS primitive plotting **Absolute Momentum**
beneath the close: `|close[i] - close[i - length]|`. Reports the
magnitude of price travel over the lookback without direction --
useful as a regime filter that triggers regardless of trend sign.

## Formula
```
raw[i]    = close[i] - close[i - length]
absMom[i] = Math.abs(raw[i])
```
`absMom[i]` is `null` during warmup (`i < length`) or when any input
is non-finite. The output is non-negative by construction.

## Defaults
- `length = 14`
- `strongThreshold = 5` (data-dependent; configurable)
- `width = 720`, `height = 460`, `panelGap = 12`

## Zones and crosses
- `strong`: `absMom >= strongThreshold`
- `weak`: `0 < absMom < strongThreshold`
- `flat`: `absMom == 0`
- `none`: warmup / non-finite
- `crossed`: `'up'` when prev `< threshold` and current `>= threshold`,
  `'down'` is the mirror. Markers fire on threshold transitions only.

## Bit-exact testability anchors
- **CONST close=K**: `raw = 0` -> `absMom = 0` everywhere across `K
  in {0, 1, 5, 100, -3}` and `length in {3, 4, 7, 10}`.
- **LINEAR UP** (`close[i] = i + 1`): `raw = (i + 1) - (i - length +
  1) = length` -> `absMom = length` (integer-exact). Verified across
  `length in {3, 4, 7, 10}`.
- **LINEAR DOWN** (`close[i] = N - i`): `raw = -length` -> `absMom =
  length` (integer-exact). Verified across the same length sweep.
- **ALTERNATING** `close[i] = i % 2`:
  - With `length = 2`: same-parity bars, `raw = 0`, `absMom = 0`.
  - With `length = 1`: every bar flips, `absMom = 1`.

## Design notes
- Both `raw` (signed) and `absMom` (magnitude) are exposed on each
  sample so consumers can recover the direction if needed.
- absMom y-axis is `0`-based with `max(observed, strongThreshold *
  1.25, 1)` so the threshold line stays visible even at low
  magnitudes.
- The `strongThreshold` default of `5` is illustrative; users should
  set it relative to the price domain of their data.
- Default colors: absMom line `#0891b2` (cyan), threshold `#9333ea`,
  bullish `#16a34a`, bearish `#dc2626`.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` and
  defensive guards.
