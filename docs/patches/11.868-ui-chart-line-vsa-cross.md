## Summary

Added `<ChartLineVsaCross />` -- a pure-SVG, dual-panel React/TS
primitive that renders the close on top and a Volume Spread
Analysis effort-vs-result oscillator on the bottom. The
indicator surfaces the divergence between relative volume
(effort) and relative bar spread (result), with markers at
every effort/result crossover of the signal line.

Cross events:

- `up` cross   -> regime `absorption` (heavy volume, little
  movement)
- `down` cross -> regime `ease` (light volume, easy movement)

Defaults: `length = 20`, `signalLength = 9`, effortColor
`#0ea5e9` (sky), signalColor `#f97316` (orange), absorption
marker `#16a34a`, ease marker `#dc2626`, zero line `#475569`.

## Formula

```
normVolume[i]   = SMA(volume, length)[i] === 0 ? null
                    : volume[i] / SMA(volume, length)[i]
spread[i]       = high[i] - low[i]
normSpread[i]   = SMA(spread, length)[i] === 0 ? null
                    : spread[i] / SMA(spread, length)[i]
effortResult[i] = normVolume[i] - normSpread[i]
signal[i]       = EMA(effortResult, signalLength)
```

Cross detection uses the standard prev-relation + strict-
inequality guard. The SMA helper carries the `min === max`
window-constant precision fix and the EMA helper carries the
SMA-seeded `min === max` precision fix plus the
`next = v === smoothed ? v : ...` short-circuit, so constant
inputs lock onto their values exactly.

## Bit-exact anchor

- **CONST high - low = D (D > 0), close = K, volume = V (V > 0)**:
  both SMAs land at their constants via the precision fix.
  `normVolume = V/V = 1`, `normSpread = D/D = 1`, `effortResult
  = 0`, `signal = EMA(0) = 0`. Relation `equal` forever,
  zero crosses. Verified across multiple (D, V, length) tuples.
- **CONST D = 0**: divide-by-zero guard leaves normSpread null
  -> effortResult null forever.
- **CONST V = 0**: divide-by-zero guard leaves normVolume null
  -> effortResult null forever.
