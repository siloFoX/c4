# Patch 11.957 - UI: chart-line-stoch-rsi-overbought-cross primitive

Version: 1.11.975

## Summary

Add `web/src/components/ui/chart-line-stoch-rsi-overbought-cross.tsx` --
a pure-SVG dual panel line chart primitive that visualizes price
alongside the Chande-Kroll Stochastic RSI %K oscillator, with explicit
cross markers and overlay arrows at every %K crossing of the canonical
80 overbought threshold. Bullish (= entry) when %K crosses up through
80; bearish (= exit) when %K crosses down through 80.

## Pipeline

1. Standard RSI on the close-only series with Wilder smoothing:
   - `delta = close - close_prev`, `gain = max(0, delta)`,
     `loss = max(0, -delta)`.
   - `avg_gain / avg_loss` from `applyLineStochRsiOverboughtCrossWilder`,
     SMA-seeded at `i = 1 + rsiLength - 1`.
   - RSI with both-zero -> 50 / loss==0 -> 100 / gain==0 -> 0 /
     standard `100 - 100/(1 + g/l)`.
2. Stochastic over the rolling RSI window of `stochLength`:
   - `range = rsi_high - rsi_low`.
   - `range == 0` -> hold previous `stochK` (seed = 50 the first time);
   - else -> `stochK = 100 * (rsi - rsi_low) / range`.
3. `classifyLineStochRsiOverboughtCrossRegime(stochK, T)`: null ->
   none, stochK >= T -> bullish, stochK < T -> bearish.
4. `detectLineStochRsiOverboughtCrossCrosses(series, stochK, T)`:
   strict `prev <= T && cur > T` -> bullish (entry); strict
   `prev >= T && cur < T` -> bearish (exit). Boundary equality does
   not fire.

## Bit-exact anchors

| Pattern                  | RSI / stochK (after warmup) | Regime  |
|--------------------------|------------------------------|---------|
| CONST close = K          | 50 / 50 (locked at seed)     | bearish |
| LINEAR UP close = i      | 100 / 50 (locked at seed)    | bearish |
| LINEAR DOWN close = -i   | 0 / 50 (locked at seed)      | bearish |

CONST anchor tested bit-exactly across K in `{0, 1, 42, 100, 1234}`.
LINEAR UP/DOWN saturate RSI to its boundary value (100 / 0), but the
stochastic of a constant RSI window holds the seed (= 50), so stochK
sits below the 80 threshold and the regime is `bearish` on every valid
bar. Cross events live in transients (decline-then-rise pushes RSI
quickly enough that the stochastic window catches a sharp climb,
firing stochK above 80 -> bullish entry; a follow-on decline drops
stochK back below 80 -> bearish exit).

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), %K panel below.
* %K panel uses a fixed `[0, 100]` range with the threshold band drawn
  at its absolute level (80 = ~80% of panel height, closer to the
  top).
* Cross markers drawn as circles on the %K panel at `(cx, cyOsc)` and
  as arrows on the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="Stoch RSI Overbought Cross
  chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only
  text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
