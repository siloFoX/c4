## Summary

Add `web/src/components/ui/chart-line-stc-cross-sig.tsx` -- a
pure-SVG dual-panel React/TypeScript chart primitive marking
discrete bullish (STC crosses up through its signal, smoothed
cycle entry trigger) / bearish (STC crosses down through its
signal, exit) STC-over-signal crossover events. Signal-line
variant of the STC family that fires on STC vs its own smoothed
average -- the canonical entry/exit trigger Doug Schaff
recommends to filter raw-STC whipsaws.

## Math anchor (bit-exact)

```
macd      = SMA(close, fast) - SMA(close, slow)
k1        = stochastic(macd, cycleLength)  (degenerate = 50)
d1        = EMA(k1, factor)
k2        = stochastic(d1, cycleLength)    (degenerate = 50)
stc       = EMA(k2, factor)
signal    = SMA(stc, kSmoothing)
bullish   : prev stc <= prev signal && cur stc >  cur signal
bearish   : prev stc >= prev signal && cur stc <  cur signal
```

- **CONST close = K**: macd=0 -> stc=50, signal=50 -> regime
  bullish at boundary (0 crosses). Verified across K = 0..1234.
- **LINEAR UP / DOWN**: macd=+/-13.5 -> cycle collapses to
  stc=signal=50 -> bullish. 0 crosses.

For 80 bars at defaults: noneCount=69, bullishCount=11.

Strict inequality on `cur` means boundary equality never
double-fires.

## API

```ts
interface ChartLineStcCrossSigProps {
  data: { x: number; close: number }[];
  fastLength?: number;     // default 23
  slowLength?: number;     // default 50
  cycleLength?: number;    // default 10
  factor?: number;         // default 0.5
  kSmoothing?: number;     // default 3
}
```
