# 11.540 -- web/src/components/ui/chart-line-fft.tsx

Pure-SVG chart that pairs a **time-domain line** on the left
with a **frequency-domain spectrum** on the right, computed via
a direct Discrete Fourier Transform. The component is the
catalog's only **frequency-domain** primitive -- the side
panel surfaces the dominant cycle (period = 1 / dominant
frequency) so operators can spot periodic structure that's
hidden in the raw time series.

## DFT formula

For input `y[0..N-1]`:

`X[k] = sum_{n=0..N-1} y[n] * (cos(-2*pi*k*n/N) + i*sin(-2*pi*k*n/N))`

We compute the half-spectrum `k = 0..N/2` (because real-valued
input has conjugate-symmetric DFT) and report magnitude
`|X[k]| = sqrt(real^2 + imag^2)`. Frequency at bin k is
`k / (N * sampleStep)` where `sampleStep = (xN-1 - x0) / (N-1)`,
so the bin units match the sample grid. Period at bin k is
`1 / frequency[k]`; bin 0 (DC) has infinite period and is
excluded from the dominant search by default.

Direct DFT is O(N^2). For chart-scale N (typically <500
points) this is fast and avoids the power-of-two length
constraint that a radix-2 FFT would impose, and it lets every
input length work without padding.

## Distinct from related primitives

The only frequency-domain primitive in the catalog. Distinct
from every other smoother / filter:

- `<ChartLineMovingAvg>` (11.513): SMA smoothing -- time-domain
  only.

  uses split panels but renders DERIVATIVE, not spectrum).

## Public surface

- `<ChartLineFft series={...} />`.
- `series: ChartLineFftSeries[]` -- `{id, label, data, color?,
  detrend?, excludeDc?}`. Per-series detrend / excludeDc
  always beats chart-level.
- `detrend` (default `true`) -- subtract the mean before
  computing the DFT (removes the bin-0 spike from a non-zero-
  centered signal).
- `excludeDc` (default `true`) -- skip bin 0 when picking the
  dominant.
- `timePanelRatio` (default `0.6`; clamped to `[0.1, 0.9]`) --
  fraction of inner width allocated to the time panel.
- `gap` (default `20`) -- gap between panels.
- `spectrumAsBars` (default `true`) -- bars vs a line on the
  spectrum panel.
- `spectrumColor` -- spectrum bar / dot stroke + fill.
- `dominantColor` -- dominant bin tinting + badge color.
- Toggle flags: `showAxis`, `showGrid`, `showDots`, `showLegend`,
  `showTooltip`, `showDominantBadge`, `showSpectrum`,
  `showDominantMarker`, `animate`.
- Standard sizing + format + a11y props.
- `formatValue` / `formatX` / `formatFrequency` / `formatPeriod`
  / `formatMagnitude`.
- `onPointClick({series, point})` (time-domain dots) /
  `onBinClick({series, bin})` (spectrum bars/dots).

## Pure helpers

- `getLineFftDefaultColor(index)` -- 10-color palette cycle.
- `getLineFftFinitePoints(points)` -- finite-only filter.
- `normaliseLineFftPanelRatio(value)` -- clamps to `[0.1, 0.9]`.
- `detrendLineFftValues(values)` -- subtracts the mean of
  finite values; returns `{detrended, mean}`. Non-finite slots
  set to 0 in the output.
- `computeLineFftDft(values)` -- direct DFT. Returns
  `{real, imag}[]` of length `floor(N/2) + 1`. Non-finite
  values are skipped from the sum.
- `computeLineFftSpectrum(points, {detrend?, excludeDc?})` --
  canonical spectrum helper. Sorts ascending; drops non-finite;
  optional detrend (default true); computes the half-spectrum;
  returns `{bins, dominantBin, totalSamples, detrended,
  excludedDc, meanValue}`. Each bin carries
  `{k, frequency, period, real, imag, magnitude,
  normalisedMagnitude}`. Period for k=0 is `Infinity`. Empty
  (or single-point) input -> empty bins, null dominantBin.
- `findLineFftDominantBin(bins, excludeDc?)` -- max-magnitude
  bin; excludes DC by default. Empty / null -> null.
- `computeLineFftLayout({series, ...})` -- splits the canvas
  into the time panel and spectrum panel; returns
  `{series, timePanel, spectrumPanel, timeXTicks, timeYTicks,
  spectrumXTicks, spectrumYTicks, xMin, xMax, yMin, yMax,
  freqMin, freqMax, magMin, magMax, totalPoints,
  visibleSeriesCount}`. Per-series carries `{id, label, color,
  detrend, excludeDc, timePoints, timePath, spectrum,
  spectrumBins, spectrumPath, dominantFrequency,
  dominantPeriod, dominantMagnitude, finiteCount, totalCount}`.
  Spectrum bins are projected to bar geometry `{px, py, barX,
  barWidth, barHeight, isDominant}`.
- `describeLineFftChart(series, options?)` -- screen-reader
  description with per-series dominant frequency and period.

- SVG `role="img"`.
- Time path + every time dot + every spectrum bar / dot are
  `role="graphics-symbol"` + `tabIndex={0}`. Spectrum aria-
  label includes the bin k, frequency, period, magnitude, and
  whether it's the dominant.

  `data-visible-series-count`, `data-total-points`,

  `data-dominant-frequency`, `data-dominant-period`,
  `data-animate`.
- Badge mirrors the same dominant frequency + period +
  magnitude.

## Tooltip

Hovering a time-domain dot shows the series label in its
colour, x, and bold y. Hovering a spectrum bin shows the
series label + bin number, bin frequency, bin period, bold
magnitude, and (when dominant) a coloured "dominant" row.
