# 11.1075 -- UI: chart-line-cci-oversold-divergence primitive (v1.11.1093)

## Scope

Add a pure-SVG dual-panel `ChartLineCciOversoldDivergence`
component that renders the close in the top panel and
Donald Lambert's (1980) Commodity Channel Index (CCI) in
the bottom panel, marking bullish (price down while CCI
up at oversold levels -- **bottom reversal warning**) /
bearish (price up while CCI down at oversold levels --
contrarian rally exhaustion, less common) price-vs-CCI
direction disagreement (divergence) trigger events with
bias coloring derived from the CCI slope at the
divergence-entry bar. **Crosses are gated to the oversold
zone**: only fire when CCI is at or below the
oversoldLevel (default -100).

Direct mirror of `chart-line-cci-overbought-divergence`
v1.11.1092 -- same zone-gated divergence pattern but
focused on bottom reversal warnings. Divergences at
oversold levels are the strongest bottom reversal warning
signals: the underlying momentum has reversed (turning
up) while the price still shows depressed CCI,
indicating that the recent decline is losing steam at a
stretched level.

## CCI oversold-divergence pipeline

```
TP[i]  = (high[i] + low[i] + close[i]) / 3
SMA[i] = mean(TP[i-period+1..i])
MAD[i] = mean(|TP[j] - SMA[i]| for j in window)
CCI[i] = (TP[i] - SMA[i]) / (0.015 * MAD[i])
         (null when MAD = 0)

regime :
  'aligned-bullish'   when priceUp   && cciUp
  'aligned-bearish'   when priceDown && cciDown
  'divergent-bullish' when priceDown && cciUp
                      (PRIMARY: bottom reversal warning
                       when CCI is in oversold zone)
  'divergent-bearish' when priceUp   && cciDown
  'none'              otherwise

bullish cross (at oversold) :
  prev regime !== 'divergent-bullish' &&
  cur regime === 'divergent-bullish' &&
  cur CCI <= oversoldLevel
bearish cross (at oversold) :
  prev regime !== 'divergent-bearish' &&
  cur regime === 'divergent-bearish' &&
  cur CCI <= oversoldLevel

bias : CCI[i] vs CCI[i-1] -> up/down/flat/none
```

Default `period = 20`, `oversoldLevel = -100` -- canonical
Lambert tuning + classical oversold threshold.

## Sibling family

The CCI zone-gated divergence pair:

- **chart-line-cci-overbought-divergence v1.11.1092** --
  top reversal warning, CCI >= +100 gate
- **chart-line-cci-oversold-divergence v1.11.1093**
  (this primitive) -- bottom reversal warning, CCI <=
  -100 gate

Other CCI siblings:

- chart-line-cci -- raw oscillator
- chart-line-cci-divergence-cross -- ungated standard
  divergence
- chart-line-cci-overbought-cross / cci-oversold-cross
  -- level cross (not divergence)
- chart-line-cci-mid-cross / cci-mid-cross-sig --
  centerline
- chart-line-cci-cross-sig -- signal SMA
- chart-line-cci-extreme-cross -- +/-200 extremes

Warmup is `period = 20` for the first CCI value. Cross
detection needs the previous bar's regime, so the first
potential cross lands at i = period + 1 = 21.

## Bit-exact anchors

All anchors use HLC input.

- **CONST band** `high = K + 1`, `low = K - 1`,
  `close = K`: TP = K (constant). MAD = 0 for every
  window. Divide-by-zero guard returns CCI = null.
  regime `none` throughout. 0 crosses. Verified across
  K in {0, 1, 50, 200, 1234}.
- **LINEAR UP** `high = i + 1`, `low = i - 1`,
  `close = i`: CCI = **+126.667** (constant; see
  period-invariant identity in the overbought-divergence
  sibling). CCI is in the OVERBOUGHT region (well above
  zero), NOT in the oversold zone. The oversold-zone
  gate filters out any divergent crosses that would
  otherwise fire. 0 crosses. `oversoldCount = 0`.
- **LINEAR DOWN** `high = -i + 1`, `low = -i - 1`,
  `close = -i`: CCI = **-126.667** (constant mirror).
  priceDown, cciFlat. CCI well below oversoldLevel (in
  oversold zone) -- the gate passes -- but the regime
  is `none` because cciFlat means neither cciUp nor
  cciDown. 0 crosses. `oversoldCount > 0`.

The LINEAR UP / DOWN anchors mirror those of the
overbought sibling: LINEAR DOWN is in oversold but
produces 0 crosses (flat CCI slope), while LINEAR UP is
outside oversold and the gate filters out crosses that
would otherwise fire on the divergent state if the zone
requirement were relaxed.

## Risk

- Pure-SVG, deterministic `.toFixed(2)`, no canvas, no
  chart lib dependency.
- 5-state regime classifier follows the divergence-
  cross family convention exactly.
- Oversold-zone gate (`curCci <= oversoldLevel`) is the
  key specialisation that mirrors the overbought
  sibling. Verified by the explicit detector-level test
  that builds the same divergent-bullish regime
  sequence with CCI values both below and above the
  threshold.
- Divide-by-zero guard returns null when MAD = 0 (CONST
  input). Matches pandas-ta / ta-lib behavior.
- The LINEAR UP / DOWN anchors specifically verify the
  zone-gating asymmetry inverted from the overbought
  sibling:
  - LINEAR UP is in OVERBOUGHT zone (outside oversold)
    -> 0 crosses (gate filters)
  - LINEAR DOWN is in OVERSOLD zone but with flat
    slope -> 0 crosses (regime none, flat momentum)
- Completes the CCI zone-gated divergence pair started
  by v1.11.1092 (overbought). Establishes a fully
  symmetric template for top/bottom reversal warning
  detection that can be extended to other bounded
  oscillators (RSI, Williams %R, Stochastic, etc.)
  with overbought/oversold zone semantics.
