# 11.517 -- web/src/components/ui/chart-line-min-max.tsx

Pure-SVG multi-series line chart with **min/max markers + connecting
reference lines**. For each series the canonical min point (lowest y)
and max point (highest y) are identified and rendered as triangle
glyphs (downward for min, upward for max) at the actual data
coordinates. Optional horizontal reference lines extend across the
full x range at the min and max y-values; optional vertical drop-lines
fall from each marker to the x-axis.

## Distinct from other reference-line primitives

- `<ChartLineThreshold>` (11.503): thresholds are supplied externally.
  Min/max are **derived from the data itself**.
- `<ChartLineBaseline>` (11.510): a single adjustable horizontal
  reference. Min/max draws **two data-derived references** (per
  series).
- `<ChartLineBand>` (11.516): chart-wide rectangle between two
  fixed y-values. Min/max anchors lines at the actual extreme y
  values.
- `<ChartLinePercentile>` (11.514): percentile bands from sample
  groups at each x. Min/max identifies the single absolute min and
  max y across the whole series.
- `<ChartLineMarker>` (11.506): configurable markers PER POINT.
  Min/max highlights only the two extreme points.

The canonical "high/low water mark" visualisation for time-series
monitoring -- "what was the peak load this week" / "what was the
trough" / "how far above the baseline did we go".

## Public surface

- `<ChartLineMinMax series={...} />`.
- Per-series `{id, label, data, color?, minColor?, maxColor?}`.
- `minColor` defaults to `#dc2626` (red), `maxColor` to `#16a34a`
  (green); per-series overrides always win.
- Toggle flags: `showAxis`, `showGrid`, `showDots`, `showLegend`,
  `showTooltip`, `showMin`, `showMax`, `showRefLines`,
  `showDropLines` (off by default), `showMarkerLabels`, `animate`.
- `markerRadius` (6), `refLineOpacity` (0.45), `refLineDashArray`
  (`'4 3'`), `dropLineOpacity` (0.35), `dropLineDashArray` (`'2 3'`)
  for visual tuning.
- Standard sizing + format + a11y props.
- Controlled + uncontrolled `hiddenSeries`.
- Callbacks: `onPointClick({series, point})`,
  `onMarkerClick({series, marker})`.

## Pure helpers

- `getLineMinMaxDefaultColor(index)` -- 10-color palette cycle.
- `getLineMinMaxFinitePoints(points)` -- finite-only filter.
- `findLineMinMaxExtrema(points)` -- returns
  `{min: {index, x, y}, max: {index, x, y}}` or `null` when no finite
  samples. **First occurrence wins on ties.** Indices are positions in
  the original array (so callers can map back to their own data).
- `buildLineMinMaxMarkerPath(kind, cx, cy, r)` -- SVG `d` string for
  an upward triangle (max) or downward triangle (min) anchored at
  `(cx, cy)` with outer radius `r`. Non-finite or `r <= 0` returns
  `''`.
- `computeLineMinMaxLayout({series, hiddenSeries?, showMin?, showMax?,
  markerRadius?, ...})` -- returns `{series, xTicks, yTicks, xMin,
  xMax, yMin, yMax, innerWidth, innerHeight, totalPoints,
  visibleSeriesCount}`. Each layout series carries `{id, label, index
  (stable original), color, minColor, maxColor, points, path,
  finiteCount, totalCount, minMarker, maxMarker}`. Each marker holds
  pixel-space `{px, py, iconPath, refY1, refY2, dropX1, dropY1,
  dropX2, dropY2, labelX, labelY}`. Points carry `isMin` / `isMax`
  flags so adopters can drive custom highlight styles.
- `describeLineMinMaxChart(series, hidden?, formatValue?)` -- builds
  the screen-reader description ("Line chart with min and max markers
  across N series. A: min ... at x=..., max ... at x=...").

## ARIA + test selectors

- Root `role="region"` + `aria-label` + `aria-describedby`.
- SVG `role="img"`.
- Per-line path, dots, and min/max markers are
  `role="graphics-symbol"` + `tabIndex={0}` with per-shape aria-label
  including the kind + value + x for markers.
- Every node carries `data-section="chart-line-min-max-..."` for test
  selectors.
- Series groups expose `data-series-id`, `data-series-color`,
  `data-series-min-y`, `data-series-max-y`,
  `data-series-point-count`.
- Dots expose `data-is-min`, `data-is-max`.
- Markers expose `data-extremum-kind` (`'min'` | `'max'`),
  `data-extremum-index`, `data-extremum-x`, `data-extremum-y`,
  `data-marker-color`.
- Root mirrors `data-show-min`, `data-show-max`.

## Tooltip

- Hovering a marker shows the series label (bold), kind + value
  coloured by the marker colour, and `at x = <fmtX>`.
- Hovering a dot shows the standard series label + x + y + optional
  `min` / `max` tag when the dot is the canonical extremum.
- Marker tooltip takes precedence when both are hovered.
