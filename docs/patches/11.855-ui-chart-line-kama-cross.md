## Summary

Added `<ChartLineKamaCross />` -- a pure-SVG, dual-panel React/TS
primitive that overlays the close with the Kaufman Adaptive
Moving Average (KAMA) in the top panel and renders the
`close - KAMA` deviation in the bottom panel. Markers fire at
every close-vs-KAMA crossover -- the canonical efficiency-weighted
trend transition. The efficiency ratio (ER) maps how directional
recent price action is, then weights the smoothing constant
between a fast and slow extreme:

- `up` cross    -> regime `trending-up`
- `down` cross  -> regime `trending-down`

Defaults: `erLength = 10`, `fastLength = 2`, `slowLength = 30`,
kamaColor `#7c3aed` (violet), deviationColor `#0ea5e9` (sky),
bullishColor `#16a34a`, bearishColor `#dc2626`, zero line
`#475569`.

## Formula

```
ER[i]    = |close[i] - close[i - erLength]| /
           sum(|close[k] - close[k-1]|) for k in (i-erLength+1..i)
fastSC   = 2 / (fastLength + 1)
slowSC   = 2 / (slowLength + 1)
SC[i]    = (ER[i] * (fastSC - slowSC) + slowSC) ^ 2
KAMA[i = erLength]          = close[i]                          (seed)
KAMA[i > erLength]          = KAMA[i-1] + SC[i] * (close[i] - KAMA[i-1])
```

When the lookback sum of |delta| is zero, ER is set to 0 (treat
as the slow extreme). The seed bar lands `close == KAMA`, so the
cross detector marks it `equal` and the next bar fires the first
cross.

## Bit-exact anchors

- **CONST close = K**: every change is 0 -> ER = 0 -> SC =
  `slowSC^2`. KAMA[seed] = K, then `K + SC * (K - K) = K`. KAMA
  stays at K forever bit-exactly, `close - KAMA = 0`, relation
  `equal`, regime `neutral`, zero crosses.
- **LINEAR UP close = i+1**: every change is +1, ER = 1, SC =
  `fastSC^2`. KAMA[seed] = close[seed]. At the next bar the
  KAMA increment is `fastSC^2 * 1`, which is less than the close
  increment of 1 -- so `close - KAMA = 1 - fastSC^2 > 0`. The
  relation flips from `equal` to `bullish` and exactly one `up`
  event fires. Bars after that stay bullish (gap grows then
  saturates). Verified across `erLength in {3, 5, 7}`.
- **LINEAR DOWN close = N-i**: mirror image. Exactly one `down`
  cross fires.
