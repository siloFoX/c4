## Summary

Added `<ChartLineAdxCross />` -- a pure-SVG, dual-panel React/TS
primitive that renders a close line in the top panel and a Wilder
ADX line in the bottom panel. Scatter markers fire at every
threshold crossover between the regime gates (low = 20, high = 40
by default). Four event kinds:

- `enter20` -- ADX newly exceeds the low threshold (weak -> trending)
- `exit20`  -- ADX newly drops below the low threshold (-> weak)
- `enter40` -- ADX newly exceeds the high threshold (-> strong)
- `exit40`  -- ADX newly drops below the high threshold (-> trending)

Defaults: `length = 14`, `lowThreshold = 20`, `highThreshold = 40`,
adxColor `#7c3aed` (violet), low-threshold line `#0ea5e9` (sky),
high-threshold line `#f97316` (orange), enter marker `#16a34a`
(green), exit marker `#dc2626` (red).

## Formula

```
TR[i]   = max(h-l, |h - prevClose|, |l - prevClose|);  TR[0] = h-l
+DM[i]  = (h - prevHigh) if positive and > (prevLow - l) else 0;
          +DM[0] = 0
-DM[i]  = (prevLow - l) if positive and > (h - prevHigh) else 0;
          -DM[0] = 0
trS     = Wilder(TR,  n)
pdmS    = Wilder(+DM, n)
ndmS    = Wilder(-DM, n)
+DI     = 100 * pdmS / trS
-DI     = 100 * ndmS / trS
DX      = 100 * |+DI - -DI| / (+DI + -DI)
ADX     = Wilder(DX, n)
```

Threshold cross detection uses a prev-relation + strict-inequality
guard, then emits the event whose threshold transition just landed
(checked high-first, then low; the bigger transition wins when both
move on the same bar).

## Bit-exact anchors

The Wilder helper carries the SMA-seeded `min === max` precision
fix plus the `next = v === smoothed ? v : (smoothed*(n-1) + v)/n`
short-circuit, so degenerate inputs land bit-exactly:

- **CONST h = l = close = K**: TR is constant 0, both DMs are 0
  -- the division-by-zero guard leaves +DI / -DI / DX / ADX
  permanently `null`. Zero events fire (and the regime is `none`
  for every bar).
- **LINEAR UP h = close = i+1, l = i**: TR = 1 every bar (TR[0] =
  h-l = 1 too); +DM = 1 after the first bar (+DM[0] = 0); -DM = 0
  everywhere. Once seeded, ADX is `Wilder(DX)` where DX bit-exactly
  equals 100 because -DI = 0 makes `DX = 100*|+DI|/+DI = 100`.
  ADX = 100 directly -- jumps from null to 100 with no prev
  relation, so no event fires.
- **LINEAR DOWN h = (N-i)+5, l = (N-i)-5, close = N-i**: TR = 10
  every bar (h-l = 10 dominates |h-prevC| = 4 and |l-prevC| = 6);
  -DM = 1 after the first bar, +DM = 0 everywhere. Mirror image:
  +DI = 0 -> DX = 100 -> ADX = 100 bit-exactly. Zero events fire
  for the same reason.
