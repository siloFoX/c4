## Summary

New `ChartLineVroc` UI primitive in `web/src/components/ui/chart-line-vroc.tsx`
-- a pure-SVG two-panel line chart with a **Volume Rate of Change**
panel from the percent change of volume across a lookback. The top
panel plots the close; the bottom panel plots the VROC.

## The computation

The VROC is the volume analogue of the price Rate of Change.
`ChartLineVrocPoint` carries `{x, close, volume}`.

- `computeLineVroc(bars, period)` -- for each bar past the lookback, the
  percent change of volume from `period` bars ago:
  `vroc = 100 * (volume - volume[-period]) / volume[-period]`. A bar
  with a non-finite volume or a zero prior volume (the divisor) yields
  null.
- `classifyLineVrocZone(vroc)` -- marks a bar **up** when the indicator
  is positive (volume expanding), **down** when negative (volume
  contracting), **flat** at zero, **none** for null.

Because volume cannot drop below zero, the VROC is bounded below at
-100 (a complete dry-up) but unbounded above; the panel uses a dynamic
domain seeded at zero so the zero line is always in view.

## Worked fixture

The whole pipeline is pure rational arithmetic with no transcendentals,
so an integer fixture computes to an exact integer VROC.

A **constant volume** series reads exactly 0 across the window. A
**doubling-volume** fixture (`v[i] = 2 * v[i-period]`: 100, 110, 120,
200, 220, 240, 400, 440 at period 3) pins the VROC to exactly +100. A
**halving-volume** fixture (`v[i] = v[i-period] / 2`: 200, 300, 400,
100, 150, 200, 50, 75) pins it to -50. A **mixed** fixture (volumes
[100, 100, 100, 150, 200, 50, 300, 100] at period 3) gives the exact
integer VROC series [null, null, null, 50, 100, -50, 100, -50],
asserted with `toEqual`. The zero-prior-volume case is checked
directly: with `v[0] = 0`, the VROC at index 3 is null. The zone
classifier is checked for up, down, flat and none.

## API

`data` (a `ChartLineVrocPoint[]`), `period` (default 14),
`pricePanelRatio` / `gap`, sizing/format/a11y props, `strokeWidth` /
`dotRadius`, the colour props, the toggle flags (`showAxis` / `showGrid`
/ `showDots` default false / `showVroc` / `showZeroLine` /
`showMarkers` / `showTooltip` / `showConfigBadge` / `showLegend` /
`animate`), controlled + uncontrolled `hiddenSeries` (keys `price` /
`vroc`), `formatValue` / `formatX`, `onPointClick` / `onSeriesToggle`.

Exported helpers: `getLineVrocFinitePoints`, `normalizeLineVrocPeriod`
(coerces the period to an integer of at least 1), `computeLineVroc`,
`classifyLineVrocZone`, `runLineVroc`, `computeLineVrocLayout`,
`describeLineVrocChart`, plus the `DEFAULT_CHART_LINE_VROC_*` constants
and the `ChartLineVrocZone` type. Every exported name carries a
`LineVroc` infix so the `export *` barrel line cannot collide.

## Accessibility

Root `<div role="region">` with `aria-label` + `aria-describedby`; SVG
`role="img"`; an off-screen `data-section="chart-line-vroc-aria-desc"`
description span; the close line, the VROC line and every dot / marker
are `role="graphics-symbol" tabIndex={0}`. Every node carries a
`data-section`; the root mirrors `data-period`, `data-vroc-final`,
`data-up-count`, `data-down-count`, `data-flat-count`,
`data-total-points` and `data-animate`.
