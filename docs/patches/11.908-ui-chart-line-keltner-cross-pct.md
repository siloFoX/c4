# Patch 11.908 -- UI: chart-line-keltner-cross-pct primitive

## Summary

Added `<ChartLineKeltnerCrossPct />` -- pure-SVG dual-panel
React/TS primitive that renders the close + upper / middle /
lower Keltner Channel (top panel) with bullish / bearish arrow
overlays at every close vs middle line cross trigger, and the
percent-normalized deviation `(close - mid) / mid` alongside
the Keltner half-width as a percent of mid (bottom panel) with
cross dot markers and a dashed zero baseline. Percent-change
variant of the Keltner Channel family that scales every signal
to the mid line for cross-instrument comparable volatility band
magnitude.

## Algorithm

```
mid[i]     = EMA(close, length)
TR[i]      = i == 0 ? 0 : |close[i] - close[i-1]|   (close-only)
atr[i]     = Wilder smooth of TR over atrLength
upper[i]   = mid + mult * atr
lower[i]   = mid - mult * atr
pct[i]     = mid > 0 ? ((close - mid) / mid) * 100 : 0
bandPct[i] = mid > 0 ? ((mult * atr) / mid) * 100 : 0
bullish   : pct crosses up   (prev <= 0, cur > 0)
bearish   : pct crosses down (prev >= 0, cur < 0)
```

Defaults `length = 20` (EMA window), `atrLength = 10`,
`mult = 2`. Regime classifier `bullish` (pct > 0), `bearish`
(pct < 0), `neutral` (pct === 0), `none` (pct null).

## Bit-exact anchor

CONST close = K (any K including 0): EMA(K) collapses to K
via the `min === max` precision short-circuit; TR = 0 -> ATR
= 0 -> upper = lower = mid = K; pct = 0 (or 0 via the
divide-by-zero guard when K = 0); bandPct = 0. Regime is
`neutral`, cross count = 0. Verified across K = 0, 1, 5, 17,
100, 1234.

LINEAR UP has a clean anchor: rolling EMA lags monotonic
close, so pct > 0 and regime is `bullish`.

## Visual

- Two stacked panels, ARPS design system colors via tokens.
- Top: close line + upper / middle / lower Keltner envelope +
  bullish/bearish polygon arrows at cross bars.
- Bottom: percent deviation line + Keltner half-width as
  percent band (with mirrored negative line for symmetric
  reference) + cross dot markers + dashed zero baseline.
- Tooltip lists x / close / upper / mid / lower / pct% /
  band% / regime / counts / crosses.
- Config badge + togglable 6-series legend (close / mid /
  upper / lower / pct / band).
- Cross marker is `role="graphics-symbol"` with `tabIndex={0}`.
