## Summary

Add `web/src/components/ui/chart-line-macd-divergence-cross.tsx`
-- a pure-SVG dual-panel React/TypeScript chart primitive that
renders the close in the top panel and a close-only MACD-style
fast-vs-slow SMA difference line in the bottom panel, marking
discrete bullish (price down + MACD up, potential bottom reversal
warning) / bearish (price up + MACD down, potential top reversal
warning) divergence cross events. Divergence variant of the MACD
family: instead of tracking a single threshold crossover, it
compares the direction of price and MACD over a configurable
look-back window and flags every transition into a disagreement
state -- the canonical reversal-warning signal analysts pair
with support/resistance for swing-trade entries.

The MACD line uses SMA smoothing (rather than the classic EMA)
so the fast-vs-slow difference is bit-exact on monotonic
patterns -- canonical EMA-MACD divergence semantics are
preserved.

## Math anchor (bit-exact)

```
fastSma  = SMA(close, fastLength)
slowSma  = SMA(close, slowLength)
macd[i]  = fastSma[i] - slowSma[i]
priceUp  = close[i] > close[i-window]
macdUp   = macd[i]  > macd[i-window]
state
  aligned-bullish    : priceUp && macdUp
  aligned-bearish    : !priceUp && !macdUp
  divergent-bullish  : !priceUp && macdUp   (price down, MACD up)
  divergent-bearish  : priceUp && !macdUp   (price up, MACD down)
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
(crosses suppressed when prev or cur state == 'none')
```

- **CONST close = K**: fastSma = slowSma = K every bar via the
  SMA `min === max` short-circuit -> macd = 0 constant. priceUp =
  false (close[i] === close[i-5]), macdUp = false (macd[i] ===
  macd[i-5]) -> regime `aligned-bearish` (neither side rising).
  0 divergence crosses. Verified across K = 0..1234.
- **LINEAR UP close = i**: fastSma = i - 5.5, slowSma = i - 12.5
  exact via the SMA constant-lag, so macd = 7 bit-exact for i >=
  25. priceUp = true (close[i] > close[i-5]), macdUp = false (7
  === 7) -> regime `divergent-bearish` (price still rising while
  MACD has flattened -- waning momentum, canonical bearish
  divergence). 0 crosses because divergent state is entered from
  `none` (prev == 'none' is never a valid cross precursor).
- **LINEAR DOWN close = -i**: macd = -7 constant. priceUp =
  false, macdUp = false -> regime `aligned-bearish`. 0 crosses.

Layout uses a symmetric range around 0 (oscMin = -1.1 * span,
oscMax = +1.1 * span where span = max(|macdMin|, |macdMax|)),
fallback to [-1, 1] when no valid MACD values or all values are
exactly 0 -- so the zero line is always centered, regardless of
whether the MACD line is biased positive (uptrend) or negative
(downtrend).

## Accessibility

- Wrapping `<div role="region" aria-label="MACD Divergence Cross
  chart" aria-describedby="...">` with the sr-only description
  copy carrying the fastLength / slowLength / divergenceWindow
  parameters plus a "reversal warning" framing.
- `<svg role="img" aria-labelledby="..." aria-describedby="..."
  tabIndex={0}>` for keyboard focus.
- Each divergence marker (both the MACD-panel dot and the
  price-panel chevron overlay) carries `role="graphics-symbol"`,
  `tabIndex={0}`, and an `aria-label="<bullish|bearish>
  divergence at <x>"` so screen readers can step through events.
- Legend buttons expose `aria-pressed` and toggle via mouse,
  focus, Enter, or Space.
