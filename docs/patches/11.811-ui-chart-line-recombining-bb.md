# 11.811 UI: chart-line-recombining-bb primitive (v1.11.829)

## Summary
Pure-SVG **single-panel** React/TS primitive that overlays the close
with a recombining Bollinger Band envelope. On every bar a standard
band `mean +/- sigmaScale * stdDev` is computed, but the upper and
lower lines collapse to the mean on bars where the close touches or
breaches the band -- the bands "recombine" on the trigger.

## Formula
```
mean[i]      = SMA(close, length)[i]
stdDev[i]    = sqrt(sum((close - mean)^2) / length)   (population)
fullUpper[i] = mean[i] + sigmaScale * stdDev[i]
fullLower[i] = mean[i] - sigmaScale * stdDev[i]
recombine[i] = close[i] >= fullUpper[i] || close[i] <= fullLower[i]
upper[i]     = recombine[i] ? mean[i] : fullUpper[i]
lower[i]     = recombine[i] ? mean[i] : fullLower[i]
```
Bars before `i = length - 1` are warmup nulls. Markers fire on
recombine bars anchored at the mean.

## Defaults
- `length = 20`
- `sigmaScale = 2`
- `width = 720`, `height = 400`, `padding = 44`

## Zones
- `recombine`: trigger fired this bar
- `above-mid`: `close > mean` (no recombine)
- `below-mid`: `close < mean`
- `at-mid`: `close == mean`
- `none`: warmup / non-finite

## Bit-exact testability anchors
- **CONST close** (`close = K`): `mean = K`, `stdDev = 0`, both
  `fullUpper` and `fullLower` collapse to `K`, and `close >= fullUpper`
  -> recombine fires every valid bar. All four series read exactly
  `K`. Verified across `K in {0, 1, 5, 100, -3}` and `length in
  {3, 5, 7, 10}` in the integration sweep.
- **ALTERNATING close [0, 1, 0, 1, ...]** with `length = 4` and
  `sigmaScale = 2`: at bar `i = 3` the SMA is `0.5`, the population
  stdDev is `0.5`, and the dyadic arithmetic is exact in IEEE 754 ->
  `fullUpper = 1.5`, `fullLower = -0.5`. Neither `0` nor `1` touches
  the band so `recombine = false` and the displayed envelope is the
  full band (bit-exact dyadics).
- **`sigmaScale = 0`**: `fullUpper = fullLower = mean`. Any close
  triggers recombine via `>=` or `<=`. Verified explicitly.

## Design notes
- Single panel keeps close and band envelope on the same y-axis: the
  user can read band touches directly.
- The `>=` and `<=` triggers (rather than strict `>` / `<`) ensure
  that the CONST case (`close == fullUpper == fullLower == mean`) is
  flagged as recombine, providing a consistent bit-exact anchor.
- Mean line is dashed `#475569` to distinguish from the band lines.
- Default colors: price `#2563eb`, band `#f59e0b` (amber), recombine
  marker `#dc2626` (red), mean `#475569` dashed.
- `noUncheckedIndexedAccess` satisfied via explicit `?? null` /
  `?? false` and defensive guards.
