# 11.562 -- web/src/components/ui/chart-line-error-bars.tsx

Pure-SVG multi-series line chart with **discrete per-point
error bars**. A connecting line joins the observations; every
point that carries error data also draws an **I-beam error
bar** -- a vertical stem with an upper and a lower horizontal
cap. Symmetric (`error`) and asymmetric (`errorLow` /
`errorHigh`) errors are both supported.

## Error model

Each `ChartLineErrorBarsPoint` may carry up to three error
fields. `resolveLineErrorBarsBounds(point)` collapses them to
a `{errorLow, errorHigh}` pair:

1. An explicit `errorLow` / `errorHigh` always wins for **its
   own side**.
2. A side with no explicit value falls back to the symmetric
   `error` field, then to `0`.
3. Negative magnitudes are clamped to `0` -- an error bar
   cannot have a negative extent.

So `{error: 2}` -> low 2 / high 2; `{errorLow: 1, errorHigh:
3}` -> low 1 / high 3; `{error: 2, errorHigh: 5}` -> low 2 /
high 5 (the explicit high wins, the low borrows the symmetric
error).

`runLineErrorBars(points)` sorts the finite points ascending
by x and produces, per sample, `upper = y + errorHigh`,
`lower = y - errorLow`, a `hasError` flag (`errorLow > 0 ||
errorHigh > 0`), and a `symmetric` flag (`errorLow ===
errorHigh`). It reports `errorPointCount` (points that carry
a bar), `meanError` (the mean half-extent `(errorLow +
errorHigh) / 2`), and `maxError` (the largest single-side
magnitude).

## Distinct from the continuous band charts

`chart-line-band` and `chart-line-confidence` fill a
**continuous shaded region** between an upper and a lower
boundary -- the uncertainty is a polygon swept across the
whole x range. ChartLineErrorBars is the **discrete**
counterpart: each point gets its own self-contained I-beam,
and a point with no error data gets **no bar at all**. The
y-axis range still expands to cover the error-bar extents
(`upper` / `lower`), not just the `y` values, so no cap is
ever clipped.

This is also distinct from the existing `chart-error-bars`
(`<ChartErrorBars>`), which is a standalone categorical
error-bar chart with no connecting line. ChartLineErrorBars
is a line chart first, with per-point error bars layered on.

All exported helpers carry a `LineErrorBars` infix so the
`export *` barrel line cannot collide with `chart-error-bars`.

## Public surface

- `<ChartLineErrorBars series={...} />`.
- `series: ChartLineErrorBarsSeries[]` --
  `{id, label, data, color?}` where each datum is
  `{x, y, error?, errorLow?, errorHigh?}`.
- `capWidth` -- width of the horizontal I-beam caps.
- `errorStrokeWidth` / `errorOpacity` -- error-bar stroke.
- Toggle flags: `showAxis`, `showGrid`, `showDots`,
  `showLegend`, `showTooltip`, `showConfigBadge`, `showLine`,
  `showErrorBars`, `animate`. Note `showDots` defaults to
  `true` here (the error bars anchor on the points).
- Controlled + uncontrolled visibility.
- Standard sizing + format + a11y props.
- `formatValue` / `formatX` / `formatError`.
- `onPointClick({series, point})`.

## Pure helpers

- `getLineErrorBarsDefaultColor(index)`.
- `getLineErrorBarsFinitePoints(points)`.
- `resolveLineErrorBarsBounds(point)` -- the low/high
  resolution rule above.
- `runLineErrorBars(points)` -- sorts ascending; per-sample
  `upper` / `lower` / `hasError` / `symmetric`; reports
  `errorPointCount`, `meanError`, `maxError`.
- `computeLineErrorBarsLayout({...})` -- projects the line
  path plus per-point `upperPy` / `lowerPy`; the y range
  covers the error-bar extents.
- `describeLineErrorBarsChart(series, options?)`.

- SVG `role="img"`.

  `role="graphics-symbol"` + `tabIndex={0}`. The error-bar

  `data-series-mean-error`, `data-series-max-error`,

  `data-symmetric`; caps expose `data-edge="upper"|"lower"`.
- Dots expose `data-x`, `data-y`, `data-error-low`,
  `data-error-high`, `data-has-error`.

## Tooltip / badge

Hovering a dot shows the series label, x, y, the bold error
line (`+/- e` for symmetric, `+high / -low` for asymmetric),
and the `[lower, upper]` range. Config badge:
`ERR mean=<meanError> max=<maxError> bars=<count>`. Legend
shows the bar count and the mean error per series.
