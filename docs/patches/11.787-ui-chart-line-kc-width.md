## Summary

`ChartLineKcWidth` is a pure-SVG dual-panel React primitive that
plots the close on the top panel and the Keltner Channel Width on
the bottom panel as a volatility proxy. KC Width normalizes the
Keltner channel height to the midline (scaled to percent):

```
middle[i] = EMA(close, length)[i]
TR[i]     = max(high - low, |high - prevClose|, |low - prevClose|)
ATR[i]    = SMA(TR, length)[i]
upper[i]  = middle[i] + multiplier * ATR[i]
lower[i]  = middle[i] - multiplier * ATR[i]
width[i]  = (upper[i] - lower[i]) / middle[i] * 100
          = (2 * multiplier * ATR[i]) / middle[i] * 100
```

Defaults: `length = 20`, `multiplier = 2`. Bars before `i = length - 1`
are warmup nulls. When `middle == 0` the width is `null` (divide-by-
zero guard). `-0` from `0 / negative_middle` is normalized to `+0` so
the bit-exact zero anchor holds for negative K too.

## Bit-exact anchor

For **CONST_FLAT** (`high = low = close = K`, `K != 0`):

- TR is zero at every bar (`high - low = 0` and `|high - prevC| = 0`,
  `|low - prevC| = 0`).
- ATR is zero (SMA of constant zero).
- Both bands collapse to the midline, so `upper - lower = 0` exactly.
- The midline is `EMA(K)` which is `K` bit-exact under IEEE 754
  (the recurrence `alpha*K + (1 - alpha)*K` reduces to `K` for
  every K).
- Dividing `0` by the non-zero midline still yields `0` (after the
  `-0 -> +0` normalization).

So `width = 0` exactly past warmup. The integration sweep verifies
this across `K` in `{1, 5, 100, -3, 7, -50, 2}`, `length` in
`{3, 5, 7, 14, 20}`, and `multiplier` in `{1, 2, 3}`.

For **CONST_FLAT K = 0**: the midline is zero, the divide-by-zero
guard fires, and width is `null` -- the singular case is surfaced
rather than producing `0/0 = NaN`.
