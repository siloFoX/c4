# Patch 11.749 -- UI: chart-line-connors-rsi primitive

Released as `v1.11.767` on 2026-05-26.

## Scope

Add `<ChartLineConnorsRsi>` -- a pure-SVG two-panel chart with a
**Connors RSI** panel computed as the average of an RSI of close,
an RSI of the consecutive up/down streak, and a percent rank of
one-bar returns.

## Definition

Inputs:

- `data: ChartLineConnorsRsiPoint[]` where
  `ChartLineConnorsRsiPoint = { x; close }`. Bars with non-finite
  fields are dropped.
- `lenRsi: number` -- lookback for the close-RSI. Default `3`.
- `lenStreak: number` -- lookback for the streak-RSI. Default `2`.
- `lenRank: number` -- lookback for the percent rank. Default
  `100`.
- `overbought: number` -- upper threshold. Default `90`.
- `oversold: number` -- lower threshold. Default `10`.

Algorithm:

```
gain[i] = max(close[i] - close[i - 1], 0)
loss[i] = max(close[i - 1] - close[i], 0)
RSI(values, L) = 100 * sumGain(values, L) / (sumGain + sumLoss)
  -- null when the window has any non-finite value or
     sumGain + sumLoss == 0

streak[0] = 0
streak[i] =
  close[i] > close[i - 1]: prev > 0 ? prev + 1 : 1
  close[i] < close[i - 1]: prev < 0 ? prev - 1 : -1
  else: 0

PercentRank(returns, lenRank) at bar i:
  count past returns in [i - lenRank, i - 1] that are strictly
  less than returns[i]; divide by the count of defined past
  returns; multiply by 100.

CRSI[i] = (RSI(close, lenRsi) + RSI(streak, lenStreak) +
           PercentRank(returns, lenRank)) / 3
```

A bar is null when any component is null.

## Bit-exact algebraic anchors

**CONST_FLAT** (`close == K`):

- `gain = loss = 0` at every bar, so the RSI denominator is zero
  -> RSI_close is null at every bar.
- `streak == 0` at every bar, deltas all zero -> RSI_streak is
  null.
- Returns are all `(K - K) / K = 0`. Percent rank of a tied
  value against tied values: count of strict-less is 0, so PR
  = 0 -- but `RSI_close` and `RSI_streak` are null, so the CRSI
  is null at every bar.

**ACCELERATING_UP** (`close = [10, 11, 12, 13, 20]`) with `lenRsi
= 3, lenStreak = 2, lenRank = 4` at bar 4:

- RSI_close: deltas `+1, +1, +1, +7`. Over the last 3 (window
  [2, 3, 4]) sumGain = 9, sumLoss = 0. RSI = `100 * 9 / 9 = 100`
  bit-exact.
- Streak: `0, 1, 2, 3, 4`. Deltas all `+1`. RSI(streak, 2) at bar
  4 over window [3, 4]: sumGain = 2, sumLoss = 0. RSI = 100
  bit-exact.
- Returns: `r[1] = 1/10, r[2] = 1/11, r[3] = 1/12, r[4] = 7/13`.
  Current `r[4]` is strictly the largest. PercentRank = `100 *
  3 / 3 = 100`.
- `CRSI[4] = (100 + 100 + 100) / 3 = 100` bit-exact.

**DECELERATING_DOWN** (`close = [31, 30, 25, 20, 10]`) with same
lengths at bar 4:

- RSI_close: deltas all negative -> RSI = 0 bit-exact.
- Streak: `0, -1, -2, -3, -4`. Deltas all `-1`. RSI_streak = 0
  bit-exact.
- Returns: all negative, with `r[4] = -0.5` strictly the smallest.
  PercentRank = `100 * 0 / 3 = 0`.
- `CRSI[4] = (0 + 0 + 0) / 3 = 0` bit-exact.

## Component

`forwardRef<HTMLDivElement, ChartLineConnorsRsiProps>` with
displayName. Renders:

- A region-role outer `<div data-section="chart-line-connors-rsi">`.
- An `<svg role="img">` with an `aria-describedby` description.
- Top panel: close line, optional dots.
- Bottom panel: CRSI line, zone-coloured markers (overbought /
  positive / negative / oversold / none), midline at 50, dashed
  overbought / oversold reference lines.
- Legend toggle for `price` / `crsi`, plus a `showCrsi` prop and
  an `onPointClick` handler.
- ARS design-system tokens for colour and motion (`motion-safe:
  animate-fade-in`).

## Acceptance

- CONST_FLAT identity: CRSI null at every bar.
- ACCELERATING_UP anchor: every component = 100 -> CRSI = 100
  bit-exact.
- DECELERATING_DOWN anchor: every component = 0 -> CRSI = 0
  bit-exact.
- Streak identities: RISING / FALLING / direction-change reset
  / flat-bar reset all bit-exact.
- RSI extremes: all gains -> 100, all losses -> 0.
- Percent rank edge cases: first bar null, non-positive prev
  close nulls return.
- RSI translation invariance: shift `+1000` leaves RSI_close /
  RSI_streak unchanged bit-exact.
- CRSI bounded `[0, 100]` on the wave fixture.
- Layout / component / ARIA / data-section hooks all covered.
