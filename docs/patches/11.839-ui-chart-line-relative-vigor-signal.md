# Patch 11.839 -- ChartLineRelativeVigorSignal primitive (v1.11.857)

## Summary

Add `<ChartLineRelativeVigorSignal />`, a pure-SVG dual-panel line
chart with the canonical Ehlers Relative Vigor Index signal-line
oscillator beneath a price line. The signal line is one further
4-bar symmetrically-weighted smoothing on top of the raw RVI value:

```
co[i]     = close[i] - open[i]
hl[i]     = high[i]  - low[i]
coSwma[i] = SWMA(co, 4)[i] = (co[i] + 2*co[i-1] + 2*co[i-2] + co[i-3]) / 6
hlSwma[i] = SWMA(hl, 4)[i]
sumCo[i]  = sum(coSwma[i - length + 1 .. i])
sumHl[i]  = sum(hlSwma[i - length + 1 .. i])
rvi[i]    = sumHl[i] === 0 ? null : sumCo[i] / sumHl[i]
signal[i] = SWMA(rvi, 4)[i]
```

- Warmup is `length + 5` bars: SWMA(4) for co/hl needs 3 bars,
  rolling sum needs `length - 1` more, signal SWMA(4) needs another 3.
- `null` propagates when `sumHl = 0` (divide-by-zero guard).
- `-0` normalized to `+0` on every arithmetic result.

## Defaults

| Knob              | Value     |
|-------------------|-----------|
| length            | 10        |
| bullishThreshold  | 0         |
| bearishThreshold  | 0         |
| rviColor          | `#0ea5e9` (sky blue) |
| signalColor       | `#14b8a6` (teal)     |

Sky blue + teal echoes the cool palette of prior breadth-style
oscillators and contrasts cleanly with the blue price line.

## Bit-exact anchors

- `CONST OHLC = K`: `co = hl = 0`; rolling sums are zero, so
  `rvi = 0 / 0 -> null` and `signal = null` everywhere.
- `CONSTANT-SPREAD open = K, close = K + D, low = K, high = K + R,
  R > 0`: `co = D` constant, `hl = R` constant. SWMA of a constant
  is the constant (`(D + 2D + 2D + D) / 6 = 6D / 6 = D`). Rolling
  sums are `L*D` and `L*R`. The ratio collapses to `D / R` (the
  `L` factor cancels in IEEE 754 since both quantities are
  integer-scaled). Signal SWMA of the constant `D/R` returns
  `D/R` bit-exactly. Verified across dyadic ratios
  `D/R in {0, 0.5, 1, -0.5, -1}` and length sweeps.

## Surface

Standard primitive surface:

- `ChartLineRelativeVigorSignalPoint = { x, open, high, low, close }`
- `runLineRelativeVigorSignal(data, options?) -> Run` with
  `ok = series.length >= length + 6`.
- `computeLineRelativeVigorSignalLayout({ data, width?, height?, padding?, panelGap?, length?, bullishThreshold?, bearishThreshold? }) -> Layout`.
  Oscillator axis auto-fits but always includes `[-1, 1]` plus the
  observed range.
- `describeLineRelativeVigorSignalChart(data, opts?) -> string` for
  the ARIA description.
- `<ChartLineRelativeVigorSignal />` `forwardRef<HTMLDivElement>`
  with three legend toggles (`price`, `rvi`, `signal`), config
  badge, threshold + zero-line overlays, markers on signal cross
  events, and a tooltip surfacing
  `close / coSwma / hlSwma / sumCo / sumHl / rvi / signal / zone /
  cross`.
