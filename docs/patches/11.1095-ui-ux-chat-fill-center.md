# 11.1095 -- UX FIX: Chat surface should fill height and center (v1.11.1113)

## Scope

At 1440 the Chat view rendered a small composer card pinned at the
top-left of a vast empty area: the chat surface neither filled the view
height nor sat in a centered readable column.

## Root cause

`Chat.tsx` renders a `Card` that is `h-full w-full`, so it fills
whatever its wrapper gives it. The App.tsx chat branch wrapper was
`flex min-h-0 flex-1 overflow-hidden p-3 md:p-6` -- it did not establish
a filled height or a capped, centered column, so the card had no room to
expand into.

## Change

- `src/App.tsx` (chat branch wrapper):
  `flex min-h-0 flex-1 overflow-hidden p-3 md:p-6` ->
  `mx-auto flex h-full min-h-0 w-full max-w-3xl flex-1 flex-col
  overflow-hidden p-3 md:p-6`. `mx-auto max-w-3xl w-full` caps the
  column at a readable ~768px and centers it on wide viewports while
  staying full-width below that (mobile). `h-full flex-1 flex-col`
  fills the view height so the transcript + composer get room.
- `src/components/Chat.tsx`: added `data-section="chat-surface"` to the
  card for the e2e (no visual change; the card already fills its
  column).

Layout-only; reuses existing token-backed classes.

## Verification (mandatory)

`web/e2e/chat-fill-center.spec.ts` (new) boots into the Chat view and
asserts:
- 1440: the chat surface fills > 60% of the viewport height, has a real
  left margin (centered, not flush top-left), roughly symmetric side
  margins, and a capped width (< 900px).
- 768: fills > 60% height and fits within the viewport.
- 375: hugs the left edge and spans > 80% width (full-width on mobile).

Result: `3 passed`. The 1440 screenshot confirms a centered max-w-3xl
column filling the height (tall transcript, composer pinned at the
bottom) instead of a small top-left card.

## Risk

- Pure layout change to the chat wrapper + a test-only data attribute.
  Mobile stays full-width (verified at 375); 768/1440 fill + center
  (verified). No behaviour change.
- The App.tsx edit is confined to the chat branch line; other routes
  are untouched.
