# 11.530 -- web/src/components/ui/chart-line-distribution.tsx

Pure-SVG multi-series line chart **with an inline marginal
histogram on the right side** showing the distribution of the y
values. The main panel takes the bulk of the canvas (`1 -
histogramRatio` ~= 80% by default) for the time series; the right
panel renders a sideways histogram with shared y axis -- bin
heights match bin widths in y space, bin bar widths scale to the
counts. Each bin segments per-series so multiple series stack into
the same bin with distinct colours. Mean + median reference lines
span both panels so the central tendency is unambiguous.

The canonical "metric over time + what's the typical range"
visualisation used in monitoring dashboards, finance, and any case
where you need both the trajectory and the distribution at a
glance.

## Distinct from other histogram / distribution primitives

- `<ChartHistogram>` (11.473): standalone histogram chart. This
  primitive **embeds** the histogram as a marginal next to a line
  chart, sharing the y axis.
- `<ChartLinePercentile>` (11.514): percentile bands derived from
  **sample groups at each x**. Distribution bins all y values
  globally.
- `<ChartLineConfidence>` (11.502): per-point bounds.
- `<ChartLineAnomaly>` (11.524): outlier highlighting via z-score.
- `<ChartLineMinMax>` (11.517): markers at extremes only.
- `<ChartLineRollingMinMax>` (11.526): sliding-window envelope.
- `<ChartLineSparkBar>` (11.523): sparkline with bar + line for
  TWO metrics. Distribution overlays a histogram of the LINE's y
  values.

## Public surface

- `<ChartLineDistribution series={...} binCount={12} />`.
- Per-series `{id, label, data, color?}`. `data` entries are raw
  `{x, y}`.
- `binCount` (default 12) -- positive integer >= 1 (non-finite /
  `<1` clamps to 1).
- `histogramRatio` (default 0.2; clamped to `[0, 0.6]`) -- share
  of the canvas allocated to the histogram panel.
- `histogramGap` (default 8) -- horizontal gap between the main
  and histogram tracks.
- `binGap` (default 1) -- vertical gap between bins inside the
  histogram.
- Toggle flags: `showAxis`, `showGrid`, `showDots`, `showLegend`,
  `showTooltip`, `showHistogram`, `showHistogramBg`,
  `showMeanLine`, `showMedianLine`, `animate`.
- Standard sizing + format + a11y props.
- Controlled + uncontrolled `hiddenSeries`.
- Callbacks: `onPointClick({series, point})`,
  `onBinClick({bin})`.

## Pure helpers

- `getLineDistributionDefaultColor(index)` -- 10-color palette
  cycle.
- `getLineDistributionFinitePoints(points)` -- finite-only filter.
- `normaliseLineDistributionBinCount(value)` -- positive integer
  `>= 1` (floor fractional, non-finite / `<1` / non-numeric -> 1).
- `computeLineDistributionMean(values)` -- arithmetic mean, drops
  non-finite; empty / non-array -> 0.
- `computeLineDistributionMedian(values)` -- sorted-ascending
  median, drops non-finite; odd n returns middle, even n averages
  the two middles; empty / non-array -> 0.
- `computeLineDistributionBins(series, binCount, rangeMin,
  rangeMax)` -- bins all finite y values across every series
  into `binCount` equal-width bins spanning `[rangeMin, rangeMax]`.
  Values exactly equal to `rangeMax` go into the LAST bin
  (inclusive upper bound); values outside the range are dropped.
  Non-array / `rangeMin >= rangeMax` -> `[]`. Returns one entry
  per bin with `total` and `perSeries` (only non-zero counts are
  emitted to keep the array compact). Input order of `perSeries`
  preserves the input series order.
- `computeLineDistributionLayout({series, hiddenSeries?,
  binCount?, histogramRatio?, histogramGap?, binGap?, ...})` --
  returns `{series, bins, xTicks, yTicks, xMin, xMax, yMin, yMax,
  innerWidth, innerHeight, mainTrackLeft, mainTrackRight,
  histogramTrackLeft, histogramTrackRight, combinedMean,
  combinedMedian, combinedFiniteCount, totalPoints,
  visibleSeriesCount, binCount, maxBinTotal}`. Each layout series
  carries `{id, label, index (stable original), color, points,
  path, finiteCount, totalCount, stats}` where stats =
  `{finiteCount, mean, median, min, max}`. Each layout bin
  carries `{index, binMin, binMax, total, y, height, segments}`
  with per-series segments inheriting the series colour. Bin
  segment widths scale to `maxBinTotal` so the longest bin always
  fills the histogram panel.
- `describeLineDistributionChart(series, binCount?, hidden?,
  formatValue?)` -- builds the screen-reader description.

## ARIA + test selectors

- Root `role="region"` + `aria-label` + `aria-describedby`.
- SVG `role="img"`.
- Line path + dots are `role="graphics-symbol"` + `tabIndex={0}`;
  bin segments are `role="graphics-symbol"` + `tabIndex={0}`;
  mean + median reference lines are `role="graphics-symbol"`.
- Every node carries `data-section="chart-line-distribution-..."`
  for test selectors.
- Series groups expose `data-series-mean`, `data-series-median`,
  `data-series-color`, `data-series-finite-count`.
- Bin segments expose `data-bin-index`, `data-series-id`,
  `data-count`.
- Root mirrors `data-bin-count`, `data-combined-mean`,
  `data-combined-median`, `data-combined-finite-count`,
  `data-max-bin-total`.

## Tooltip

Two tooltip kinds:

- **Point tooltip** (hovering a dot) shows the series label in its
  colour, x, and bold y.
- **Bin tooltip** (hovering a histogram segment) shows the bin
  range, the total count (bold), and a per-series row in each
  series's colour with its count.

Bin tooltip takes precedence when both are hovered.
