## Fisher pipeline (Ehlers)

```
minClose[i] = min(close[i-length+1..i])
maxClose[i] = max(close[i-length+1..i])
range[i]    = maxClose - minClose
raw[i]      = 2 * (close[i] - minClose) / range - 1
            = 0 when range === 0 (degenerate fallback)
clamped[i]  = clamp(raw[i], -clampLimit, clampLimit)
fisher[i]   = atanh(clamped[i]) = 0.5 * ln((1+c) / (1-c))
```

The clamp keeps the input away from the atanh singularity at
+/-1; Ehlers' canonical value is 0.999 (-> max |fisher| =
atanh(0.999) ~= 3.80). Warmup is `length - 1`: the rolling
min/max needs `length` valid closes to seed.

A single-step Fisher (no double-smoothing) is used so the
saturated-trend anchors are bit-exact in steady state. The
classic Ehlers form additionally smooths the raw value and the
fisher output through a 0.5/0.5 IIR; that variant is omitted
here because its slow convergence breaks the bit-exact contract
the rest of the divergence family follows.

## Divergence regime model

5-state classifier exactly like the other divergence primitives:

```
priceUp  = close[i]  > close[i - divergenceWindow]
fisherUp = fisher[i] > fisher[i - divergenceWindow]
state    = aligned-bullish    if priceUp && fisherUp
         | aligned-bearish    if !priceUp && !fisherUp
         | divergent-bullish  if !priceUp && fisherUp   (price down, Fisher up)
         | divergent-bearish  if priceUp && !fisherUp   (price up, Fisher down)
         | none               if either input is null
bullish-cross : prev != 'divergent-bullish' && cur == 'divergent-bullish'
bearish-cross : prev != 'divergent-bearish' && cur == 'divergent-bearish'
```

Defaults: `length = 9` (Ehlers' canonical Fisher window),
`divergenceWindow = 5`, `clampLimit = 0.999`. Crosses suppress
when either `prev` or `cur` is `none`, so warmup transitions
never fire spuriously. Strict inequality guarantees boundary
equality (Fisher saturated) falls into the aligned-bearish
state instead of double-firing.

## Bit-exact anchors

| input              | fisher (post-warmup)      | regime (post i=13)  | crosses |
| ------------------ | ------------------------- | ------------------- | ------- |
| CONST close = K    | 0 (degenerate fallback)   | aligned-bearish     | 0       |
| LINEAR UP, close=i | +atanh(0.999) ~= +3.80    | divergent-bearish   | 0       |
| LINEAR DOWN        | -atanh(0.999) ~= -3.80    | aligned-bearish     | 0       |

CONST K verified across K = 0..1234 -- the rolling min equals
the rolling max so `range = 0` triggers the degenerate
fallback. priceUp = false (close[i] === close[i-5], strict
`>`), fisherUp = false (0 === 0), regime = aligned-bearish.

LINEAR UP exercises the saturated-trend case: for `length = 9`
the rolling window covers `close[i-8..i] = [i-8..i]`, so
`min = i-8`, `max = i`, `range = 8` and
`raw = 2 * 8/8 - 1 = 1`. The clamp pulls it to 0.999 and atanh
produces `atanh(0.999) ~= 3.8002` constant from i=8 onward.
priceUp = true, fisherUp = false (constant -> not strictly
greater) -> regime `divergent-bearish` for the full 47 valid
samples on the 60-bar fixture. The cross count stays at 0
because the regime is entered from `none`.

LINEAR DOWN is the mirror image: `raw = -1` -> clamped to
`-0.999` -> `fisher = -atanh(0.999) ~= -3.8002`. priceUp =
false, fisherUp = false -> regime `aligned-bearish`.
