## Summary

`ChartLineBbWidth` is a pure-SVG dual-panel React primitive that
plots the close on the top panel and the Bollinger Band Width on the
bottom panel. BB Width normalizes the channel height to the midline:

```
middle[i] = SMA(close, length)[i]
stdDev[i] = sqrt(sum((close - middle)^2) / length)      (population)
upper[i]  = middle[i] + numStdDev * stdDev[i]
lower[i]  = middle[i] - numStdDev * stdDev[i]
width[i]  = (upper[i] - lower[i]) / middle[i] * 100
          = (2 * numStdDev * stdDev[i]) / middle[i] * 100
```

Defaults: `length = 20`, `numStdDev = 2`. Bars `0 .. length - 2` are
warmup (`width = null`). When `middle == 0` width is `null` (divide-by-
zero guard). The expression normalizes `-0` to `+0` so the bit-exact
zero anchor holds for negative `K` too.

## Bit-exact anchor

For **CONST close** (`close = K` constant, `K != 0`): the SMA collapses
to `K`, every centered residual `close - middle` is exactly zero, so
`stdDev = 0`, both bands coincide with the close, and the numerator
`upper - lower` is exactly zero. Dividing by the non-zero midline still
yields exactly zero (after the `-0 -> +0` normalization), so:

```
width = (0) / K * 100 = 0   (exact)
```

The integration test sweeps `K` across positive and negative non-zero
values, `length` across `{3, 5, 7, 14, 20}`, and `numStdDev` across
`{1, 2, 3}` -- every post-warmup bar reads `0`.

`CONST close = 0` is the singular case: `middle = 0` triggers the
divide-by-zero guard and `width` is `null`.
