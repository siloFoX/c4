# Patch 11.827 -- ChartLineNetVolume primitive (v1.11.845)

## Summary

Add `<ChartLineNetVolume />`, a pure-SVG dual-panel line chart that
renders a Net Volume oscillator beneath a price line. Net Volume
buckets each bar's volume into the up or down side based on the
sign of its close-on-close change, sums each bucket over the
lookback window, and scales their difference by the total window
volume:

```
dir[i] = sign(close[i] - close[i - 1])  for i >= 1
dir[0] = 0  (no prior bar)
upVol[i]    = sum_{k = i - L + 1 .. i}(vol[k]  if dir[k] > 0)
downVol[i]  = sum_{k = i - L + 1 .. i}(vol[k]  if dir[k] < 0)
totalVol[i] = sum_{k = i - L + 1 .. i}(vol[k])
net[i]      = totalVol[i] === 0
                ? null
                : (upVol[i] - downVol[i]) / totalVol[i] * 100
```

- Warmup: `i < length - 1` is `null`.
- `null` returned whenever totalVol is zero (divide-by-zero guard).
- `-0` normalized to `+0` on every arithmetic result.
- Output bounded in `[-100, 100]`.

## Defaults

| Knob       | Value     |
|------------|-----------|
| length     | 20        |
| overbought | +30       |
| oversold   | -30       |
| netColor   | `#0d9488` (teal) |

The teal color distinguishes the net-volume panel from the prior
purple/indigo/cyan stochastic and Hilbert oscillators.

## Bit-exact anchors

Integration sweeps `(K, length, volume)` and asserts:

- `CONST close = K, volume = V > 0`: every defined bar is `flat`,
  so `upVol = downVol = 0`, `net = (0 - 0) / (L * V) * 100 = 0`
  bit-exact post-warmup.
- `CONST close = K, volume = 0`: `totalVol = 0` -> divide-by-zero
  guard returns `null` everywhere.
- `LINEAR UP close = i + 1, volume = V > 0`: at index >= `length`,
  every window excludes the index-0 bar (which is `flat` because
  there is no prior close), so `upVol = L*V`, `downVol = 0`,
  `net = (L*V) / (L*V) * 100 = 100` bit-exact across `(L, V)`.
- `LINEAR DOWN close = N - i, volume = V > 0`: symmetric -> `net =
  -100` bit-exact from index `length` onward.

The first defined window at index `L - 1` includes index 0
(which is flat), so `net[L-1] = ((L - 1) * V) / (L * V) * 100 =
(L - 1) / L * 100` -- explicitly tested for `L = 4` (75).

## Surface

Standard primitive surface:

- `ChartLineNetVolumePoint = { x, close, volume }`
- `runLineNetVolume(data, options?) -> Run` with `ok = series.length
  >= length`.
- `computeLineNetVolumeLayout({ data, width?, height?, padding?, panelGap?, length?, overbought?, oversold? }) -> Layout`.
  Net axis fixed to `[-100, 100]`.
- `describeLineNetVolumeChart(data, opts?) -> string` for the ARIA
  description.
- `<ChartLineNetVolume />` `forwardRef<HTMLDivElement>` with legend
  toggle (`price`, `net`), config badge, threshold + midline
  overlays, markers on cross events, and a tooltip surfacing
  `close / vol / dir / up / down / total / net / zone`.
