## Summary

Added `<ChartLineRocCrossSig />` -- a pure-SVG, dual-panel
React/TS primitive that renders the close in the top panel and
overlays the Rate of Change (ROC) with its EMA-smoothed signal
in the bottom panel, marking bullish / bearish cross trigger
events. Trigger-focused ROC variant that isolates the actionable
momentum trigger events when ROC crosses its signal, distinct
from the base ROC level. Cross markers are also painted as
arrow overlays on the price panel for direct charting-overlay
use:

```
ROC[i]    = close[i - length] === 0
              ? null
              : (close[i] - close[i - length]) /
                close[i - length] * 100
signal[i] = EMA(ROC, signalLength)
bullish  : (ROC - signal) crosses up    (prev <= 0, cur > 0)
bearish  : (ROC - signal) crosses down  (prev >= 0, cur < 0)
```

Defaults: `length = 14` (canonical ROC window), `signalLength =
9`. Regime classifier: `bullish` (ROC > signal), `bearish` (ROC
< signal), `neutral` (ROC === signal), `none` (either side
null).

## Bit-exact anchor

- **CONST close = K (K > 0)**: `close[i] - close[i-length] = K
  - K = 0` so `ROC = 0 / K * 100 = 0` every bar after warmup.
  signal EMA of 0s = 0. ROC === signal -> regime `neutral`,
  cross count = 0. Verified across K = 1..1234. CONST K = 0
  triggers the divide-by-zero guard -> ROC = null, regime none.
