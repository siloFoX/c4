## Summary

Add a new `<ChartIcicle>` UI primitive in
`web/src/components/ui/chart-icicle.tsx` -- pure-SVG
**icicle** chart, the rectangular variant of
`<ChartSunburst>` (11.483). Each level of the
hierarchy is a band; within a band, sibling
rectangles tile the parent's extent with sizes
proportional to their share of the parent. Two
orientations: `horizontal` (root at top, children
below) or `vertical` (root at left, children to the
right). Click any cell to make it the new focus --
the focus subtree expands to fill the full canvas
extent. A breadcrumb above the canvas lets adopters
jump back to any ancestor.

## API

## Behaviour

- Renders `<div role="region">` wrapping
  `<svg role="img">` with the same `aria-label`. A
  breadcrumb nav (one link per focus level) renders
  above the canvas by default.
- Value resolution (`getIcicleNodeValue`): leaves
  use their explicit `value` (positive finite);
  non-leaf nodes prefer the sum of their children's
  values, falling back to the explicit value only
  when child totals collapse to 0. Childless
  without value -> 0.
- Layout (`computeIcicleLayout`):
  - bands = `maxDepth - focusDepth + 1`.
  - In **horizontal** orientation, band size is
    `height / bands`; the focus cell spans the
    full width; children stack below the focus
    band with widths proportional to share.
  - In **vertical** orientation, band size is
    `width / bands`; the focus cell spans the
    full height; children stack to the right of
    the focus band with heights proportional to
    share.
  - `cellGap` is split half on each side of every
    cell so neighbours have a thin divider.
- Click-to-zoom:
  - Uncontrolled: `defaultFocusPath` seeds initial
    focus; cell click updates internal focus.
  - Controlled: `focusPath` always wins; cell click
    fires `onFocusPathChange(nextPath)` and
    `onCellClick({cell, becameFocus})` so the
    parent can persist next focus.
  - Breadcrumb click rewinds focus to that
    ancestor.
- Each cell label paints inside the cell when the
  rendered dimensions exceed both `labelMinWidth`
  and `labelMinHeight`. `showLabels=false`
  suppresses all labels.
- Hover/focus on a cell opens a tooltip with the
  breadcrumb path, formatted value, share against
  the focus value, and share of the overall total.
  `showTooltip=false` suppresses.
- Mount animation:
  `motion-safe:animate-fade-in` per cell.
  `animate=false` removes the class.
- Default ARIA description: `"Icicle chart
  (<orientation>) with N nodes across L levels,
  total <total>. Focused on <focus.label> (depth
  D)."` Null root / zero total -> `"No data"`.

- `computeIcicleLayout` returns the cells array
  (one per visible node), the flat tree, the
  resolved focus node, the focus value, the
  observed max depth (relative to the focus), and
  the computed band size. Empty cells are returned
  for null root, non-positive dimensions, or zero
  focus value.

## ARIA + data attributes

- Root: `<div role="region">` + `aria-label`.
- Breadcrumb: `<nav aria-label="Icicle
  breadcrumb">` with one `<button aria-label="Zoom
  to <label>">` per entry.
- SVG: `role="img"` + same `aria-label`.
- sr-only span:
  `data-section="chart-icicle-aria-desc"`.
- Each cell rect: `role="graphics-symbol"` +
  `tabIndex={0}`. aria-label = `"<label>: <value>
  (<percent>)"`.
- `data-section` on every node:
  chart-icicle / -breadcrumb / -breadcrumb-item /
  -breadcrumb-button / -breadcrumb-sep / -canvas /
  -aria-desc / -svg / -cells / -cell / -rect /
  -cell-label / -tooltip / -tooltip-label /
  -tooltip-value / -tooltip-share / -tooltip-global.
- Root mirrors `data-node-count`,
  `data-cell-count`, `data-focus-id`,
  `data-focus-depth`, `data-max-depth`,
  `data-orientation`, `data-animate`.
- Cell groups mirror `data-cell-id`,
  `data-cell-depth`, `data-cell-parent`,
  `data-cell-value`, `data-cell-share`,
  `data-cell-global-share`, `data-cell-color`,
  `data-cell-is-focus`, `data-hovered`.

  order.
- Animated zoom transitions -- snap v1; mount fade
  only.

  only; breadcrumb covers zoom-back.

  orientation across the whole chart.
