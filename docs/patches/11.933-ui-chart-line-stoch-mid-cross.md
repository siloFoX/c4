# 11.933 -- UI: chart-line-stoch-mid-cross primitive (v1.11.951)

## Summary

Adds `ChartLineStochMidCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and a close-only
Stochastic %K line in the bottom panel. Marks bullish (cross
up through midline 50) / bearish (cross down through midline
50) %K level centerline regime trigger events. Single-
threshold cross variant of the Stochastic family that flags
discrete %K level 50 midline crossover events distinct from
the canonical overbought (80) / oversold (20) extreme bands.
Sister of 11.932 `chart-line-rsi-mid-cross` for the
Stochastic family.

## Pipeline

For close-only data with default `length = 14`,
`kSmoothing = 3`:

```
range[i] = max(close[i-n+1..i]) - min(close[i-n+1..i])
rawK[i]  = range > 0
             ? ((close - lowestClose) / range) * 100
             : 50
k[i]     = SMA(rawK, kSmoothing)
bullish  : prev k <= 50 && cur k > 50   (momentum up)
bearish  : prev k >= 50 && cur k < 50   (momentum down)
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> range = 0 every bar
  -> rawK = 50 via the neutral fallback. SMA of 50s = 50 via
  the `min === max` short-circuit. k = 50 sits on the
  threshold but the strict-inequality detector never fires.
  regime `bullish` (k >= 50). cross count = 0. Verified
  across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` (length 14) -> close always sits at
  the window high so rawK = 100 once warm. k = 100 constant.
  regime `bullish`. 0 crosses (k jumps from null to 100 so
  prev-null skips the strict-inequality detector).
- LINEAR DOWN `close = -i` -> close always sits at the window
  low so rawK = 0 constant. k = 0 constant. regime `bearish`.
  0 crosses.
- Long flat (rawK = 50 stable) followed by a sustained rise
  -> SMA-3 smooths and produces exactly one bullish midline
  cross when %K rises strictly above 50.
- Long flat (rawK = 50 stable) followed by a sustained
  decline -> %K falls strictly below 50, firing exactly one
  bearish midline cross.

## Regime classifier

| k range            | regime      |
|--------------------|-------------|
| k >= 50            | bullish     |
| k < 50             | bearish     |
| null               | none        |
