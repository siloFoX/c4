## Summary

New `ChartLineQqe` UI primitive in
`web/src/components/ui/chart-line-qqe.tsx` -- a pure-SVG two-panel line
chart with a **Quantitative Qualitative Estimation** panel from a smoothed
RSI and an adaptive trailing band. The top panel plots the price; the
bottom panel plots the smoothed RSI, the QQE trailing line and a 50
midline.

## The computation

The QQE works on the RSI. It is a close-only indicator, so
`ChartLineQqePoint` carries `{x, value}`.

- `computeLineQqeRsi(closes, rsiPeriod)` -- Wilder's RSI.
- `computeLineQqeEma(values, period)` -- a standard EMA (alpha
  `2/(period+1)`); `computeLineQqeWilder(values, period)` -- Wilder
  smoothing (alpha `1/period`). Both seed from the first finite value.
- `computeLineQqeTrail(rsiMa, delta)` -- the adaptive trailing-band state
  machine. Given the smoothed RSI and the band half-width it ratchets a
  long band up (it can only climb while the RSI leads it) and a short band
  down, flips the trend when the smoothed RSI crosses the active band, and
  emits the QQE line (the long band in an uptrend, the short band in a
  downtrend) and the per-bar trend (+1 up / -1 down / 0 undefined).
- `computeLineQqe(closes, options)` -- composes the pipeline: RSI, EMA
  smoothing into the smoothed RSI, the absolute change of the smoothed RSI
  Wilder-smoothed and scaled by a factor into the band width, then the
  trailing band.

Each defined bar is classified by the trend: **up** or **down**.

## Worked fixture

The RSI and the EMAs are transcendental for a mixed series, but the
deterministic core has exact anchors:

- **The trailing-band state machine** is a pure deterministic function. It
  is hand-traced from `rsiMa = [50,54,58,56,48,44,50,56]` and a constant
  band half-width of 4: the band ratchets, the trend flips down at bar 4
  when the smoothed RSI drops below the long band and back up at bar 6
  when it tops the short band. The QQE line `[46,50,54,54,52,48,46,52]`
  and the trend `[1,1,1,1,-1,-1,1,1]` are asserted with `toEqual`.
- **The RSI** has monotone anchors: a strictly rising series pins it at
  100, a strictly falling one at 0, a flat one at 50.
- **The EMAs** are checked with dyadic alpha (`[4,8,16]` -> `[4,6,11]` for
  EMA period 3 and Wilder period 2).
- **The full pipeline**: a flat series leaves the RSI at 50 and the ATR of
  the RSI at 0, so the QQE line is exactly 50 and the trend stays up.

The run / component use a choppy fixture and are checked structurally.

## API

`data` (a `ChartLineQqePoint[]`), `rsiPeriod` (default 14), `smoothPeriod`
(default 5), `wilderPeriod` (default 27), `factor` (default 4.236),
`pricePanelRatio` / `gap`, sizing/format/a11y props, `strokeWidth` /
`dotRadius`, the colour props, the toggle flags (`showAxis` / `showGrid` /
`showDots` default false / `showRsi` / `showQqe` / `showMidline` /
`showMarkers` / `showTooltip` / `showConfigBadge` / `showLegend` /
`animate`), controlled + uncontrolled `hiddenSeries` (keys `price` / `rsi`
/ `qqe`), `formatValue` / `formatX`, `onPointClick` / `onSeriesToggle`.

Exported helpers: `getLineQqeFinitePoints`, `normalizeLineQqePeriod`,
`computeLineQqeEma`, `computeLineQqeWilder`, `computeLineQqeRsi`,
`computeLineQqeTrail`, `computeLineQqe`, `runLineQqe`,
`computeLineQqeLayout`, `describeLineQqeChart`, plus the
`DEFAULT_CHART_LINE_QQE_*` constants and the `ChartLineQqeZone` type.
Every exported name carries a `LineQqe` infix so the `export *` barrel
line cannot collide.

## Accessibility

Root `<div role="region">` with `aria-label` + `aria-describedby`; SVG
`role="img"`; an off-screen `data-section="chart-line-qqe-aria-desc"`
description span; the price line, the smoothed RSI line, the QQE line and
every dot / marker are `role="graphics-symbol" tabIndex={0}`. Every node
carries a `data-section`; the root mirrors `data-rsi-period`,
`data-smooth-period`, `data-wilder-period`, `data-factor`,
`data-trend-final`, `data-up-count`, `data-down-count`,
`data-total-points` and `data-animate`.
