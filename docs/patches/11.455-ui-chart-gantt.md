## Summary

Add a new `<ChartGantt>` UI primitive in
`web/src/components/ui/chart-gantt.tsx` -- pure-SVG
horizontal Gantt chart. Each row is a task bar stretched
between `start` and `end`; optional progress fill shades
the completed portion. Dependency arrows route from one
task's end to another task's start. A `today` marker line
highlights the current date when supplied. Per-task
tooltip surfaces label, start, end, and optional progress.
Per-task click handler.

## API

```ts
interface ChartGanttTask {
  id: string;
  label: string;
  start: string | number;      // ISO date or epoch
  end: string | number;
  progress?: number;           // 0..1
  color?: string;
  group?: string;
  dependencies?: readonly string[];   // task ids
}

interface ChartGanttProps {
  tasks: readonly ChartGanttTask[];
  width?: number;                    // default 720
  height?: number;                   // default 280
  rowHeight?: number;                // default 20
  rowGap?: number;                   // default 6
  labelWidth?: number;               // default 120
  startDate?: string | number;
  endDate?: string | number;
  today?: string | number;
  showToday?: boolean;               // default true
  showDependencies?: boolean;        // default true
  showLabels?: boolean;              // default true
  showTooltip?: boolean;             // default true
  showAxisTicks?: boolean;           // default true
  showProgress?: boolean;            // default true
  animate?: boolean;                 // default true
  className?: string;
  ariaLabel?: string;                // default 'Gantt chart'
  ariaDescription?: string;
  formatDate?: (d: string | number) => string;
  tickCount?: number;                // default 5
  defaultBarColor?: string;          // default '#2563eb'
  todayColor?: string;               // default '#dc2626'
  onTaskClick?: (args: {
    task: ChartGanttTask;
    index: number;
  }) => void;
}
```

## Behaviour

- Renders a `<div role="region">` wrapping `<svg
  role="img">` carrying the same `aria-label`.
- Date scale: `getGanttDateBounds` returns min(start) /
  max(end) across tasks. `startDate` / `endDate` props
  override either bound. Empty / unparseable data falls
  back to `(0, 1)`. Collapsed range expands by 1 so
  the chart still renders.
- Geometry: tasks stack vertically with `rowHeight` per
  row and `rowGap` between rows. Each row has a
  left-aligned label column of `labelWidth` px; the
  track starts at `labelWidth + 8`. The chart's
  rendered height auto-expands when `tasks.length *
  (rowHeight + rowGap)` exceeds the configured
  `height`.
- Bar geometry: `x1 = xFor(start)`, `x2 = xFor(end)`,
  width clamped to >= 1 px. Tasks with unparseable
  dates are skipped without crashing.
- Progress fill: when `progress > 0` (clamped to
  [0, 1]) a translucent dark overlay covers the
  completed portion. `showProgress=false` suppresses.
- Dependency arrows: when `task.dependencies` includes
  a known task id, an elbow-shaped path routes from
  the dependency's end -> elbow -> current task's
  start. An SVG marker (`#chart-gantt-arrow`) draws
  the arrowhead. `showDependencies=false` suppresses.
- Today marker: when `today` is provided AND in
  range, a dashed vertical line + "today" label paint
  in the configured `todayColor`. Out-of-range or
  missing today -> no marker. `showToday=false`
  suppresses.
- Tooltip: hover opens a tooltip with the task label,
  formatted start, formatted end, and progress (when
  present).
- Mount animation: `motion-safe:animate-fade-in` per
  row group on mount. `animate=false` removes the
  class.
- Default ARIA description: `"Gantt chart with N
  tasks. <label> <start> to <end>. ..."` --
  `ariaDescription` overrides; empty input produces
  `"No data"`.

## Pure helpers (exported)

```ts
DEFAULT_CHART_GANTT_WIDTH        = 720
DEFAULT_CHART_GANTT_HEIGHT       = 280
DEFAULT_CHART_GANTT_ROW_HEIGHT   = 20
DEFAULT_CHART_GANTT_ROW_GAP      = 6
DEFAULT_CHART_GANTT_LABEL_WIDTH  = 120
DEFAULT_CHART_GANTT_TICK_COUNT   = 5
DEFAULT_CHART_GANTT_BAR_COLOR    = '#2563eb'
DEFAULT_CHART_GANTT_TODAY_COLOR  = '#dc2626'

parseGanttDate(d): number | null
getGanttDateBounds(tasks, startOverride?, endOverride?):
  { min, max }
getGanttRangeRatio(value, min, max): number
getGanttTaskColor(task, defaultColor): string
getGanttTicks(min, max, count?): number[]
formatGanttDate(epoch, formatter?): string
buildGanttDependencyPath(fromX, fromY, toX, toY): string
describeGanttChart(tasks, formatDate?): string
```

- `parseGanttDate` converts ISO strings and epoch
  numbers to epoch ms; returns null for non-finite or
  unparseable inputs.
- `getGanttDateBounds` ignores tasks with unparseable
  dates; honours explicit start / end override;
  expands collapsed range.
- `getGanttRangeRatio` clamps `(value - min) /
  (max - min)` into `[0, 1]`; collapsed range returns
  0.
- `getGanttTaskColor` per-task override > default.
- `getGanttTicks` evenly-spaced inclusive endpoints;
  collapsed -> `[min]`; min count 2.
- `formatGanttDate` uses formatter when supplied;
  default `YYYY-MM-DD`. Returns empty string for
  non-finite epoch.
- `buildGanttDependencyPath` emits an `M / L / L / L`
  elbow path; elbow x is the max of `fromX + 6` and
  the midpoint so it always sits to the right of the
  source.
- `describeGanttChart` produces the default ARIA
  summary; unparseable dates render as `?`.
