# Patch 11.829 -- ChartLineHlOsc primitive (v1.11.847)

## Summary

Add `<ChartLineHlOsc />`, a pure-SVG dual-panel line chart that
renders a High-Low Oscillator beneath a price line. The oscillator
subtracts the rolling-mean low from the rolling-mean high across the
lookback, giving a volatility-band proxy whose magnitude reflects
how wide the average per-bar range has been:

```
highSma[i] = mean(high[i - length + 1 .. i])
lowSma[i]  = mean(low[i  - length + 1 .. i])
osc[i]     = highSma[i] - lowSma[i]
```

- Warmup: `i < length - 1` is `null`.
- For typical OHLC data where `high >= low`, the output is non-negative.
- `-0` is normalized to `+0` on every arithmetic result.

## Defaults

| Knob          | Value     |
|---------------|-----------|
| length        | 14        |
| highThreshold | 1         |
| lowThreshold  | 0         |
| oscColor      | `#ea580c` (orange) |

Orange is visually distinct from the prior amber volume-ratio and
teal net-volume oscillators shipped earlier in this batch.

## Bit-exact anchors

Three classes of bit-exact anchor cover the SMA-of-difference
identity:

- `CONST high = low = K`: both SMAs collapse to `K`, so
  `osc = K - K = 0` bit-exact post-warmup. Subtraction of identical
  IEEE 754 doubles is exact regardless of `K`.
- `CONSTANT-SPREAD high = low + D` (with any `baseLow`): the linearity
  of SMA gives
  `SMA(high) - SMA(low) = SMA(high - low) = SMA(D) = D` bit-exact
  post-warmup. Verified across `(K, D, length)` sweeps.
- `LINEAR + CONSTANT-SPREAD high = i + 1 + D, low = i + 1`: even
  though both SMAs are non-trivial dyadic constants in `i`, their
  difference collapses to `SMA(D) = D` bit-exact across `length`.

## Surface

Standard primitive surface:

- `ChartLineHlOscPoint = { x, high, low, close }`
- `runLineHlOsc(data, options?) -> Run` with `ok = series.length >= length`.
- `computeLineHlOscLayout({ data, width?, height?, padding?, panelGap?, length?, highThreshold?, lowThreshold? }) -> Layout`.
  Osc axis auto-fits but always includes 0 and both threshold lines.
- `describeLineHlOscChart(data, opts?) -> string` for the ARIA
  description.
- `<ChartLineHlOsc />` `forwardRef<HTMLDivElement>` with legend
  toggle (`price`, `osc`), config badge, threshold + zero-line
  overlays, markers on cross events, and a tooltip surfacing
  `close / high / low / highSMA / lowSMA / osc / zone / cross`.
