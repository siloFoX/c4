# 11.931 -- UI: chart-line-cci-oversold-cross primitive (v1.11.949)

## Summary

Adds `ChartLineCciOversoldCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and a close-only
Commodity Channel Index (CCI) line in the bottom panel. Marks
bullish (exit upward from oversold) / bearish (entry downward
into oversold) trigger events at the canonical -100 oversold
threshold. Single-threshold cross variant of the CCI family
that flags discrete CCI level -100 entry / exit events.
Mirror of 11.930 `chart-line-cci-overbought-cross` using the
oversold level (-100) instead of the overbought level (+100).

## Pipeline

For close-only data with default `length = 20`:

```
sma[i]  = SMA(close, length)
md[i]   = avg |close[i-n+1..i] - sma[i]|
cci[i]  = md > 0
            ? (close - sma) / (0.015 * md)
            : 0
bullish : prev <= -100 && cur > -100  (exit upward)
bearish : prev >= -100 && cur < -100  (entry downward)
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> sma = K, md = 0 ->
  0/0 guard returns cci = 0. 0 > -100, regime is `neutral`,
  cross count = 0. Verified across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` (length 20) -> cci = 126.667 constant
  once warm. 126.667 > -100, regime `neutral`, 0 crosses.
- LINEAR DOWN `close = -i` -> cci = -126.667. -126.667 < -100,
  regime `oversold`, 0 crosses (cci jumps from null to
  -126.667 so prev-null skips the strict-inequality detector).
- Single negative spike at the end of an otherwise constant
  window of length 14 -> cci = -466.67 at the spike index,
  which crosses the -100 band in one bar and fires exactly 1
  bearish entry cross.

## Regime classifier

| cci range            | regime      |
|----------------------|-------------|
| cci <= -100          | oversold    |
| cci > -100           | neutral     |
| null                 | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- Two horizontal reference bands inside the oscillator panel
  (threshold at -100, mid at 0).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind. Bullish (exit
  upward from oversold) arrow points up, bearish (entry
  downward into oversold) arrow points down -- inverted vs
  overbought variant since the oversold semantics are
  reversed.
- Default oscillator range `+/- 300` expands to wrap any
  observed CCI value (e.g. -466.67 spike) so the marker never
  clips off the panel.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.

## Defaults

- length = 20
- threshold = -100
- oscillator range = +/- 300 (auto-expands)
- width 720, height 460, padding 44, panelGap 12
