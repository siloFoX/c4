## Summary

Added `<ChartLineSupertrendCross />` -- a pure-SVG, dual-panel
React/TS primitive that overlays the close with the Supertrend
line in the top panel and renders the `close - Supertrend`
deviation in the bottom panel. Markers fire at every Supertrend
direction flip -- the canonical volatility-adaptive trend
transition.

Cross events:

- `flip-up`   -- direction newly up (Supertrend snapped to the
  lower band beneath price)
- `flip-down` -- direction newly down (Supertrend snapped to the
  upper band above price)

Defaults: `length = 10`, `multiplier = 3` (classic Supertrend
parameters), upSupertrendColor `#16a34a`, downSupertrendColor
`#dc2626`, deviationColor `#0ea5e9`, flipUpColor green,
flipDownColor red, zero line `#475569`. The supertrend line is
recoloured each render based on the current direction.

## Formula

```
hl2[i]         = (high[i] + low[i]) / 2
ATR[i]         = Wilder(true_range[i], length)
basicUpper[i]  = hl2[i] + multiplier * ATR[i]
basicLower[i]  = hl2[i] - multiplier * ATR[i]
finalUpper[i]  = (basicUpper[i] < finalUpper[i-1]
                  || close[i-1] > finalUpper[i-1])
                    ? basicUpper[i]
                    : finalUpper[i-1]
finalLower[i]  = (basicLower[i] > finalLower[i-1]
                  || close[i-1] < finalLower[i-1])
                    ? basicLower[i]
                    : finalLower[i-1]
direction[i]   = ... (see below)
supertrend[i]  = direction[i] === 'up' ? finalLower[i] : finalUpper[i]
```

Direction kicks in on the first bar with a valid ATR. The seed
direction is `up` when `close >= hl2`, otherwise `down`. Each
subsequent bar:

- If `direction[i-1] === 'up' && close[i] < finalLower[i]` ->
  flip to `down`.
- If `direction[i-1] === 'down' && close[i] > finalUpper[i]` ->
  flip to `up`.
- Otherwise direction stays unchanged.

## Bit-exact anchor

- **CONST h = l = close = K**: TR = 0 every bar, ATR = 0, hl2
  = K. Both basic and final bands collapse to K. Supertrend = K
  bit-exactly, close = K, deviation = 0, relation `equal`,
  direction stays `up`, zero crosses. Verified across multiple
  `(length, multiplier, K)` tuples.
