## Indicator

The McClellan Oscillator (Sherman and Marian McClellan) is a
market-breadth indicator. Its input is the per-period
advance-decline value -- advancing issues minus declining issues
-- so `ChartLineMcclellanPoint` carries `{x, value}` where the
value is that breadth figure.

- `computeLineMcclellanEma(values, period)` -- an exponential
  moving average of the breadth. It seeds with the first value
  and advances `ema += alpha * (value - ema)`, so a constant
  input yields a bit-exact constant output.
- `computeLineMcclellan(values, fastPeriod, slowPeriod)` -- the
  McClellan Oscillator: the spread between a fast and a slow EMA
  of the breadth (classic 19 and 39). It is positive when the
  fast average leads the slow.
- `runLineMcclellan` -- sorts the finite points by x, runs the
  pipeline, and returns per-bar samples classified `positive` /
  `negative` / `zero`, the zone counts and the final reading;
  `ok` is false for fewer than two points.
- `computeLineMcclellanLayout` -- stacks the breadth panel above
  a McClellan Oscillator panel on a dynamic, zero-centred scale,
  with the oscillator line and one marker per bar.
- `describeLineMcclellanChart` -- an off-screen sentence summary
  for assistive technology.

## Testability

The McClellan Oscillator is a difference of two EMAs, and an EMA
smoothing constant `2 / (period + 1)` is exact in floating point
exactly when `period` has the form `2^k - 1`. The fixture uses a
fast period of 3 (alpha 0.5) and a slow period of 7 (alpha 0.25),
both exact dyadics, with integer breadth values -- so both EMAs
and the oscillator spread land on exact binary fractions, the EMA
arrays and the spread identity asserted with `toEqual` / `toBe`.
A constant breadth series drives the oscillator to an exact zero,
and the spread of the early bars resolves to clean values (the
first bar is always zero, the second is a quarter of the first
breadth step). The breadth swings positive and negative so the
oscillator crosses zero and both signs appear.
