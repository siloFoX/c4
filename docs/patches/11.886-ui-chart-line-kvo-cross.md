## Summary

Added `<ChartLineKvoCross />` -- a pure-SVG, dual-panel React/TS
primitive that renders the close in the top panel and overlays
the Klinger Volume Oscillator (KVO) with its EMA-smoothed
signal in the bottom panel, marking bullish / bearish cross
trigger events for *volume force trend regime shifts*. KVO is
the MACD-style spread of two EMAs applied to a signed volume
force series, so this primitive isolates the actionable trend
flips of accumulation versus distribution and paints them as
arrow overlays on the price panel for direct charting-overlay
use:

```
trend[i]    = i === 0 ? 0
                     : close[i] > close[i-1] ? 1
                     : close[i] < close[i-1] ? -1
                     : 0
volForce[i] = volume[i] * trend[i]
emaFast[i]  = EMA(volForce, fastLength)
emaSlow[i]  = EMA(volForce, slowLength)
KVO[i]      = emaFast[i] - emaSlow[i]
signal[i]   = EMA(KVO, signalLength)
bullish    : (KVO - signal) crosses up    (prev <= 0, cur > 0)
bearish    : (KVO - signal) crosses down  (prev >= 0, cur < 0)
```

Defaults: `fastLength = 34`, `slowLength = 55`, `signalLength =
13` (canonical KVO). Regime classifier: `bullish` (KVO >
signal), `bearish` (KVO < signal), `neutral` (KVO === signal),
`none` (either side null).

## Bit-exact anchors (three)

- **CONST {close = K, volume = V}**: every trend = 0 ->
  volForce = 0 -> emaFast = emaSlow = 0 -> KVO = 0 every bar
  -> signal EMA of 0s = 0. KVO === signal -> regime `neutral`,
  cross count = 0. Verified across multiple K and V tuples.
- **LINEAR UP step > 0 with V > 0**: trend = 1 every bar,
  volForce = V (constant). emaFast and emaSlow both collapse
  to V via the `min === max` precision fix -> KVO = V - V =
  0 -> signal = 0 -> regime `neutral`, cross count = 0.
- **LINEAR DOWN step < 0 with V > 0**: trend = -1 every bar,
  volForce = -V (constant). emaFast = emaSlow = -V -> KVO =
  -V - (-V) = 0 -> signal = 0 -> regime `neutral`, cross
  count = 0.

All three anchors hit `KVO = signal = 0` bit-exact.

## Relationship to chart-line-obv-cross-sig (11.885)

| primitive | volume aggregation | output |
| --- | --- | --- |
| `chart-line-obv-cross-sig` (11.885) | cumulative running sum | OBV (running total) |
| `chart-line-kvo-cross` (11.886) | dual EMA spread of signed volume | KVO (MACD-style oscillator) |

OBV is a cumulative running balance; KVO is the MACD-style
fast/slow EMA spread of the same signed-volume input, which
makes it a true oscillator centered on zero.
