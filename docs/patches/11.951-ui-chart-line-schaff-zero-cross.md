# Patch 11.951 - UI: chart-line-schaff-zero-cross primitive

Version: 1.11.969

## Summary

Add `web/src/components/ui/chart-line-schaff-zero-cross.tsx` -- a pure-SVG
dual panel line chart primitive that visualizes price alongside Doug
Schaff's Trend Cycle (STC) panel, with explicit cross markers and overlay
arrows at every STC midline-50 crossover. STC applies two passes of
stochastic + smoothing on top of the MACD line, producing a fast,
bounded `[0, 100]` cycle oscillator. The midline (50) splits the cycle
into bullish-trend vs. bearish-trend halves -- a sharper trigger than
MACD's own zero line because the double smoothing rejects high-frequency
noise.

## Pipeline

1. EMAs over the close: `ema_fast = EMA(close, fast)` and
   `ema_slow = EMA(close, slow)` (SMA-seeded).
2. MACD: `macd = ema_fast - ema_slow`.
3. First stochastic + smoothing pass:
   - Rolling min/max over `cycle` bars of MACD.
   - `k1 = 100 * (macd - macd_low) / (macd_high - macd_low)`; when
     `macd_high == macd_low`, hold the previous k1 (seed = 50).
   - `d1 = prev_d1 + 0.5 * (k1 - prev_d1)` (EMA-style smoothing,
     seed = 50).
4. Second stochastic + smoothing pass on d1, using the same window /
   smoothing structure (seeds = 50). The final `d2` is the STC.
5. `classifyLineSchaffZeroCrossRegime(stc, T)`: null -> none,
   stc >= T -> bullish, stc < T -> bearish.
6. `detectLineSchaffZeroCrossCrosses(series, stc, T)`: strict
   `prev <= T && cur > T` for bullish, strict `prev >= T && cur < T`
   for bearish. Boundary equality does not fire.

The seed value of 50 for both `k` holds and both `d` initialisations is
what makes CONST/LINEAR series bit-exactly converge to `stc = 50`. The
indicator's interesting behaviour lives during transitions, not steady
states.

## Bit-exact anchors

| Pattern                  | STC (after warmup) | Regime  |
|--------------------------|--------------------|---------|
| CONST close = K          | 50                 | bullish |
| LINEAR UP (close = i)    | 50 (MACD constant) | bullish |
| LINEAR DOWN (close = -i) | 50 (MACD constant) | bullish |

CONST anchor tested bit-exactly across K in `{0, 1, 42, 100, 1234}`.
LINEAR ramps also yield `stc = 50` because both EMAs reach their
steady-state lag exactly, leaving MACD constant -- so the stochastic
sees zero range and falls back to the 50 seed.

The directional information lives in transients (decline-then-rise,
rise-then-decline patterns) where MACD passes through zero and the
double-stochastic captures the cycle.

## Layout

* Default dimensions: 720 x 460.
* Two panels: price panel on top (~55% of plot area), STC panel below.
* STC panel uses a fixed `[0, 100]` range (STC is bounded by
  construction) so the threshold band sits exactly at the midpoint when
  `threshold = 50`.
* Threshold band rendered as a horizontal rule at `y = thresholdY` in
  the STC panel.
* Cross markers drawn as circles on the STC panel at `(cx, cyOsc)` and
  as arrows on the price panel at `(cx, cyPrice)`.
* SVG path strings use `.toFixed(2)` for deterministic snapshot tests.

## Accessibility

* Wrapping `<div role="region" aria-label="STC Zero Cross chart">`.
* `<svg role="img">` with an aria-described-by linked `<desc>` sr-only
  text.
* Each cross marker: `role="graphics-symbol"` + `tabIndex={0}` +
  descriptive `aria-label` ("bullish cross at x=...").
* Legend toggles: `<button role="switch" aria-pressed="...">`.
* Motion-safe fade-in via `motion-safe:animate-fade-in` (opt-out with
  `animate={false}`).
