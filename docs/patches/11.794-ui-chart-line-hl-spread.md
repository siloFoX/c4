## Summary

`ChartLineHlSpread` is a pure-SVG dual-panel React primitive that
plots the close on the top panel and a High-Low Spread oscillator
on the bottom panel. The spread is the per-bar range scaled to
percent of close, smoothed by an SMA across the lookback:

```
raw[i]    = (high[i] - low[i]) / close[i] * 100   (close != 0)
spread[i] = SMA(raw, length)[i]
```

Defaults: `length = 14`. Bars before `i = length - 1` are warmup
nulls. When `close[i] == 0` the raw value is `null`, which nulls
every spread bar whose window includes it. `-0` from
`0 / negative_close` is normalized to `+0` so the bit-exact zero
anchor holds for negative closes.

## Bit-exact anchors

- **CONST_FLAT** (`high = low = close = K`, `K != 0`):
  `high - low = 0`, so `raw = 0 / K * 100 = 0` exactly at every
  bar, and the SMA collapses to `0`. The integration sweep
  verifies this across many `K` (including negatives).
- **CONST_FLAT K = 0** is the singular case: divide-by-zero
  guard fires and spread is all-null.
- **CONST_BAR with dyadic `(H-L)/C`** -- when the ratio
  `(H - L) / C` is representable in IEEE 754 (powers of 2 over
  powers of 2), the spread is bit-exact constant past warmup. The
  test sweeps several anchors:
  - `H = 20, L = 10, C = 20` -> `10/20 * 100 = 50`
  - `H = 16, L = 8, C = 16` -> `8/16 * 100 = 50`
  - `H = 30, L = 10, C = 20` -> `20/20 * 100 = 100`
  - `H = 5, L = 0, C = 20` -> `5/20 * 100 = 25`

For non-dyadic ratios (e.g. `(H-L)/C = 0.4`) the per-bar value
has 1-ULP drift, but the SMA of a constant value still stays at
that value bit-exact -- the constant lifts through the SMA cleanly
because all `length` samples are identical.

## Zone classifier

Mirrors the Donchian / BB / KC width buckets:

- `wide` -- spread >= 0.75 * spreadMaxSeen
- `normal` -- 0.25 * spreadMaxSeen <= spread < 0.75 * spreadMaxSeen
- `narrow` -- 0 < spread < 0.25 * spreadMaxSeen
- `flat` -- spread == 0 (singular)
- `none` -- spread is null (warmup or close == 0)

Sane fallback when `spreadMaxSeen` is zero (e.g. CONST_FLAT
series): classify any non-zero spread as `normal`.
