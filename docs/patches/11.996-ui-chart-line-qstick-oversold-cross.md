## Summary

Add `web/src/components/ui/chart-line-qstick-oversold-cross.tsx`
-- a pure-SVG dual-panel React/TypeScript chart primitive marking
discrete bullish (QStick crosses up through oversold threshold
-0.5, candle body momentum oversold exit) / bearish (QStick
crosses down through oversold threshold -0.5, oversold entry)
QStick oversold-crossover events. Mirror of
`chart-line-qstick-overbought-cross` (11.995) with default
threshold flipped from +0.5 to -0.5.

## Math anchor (bit-exact)

```
body[i]    = close[i] - open[i]
qstick[i]  = SMA(body, length)
bullish    : prev qstick <= -0.5 && cur qstick >  -0.5
bearish    : prev qstick >= -0.5 && cur qstick <  -0.5
```

- **CONST open == close == K**: body = 0 -> qstick = 0. 0 >=
  -0.5 -> regime `bullish` (above oversold, mirror flip of
  overbought-cross's bearish at the same qstick=0 reading). 0
  crosses. Verified across K = 0..1234.
- **CONST body = +1**: qstick = 1 -> regime `bullish`. 0 crosses.
- **CONST body = -1**: qstick = -1 < -0.5 -> regime `bearish`
  (in oversold zone). 0 crosses.

For 50 bars at defaults (length=8, threshold=-0.5):
- qstick null for i = 0..6
- post-warmup regime count = 43

Layout auto-fits with 10% padding and expands DOWN to include
the threshold band on degenerate (CONST K=50 -> oscMin=-0.55,
oscMax=0.05; body=-1 -> oscMin=-1.05, oscMax=-0.45).

## API

Same shape as overbought variant, with default threshold = -0.5.
