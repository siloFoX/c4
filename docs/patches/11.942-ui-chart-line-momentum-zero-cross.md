# 11.942 -- UI: chart-line-momentum-zero-cross primitive (v1.11.960)

## Summary

Adds `ChartLineMomentumZeroCross`, a pure-SVG dual-panel
chart that renders the close in the top panel and the close-
only Momentum oscillator line in the bottom panel. Marks
bullish (cross up through zero) / bearish (cross down through
zero) close-minus-close-lookback baseline regime transition
events. Zero-line cross variant of the Momentum family that
flags the discrete Momentum crossing of the zero baseline.
Simplest form of the momentum family -- no percentage
scaling, just the raw `close[i] - close[i-length]`
difference.

## Pipeline

For close-only data with default `length = 10`:

```
mom[i] = close[i] - close[i-length]
bullish : prev mom <= 0 && cur mom > 0   (momentum up)
bearish : prev mom >= 0 && cur mom < 0   (momentum down)
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> close[i] = close[i-
  length] = K -> mom = 0 every bar. mom = 0 sits on the
  threshold but the strict-inequality detector never fires.
  regime `bullish` (mom >= 0). cross count = 0. Verified
  across K in {0, 1, 42, 100, 1234}.
- LINEAR UP `close = i` -> mom[i] = i - (i - length) =
  length, constant. length > 0, regime `bullish`. 0 crosses
  (mom is exactly `length` for every valid bar). This is a
  particularly clean bit-exact anchor compared to ROC because
  there is no percentage scaling.
- LINEAR DOWN `close = -i` -> mom = -length constant. regime
  `bearish`. 0 crosses.
- Decline followed by sustained rise -> Momentum transitions
  from negative to positive -> 1 bullish zero-line cross.
- Rise followed by sustained decline -> 1 bearish cross.

## Regime classifier

| mom range            | regime      |
|----------------------|-------------|
| mom >= 0             | bullish     |
| mom < 0              | bearish     |
| null                 | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- One horizontal reference band inside the oscillator panel
  (threshold at zero, the baseline).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind.
- Oscillator range auto-fits to observed Momentum values
  with 10% padding above and below; Momentum is unbounded.
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.

## Defaults

- length = 10
- threshold = 0
- oscillator range = auto-fit (Momentum is unbounded)
- width 720, height 460, padding 44, panelGap 12
