## ADX mid-cross-sig pipeline

```
adx     = Wilder ADX over HLC (length-bar smoothing)
signal  = SMA(adx, kSmoothing)
bullish : prev signal <= 25 && cur signal >  25
bearish : prev signal >= 25 && cur signal <  25
regime  : bullish (signal >= 25), bearish (signal < 25), none (null)
bias    : up / down / flat / none from signal[i] vs signal[i-1]
```

Signal-line variant of the ADX midline family. The signal SMA
suppresses ADX whipsaws so the trigger fires only when the
smoothed trend strength crosses the canonical 25 trending
threshold. The 25 midline is the classical interpretation
boundary -- "ADX > 25 = trending market" -- and crossing it up
flags an emerging trend, crossing it down flags a dying one
(regardless of direction; ADX measures strength only).

Warmup is `2 * length - 1 + kSmoothing - 1 = 29` for the
default tuning: ADX itself seeds at `2 * length - 1 = 27`,
then the signal SMA needs another `kSmoothing - 1 = 2`
valid ADX samples.

Defaults: `length = 14` (Wilder's canonical ADX window),
`kSmoothing = 3` (signal smoothing), `threshold = 25`
(midline). Bias-coloured cross markers (up/down/flat/none)
match the convention already used by
`chart-line-stc-overbought-cross`,
`chart-line-stc-oversold-cross`, and
`chart-line-stc-divergence-cross`.

## Bit-exact anchors

| input                              | signal (post-warmup) | regime | bias post-warmup | crosses |
| ---------------------------------- | -------------------- | ------ | ---------------- | ------- |
| CONST HLC = K                      | 0                    | bearish | flat (from WARMUP+1) | 0   |
| LINEAR UP (h=i+1, l=i-1, c=i)      | 100 (saturated)      | bullish | flat                 | 0   |
| LINEAR DOWN (h=-i+1, l=-i-1, c=-i) | 100 (saturated)      | bullish | flat                 | 0   |

CONST HLC = K verified across K = 0..1234 -- TR = 0 every bar,
so smoothed +DM / -DM / TR all collapse to 0; both DI denominators
go to 0 and the degenerate fallback returns 0; DX = 0; ADX = 0;
signal = SMA(0, 3) = 0 via the SMA `min === max` short-circuit.
signal < 25 -> regime `bearish`. From the second valid signal
(`i = WARMUP + 1 = 30`), `signal[i] === signal[i-1] === 0` gives
bias `flat`.

LINEAR UP exercises the saturated-trend case: TR = 2, +DM = 1,
-DM = 0 -> +DI = 50, -DI = 0, DX = 100, ADX = 100. signal =
SMA(100, 3) = 100 via the same SMA short-circuit. signal > 25
-> regime `bullish`. The very first valid signal already sits
above 25 so there is no crossing transition -- the 0-trigger
anchor exercises the suppression of warmup boundary events.

LINEAR DOWN is the algebraic mirror: TR = 2, +DM = 0, -DM = 1
-> +DI = 0, -DI = 50, DX = 100, ADX = 100, signal = 100,
regime `bullish` (ADX measures trend strength only -- a strong
downtrend reads bullish on the ADX). 0 triggers.

The bias / threshold-crossing classifier is exercised in
dedicated unit tests for `detectLineAdxMidCrossSigCrosses`,
which checks that a signal rising through 25 fires `bullish`
with `up` bias, falling through 25 fires `bearish` with `down`
bias, boundary equality uses `prev <= 25 && cur > 25`, and that
the trigger does not double-fire on consecutive bars above the
threshold.
