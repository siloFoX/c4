## Summary

Added `<ChartLineSupertrendCrossPct />` -- a pure-SVG, dual-
panel React/TS primitive that overlays the close with the
Supertrend trailing band in the top panel and renders the
`close - Supertrend` deviation scaled to the close as a percent
in the bottom panel. The percent normalisation makes the
adaptive volatility trend regime comparable across instruments
at different price magnitudes:

```
TR[i]       = i === 0 ? 0 : |close[i] - close[i-1]|
ATR[i]      = Wilder smooth of TR over length
upperBand   = close + factor * ATR
lowerBand   = close - factor * ATR
ST[i]       = trend flip with stickiness when close crosses the
              active band
stPct[i]    = close[i] === 0
                ? null
                : (close[i] - ST[i]) / close[i] * 100
```

Defaults: `length = 10`, `factor = 3` (canonical Supertrend).
Regime classifier: `above` (stPct > 0; close above Supertrend
= uptrend), `below` (close below Supertrend = downtrend),
`at` (close === Supertrend), `none` (null).

## Bit-exact anchor

- **CONST close = K (K > 0)**: TR = 0 every bar -> ATR = 0 ->
  upperBand = lowerBand = K -> Supertrend = K every bar.
  `close - Supertrend = 0` -> `stPct = 0 / K * 100 = 0` every
  bar. Verified across K = 1..1234 and length = 5..21. K = 0
  triggers the divide-by-zero guard -> stPct = null.

## Relationship to chart-line-supertrend-cross (11.859)

| primitive | bottom panel | use-case |
| --- | --- | --- |
| `chart-line-supertrend-cross` (11.859) | price-vs-Supertrend cross dots | discrete adaptive volatility flip events |
| `chart-line-supertrend-cross-pct` (11.895) | (close - Supertrend) / close * 100 magnitude channel | continuous adaptive volatility trend regime scaled to price |

11.859 marks the discrete bullish / bearish flip events when
price crosses the Supertrend band. 11.895 surfaces the
continuous percent distance from Supertrend so the adaptive
volatility trend regime can be compared across instruments at
different price magnitudes.
