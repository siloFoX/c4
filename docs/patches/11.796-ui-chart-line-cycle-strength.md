## Summary

`ChartLineCycleStrength` is a pure-SVG dual-panel React primitive
that plots the close on the top panel and a Cycle Strength
oscillator on the bottom panel. The oscillator measures the
amplitude of the lagged-difference (cycle) component relative to
the total signal energy:

```
detrended[i]   = close[i] - close[i - cycleLag]   (i >= cycleLag)
cycleEnergy[i] = sum(detrended[j]^2, [i - length + 1, i])
totalEnergy[i] = sum(close[j]^2,    [i - length + 1, i])
strength[i]    = cycleEnergy[i] / totalEnergy[i]  (typically [0, 1])
```

Defaults: `length = 14`, `cycleLag = 7`. Bars before
`i = cycleLag + length - 1` are warmup nulls because the cycle
energy window must lie entirely within bars where `detrended` is
defined. When `totalEnergy == 0` (every close in the window is
zero) strength is `null`.

A reading near zero means the lagged-difference component is
small relative to total energy (the signal is mostly trending or
flat). A reading near 1 means most of the energy lives in the
cyclic component.

## Bit-exact anchor

For **CONST close** (`close = K`, `K != 0`): every
`detrended[i] = K - K = 0` for `i >= cycleLag`, so
`cycleEnergy = 0` exactly. `totalEnergy = length * K^2 > 0`
(sum of squares is always non-negative; positive for K != 0).
Therefore:

```
strength = 0 / (length * K^2) = 0
```

bit-exact past warmup, regardless of K's sign or magnitude. The
integration sweep verifies this across `K`, `length`, and
`cycleLag` combinations.

For **CONST close = 0** the total energy is zero, the
divide-by-zero guard fires, and strength is `null` -- the
singular case is surfaced rather than producing `0/0 = NaN`.

## Zone classifier

- `cyclic` -- strength >= 0.5 (most energy in the cycle
  component)
- `mixed` -- 0.1 <= strength < 0.5
- `trending` -- 0 < strength < 0.1 (cyclic component is small;
  signal is mostly trending)
- `flat` -- strength == 0 (singular)
- `none` -- strength is null (warmup or close-all-zero)
