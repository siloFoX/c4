# 11.1048 -- UI: chart-line-rsi-mid-cross-sig primitive (v1.11.1066)

## Scope

Add a pure-SVG dual-panel `ChartLineRsiMidCrossSig`
component that renders the close in the top panel and the
Relative Strength Index (RSI) plus its smoothed SMA signal
line in the bottom panel, marking bullish / bearish RSI-
over-signal crossover trigger events with bias coloring
derived from the RSI slope at the trigger bar.

J. Welles Wilder Jr's 1978 RSI is the canonical bounded
momentum oscillator: it normalises the Wilder-smoothed
ratio of average gain over average loss into a `[0, 100]`
range, making `> 70` the conventional overbought zone,
`< 30` the oversold zone, and `50` the natural momentum
centerline. This primitive watches the RSI vs its own
SMA-smoothed signal line to detect momentum centerline
trend trigger events.

## RSI pipeline

```
gain[i]      = max(close[i] - close[i-1], 0)
loss[i]      = max(close[i-1] - close[i], 0)
avgGain[i]   = Wilder smoothed average of gain over
               the period window:
                 avgGain[period]   = SMA(gain, period)
                 avgGain[i]        = (avgGain[i-1] *
                                      (period - 1) +
                                      gain[i]) / period
avgLoss[i]   = analogous for loss
rsi[i]       = (avgGain + avgLoss) > 0
               ? 100 * avgGain / (avgGain + avgLoss)
               : 50
signal[i]    = SMA(rsi, signalLength)

bullish      : prev rsi <= prev signal &&
               cur rsi > cur signal
bearish      : prev rsi >= prev signal &&
               cur rsi < cur signal
regime       : 'bullish' when rsi >= signal
               'bearish' when rsi <  signal
               'none'    when either is null
bias         : rsi[i] vs rsi[i-1] -> up / down / flat /
               none
```

Default `period = 14`, `signalLength = 3` -- canonical
Wilder tuning paired with the family-conventional short
SMA signal.

Warmup is `period + signalLength - 1 = 16` for the
default tuning: avgGain / avgLoss are seeded with the SMA
of the first `period` gains / losses at i = period, RSI is
valid from i = period = 14, then the signal SMA needs
`signalLength - 1 = 2` more bars.

The neutral-zone guard `(avgGain + avgLoss) > 0 ? formula
: 50` prevents 0/0 NaN on CONST input (all gains and
losses are 0; the centerline is the natural fallback,
canonical handling in TA-Lib and pandas-ta).

## Bit-exact anchors

All anchors use close-only input.

- **CONST band** `close = K`: gain = loss = 0 -> avgGain
  = avgLoss = 0. The neutral-zone guard returns `rsi =
  50` (instead of 0/0 NaN). signal = SMA(50, 3) = 50.
  RSI === signal -> regime `bullish` (>=) for every
  valid bar. 0 crosses. Verified across K in {0, 1, 50,
  200, 1234}.
- **LINEAR UP** `close = i`: gain = 1, loss = 0 ->
  avgGain converges to 1 (SMA seed is exactly 1, Wilder
  recurrence preserves the steady state). avgLoss = 0.
  `rsi = 100 * 1 / (1 + 0) = 100`. signal = SMA(100, 3)
  = 100. RSI === signal -> regime `bullish` (>=). 0
  crosses. RSI saturates at 100 on a steady linear
  uptrend -- the canonical strongest-momentum reading.
  This matches Wilder's 1978 result on monotonic data.
- **LINEAR DOWN** `close = -i`: gain = 0, loss = 1 ->
  avgGain = 0, avgLoss = 1. `rsi = 100 * 0 / (0 + 1) =
  0`. signal = SMA(0, 3) = 0. RSI === signal -> regime
  `bullish` (>=) for the same `===` reason. 0 crosses.

## Risk

- Pure-SVG, deterministic `.toFixed(2)`, no canvas, no
  chart lib dependency.
- 3-state regime classifier follows the cross-sig family
  convention (bullish on `===`, bearish on `<`); the
  divergence-cross sibling family uses a 5-state model.
- Neutral-zone guard on `(avgGain + avgLoss) = 0`
  prevents NaN on CONST input -- returns the canonical
  midline value of 50. This matches TA-Lib's `RSI`
  function and pandas-ta's behaviour. Some libraries
  return 0 on zero-loss-and-zero-gain -- this is a
  deliberate deviation chosen for visual stability (a
  flat 50 reads as "neutral momentum" while a flat 0
  reads as "strongest oversold").
- On steady LINEAR input RSI saturates at 100 (up) or 0
  (down) and produces zero crosses, which is correct
  canonical behaviour -- real-world price data with noise
  and trend-direction changes produces typical RSI cross
  patterns.
- Complementary to the sibling CCI mid-cross-sig
  primitive -- RSI adds a bounded momentum-flavoured
  oscillator while CCI offers a mean-reversion-flavoured
  one; both share the cross-sig family pattern of CCI/RSI
  vs its own SMA signal line.
