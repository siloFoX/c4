# 11.944 -- UI: chart-line-cmo-zero-cross primitive (v1.11.962)

## Summary

Adds `ChartLineCmoZeroCross`, a pure-SVG dual-panel chart
that renders the close in the top panel and the close-only
Chande Momentum Oscillator (CMO) line in the bottom panel.
Marks bullish (cross up through zero) / bearish (cross down
through zero) momentum baseline regime transition events.
Zero-line cross variant of the CMO family that flags the
discrete CMO crossing of the zero baseline. CMO is similar
to RSI but uses a difference-over-sum formulation that
produces a bounded `[-100, +100]` range with zero as the
natural baseline (vs RSI's 50 midline).

## Pipeline

For close-only data with default `length = 14`:

```
delta[i] = close[i] - close[i-1]
gain[i]  = max(delta, 0)
loss[i]  = max(-delta, 0)
sumGain  = sum gain over length window
sumLoss  = sum loss over length window
cmo[i]   = sumGain + sumLoss > 0
             ? (sumGain - sumLoss) / (sumGain + sumLoss) * 100
             : 0
bullish  : prev cmo <= 0 && cur cmo > 0   (momentum up)
bearish  : prev cmo >= 0 && cur cmo < 0   (momentum down)
```

## Bit-exact anchor

- CONST `close = K` for any finite K -> delta = 0 every bar
  -> sumGain = sumLoss = 0 -> cmo = 0 via the 0/0 short-
  circuit. cmo = 0 sits on the threshold but the strict-
  inequality detector never fires. regime `bullish` (cmo >=
  0). cross count = 0. Verified across K in {0, 1, 42, 100,
  1234}.
- LINEAR UP `close = i` (length 14) -> delta = +1 every bar
  -> sumGain = length, sumLoss = 0 -> cmo = length / length *
  100 = 100 constant. regime `bullish`. 0 crosses.
- LINEAR DOWN `close = -i` -> cmo = -100 constant. regime
  `bearish`. 0 crosses.
- Balanced alternation `[10, 11, 10, 11, ...]` -> sumGain =
  sumLoss in each window -> cmo = 0, regime `bullish` at
  boundary, 0 crosses.
- Decline followed by sustained rise -> CMO transitions
  from negative to positive -> 1 bullish zero-line cross.
- Rise followed by sustained decline -> 1 bearish cross.

## Regime classifier

| cmo range            | regime      |
|----------------------|-------------|
| cmo >= 0             | bullish     |
| cmo < 0              | bearish     |
| null                 | none        |

## Layout / a11y

- ARIA region + role=img SVG + sr-only desc.
- `data-section` attributes on every drawn group for
  end-to-end assertions.
- One horizontal reference band inside the oscillator panel
  (threshold at zero, the baseline).
- Cross markers in oscillator panel + chevron overlays in
  price panel; each carries `role="graphics-symbol"`,
  `tabIndex=0`, and a discrete colour per kind.
- Oscillator range fixed at `-100 .. +100` (CMO is bounded).
- Motion-safe fade-in animation, motion-reduce respected.
- Controlled / uncontrolled hidden-series legend with `Enter`
  / `Space` keyboard toggle parity.
