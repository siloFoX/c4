## Operating rules

- One task at a time, throttleMs (10 minutes) between dispatches.
- Worker uses the assigned branch; commit must stay on that branch (no push from worker).
- Manager (c4-mgr-auto) verifies each commit, fast-forwards into main, pushes to origin via SSH.
- Circuit breaker: 3 consecutive failures stop the loop until manual resume (c4 autonomous resume).
- Escalation: any halt surfaces in /api/autonomous/escalations and on the Web UI Autonomous tab.
